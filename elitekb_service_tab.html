Service Tab

 

Under the Service Tab, you can create recurring Service Calls for this contract.  This would be used for anything that needs periodic servicing such as portable toilets or event generators.

 

 

Description – This is the Service Call’s description.  It will be entered in the Contract Info of the created service call.  This should correspond to what needs to be done such as “Pump Toilet”.

Frequency – This is the frequency that the service needs to be done.  The options are Days on Rent, Weekly, or Monthly.

Every _ Days – This is the number of days between services.

Sun, Mon…- Check box the day(s) that you want to complete the service.  If you want to only service on Tuesday or Friday, then only check those days.  If a service is due on a different day, it will roll to the earlier acceptable day.

Start Date – This field is used to set the start date of the service.  You would only need to use this field if you didn’t want to start the service until some date in the future.  If you want to start the service on the contract open date, just leave this field blank.

Max Times – This field sets the maximum number of services that you want to do.  For example, if you only wanted to do 2 services, enter 2 in this field.  If you want to do the services for as long as the contract is opened, leave this field blank.

IRO Customer – You can specify an IRO Customer to use.  For example, if you have toilet routes, you could set up a Repairs customer for each of the drivers with a Key of REPAIRS-SERV and something unique.

Trip Name – This field sets the trip name for the Service Call.  If you use the Dispatch Center, this allows you to have your entire trip setup for dispatch.

Truck – This field sets the truck to be used for the Service Call.  This is mainly used if you have the Dispatch Center and want to pre-build your daily trips.

Stop Time – This field sets the stop time for the Service Call.

Service For – You must select the item that you are servicing.  Please note that if you close that line item, no new service calls will be created since the item is no longer out on rent.

Add Item Key – You can add up to three other items that will go on the Service Call.  This could be used for a service fee that is automatically added each time the item is serviced.

Delivery Instructions – You can add any delivery instructions that you want on the Service Calls.

 

Notes:

You can have an unlimited number of recurring Service Calls defined on each contract.  If you are creating specific trips with the service, you may need to add a recurring service call for each day of the week.
