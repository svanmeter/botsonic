There was a problem retrieving printer information for this object. The object may have been sent to a printer that is unavailable 

This is due to an issue within Windows and can normally be resolved using the following process: 
a. Exit Syrinx. 
b. Go to Printers and Faxes or Devices and Printers. 
c. Right-click the problem printer, and then click Properties. 
d. In the Properties dialog box, click Printing Preferences. 
e. Click each tab in the Printing Preferences dialog box without making any changes. 
f. Click OK to close the Printing Preferences dialog box. 
g. Click OK to close the Properties dialog box. 
h. Restart Syrinx, and then try to print the document again.
