Language Translations

When you choose a language other than English. Spanish, French, undefined 1, or undefined 2. Whichever language you select in your parameters, you can have different print out fields, backside legalese, CC authorization, or marketing page. When you choose the language in your parameters, POR will use whatever is in those fields, when printing a contract.

 

 

Language Translation Text Boxes

New multi-language text boxes have been added in many of the text fields with the item and customer records. This allows you to set the description of an item in English, French, or Spanish. Then when you print a contract for a customer in French, not only will all of the static headings be printed in the selected language, but all of the item descriptions, specs, and notes will be printed in the selected language.

*Note-This only applies to SQL databases, not JET.

 

Removal of Language Textboxes

You can remove these text boxes by right-clicking on each column headings.

 

Select from the drop-down box, undefined. You will need to do this for each column.

Press OK

The boxes should now be removed
