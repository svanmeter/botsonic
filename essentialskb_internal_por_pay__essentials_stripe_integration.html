INTERNAL POR Pay | Essentials Stripe Integration


 

This article goes over how to connect Stripe and Stripe terminals in Essentials. POR Pay is a new micro application that integrates natively with Expert/Elite/Rental Hosting, Syrinx, and Essentials. The following are the currently supported setup and migration processes: Stripe for NEW or existing Essentials customers. Not all processors support all of POR Pay’s features. Also, processors only support some countries and currencies.
 


Topics in this article include
 

  • Setting up Stripe in Essentials

  • POR Pay Migration for existing Stripe Customers in Essentials - (POR Pay adds terminal support for Stripe)

  • (Optional) Terminal Order and Setup (new or existing customer, post-migration)

  • Processing Payment with POR Pay → Stripe


 

Setting up Stripe in Essentials 

 

1. Go to Accounting → Credit Card Processing.

 


 

2. In the Update Merchant Credentials section, select Payment App, and click Save.

 
 

3. Go back to Accounting → Credit Card Processing.



 

4. In the Card Processing section, click Connect with Point of Rental.

 
 

5. The POR Payments interface will display (pictured below). If not displayed, refresh the page.

 
 

6. From the POR Payments interface, expand the Organization Info accordion, fill out the organization info, and click Save.

 
 

7. Expand the Payment Processors accordion and click Add Payment Processors.

 
 

8. In Processor Selection, choose Stripe from the dropdown and click Select.

 
 

9. Give your processor a Label (anything works to identify the processor and multiple processors are supported), select the appropriate Currency, and click Save.

 
 

10. Find your processor in the list (red highlighted) and click Complete.

 
 

11. Click Finish Stripe Registration.

 

12. Complete the Stripe setup wizard, providing information and documents when asked.

 

13. When the Stripe process is complete, the processor on the list will be gray and show a status of ‘Stripe credentials are valid.’

 
 

14. The setup process is complete. You will now see the option to use the configuration in all application functions where payments are supported (counter, storefront, email, etc.). When multiple processors are set up, you will have a dropdown to select the appropriate processor.



 
 POR Pay Migration for existing Stripe Customers in Essentials

The migration script is in beta, with alpha testing approved. The worst-case scenario in running the script is existing tokens no-longer being valid and some CC reporting issues. These issues can be manually hot-fixed by development if reported. The business scenario for this is POR Pay adds terminal support for Stripe. 
 

1. To initiate the process, create a Tier 2 support ticket with the following information:

  1. Title - Enter the title "POR Pay Essentials Setup for Client XXX"

  2. Description -

    1. Server or Location 

    2. Company ID (company details)

 

2. Advise customer we will coordinate on a migration date/time after hours.
 

3. Push ticket from Salesforce to Azure DevOps (AZD).
 

4. Hop in the ‘#por-pay’ Slack channel chat and include @Ben Mohorc. If you don’t have access, Kirk Bennett can provide.

 

5. Development will facilitate the migration. The customer can order the terminal in preparation for the support.

 

(Optional) Terminal Order and Setup (new or existing customer, post-migration)


The client has access to execute this process.
 

1.  After registration, the client can order a Stripe terminal from their Stripe dashboard → Terminal → Shop https://dashboard.stripe.com/terminal/shop
 

2. POR’s Integration uses the ‘JavaScript SDK,’ so ensure the customer orders the device in their region that supports that integration (BBPOS WisePOS™ E is supported).

 
 

3. From the Stripe dashboard, click settings (top right gear icon ).

 

4. From the ‘Business settings’ section, click Public Details.

 

5. Fill out the ‘Support address’ section.

 
 
 

6. Connect your Stripe terminal to the internet and generate pairing code. ( - Stripe Instructions)

 

Note: The Stripe terminal must connect to the internet from the same network running Essentials.

 

7. Log in to your Essentials rental management platform.

 

8. Navigate to Accounting → Credit Card Processing.

 
 

9. Find your processor from the list and click Add Terminal.
 

 

10. Enter a Label (anything you want to identify later), the Registration Code from the terminal, and click Save.
 

11. Navigate to Stations → Registers.
 

 

12. Click the pencil icon for the register you want to add the terminal to.

 

13. From the ‘Register Configurations’ section → click the plus icon .

 

14. Enter the Name you want to identify the terminal by → select your terminal from the dropdown and click Save.
 

 

15. Back in the Stations → Register section, check the 'Associated’ box for the terminal you just set up and click Save, completing the terminal setup process.
 

 
 

Processing Payment with POR Pay 


Once setup is complete, you can leverage POR Pay anywhere in Essentials where payments are collected (counter, storefront, auto billing, pre authorization, refunds, and automatic billing).

 

1. For this example, we will collect a payment from Workbench → Click an existing Reservation → Collect Payment.

 
 

2. Select ‘Card’ as the payment method, then select how to charge the card. Select ‘Manually Enter Card’, for new card form (other options are ‘Terminal’, if set up, and card's on file,
if card was previously run with POR Pay), enter the amount, and click Save.
 


3. Enter the card information, verify the customer information, scroll to the bottom, and click Process.


 

4. You will then be redirected to the contract upon payment success.
