Salesperson Enhancements - Salesperson assigned at salesperson level that will flow to any new transaction written to that customer as the default salesperson (note: can be changed at the transaction level)

 


 

Also, if no salespersons are set up in employee setup, there will not be a selection for salespersons at the transaction level

Ability to pin filters - At the workbench/customer/inventory level you can now pin a filter that will stay as the default. Even if you switch filters or close the default filter. If you refresh the page or move away from it and come back your pinned filter will be activated again.

 

 

Absolute Time - Currently the software at the transaction level will show you ‘tomorrow’ and ‘today’ instead of actual dates. You now have the ability to turn that off and have only dates/times like 10/4/2020 2:20 pm.
