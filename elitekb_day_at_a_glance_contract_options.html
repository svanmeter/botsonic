Day at a Glance Contract Options

 

If you click on a contract displayed in Day at a Glance, you will get a menu with various options depending on the status and type of the contract. The tops are as follows:

 

Quote Contracts:

Print Contract - This option will print the contract to the End of Day printer.

Void Reservation - This option will only be displayed for Reservation Contracts It will allow you to cancel the reservation.

Modify Contract - This option will go into the Modify contract mode.

Inquire Contract - This option will go into the Inquire Reprint contract Mode ( *No Modification can be made).

Pay Contract - This allows you to pay the contract from a quote status. You will be asked if you would like to change the quote to a reservation. If you answer yes, you will be taken to the contract payment screen.

Clone Contract - This allows you to copy the existing contract and change the contract status.

Customer Record - This option displays the customer record to view information at a glance.

Customer Dashboard - This option will display the customer's dashboard, allowing you to view (contract statuses, contract aging, reports, statements, View calls, Tasks).

Logistic Ticket - This option will display any logistic tickets created through the contract. To list parent contracts is associated with a logistic contract.

Update Operations - This option will allow the Transaction Operation to be updated.

Update Item Sub-status - This option will allow the Transaction Item Sub-status to be updated.

Update Employee Assigned - This option allows one to assign an employee to a contract.

 

Reservation Contracts:

Will allow the same features as Quote Contract will the following additions:

Send Out Reservation - This option will send out all reservations for the current day. This option will only be displayed for Reservation contracts.

Marked as Locked - This option will lock the reservation so changes cannot be made to this contract without an override password

Marked as Unlocked - This option will unlock the contract so that changes can be made to the contract without any password.

Confirm Reservation - This option will set the reservation as being confirmed. When confirmed, the reservation date will be shown in red. This option is only available to non-delivery reservation contracts. For delivered contracts use the Update Delivery Pickup option.

Unconfirmed Reservation - This option will set the reservation as being unconfirmed. When confirmed, the reservation date will be shown in black. This option is only available to non-delivery reservation contracts. For delivered contracts use Update Delivery Pickup option contracts.

Create a Reminder - This option allows you to create a Reminder to either yourself or to other employees.

Show Call Logs - This option will display the customer's call logs. You can view or add a new call.

Transaction Edit - Display all of the Transaction Edits for this contract.

Print Item Prep Report - This option prints the Item Prep Report for just this contract.

Print Prep Labels - This option allows you to print prep labels.

Print Load Slip - This option reprints the Load Slip for the selected contract.

 

Open Items Tab

This will display the same items as the reservation with the following additions:

Send back to Reservation - This will allow the open contract status to be changed back to a Reservation contract.

 

Came Back Tab

This will display all the same options as a quote with the following additions:

Close Contract - This option will go into the Close contract mode.

 

Non-Serialized Tab

This will display all the same options as a quote with the following additions:

Select Serialized - This option allows the selection of serialized items when a header item was sent out. This only shows up under the Non-Serialized tab and if the parameters are set to not force serialized items on sending out.
