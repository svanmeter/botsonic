Multiple Transaction Payments

This article explains how to handle multiple transaction payments within the system.

Topics included in this article

  • Processing multiple payments for a single transaction
  • Splitting payments between different methods
  • Recording partial payments
  • Managing payment adjustments

Processing multiple payments for a single transaction

Our system allows for multiple payments to be processed for a single transaction, providing flexibility for both the business and the customer.

  1. Select the transaction to which payments need to be applied.
  2. In the payment section, choose to add a new payment.
  3. Enter the payment amount and select the payment method.

Splitting payments between different methods

Payments can be split across different methods, such as cash, credit, or check.

  1. Within the transaction, select the option to split the payment.
  2. Enter the amounts for each payment method being used.
  3. Process each payment separately according to the entered amounts.

Recording partial payments

Partial payments can be recorded in cases where the total amount is not paid in one go.

  1. Select the option to record a partial payment for the transaction.
  2. Enter the amount of the partial payment being made.
  3. The system will update the remaining balance accordingly.

Managing payment adjustments

If adjustments are required for any payment, they can be managed directly within the transaction.

  1. Access the payment details within the transaction.
  2. Choose to edit or remove a payment if adjustments are necessary.
  3. Ensure that all payment information is accurate after adjustments.
