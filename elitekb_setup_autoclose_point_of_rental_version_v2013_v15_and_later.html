Setup Auto Close through Task Runner in Point of Rental version 2013 (v15) and Later

 

 

Starting with Point-of-Rental v2013, setting up auto-close is easier than ever.  It is the alternative to manually doing an end-of-day. If your company has more than one store, only the main store needs an auto-close setup. The other stores can still manually close or set their Parameters to auto-close after the main store auto-closes. This can be done through the drawer tab in configurations. However, there are different options for multi-store auto-closing. 

   

Note: You do not have to be out of a Counter System if autoclose has been set up through Task Runner. You will need to be out of the Counter System if you manually select the Autoclose from the End of Day menu.

 
 

Topics covered in this article

 

  • Multi-store closing options
  • Create a new task
  • Set up user profiles
  • Configure task runner user profiles
  • Auto-Close Reports


 


Multi-store Auto-Close Setup Options

If your company has multiple stores and Store 1 is set to auto-close, you have multiple options:

Option 1 - The other stores can be set to auto close, in which case they will close immediately after store 1.
 

Option 2 - If the other stores are not set to auto close, they can be closed through the End of Day process. If they have not been closed by 11 pm, Point of Rental will automatically close them to prevent problems the next day.
 

Option 3 - You can schedule an Auto Close to close at a certain time each night. Select the following link to Configure Task Runner.




Create a New Task


Expert: Program Menu > Configurations > System Configuration > Configure Task Runner

Elite\Launchpad: ⚙️ > App Configuration > System Configuration > Configure Task Runner

 

 

To create a new task, click on Add Task.

 

 

A window opens to Create a New Task.

 

 

Enter the Following information into the respective text boxes. The Start Time can be different depending on when you want auto close to happen (Note: You do not want to have autoclose happen if other tasks are running, for example, schedule backups). The drive letter for the File to Run and Start in Path text boxes can be different too, depending on the drive letter your Point of Rental Software is installed in.

 

Copy and paste the following information into the corresponding fields:

Name: Close for End of Day

Description: End of Day Auto close

File to Run: <DataDrive>:\POR\Program\Aracd1.exe

Start in Path: <DataDrive>:\POR\Users 

Command Arguments: AUTO 1

 

 

Click ‘Save + Close’ Button when done. You will be taken back to the Configuration Task Runner window where you can see your newly created task.

 

 

Make sure PORTaskRunner is running. If it is not, then click the start button. If this service is not running, then none of the tasks will run.

 



Set Up User Profiles

If you are a Multistore Configuration, you will have to set up User Profiles to make sure that all your stores close properly. To do that, go to:

 

Expert: Program Menu > Security > User Profiles

Elite\Launchpad: ⚙️ > Legacy Options > Security > User Profiles

 

 

Verify there is no STR*Admin as the Terminal name. (Note: * is the Store Number)

If there is not, click Add.

 

 

 

When you get asked to enter a username, enter: Str*Admin (Note: * is the Store Number). Click OK.

 

 

When the Printer Selection comes up, go to the Advanced Tab and click Modify. Click on the dropdown for store and choose the store for the user you are making.

 

 

Click OK when you are done. 

 

Note that if you integrated with credit cards, using Point, Cayan, or Moneris, you will need to make sure a terminal is assigned for each of the profiles.


Configure Task Runner User Profiles

If you have Automatic Reports that you need to print off or email, you will need to configure the Taskrunner User Profile in Point of Rental. To do this go to:

 

Expert: Program Menu > Security > User Profiles

Elite\Launchpad: ⚙️ > Legacy Options > Security > User Profiles

 

 

Verify there is no taskrunner as the terminal name.

If there is not, click Add.

 

 

When you get asked to enter a username, enter: ‘taskrunner’ Click OK.

 

 

When the Printer Selection window comes up, choose the ‘Other Printing' tab and verify all your printers that you want to print to. Click OK when you are done.

 


Auto-close Reports


Any reports that are set up to print at the end of the day WILL NOT if the drawer is set to Auto-Close, including system-generated and Automatic Reports. System-generated reports cannot be manually generated. You can manually generate Automatic Reports.

However, if you manually select the auto-close button, then only the Total Summary Report will print. If you have your email set up in your account numbers, then you will receive an email at the end of the day for the GL numbers.
