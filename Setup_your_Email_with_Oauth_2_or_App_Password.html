Setup your Email with Oauth 2 or App Password



An upcoming Google update will affect users who use Google as an email provider from being able to send emails from third-party apps like Point of Rental without proper setup. This article will show you how to configure and set up your Google or Microsoft 365 email account to be used within your Point of Rental software.


Topics in this article include

  • Find your email credentials in Point of Rental
  • Registering for Oauth 2
  • Set up a Google Application Password
  • Set up a Microsoft 365 Application Password
  • Changing your Password in Point of Rental to the App Password



 

Find your email credentials


Depending on your product you have multiple areas to set up email. This section will show you each area that you might have the ability to set up email in your program.

 

Elite
 

  • Global Email: ⚙️> Core Options > App Configuration > Reports and Notification > System Email Setup > Global Default Setting > Edit

  • Store Email: ⚙️> Core Options > App Configuration > Reports and Notification > System Email Setup > “Pick the Store that you are wanting” > Edit

  • System Notifications: ⚙️> Core Options > App Configuration > Reports and Notification > System Email Setup > System Notifications > Edit

  • Mass Email: ⚙️> Core Options > App Configuration > Reports and Notification > System Email Setup > Mass Email Setting > Edit

  • Internet Fax Setup: ⚙️> Core Options > App Configuration > Reports and Notification > System Email Setup > Internet Fax Setup

  • Employee Email: ⚙️> Core Options > Security > Employee Security > Choose a user (Bret Crosby) > Email Setup

 

Expert

 

  • Global Email: Program Menu > Configuration > Reports and Notification > System Email Setup > Global Default Setting > Edit

  • Store Email: Program Menu > Configuration > Reports and Notification > System Email Setup > “Pick the store that you are wanting” > Edit

  • System Notifications: Program Menu > Configuration > Reports and Notification > System Email Setup > System Notifications > Edit

  • Mass Email: Program Menu > Configuration > Reports and Notification > System Email Setup > Mass Email Setting > Edit

  • Internet Fax Setup: Program Menu > Configuration > Reports and Notification > System Email Setup > Internet Fax Setup

  • Employee Email: Program Menu > Security > Employee Security > Choose a user (Bret Crosby) > Email Setup


 

Registering of Oauth2


Selecting ‘Register for Oauth2’ will log you into Google or Microsoft 365 and guide you through the login process. If you are using an App Password then you can skip this section. If at any time, you need to remove your Oauth2 setup, click on the ‘Register for Oauth2’ button again and a pop-up will appear asking if you'd like to delete your Oauth2 credentials, so you can revert back to the SMTP method.
 

1. After all the information is connected, click on the Register for Oauth2 button.
 


2. A popup window will come up that will take you to a browser. Click OK.
 


 

3. Sign in to your account. Follow the prompts to give access to it.
 


 

4. When you log in you will be taken to a website that says the following.
 


 

5. Close out of your browser and go back to Point of Rental.

6. You will see a message for Click OK once you have completed the authorization process.
 


 

7. Now you will see that Oauth2 is enabled.
 


 

Set up a Google application password

 

To continue to use Gmail with third-party apps like Point of Rental, Google will require you to use an app password instead of your Google account password. The app password is a 16-digit passcode that gives less secure apps or devices permission to access your Google account. App passwords can only be used with accounts that have 2-step verification turned on. For more information on this please visit this Google support article here.

 

1. Go to your Google account.

2. Select Security.

3. Under "Signing into Google," select App Passwords. You may need to sign in. If you don’t have this option, it might be because:

a. 2-Step Verification is not set up for your account.

b. 2-Step Verification is only set up for security keys.

c. Your account is through work, school, or other organizations.

d. You turned on Advanced Protection.
 

4. At the bottom, choose Select app and choose the app you are using.

5. Select device and choose the device you are using. Press Generate.

6. Follow the instructions to enter the app password. The app password is the 16-character code in the yellow bar on your device.

7. Select Done.
 

Tip: DO NOT CHANGE your Google account password to the 16-digit Google app password. To continue to use Gmail with third-party apps like Point of Rental, Google will require you to use an app password instead of your Google account password.

 

Set up a Microsoft 365 application password

 

To continue to use Microsoft 365 with third-party apps like Point of Rental, Microsoft will require you to use an app password instead of your Microsoft account password. The app password is a 16-digit passcode that gives less secure apps or devices permission to access your Microsoft account. App passwords can only be used with accounts that have 2-step verification turned on. For more information on this please visit this Microsoft support article here. Please note that some steps may differ. If you need assistance please contact your IT or Network Administrator for assistance.

Tip: DO NOT CHANGE your Microsoft 365 account password to the 16-digit Google app password. To continue to use Microsoft 365 with third-party apps like Point of Rental, Microsoft will require you to use an app password instead of your Microsoft 365 account password.
 

 

Changing your Password in Point of Rental to the App Password

 

1. Now that you have your app password you can add that password to your Email Setup in Point of Rental. You will need to do this anywhere that email password is stored in Point of Rental. Click Here to learn more about where Point of Rental Stores email passwords.

2. From the Email Setup window change the password to your app password.
 


3. Click OK when finished.
