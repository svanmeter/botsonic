Transfer to a 901 Location




If you have inventory stored off-site or in remote warehouses, such as service trucks or jobsite trailers, this feature helps you track and move inventory from stores and multiple inventory locations without creating transactions at those sites. In addition, reports can be generated to show inventory available at the locations.



Topics in this article include
 

  • Setup
  • Customer Record
  • Transferring Items




 

Setup


Here's how to get there
Expert: Program Menu > Configurations > System Configurations > Delivery Truck
Elite: Configuration Gear > App Configurations > System Configurations > Delivery Truck


You will need to identify what is going to be your 901 location (Warehouse, Service Truck, Trailer). These would be places where you would NOT have a cash drawer. From the Delivery Truck screen, you can select a truck, if the equipment will be stored in the truck.

 

1. Select or enter a truck you would like to designate as the 901 location.
 


2. Check the box to mark this vehicle as a service truck and enter 901 in the number field.
 


Note: If you are using a warehouse as the 901 location, then in the name field, enter the name warehouse and follow steps 1 and 2 as listed above.


 

Customer Record
 

Before you begin, it is important to make sure you have your transfer customers set up in your database.

Here's how to get there
Expert: Program Menu > File Maintenance > Customer File > Add Customer
Elite: Configuration Gear > File Maintenance > Customer File > Add Customer

For this type of transfer, you will need to enter XFER-901 in the customer key.
 


Note: The transferred contract will now automatically select Use Same Address from the Transfer customer’s record. Make sure you set the store’s address in the XFER customer for each store. You also need to be sure that any serialized items you are transferring have a header record. The header record should have a store location of 000 and a current location of 000.

 

 

Transferring Items

Here's how to get there
Expert: Program Menu > Counter System > New Contract
Elite: Left Menu > Transactions > New Contract
 

1. Select Transfer Contract as your contract type.

 

The Select Store to Transfer to screen will appear. This will display the 901 created as a store location.
 


 

2. Select 901 and a message will appear asking you "Are you transferring Now".

 

 

Answering Yes - You will be taken to the Inventory Selection screen where you can start selecting the inventory to be transferred.

 

Answering No - You will be taken to the Event Date and Time where you can select a date to do the transfer. You will be taken to the Inventory Selection Screen.
 

3. Enter the items you are transferring to the 901 location.
 

From Day at a Glance, you will be able to see the item being transferred under the Transfer Out tab.
 

4. Right-click on the transfer contract and select closed.
 

Note: If you are in store one, you will select from store one the transfer contract and close. This will add the items to the 901 location.
