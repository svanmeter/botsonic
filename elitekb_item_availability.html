Item Availability


 

Overview

This article is going to go over some of the features of Item Availability.
 

 
Topics included in this article

  • Inventory Selection Screen
  • Viewing a Header Item
  • Viewing a Serialized Item
  • Sales



Here's how to get there
Expert Program Menu: Program Menu > Item Availability > Item Availability
Expert Counter System: Counter System > Item Availability 
Elite: Left Menu > Item Availability

 

 

Inventory Selection Screen

Once in Item Availability, you will be prompted to look up an item in the Inventory Selection screen.


Column Edit Box - The box in the right corner will open a ribbon box allowing you to customize the columns for the individual user. The box on the top will allow you to save the options selected in the ‘Show Items’ box.

Recently Viewed - This is a search history that allows you to see previously searched items. This will store previously accessed record items in the drop-down list.
 

Note: This will only store record items that have been opened.
 

Selecting the item from the drop-down menu will open the item in Item Availability. If the item you are looking for is not in the drop-down list, you can enter the item key, name, number, part number, search, or use MORE if an exact match could not be found. You can also use the category selection to list all the items in a category.

 

Viewing a Header Item

 

Note: You must have allowed the wait list box to be checked under the options tab in the item record.
 

Note: You must have the instruction file defined under the options tab in the item record.

 

Up & Down Arrows - The up and down arrows can be used to select the next or previous item. If you click the up arrow next to “Key,” then the previous item by “Key” will be displayed. If you click the down arrow next to “Part Number,” the next item by “Part Number” will be displayed.

Key, Name, Part No, Number - Click on any of these green labels and an Item MORE screen displaying the items from the current item forward will be displayed. If you click on “Key,” then the items will be displayed in “Key" order. If you click on “Name,” then the items will be displayed in “Name” order.

Serialized - This button will display all the serialized items under this header. This button only displays for Rental Header Items.
 




Viewing a Serialized Item

If viewing a serialized rental item in Item Availability, it will also display the label showing if it is Available, Unavailable, Reserved, Down for Repair, on an IMO or ISO, or On order, so you can quickly see at a glance what the item's status is.
 

 

Header - This button will display the Rental Header item for this serialized item. This button only displays serialized items.

Kits - This button will display the kit associated with this item. The button only displays for items with kits defined.

Bin Location - Double-clicking in the gray space to the right of the item number will prompt the user to enter an ID and password. If they have the security to do so, they can update the bin location from here. This security can be changed on Item File, in File and Field Security, through the ‘bin location’ option.

All Stores - This button displays all the items that match the current item's key from all stores. This button only displays for multi-store systems.
 


 


 

Previous - This button will divide the calendar in 4-week periods. This button is only available if there is a contract dated before the period displayed.

Today - This button will set the calendar to start for the current week.

X Total Contracts - This button will display the details for ALL the contracts that have the current item on it. X denotes the total number of Open & Reservation contracts written for this item.
 

Note: Contracts will show overdue after.
 


 

Pictures - This allows you to right-click in the box and paste a picture of the item. Once the image has been entered, double-click on the image to enlarge the picture.
 

Note: On SQL databases, users can also drag and drop images from file explorer into their intended item record.
 

Quote - This button allows you to get a price quote for a certain quantity and a specific item period for the current item. You will also be able to change quotes to a reservation or open a ticket.

Missed - This button allows you to add the item to the Missed Rental Report. This will be done when someone calls in for an item and the item is not available for the date desired. Click the Missed button to add the item. When you click on this button, the following screen will be displayed.
 


 

Enter the appropriate information in Customer, Quantity Desired, and Approximate Rental Amount, as well as the reason for the missed rental and any additional notes. Then click the OK button.

Tear Sheet - The button will print out an item information sheet that includes the item rates, pictures, and notes for the item. This can be given to a customer for pricing purposes.

Wait List - This button will put a customer on the waiting list for this item. This button will only be displayed if the item has “Allow Wait List” checked. When you click the button, the following screen is displayed:
 

Note: If the waiting list button is missing, then the item you are viewing in item availability does not have the box checked to allow a wait list in the file record.
 


 

Hourly - This button will display the hourly availability for the current item. The hourly availability will be displayed for a 9-day period. This button will only be available if there are contracts on this item for the current period. The screen will look similar to the following screen:
 


 

Details - This button will display all the contracts that occur within the current period. The detail screen will look like the following screen:
 


Note: When looking at the contract details for a specific store, all items being transferred into that store via Transfer Contracts will be displayed as well.

 
 

Sales

When viewing a sales item, the purchase price plus any additional expenses are displayed at the bottom of the rates column. A breakdown of this is visible when the operator hovers the mouse over the cost.
