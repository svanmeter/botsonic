Crystal Report Viewer


The top-menu options are defined below from left to right.  

Email Report - Allows the user to send the generated report through email. Its icon is displayed as an envelope. 

Export to Excel - Exports the entire report including headers and column titles.

Export to Excel (data only) - Should only export the data. However, depending on the report format some Page titles could come over. Also, if the report was not specifically formatted for Excel the report will not export cleanly. An Excel icon on the top right of the report means that the report can be exported and usually there are hidden fields.

Export to PDF - Will export the report as a PDF.

Export to Crystal - Will export the report to Crystal. Crystal needs to be installed. The file will be saved as .rpt.

Export Report - Will save the .rpt to a specified directory. This is used to troubleshoot reports. The .rpt can be emailed to Point of Rental for troubleshooting.

Print Report - Will allow you to print your generated report. 

Copy - Select the component of the report you would like to copy. Once selected you'll press this button. You can paste it elsewhere on the document or in a separate file.
 

Note: The copy feature is disabled until a portion of the report is selected. 


Toggle Parameter Panel (?) - Brings up the original parameters from the User Interface if available. Not all reports have this functionality. This allows the report to be re-run without returning back to the menu.

Toggle Group Tree - Hides the group tree. If it is hidden it will be revealed. 

 Arrows - There are four arrows: Go to First Page, Go to Previous Page, Go to Next Page, Go to Last Page. Each arrow will respectively take you to the first, previous, next, or last page in the report. 

Page numbers - Enter a specific page number for the report in the text bar to be taken to that page. This also informs you of the total page count for the report. 

Zoom (Magnifying Glass) - Will allow you to zoom in on your report. Includes various increments, page width, and whole page. 



Other Features
 
Find Text (Binoculars) or <Control F> - Brings up a box to enter text to find in the report.

The report name is always at the top left.
 
Note: There could be Tool Tips at the top right of the page which gives more information about the report.

 

Group Tree to the Left


+ : Open the group to show subgroups.

– : Go back up, 1 group.
 
Note: Double-clicking on a group tree name will take the user straight to that page on the report.
