PO duplizieren

 

Übersicht

 

Die Option „PO duplizieren“ wird verwendet, um eine frühere Bestellung zu duplizieren. Derselbe Lieferant, dieselben Artikel und Mengen sowie alle Ausgabeninformationen und Notizen werden in die neue Bestellung übernommen. Hinweis: Dadurch werden Nicht-Massenartikel beim Duplizieren auf den Sammelartikel zurückgeführt.

 

 

Duplizieren einer PO

 

1. Es gibt zwei Möglichkeiten, die zu duplizierende PO auszuwählen. Sie können entweder mit der rechten Maustaste auf die Bestellung klicken oder auf der Registerkarte „Suchen“ oben im Programm „Bestellung“ die Option „Duplizieren“ wählen.

 a. Duplizieren durch Rechtsklick auf eine Bestellung: 




b. Wählen Sie auf der Registerkarte „Suchen“ die Option „Duplizieren“:



Suchen Sie nach der Bestellung, die Sie duplizieren möchten, indem Sie die erforderlichen Informationen in das Bestellungssuchfenster eingeben:



2. Nachdem Sie die Bestellung ausgewählt haben, die Sie duplizieren möchten, wählen Sie „Ja“, wenn Sie gefragt werden, ob Sie diese Bestellung duplizieren möchten.




3. Es wird eine Meldung angezeigt, die bestätigt, dass die Bestellung dupliziert wurde.




4. Sie sehen die duplizierte Bestellung auf dem Bildschirm unter der Registerkarte „Öffnen".
