Shared Users in Elite


 

 

 

Overview:

 

The Shared User feature allows multiple users to use a single terminal (counters and shops, for example) but maintain user security integrity by requiring credentials at critical points.

 

Topics in this article include:

 

  • How do you know if a terminal is set up?
  • Configure a terminal for Shared Sign In

  • Configure the terminal

 

How do you know if a terminal is set up?

You can tell if a terminal is Shared Sign In by the log in changes to Shared Sign In.

Information Tab

 

 

To create a new Shared Operator, in the Information tab check the Shared User box. When this box is checked, only the Operator Name is required.

 

Click Save when done.

 

 

Security for Shared Operators

Select in the header and then

 

 

Shared Operators can be found in the dropdown of users in the browser security feature in configurations. Setting security for a Shared Operator will control initial options for anyone that uses that terminal. Accessing options from a shared terminal will ask for user credentials when accessing a feature. A common practice is to allow options at the Shared Operator and then limit them at the Group and Employee level. This way a manager that has permissions can see and use options at the terminal but an employee with fewer permissions will see the option, but their credentials will not allow them to proceed.

 

 

Configure the Terminal

From Elite, go to your Left Menu > User Tools > Operator Information. 

 

Now click Print Setup.

 

Then go to the Advanced Tab.

 

 

Then click on Modify and type in your Operator Password, if needed.

 

Under Shared User, click the drop-down box and the Shared User.

 

 

Click OK, then click Cancel to close the window.

 

Log out of your Elite software and type in a Login ID and Password then click Sign In. This will change the icon to Shared Sign In and will log you in as the shared user that you created.
