Program Menu Functions Overview

 

Quit: This selection terminates the POINT-OF RENTAL program.

 

1.  Counter System

     This selection runs the counter system program that is used for all rental contract processing.

 

2.  End of Day Processing

 

            Close Drawer for End of Day – This program performs the end of day processing for store.

            Auto Close Drawer – This program performs the end of day processing for your store with out asking any drawer counts. The amount entered in Parameters for default carry over cash will be used during close.

            Print Transaction Edit – The reports lists all the transactions that have occurred on the day that you selected.

            Print Operator Edit Report - This report lists all of the monetary transactions fore each operator for the day that you select.

            Print Check and Card Report - This report lists all of the checks and credit cards for a time period that you select.

            Print Contracts Posted – This report lists all the transactions that were posted to totals on the day that you select.

            Unpaid Contracts Report – This report shows all of the unpaid contracts. This report will print automatically during the End of Day.

            Overdue Items Report – This report shows all of the items that are overdue. This report will print automatically during the End of Day.

            Open Items Report - This report shows all of the items that are currently on contracts This report can be printed automatically during the End of Day.

            Reservation No-Show Report - This report shows all of the reservation contracts that weren’t sent out. This report will print automatically during End of Day.

            Credit Card Inquire – This option displays the total amount of credit cars run through the credit card interface since the last settlement.

 

3.  Accounting System

 

            Account Payments – This program allows you to enter account payments that have been received.

            Print Statements – This program prints statements for your customers. This program is run on demand.

            Print Invoices – This feature allows you to print all of the closed contracts for a selected date range.

            Print Aged Customer Balance – This program prints a summary aging report, with under 30, 60, 90 and over 90 day totals.

 

            Accounting Reports – This option gives another menu with the following accounting reports:

                        Sales Tax Report – This report lists taxable and non taxable rentals, sales and damage waiver amounts for each of your tax jurisdictions. You choose whether you want a monthly, quarterly or yearly report.

                        Contracts Posted Detail Report – This report list each contract that was posted to income and the rental, sale, damage waiver, other and tax breakout.

                        Taxable Contracts Report  - This report lists each contract that had sales tax charged.

                        Non Taxable Contract Report – This report lists all transactions that have non-taxable amounts for the period you select.

                        Tiered Sales Tax Details – This reports lists all transaction that have amounts over the maximum tax amounts listed for tier 1 tax amount.

                        Percentage Report – This report lists details or summary the total amounts collected for Item Percentage.

                        Contracts by Customer Type – This report lists all transactions written to customers with a customer type defined for the period you select.

                        Daily Totals Report – This report lists all of your totals in a daily spreadsheet style format.

                        Reprint Category Totals -  This option allows you to reprint the store’s total sheet for the day select.

                        Reprint Store Totals – This report will reprint the store total sheet for the day selected.

                        Reprint General Ledger Entries – This report will reprint the General Ledger Entries for the day selected.

                        Monthly Deposits Report – This report will list the bank deposits and credit card deposits each day for the selected month.

                        Reprint Bank Deposit Slip – This option allows the reprinting of a bank deposit slip for the selected day.

                        Credit Card Report – This report will break ou the Visa, MasterCard, American Express and Discover amounts on a daily basis for the time period that you select.

                        Disbursements Report – This report will list the individual disbursement for the time period selected.

                        Active Deposits Details – This report gives a detailed listing of all customer deposits held on transaction.

                        Suspense Fund Details – This report gives a detailed listing of all transactions that are held in the Suspense Fund total.

                        In-Store Credit Details – This report gives a detailed listing of all credits held for all customers.

                        Account Payments Summary – This report gives a listing of all payments entered for a giving time period through Account Payments.

                        Average Days to Pay Report – This report gives a graph showing the average days it took to collect money on contracts written broken out on a monthly and yearly basis.

                        Yesterday’s Closed Account Contracts – This report lists all account contracts that were closed on the previous day.

 

            Accounting Inventory Reports – This option gives another menu with the following options:

                        Item Income Report – This program produces a report of income generated by your inventory for the current month, current year and lifetime.

                        Item Income Trend Report -  This report will show the income in detail or summary by category for four different time periods that you select.

                        Income from Non-Home Store – This report will show the income in detail or summary for all income generated at a store other  than the item’s home store.

                        Item Markup Report This report gives the amount and the percentage markup of each sales items.

                        Item Purchase History – This report gives the purchase history for rental or sale items for a selected time period.

                        Sales Inventory Value Report – This program produces a report of the current value of your inventory for sale item.

                        Parts Inventory Value Report – This program produces a report of the current value of your parts items

                        Rental Inventory Value Report – This program produces a report of the current value of your rental items.

                        Yearly Capital Expenditure – This report shows the purchase of either rental or sales inventory broken out on a yearly basis.

                        Cost of Good Sold Report – This report shows the cost and the income for your entire sales inventory.

 

            Fixed Assets Reports – This option gives another menu with the following options.

                        Rental Inventory Value Report – This program produces a report of the current value of your rental equipment.

                        Yearly Inventory Value – This report shows the purchased of either rental or sales inventory broken out on a yearly basis.

                        Sold Asset Report -  This option generates a report of the entire rental items old during the time period you select.

                        Sale Transfer to Rental – This report shows the listing of sale items that were transferred to the rental equipment inventory for the time period you selected.

                        Item Purchase History – This report gives the purchase history for rental or sale items for a selected time period.

                        Depreciation Report – This option generates a report of all the asset items listing the purchase price, current depreciation, and accumulated depreciation.

                        Capitalized Equipment Listing – This option generates a report of all the capitalized equipment.

                        Depreciation Schedules – You can set up to 5 depreciation schedules for each item record.

 

            Sales Rep Report – This option gives another menu with the following equipment.

                        Sales Rep Detail Report -   This report gives you a detail listing of all contracts for each sales rep for the time period entered. This report can be run for either a Cash or Accrual accounting basis.

                        Sales Rep Summary Report – This report gives you a summary listing of total contract values by month for each sales rep.  This report can be run for either Cash or Accrual accounting basis.

                        Sales Rep Profit Report – This report details  out each sale by the sales rep and the profit associated with the sell.

                        Reservation Value by Sales Rep – This report details each reservation for each Sales Rep and the totals the income.

                        Sales Rep Customer Report – The report gives you a listing of all the customer’s name and address assigned to a particular sales rep.

                        Sales Rep Customer Aging Report – This report gives you a listing of each customer’s 30-60-90 day aging for each sales rep.

                        Open Items by Sale Rep Report -  This report gives you a listing of each rental item that I is out on rent for each sales rep.

                        Detail Job Report by Sales Rep – This report gives you all reservations pending with delivery dates, items details, expected return date and all accounts receivable not yet paid by sales rep.

                        Off Rent Report by Sales Rep – This report gives you all items that  have been called off rent grouped by Sales Rep. This report will display the item, date called off rent and number of days it has been waiting for pick up.

                        Dunning Letters – This program prints letters to customers who have over due rental items, past due balances or other user specified criteria.

                        Bad Check Letter – This program is used to print a bad check letter for a customer with a returned check. The customer record is marked  and a notation is entered under comments.

                        Check/Card Lookup – This program is used to find transactions that were paid with a particular check number, credit card number or dollar amount.

                        Customer Credit Limit Report -  This report lists the credit limit of Account Customers and the remaining credit on their account.

 

4.  Purchase Orders (Optional, may not be operational on your system)

     This program allows you to write and receive purchase orders for resale inventory, rental equipment, and miscellaneous items.

 

5.  File Search

     This option allows a search of the various data files (e.g. Customers) for specific combinations of letters or numbers. For example, you may search the Customer file for “ROBERT”

 

6.  File Maintenance

    This program lets at and/or make changes to various data files in your system, such as Customer, Items, and Totals.

 

7.  Inventory Price Adjustments

     This program allows you to make mass changes to rental and /or sales process for one category or all categories by  a specified percentage.

 

8.  Database Mass Change

     This program makes mass changes of up to 10 data fields  in all or selected groups of item or customer records.

 

9.  Configuration

 

            Store Configuration:

Parameters – This program allows you to customize POINT OF RENTAL programs to better fit your method of operation. * You may specify rental ‘overtime’ formulate, grace periods, tax rates, and many others. You can also setup a rental and sale discount tables that interact with the customer record.

                        Printers Selection – This program allows you to select which printers will be used to print contracts, receipts, reports, labels, and End of Day Processing summaries for each terminal.

                        Calendar Maintenance – This program allows out to define the open and close dates and times for your store as well as defining the end of your accounting periods.         

                        Laser Contract Format – This function allows you to define the different laser contract formats that are available when printing a contract.

                        Work Order Formats – this function allows you to define the different laser contracts formats that are available when printing Work Orders.

                        Printout Fields – Under this selection you can edit all of the user defined fields that can printout in the contract when printed.

                        Edit System Letters – Under this selection you can edit and/or create files used by the system, such as New Customer Letter, Instruction Header, and Bad Check Letter.

                        Email Letters – Under this selection, you can create generic letters that can be attached to emails sent through the POR System based on Contract Status: Quote, Reservation, Open, ON Hold, Closed or General.

                        Store Groups – Grouping of stores and/or separate inventory locations defined on this screen.

                        Store Division -  Store divisions such as Party, Equipment, Service are defined on this screen.

                        Delivery Time Window – The delivery time windows such as  9am-11am are define on this screen

                        Delivery Truck – The delivery trucks are entered here.

                        Edit Zip Codes – The ZIP Code ranges for cities in your area are entered here.

                        Call Log Types – Types used to group certain call logs together are defined here.

                        Sales Reps – Each of the sales reps names are entered here. You can also set whether a sales rep is active or inactive

                        Mass Fax/Email Setting – This option allows you to set default settings such as fax cover page, contract format, default printer, subject line and message.

 

            Accounting Configurations:

                        Account Numbers – The General Ledger account numbers are entered here. This is necessary if using the G/L interface to various PC General Ledger program.

                        Disbursement Items – This option allows you to add or change the disbursement items for your system.

                        Discount Tables – The customer discount percentages are entered here.

                        Default Vendor Mark Up – This option allows you define the percentage markup on each vendor based on the value of the item you are purchasing.

                        Multi-Currency – This option is used to define the name and the default conversion rate for different currencies used in your business.

                        PO Payment Methods – This different payment methods available in Purchase Orders entered here.

                        Tax Table – The tax percentages for rental, sale and damage waiver are entered here for each taxing jurisdiction for your store.

 

            Item Configurations:

                        Item Categories -  The item category descriptions are enter here.

                        Item Departments – This field allows you to group items by particular departments

                        Item Messages – The general messages that are printed out on the contract for each item are entered here.

                        Missed Rentals Reasons – The reasons why an item wasn’t available for rental are defined in this section.

 

            Customer Configurations:

                        Customer Types – The customer types are entered here.

                        Customer Groups – The customer groups are entered here.

                        Customer Price Type – The customer pricing types are entered  here

                        Customer Heard about us – The different options for how a customer heard abut your store are entered here.

                        Customer Message – The customer message are entered here.

 

            Transaction Configurations:

                        Transaction Types - The Transaction Types you want to use to break out your contracts by are entered here.

                        Transaction Operations – Transaction Operations allow you to setup sub-status for contracts which allow the contract status to be updated as each process I completed.

                        Transaction Items Substatuses – The different substatuses that are available for each transaction line item is defined here.

                        Contract Questions -  Up to 16 questions can be defined for the counter operations to type answers. Questions can vary depending upon the type of contract or even the customer

 

            Other Updates:

                        Update Website – This option allows you to manually update your intergraded website hosting by Point of Rental Systems. If you do not host your website with Point of Rental, this option will give a trail view.

                        Parts Database Import – This option allows you to import your Parts Database from the CD your vendor sends you.

                        Database Record Count -  This screen displays the number of records in each table of the database.

                        Check for Updates - This function checks whether any program or report has been updated. If so, it shown loads the latest version to your compute. An Internet connection is required.

 

            Licensing – This function allows you to see how Enterprise Software is licensed including drop-dead dates, number of users, and number of stores licensed. This is where you would add licensed to the POINT OF RENTAL.

 

 

10.  Security

            Passwords – The owner’s level, manager’s level and key counter’s level of passwords are entered here.

            Operators Id’s - This function allows you to add/or modify employee Id’s.

            Data File Security – This program allows you to specify the security level for each data field. For example, you may designate the Item Income fields as being viewable only by the owner.

            Menu Security – This function allows you to set the menu options that are available on a terminal by thermal basis.  These setting can also be used to define menu security by operator id.

            Clear Data Locks - This function will display all file and record locks currently set and optionally allow you to clear them. This function should only be used if a power outage caused the server to shutdown while people were using the system.

            User Profiles - This screen displays all of the Point of Rental users along with their printer selection settings.

 

11.  Custom Reports – This program enables you to design your own reports using data from the item, customer or contract files.

Training Database -  The training option runs POINT-OF-RENTAL with a training database that does not affect you store’s “real” database. This can be used to train new employees or for trying something new. The training database is updated with current data by backing up the hard drive to tape using the POINT-OF-RENTAL backup icon. When the Training Database option on the Program Menu is checked, you will be using the Training Database. With the checkbox is unchecked, you will be using your real data.  When in the Training Database, the background will be bright red with the words Training on it. *Note:  The training database is a copy of your real database. Your real database will be copied into your training database every time you perform a system backup. The training database will have your actual items, customer, transactions, etc as of the last system backup. You can add or modify items, customers, or transaction in your training database and it will not affect your real data.
