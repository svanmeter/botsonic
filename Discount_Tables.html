Discount Tables

 



This article will discuss discount tables. Discount Tables allow you to define the types of discounts given to a customer or placed on a contract. If you want to use automatic customer discounts, you must define them in the Discount Code Table. To add an automatic discount to a customer, select the appropriate discount description in the Discount field of the customer record.


Topics in this article include

  • Getting there

  • Discount Code Table





Getting there

Here's how to get there
Expert: Program Menu > Configuration > Accounting Configurations > Discount Tables
Elite: Configuration Gear > App Configurations > Accounting Configurations > Discount Tables


Discount Code Table

For each discount code, you may define the rental discount percentage and the sale discount percentage. Discount codes cannot be used to increase the Sell price of sales items. When using negative amounts for Sales discounts, the system will not increase the price over the current Sell Price. Instead, it will increase the price over the last purchase price. This is helpful when liquidating slow-moving stock and the decision is to sell at cost + a set amount.

After you enter the owner or manager’s level password, the customer discount table will be displayed and you will be able to make changes to it.
 


 

Add/Edit Discount Codes

Click on the discount code you want to change or click the ADD button to add an additional discount code. In either case, the following screen will be displayed:
 



1. You can enter a description, rental, and sale amount for your discount. If you have marked any categories as non-discountable they will show up in the Category Exceptions list. 

Make sure your percentages are expressed as whole numbers. For example, a discount of 25% would be tiered as 25. If you are adding a discount, the discount code will be automatically assigned.

2. Then press OK to save your changes.
