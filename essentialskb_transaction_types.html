With Transaction Types, you have the ability to define a transaction type to assign for each applicable transaction, such as ‘Annual Event,’ ‘Company Party,’ ‘Stadium Project,’ and then later be able to report income off of them.


 

Topics in this article include:

  • Setup in Configurable Dropdowns

  • Assigning to Contracts

  • Reporting off of Transaction Types


 

 

Setup in Configurable Dropdowns
 

To configure your Transaction Types, go to Configurations > Configurable Dropdowns, and add your Transaction Types.
 



 

Assigning to Contracts
 

Then you can assign them to individual transactions. Edit Transaction type, choose your selection, then click the check to save.



 

Reporting off of Transaction Types
 

There are several reports that now give you access to the transaction type.
 
  • Transaction History
  • Transaction List
  • Transaction List by Current Closed Date
  • Transaction List by Item ID
  • Transaction List by Invoice Date
     
When you run any of these reports, you will have a filter to choose only 1 of them for your time period.
 


Also, after the report runs, in manage columns you will have a column selection you can turn on for Transaction Type.
