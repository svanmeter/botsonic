WebPay Setup - Final Steps

 

Now that the Web team has started the implementation of your WebPay module, you’re ready to set up WebPay email templates. Please note that WebPay only supports payments for quotes and reservations. WebPay does not support payments for any other contract type like Open or Work Order contracts. 



 

Create one or more email templates 

  If you plan to accept online payments for quotes AND reservation contracts, then you will need to create a new email template for each. The new templates will include a WebPay link for your customers to use, so they can make a payment online. This email template will replace the current emails that you send to customers with their quotes  and reservations. 

Follow the steps below to create an email template.

 
  1. For Elite: Go to the Configuration Gear (top-right corner) > select App Configuration under Legacy Options >  System Configurations > Email Letters. 

For Expert: Go to the Program Menu> #09 Configuration > System Configurations > Email Letters

  1. Modify or create a new email template to include your Web Payment Link. You can edit any existing templates from the list by clicking on a template. Or you can create a new email template by clicking the Add button located in the bottom left corner. 
  1. Position your cursor in the body of the message where you want to place the link.
  1. Add the WebPayment link to the email body. To do so, press the Special Fields button and select {WebPaymentLink}. When this message is used, the tag is replaced with a web link to your payment site.

 
  1. Setup varying messages for different situations, such as Quote or Reservation, so an appropriate message is delivered to your customer. It may help to copy the body of the email from your current template and add text about the online payment to any new template you create. 
  1. For now (until we complete our testing), make sure that you DO NOT set this email template as the default template for your staff to use. 

If you need assistance, please see our Net Help document “Web Payment Configuration/Setup” or please contact our Point of Rental Support.

 

Create a test quote and/or reservation

Now that you’ve created an email template with a Webpayment link, you’re ready for the next step - testing the WebPay link using an email template. 

 
  1. Create a test quote and/or reservation from Transactions> New Contract. 
  1. Make the value of the contract $1.00. 
  1. Using your new email template with the WebPayment link, email the contact to webpay@rentalhosting.com.  
 

What’s next? 

  • The Web Team will test the online payment feature, then contact you to process a refund. 
  • Once we confirm the initial payment and the refund processes correctly, you’ll be live and ready to accept payments!
