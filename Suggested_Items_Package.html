Empfohlenes Artikelpaket


Mit dieser Option können Sie die empfohlenen Artikel für den aktuell ausgewählten Artikel anzeigen oder auswählen. Beim Einrichten der empfohlenen Artikel ist die von Ihnen eingegebene Menge die Standardmenge, wenn dieser Artikel vermietet wird. Wenn Sie zwei der aktuellen Artikel vermieten, beträgt die Standardmenge das Doppelte der von Ihnen eingegebenen Menge. Sie können eine Null für die gewünschte Menge eingeben. Dann wird der empfohlene Artikel auf dem Bildschirm mit einem Standardwert von Null angezeigt. Der Schaltermitarbeiter kann Null annehmen oder die Menge eingeben, die der Kunde erhält. Auf diese Weise können Sie alle zugehörigen Artikel auf einem Bildschirm ablegen, damit der Schaltermitarbeiter schnell und einfach alle Artikel eingeben kann, die der Kunde erhält.

 

EINRICHTUNG

1. Wählen Sie im Programmmenü>Dateiverwaltung >Artikeldatei > Bestandsauswahl, Artikel auswählen. Wählen Sie im Dateidatensatz Tools> Empfohlene Artikel.

2. Klicken Sie mit der rechten Maustaste irgendwo in das Feld und wählen Sie Neu. Der Bildschirm Bestandsauswahl wird angezeigt. Suchen Sie nach dem Artikel und wählen Sie es aus. Standardmäßig ist der Typ der Auswahl 'Menge’ und die ‘Menge’ ist 0. Der Typ wird durch einfaches Klicken mit der linken Maustaste innerhalb des Feldes und über das Dropdown-Menü geändert. Verkaufsartikel sind grau hinterlegt.



3. Der Benutzer hat dann die Möglichkeit, einen Sondertarif, einen Rabattbetrag oder einen Rabattprozentsatz festzulegen.

*Sondertarif-Hinweis – Bei empfohlenen Artikeln, die kostenlos oder zur Hälfte mit der Miete angeboten werden, markieren Sie NICHT das Kästchen Sondertarif (dieses Kästchen sollte nur angewandt werden, wenn Sie eine andere Preisstruktur für einen Artikel verwenden möchten, der als zugehöriger Artikel ausgewählt wurde). Dadurch wird verhindert, dass die Tarife aktualisiert werden, wenn in Zukunft eine Tarifhöhung vorgenommen wird. Verwenden Sie hierfür entweder das Feld Rabatt oder Rabatt %.

4. Der zugewiesene Aktionstyp bestimmt, wann der empfohlene Artikel dem Benutzer zum Hinzufügen zum Vertrag angeboten wird. Bei Hzfg.(wenn der Artikel zu einem Vertrag hinzugefügt wird), Bei Senden (wenn die Reservierung gesendet wird) oder Bei Schließen (wenn der Vertrag geschlossen wird). Wenn ein Artikel zu einem offenen Vertrag hinzugefügt wird, werden sowohl die Optionen Bei Hzfg. als auch Bei Senden angezeigt.



Standardliste der empfohlenen Artikel

Ermöglicht dem Schaltermitarbeier oder Verkäufer, empfohlene Artikelmengen auszuwählen, während er den Vertrag schreibt.



Mengen-, Kontrollkästchen- und Mehrfachauswahloptionen

Diese Optionen können verwendet werden, wenn der Multi-empfohlene Artikel immer mit der gleichen Menge hinzugefügt wird oder Sie eine bestimmte Menge zu einem reduzierten Preis anbieten. Zum Beispiel würde ein Kompaktlader fast nie mit mehr als einem Anhänger vermietet werden.

•Nachdem Sie die Menge für den empfohlenen Artikel eingegeben haben, klicken Sie mit der linken Maustaste in das Feld „Typ“ des Artikels und wählen Sie Ihre Auswahl aus dem Dropdown-Menü aus.



Menge

Ermöglicht dem Counter, die Menge des empfohlenen Artikels manuell einzugeben.

 

Nein anklicken

Ermöglicht dem Counter, das Kontrollkästchen des Artikels manuell zu aktivieren, wodurch der Artikel dem Vertrag hinzugefügt wird. Keines oder alle Kästchen können ausgewählt werden.

*Hinweis: Bewegen Sie den Mauszeiger über das Kontrollkästchen, um die Menge anzuzeigen, die dem Vertrag hinzugefügt wird, falls er ausgewählt ist.

 

Ja anklicken

Dies sieht genauso aus wie bei „Nein“, nur dass die Kästchen automatisch markiert werden. Wenn Sie auf OK klicken, werden diese automatisch hinzugefügt. Wenn Sie die Kontrollkästchen deaktivieren oder auf Abbrechen klicken, werden sie nicht im Vertrag aufgenommen.

 

Mehrfachauswahl

Dies ist nützlich, wenn ein Kunde die Wahl zwischen einem kostenlosen Anhang für einen Mietartikel hat und alle zusätzlichen Anhänge zu Standardmietpreisen berechnet werden. Mehrfachauswahl verhindert, dass der Benutzer mehr als ein Mehrfachauswahl-Artikel in einer empfohlenen Artikelliste auswählt. Zum Beispiel kann ein Kunde einen kostenlosen Löffel oder ein kostenloses Set Gabelverlängerungen mit der Miete eines Kompaktladers erhalten. Der Benutzer kann nie mehr als ein Artikel auswählen.

*Hinweis: Sie können Anhänge ein zweites Mal erstellen, damit der Benutzer zusätzliche Anhänge zu Standardmietpreisen auswählen kann.

 

Mehrfach Ja

Wählen Sie einfach den am häufigsten verwendeten Anhang im Voraus aus. Während Sie mehr als einen Artikel auf Ja setzen können, wird nur der letzte Artikel, der auf Ja gesetzt ist, als ausgewählt angezeigt.

*Wichtiger Hinweis – Jeder Artikel in einer Liste empfohlener Artikel mit einem Artikeltyp des Mietzubehörs wird dem Vertrag ohne Kosten hinzugefügt, unabhängig davon, auf welche Preise er eingestellt ist.

Beispiel (kombiniert alle drei Methoden)

Dieses Beispiel gibt dem Kunden die Wahl zwischen einem einfachen oder einem Tieflöffel oder einem Set Gabelverlängerungen kostenlos (Mehrfachauswahl). Wenn sie zusätzliche Artikel für eine Gebühr möchten, können sie unten wählen (Nein anklicken). Da die Menge des Gehörschutzes mit jeder Vermietung variieren kann, werden sie bei Menge belassen, sodass der Benutzer dies beim Schreiben des Vertrages eingeben kann.



Erweiterte Verwendung von empfohlenen Artikeln

Große Liste der empfohlenen Artikel:

•Erstellen Sie einen Artikel mit der Artikelart Vermietung – Paket

•Fügen Sie den Rest der empfohlenen Artikel zu diesem Paket unter den empfohlenen Artikeln hinzu

•Fügen Sie das Paket dem Originalartikel unter Tools>Paketartikeln hinzu.

Der Benutzer geht durch die erste Liste der empfohlenen Artikel auf einem Vertrag, und dann die empfohlenen Artikel aus dem Mietpaketartikel. Es wird dem Benutzer einfach als zwei Seiten mit empfohlenen Artikeln angezeigt.

 

Allgemeine empfohlenen Artikel

Artikel wie vergrößerbare Zelte können alle die gleichen Befestigungsmaterialien aufweisen. Anstatt Stangen, Wasserfässer, Zementgewichte und andere Sicherungsartikel einzeln zu jedem Zelt hinzuzufügen, kann ein Artikel mit dem Artikeltyp Mietpaket erstellt werden.

•Erstellen Sie einen Artikel mit dem Artikeltyp Mietpaket (nennen Sie es in diesem Fall „Zeltsicherung“)

•Fügen Sie die allgemeinem Artikel zu den empfohlenen Artikeln zu diesem Paket hinzu

•Fügen Sie das Paket zu den mehreren Mietartikeln hinzu, die diese empfohlenen Artikel unter Tools> Paketartikel verwenden würden

Auf diese Weise müssen alle Änderungen am Zeltsicherungsbestand (z. B. ein neuer Zeltpfahl für Strände) nur einmal am Sicherungspaket vorgenommen werden, anstatt an jedem Zelt im Bestand.

 

Vertrag Empfohlene Artikel

 

In einem Vertrag können Sie mit der rechten Maustaste auf die Option Empfohlene Artikel klicken, das System fragt die Liste der empfohlenen Artikel für den Artikel und dann für den Sammelartikel, falls vorhanden.



Dies gilt auch für Reparaturkits. Wenn Sie also einen Reparaturkit oder eine empfohlene Liste sowohl für den serialisierten Artikel als auch für den Sammelartikel haben, erhalten Sie beides.
