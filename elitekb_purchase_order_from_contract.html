Bestellung aus Vertrag

 

1.  Erstellen Sie den Vertrag (Angebot, Reservierung, Offen, Geschlossen), wählen Sie die Artikel aus, für die Sie eine Bestellung erstellen möchten. Dieser Vorgang kann sowohl für Miet- als auch für Verkaufsartikel auf einem Vertrag ausgeführt werden.

 

 

2.  Halten Sie die STRG-Taste gedrückt und klicken Sie dann mit der Maus auf den Artikel, um die Auftragspositionen auszuwählen. Klicken Sie mit der rechten Maustaste, um ein Feld zu füllen, in dem Sie „Bestellung für ausgewählte Artikel erstellen“ auswählen können.

 

 

* Hinweis – Ausgewählten Artikeln sollten Händler aus dem Artikeldatensatz zugewiesen werden.

 

 

    *Hinweis: Wenn für einen der drei bevorzugten Händler eine Bestellung aussteht, zeigen Sie alle Bestellungen an. Das Counter-Personal kann jede dieser Bestellungen auswählen, um diesen Artikel hinzuzufügen. Wenn sie auf „Abbrechen“ klicken, erhalten sie eine Liste der bevorzugten Händler für diesen Artikel. Das Counter-Personal kann dann auswählen, welcher Händler aufgrund der neuen Bestellung verklagt werden soll.

 

Wenn Sie für Bestellung für ausgewählte Artikel erstellen auswählen, wird die folgende Nachricht angezeigt. Wenn für diesen Händler bereits eine Bestellung erstellt wurde, werden Sie gefragt, ob Sie diese zur aktuellen Bestellung hinzufügen möchten. Wählen Sie „Ja“ zum Aktualisieren

 

 

Es wird eine Liste der Bestellungen für diesen Händler angezeigt. Sie können auswählen, zu welcher Bestellung Sie die Artikel hinzufügen möchten.


 

 

Sobald Sie die Bestellnummer ausgewählt haben, werden die Artikel aktualisiert. In einer Nachricht werden Sie darüber informieren, dass die ausgewählte Bestellung mit den neuen Artikeln aktualisiert wurde.


 

 

Kein Händler ausgewählt

 

1. Wenn der Auftragsposition kein Händler zugeordnet ist, werden Sie beim Auswählen von Bestellung für ausgewählte Artikel erstellen aufgefordert, einen Händler aus dem Händlersuchbildschirm auszuwählen. Wählen Sie außerdem aus, ob Sie die Artikel aus dem Vertrag herausbuchen möchten. Für die erste Auftragsposition im Vertrag wird eine neue Bestellung erstellt.


 

Wenn der Vertrag mehrere Artikel enthält, werden Sie aufgefordert, einen Händler für die zusätzlichen Artikel auszuwählen. Wenn Sie denselben oder einen anderen Händler auswählen, der derzeit Bestellungen hat, erhalten Sie eine Nachricht, dass für den Händler Bestellungen ausstehen. Möchten Sie sie zur bestehenden Bestellung hinzufügen?


 

Wenn Sie ja wählen, wird Ihnen eine Liste der Händlerbestellungen angezeigt. Wählen Sie die Bestellung aus, zu der Sie den Artikel hinzufügen möchten. Sobald die Bestellung ausgewählt wurde, erhalten Sie eine Nachricht, dass die ausstehende Bestellung aktualisiert wurde.


 

 

Verschiedene Händler

 

Wenn Ihr Vertrag mehrere Artikel enthält und für jeden Artikel ein anderer Händler aufgeführt ist.

 

1. Wenn der Händler „Bestellungen“ bereits geöffnet hat, werden Sie aufgefordert, diese zur bestehenden Bestellung hinzuzufügen. Sobald Sie die Bestellung auswählen, zu der die Artikel hinzuzufügen sind, werden Sie in einer Nachricht darüber informiert, dass die Artikel aktualisiert wurden.

2. Wenn es sich bei der zweiten Auftragsposition um einen anderen Händler handelt, werden Sie aufgefordert, einen Händler aus der Händlerliste auszuwählen:

A. Wenn der Händler bereits „Bestellungen“ geöffnet hat, werden Sie aufgefordert, auszuwählen, zu welche Bestellung die Artikel hinzugefügt werden sollen. Sobald die Bestellung ausgewählt wurde, wird eine Nachricht angezeigt, dass die Bestellung aktualisiert wurde.

B. Wenn keine Bestellungen vorhanden sind, wird eine für Sie erstellt und Sie werden aufgefordert, einen Händler aus der Händlerliste auszuwählen. Sobald der Händler ausgewählt wurde, wird eine Nachricht angezeigt, dass die ausstehende Bestellung hinzugefügt wurde.
