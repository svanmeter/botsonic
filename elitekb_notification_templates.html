Notification Templates

 

Using notification templates, you can customize what messages get sent to your customers when select notifications go out.

 

Configurations>System Configurations>Notification Messages

 

 

Unique messages can be set up depending on whether the customer is receiving the notification as an e-mail or a text.  The list of available special fields available are listed in the ‘Special Fields’ button at the bottom right.  Only one template can be set for each action.
