Vertrag ändern

 

Anleitung: Programmmenü > Counter System > Vertrag ändern

 

Ermöglicht Ihnen, ein Angebot, einen Arbeitsauftrag, eine Reservierung, einen offenen Vertrag oder einen geschlossenen Vertrag zu ändern (zu ändern). Es gibt viele Gründe, Verträge zu ändern. Zum Beispiel kann der Kunde seine Meinung bezüglich der Mengen und sogar der zu reservierenden Artikel ändern. Oder vielleicht gibt ein Vorkassekunde Artikel zu spät zurück und möchte seinen Vertrag von einer Woche auf einen Monat verlängern.

 

1. Im Bildschirm „Transaktionsauswahl“ müssen Sie die Vertragsnummer eingeben oder nach dem Vertrag suchen, indem Sie eine der Suchoptionen im Pulldown-Menü verwenden, das durch einen Mausklick auf das Feld „Suchen nach“ erreicht wird. Unter „Feldsuche“ können Sie in einem Feld nach einem bestimmten Wort oder einer bestimmten Phrase suchen, indem Sie an einer beliebigen Stelle innerhalb des Felds klicken. Wenn Sie ENTER drücken, kehrt das Programm zum Hauptmenü des Counter System zurück.

Sie können einen der Verträge auswählen, indem Sie auf die Vertragsnummer klicken. Wenn Sie auf das Pluszeichen klicken, werden alle Artikel zu diesem Vertrag angezeigt. Wenn Sie auf den Kundennamen klicken, wird der Kundendatensatz angezeigt.

Station für Anzeige: Mit dieser Option können Sie nur Verträgen für die gewünschte Station auswählen.

Status: Mit diesem Kontrollkästchen können Sie nur Verträge mit einem bestimmten Status auswählen. Hinweis: Einige Status sind nicht verfügbar, wenn bestimmte Funktionen ausgeführt werden, z. B. ist der Status „geschlossen“ beim Schließen von Verträgen nicht verfügbar.

Historisch verwenden: In Jet zeigt diese Option nur historische Verträge an. In SQL werden sowohl aktive als auch historische Verträge angezeigt.

Zahlung: Mit diesem Kontrollkästchen können nur Verträge für Vorkasse- und/oder Rechnungskunden angezeigt werden.

Suche: Mit dieser Schaltfläche wird die Suche nach allen Verträgen eingeleitet, die die Auswahlkriterien basierend auf den markierten Kontrollkästchen erfüllen.

* Hinweis: Alle Beträge, die von einem Kreditkunden fällig werden, werden grün dargestellt. Alle Beträge, die bei einem geschlossenen Vertrag für einen Vorkassekunden fällig sind, werden rot dargestellt. Bei Verträgen, die zur Lieferung oder Abholung gekennzeichnet sind, wird der Vertragsstatus fett gedruckt.

2. Wenn die ausgewählte Kontaktnummer nicht gefunden werden kann, werden Sie gefragt, ob Sie den Vertrag aus dem historischen Feld abrufen möchten. Wenn Sie mit Ja antworten, sucht das System nach dem Vertrag. Wenn der Vertrag gefunden wird, wird er in die aktive Vertragsdatei übertragen, wo Sie ihn ändern oder erneut drucken können. Wenn Sie „Vertrag aus historischer Datei abrufen“ mit Nein beantworten, kehren Sie zum Bildschirm „Transaktionsauswahl“ zurück.

3. Wenn die ausgewählte Vertragsnummer gefunden wird, wird der Datensatz der Vertragsdatei zu Ihrer Überprüfung angezeigt. Sie können den Vertrag ablehnen. In diesem Fall kehrt das Programm zum Bildschirm „Transaktionsauswahl“ zurück.

4. Wenn der Vertrag bereits geschlossen wurde, werden Sie darüber informiert und dann gefragt, ob Sie fortfahren möchten.

 

Nachdem Sie einen offenen Vertrag ausgewählt haben, sehen Sie Folgendes:

Es gibt mehrere Möglichkeiten, einen offenen Vertrag zu ändern. Sie können auf das Element, das Sie ändern möchten, doppelklicken, um ein Fenster zu öffnen und eine Änderung durchzuführen. Sie können Änderungen vornehmen an (Schlüssel, Name, Status, Menge, Stunden, Fälligkeitsdatum, Fälligkeitszeit, Jeder Rabatt %, Sortierung, Lieferung, Abholung, Schadensverzicht)

 

Beispiel: Zu Menge wechseln – Doppelklicken Sie auf das Mengenfeld – geben Sie eine neue Menge in das Eingabefeld ein, um Ihre Änderungen vorzunehmen.


Wenn Sie einen Vertrag ändern, können Sie fast alle Felder innerhalb des Vertrags ändern. Eine neue Funktion, die hinzugefügt wurde, ist die Registerkarte für die Minderlieferung der historischen Schaltfläche.

Änderungen können auch von Online-Artikel beeinflusst werden, indem Sie mit der rechten Maustaste darauf klicken.

Teilweise Rückgabe - Gibt dem Benutzer die Möglichkeit, eine Teilmenge zu schließen und die Artikel in geschlossene und offene Artikel aufzuteilen

Anlagenverkäufe - Erleichtert den Verkauf des Mietgegenstands im Vertrag

Bruch - Erleichtert den Verkauf des Mietgegenstandes im Vertrag aufgrund von Verlust oder Beschädigung seitens des Kunden. Wenn eine Schadensverzichtserklärung gezahlt wurde, besteht die Möglichkeit, diese zur Deckung der Kosten des Schadens zu verwenden, anstatt den Kunden in Rechnung zu stellen.

Kopfzeile auswählen – Ermöglicht Benutzern, die Kopfzeile wieder in den Vertrag einzufügen. (Wenn beispielsweise eine neue Serialisierung ausgewählt werden muss, die nicht einem Tausch entspricht.

Artikel tauschen - Artikel kann gegen einen anderen getauscht werden, der an dieselbe Kopfzeile angehängt oder derselben Gruppe zugeordnet ist. Stunden und Preise werden für den ursprünglichen Artikel auf Null gesetzt und dem zweiten zugewiesen.

Artikel schließen - Gibt den Artikel in das Inventar zurück

Unterstatus ändern - Ermöglicht dem Benutzer, den betrieblichen Unterstatus eines Artikels zu ändern

Betriebsstundenzähler ändern - Ermöglicht dem Bediener, den ausgehenden Betriebsstundenzählerstand zu aktualisieren

Vertragslink ändern - Ermöglicht dem Benutzer, einen zugehörigen Vertrag mit diesem Artikel zu verknüpfen

Sonderpreis festlegen - Ermöglicht dem Bediener, die Standardpreise für den Artikel zu überschreiben, wenn eine Sicherheitsfreigabe erteilt wurde

Menge nach Station anzeigen - Zeigt die Menge an, die sich im Besitz und derzeit an jedem Standort für Massenartikel befindet

Serviceauftrag erstellen – Erstellt einen Serviceauftragsvertrag für den aus dem Vertrag ausgewählten Artikel. Option zum Ändern der Standort-/Vertragsinformationen angegeben. Füllt die Vertragsnummer für den Serviceabruf in das Vertragslinkfeld für den Artikel aus.

Artikel entfernen - Die Artikelmenge im Vertrag wird auf Null gesetzt

Artikelverfügbarkeit – Zeigt die Verfügbarkeit des Artikels an seinem aktuellen Standort an

Kategorie-Verfügbarkeit - Zeigt die Verfügbarkeit aller Artikel in derselben Kategorie an

Gruppenverfügbarkeit - Zeigt die Verfügbarkeit für alle Artikel in derselben Gruppe an

Paketverfügbarkeit - Zeigt die Verfügbarkeit für alle Artikel im Paket an

Andere Verträge anzeigen – Listet alle anderen aktiven Verträge auf, in denen sich der Artikel derzeit befindet

Artikel anzeigen - Zeigt die Artikeldatei an

Auftragsposition Kommentare – Für die vertragsspezifische Position kann ein Kommentar hinzugefügt oder geändert werden

Artikel verschieben - Fordert auf, das Element an eine andere Zeilenposition zu verschieben

Alle Zeilen auswählen/Alle Zeilen abwählen – Führt diese Aktion für alle Artikel im Vertrag aus

-Spezifischere Elemente können zusammen ausgewählt werden, indem Sie die Strg-Taste gedrückt halten und auf die gewünschten Elemente klicken

-Sobald alle oder eine Gruppe von Elementen ausgewählt wurden, wird durch Rechtsklick auf eines der Elemente ein zusätzliches Menü angezeigt

Ausgewählte Artikel aus Vertrag löschen - Entfernt eine Gruppe von Artikeln zusammen

Bestellung für ausgewählte Artikel erstellen - Erstellt Bestellung aus dem Vertrag für

 

Registerkarte „Lieferung“

Unter der Registerkarte „Lieferung“ können Sie die Lieferadresse, Lieferung und die zugewiesenen Pickups sowie den Ort ändern.

* Hinweis - Neue historische Schaltfläche wurde hinzugefügt, mit der Sie frühere Lieferadressen auswählen können.


 

Wenn Sie auf die Schaltfläche „Historisch“ klicken, werden frühere Lieferadressen angezeigt, aus denen Sie wählen können.

Bildschirm Zahlung:

Als nächstes wird der Bildschirm „Zahlungsoptionen“ angezeigt. Unter „Vertrag“ ändern können Sie Zahlungs- und/oder Einzahlungsbeträge ändern.



 

1.

Miet-/Verkaufszahlungsbetrag (Erstattungsbetrag) - Sie können einen (-) Betrag angeben, der auf den zuvor gezahlten Betrag begrenzt ist, oder einen positiven Betrag bis zur Höhe des geschuldeten Betrags wählen. Der negative Betrag dient dazu, einen für diesen Kontakt gezahlten Betrag, d. h. normalerweise einen irrtümlich gezahlten Betrag, zu erstatten. Hinweis: Wenn der Setup-Parameter so eingestellt ist, dass die Eingabe von Zahlungsbeträgen erzwungen wird, müssen Sie den Zahlungsbetrag eingeben, wenn etwas sich in Ihrem Besitz befindet.

 

2.

Miet-/Verkaufszahlungsmethode (Rückerstattung) - Wählen Sie eine der gültigen Zahlungsoptionen für diesen Kunden aus. Auch hier können Sie auf bestimmte Codes beschränkt sein.

Hinweis* Um eine zuvor falsch geschaltete Zahlungsmethode zu ändern, müssen Sie den vollen Betrag der falschen Zahlungsmethode zurückerstatten, dann den Vertrag ein zweites Mal ändern und mit der richtigen Methode anrufen.

 

3.

Betrag der Anzahlung (Rückerstattung) - kann nur geändert werden, wenn der Vertrag offen ist. Dies ist der Betrag, der bei dieser Transaktion eingezogen oder zurückerstattet wird. Hinweis: Wenn der Parameter-Setup so eingestellt ist, dass die Eingabe von Einzahlungen erzwungen wird, müssen Sie bei der Eröffnung eines Kontrakts einen Einzahlungsbetrag eingeben

 

4.

Einzahlungsmethode (Rückerstattung) - Wählen Sie eine der gültigen Zahlungsoptionen für diesen Kunden aus. Auch hier können Sie auf bestimmte Codes beschränkt sein.

 

5.

Eingezahlter Betrag - Wenn etwas geschuldet ist, können Sie den eingezahlten Betrag eingeben. Die Änderung wird berechnet und angezeigt.




Zurück zu: Bringt den Benutzer zu dem Bildschirm zurück, auf dem er sich vor Vertragsabschluss befand. Bsp.: Tag auf einen Blick, Kunden-Dashboard usw.

Vertrag neu ausdrucken: Bringt den Benutzer zum Druckauswahlbildschirm zurück.

Vertrag ändern: Bringt den Benutzer im Änderungsmodus zurück in den Vertrag.

Vertragskopf klonen: Klont den Kunden und die Daten eines Vertrags, sodass der Benutzer schnell und einfach einen Schwestervertrag abschließen kann.

IRO erstellen: Möglichkeit, eine IRO direkt nach Vertragsschluss zu erstellen.

Kunden-Dashboard: Bringt den Benutzer zum Dashboard des Vertragskunden.

Logistiktickets: Ermöglicht dem Benutzer, vollständige Logistiktickets für diesen Vertrag anzuzeigen, zu erstellen oder zu ändern.
