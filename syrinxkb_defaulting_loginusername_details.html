Syrinx remembers the login details of the last successful login using the connect database screen. So when starting Syrinx if you only get the small "Logon" screen which only displays the "Name" and "Password" fields, press "cancel" to bring up the full "Connect Database" screen and then login with the user that you want it to remember. Then next time you login the small "Logon" screen will default the user name to the user you used.
