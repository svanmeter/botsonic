Delivery and Service Trucks




In this article, we are going to show you how to set up delivery trucks and service trucks in the Point of Rental system as well as assign weights, GVW, and cubic capacity.




Topics in this article include
 

  • Getting there

  • Delivery Truck Screen

  • Using the Service Truck Module





Getting there

How to get there

Expert: Program Menu > Configurations > System Configurations > Delivery Truck

Elite: Configuration Gear > App Configurations > System Configurations > Delivery Truck


 

Delivery Truck Screen


All trucks used for deliveries are defined in this section. The delivery trucks will be displayed.  Click on any truck to view or modify the record.
 

 

1. If you want to edit a Current Truck, click on the desired truck you wish to update. If you want to add an additional truck, click the Add button. In either case, this screen will show up.

 


2. Enter the below information. 


Number - If you are adding a truck, the number will be automatically assigned.
 
Store - A store number of 000 will allow this truck to be shared between all the stores. A store number of 001 would only allow store 1 to select this truck for deliveries.
 
Name - Enter a description of the truck.
 
Service Truck - If you have the optional Service Truck module, you can mark this truck as a service truck. A service truck is one that has inventory located in the truck. Once it is marked as a service truck, you can transfer inventory to and from the truck. In addition, you can run reports that show all the inventory in the truck.
Notes - You can make any notes that you would like other operators to know about this truck.
 
Inactive - Check this box if this truck is no longer available for deliveries.
 
Year - This is the year of the truck.
 
Weight - The weight class of the vehicle. This is used in delivery reports and the Dispatch Center to calculate the weight of the load. This field does not consider any type of unit of measurement.
 
GVW - This is the gross vehicular weight. Used in delivery reports and the Dispatch Center to warn if a route has exceeded the weight limitations.
 
Cubic Cap - This allows you to enter the max capacity of the inside of the storable space by cubic inch. The cubic size does not consider any type of unit of measurement. If you wish to track by cubic meters, enter that item's whole number size and vice versa for cubic feet measurements.

 

3. After you have made any necessary changes, press the OK button.


 

Using the Service Truck Module
 

If you have inventory stored off-site or in remote warehouses, such as service trucks or jobsite trailers, this feature helps you track and move inventory from stores and multiple inventory locations without creating transactions at those sites. In addition, reports can be generated to show inventory available at the locations.

 

If you have the Service Truck Module you can transfer inventory into your trucks. To do this from the New Contract Screen, click on Transfer Contracts.

 

 

From here, you will be able to choose the Service Truck and finish creating the contract as normal.
