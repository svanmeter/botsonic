Overview
There is now a new add-in available for Outlook that integrates with Syrinx and allows users to save
email messages as files and save these as calls against customers in the system.

Note: The Outlook plugin requires will only work with a 32 bit version of Office.


Setting Up
There is a new option when installing Syrinx v4 to include the outlook plugin as part of the install. This
must be checked to install the files with Syrinx:

 


To use the outlook plugin – you must set up a new documents folder – ‘Email Files Folder’ within the
Documents Root Folders screen in Syrinx Configuration. The physical folder will also need to be created within the appropriate directory.  The email files will be saved in this folder.

 

 

Configuration
There is a new menu item in the Start Menu called ‘Outlook Plugin Config’, which provides a shortcut
to the SyrinxOutlookAddIn.CONFIG file which is located with the main Syrinx install files:

 


If the shortcut is not available in your version of Windows, the Outlook Addin configuration file, SyrinxOutlookAddIn.CONFIG, is found in C:\ProgramData\Higher Concept\SyrinxOutlookAddIn.


This will open up notepad where you can edit the Server and Database names which the Outlook add-in
will use to communicate with your server/database:

 


Note: If Syrinx has been installed on the machine before, this configuration file will use the latest
server and database details that are stored in the registry – so the configuration file may not need
amending.

 

Usage
You will now see that when you right-click on one or more selected emails you will have a new option
“Save to Syrinx”


Clicking this will search Syrinx for any contacts that have the email address of the selected emails.

 


If no contacts exist in Syrinx with that email address you will be asked to find or enter the contact

 


If you pick a contact, the email address of the selected email will get assigned to this user (provided
they don’t have an email address). Otherwise you can enter a new contact which will get added to
that customer.


The next screen requires the user to enter a description for the call that will get logged against this
customer:


The contact that has been selected or created will also get a call logged against them with the type
“Email Received”


The email will finally get saved in the folder that was set-up as the root folder for email files as an
outlook attachment.


Sent Box Monitoring
The add-in also monitors the sent box in Outlook, so any emails sent will get saved and added to a
contact in the same way as mentioned above. The only difference is that the call log type will be “Email
Sent” instead of “Email Received”.
