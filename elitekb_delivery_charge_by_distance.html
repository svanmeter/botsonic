Delivery Charge by Distance

Calculate the distance from the store to the delivery address. It will use the |MILE item to charge for the distance.

* Note: Mile distance will be calculated using Google Maps.

Create an item record with the name |MILE for the item key. Mark the type as a Sale Normal. Enter the price per mile in the sell price field.

 

Enter the minimum delivery fee in the retail price field. (ex. $30.00 delivery fee). This would be the default price if the delivery amount is below minimums.

 

When you enter |MILE on the contract a map will appear showing you the distance to the customer or job site. You will be able to select different routes based on the fastest route, etc. You can select the route you want by clicking on the left window pane on the route description. The distance will be calculated from the contract’s store to the delivery address.

* Note You must have a valid address and zip code in the delivery address, located under the delivery tab, if there is no address and zip code is entered, it will not calculate the miles.

 

Example 1 - Over Minimum Price, will calculate per mile times the quantity entered. (ex. $1.00 x 50 miles = $50.00)

Example 2 - Below the minimum will charge the minimum default price. (ex. $1.00 x 15 miles = $ 30.00 or the default rate).
