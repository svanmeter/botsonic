Overview

Bulk item availability allows you check the stock levels of bulk items from within the ‘Enter Hire Contract’ and ‘Hire Stock List’ screens to see if you can fulfil the order. This is in addition to the ‘Stock Bookings Screen’ and ‘Bulk Stock Items’ report. This is particularly useful for items such as alloy towers that would be made up of many components.

System Setup.

This feature is turned off by default.  To turn the feature on, use the ‘Options’  tab of the Syrinx Configuration\System Options\Contract Entry:

Setting this option turns the feature on, and also defines the threshold that determines when a warning is issued.

  • Set the option to -1 (minus one) if you only want to be warned when there are not enough items available for the hire.
  • Set the option to 0 (zero) if you only want to be warned when the hire will use the last remaining item(s).
  • Set the option to a positive number if you want to be warned whenever a hire will take you down to your last n items.

Contract Entry – Show Bulk Item Availability.

When this option is set, the ‘Enter Hire Contract’ screen has a ‘Bulk Availability’ button in the lower left of the screen:

When you press this button the availability of bulk items between the contract’s start and planned end dates are analysed, and the results are displayed in the ‘Bulk Item Availability’ screen:

 

The ‘Qty Owned’ column shows the total quantity owned by the depot that the contract is for, the ‘Qty UnAvail’ shows the qty that has been manually set as unavailable on the stock record, the ‘Qty L/D/S column shows the quantity on any unconfirmed loss/damage or sales notes, and the ‘Qty Req’d’ column shows the quantity required for this hire contract.

The next three columns indicate the situation on the start date of the hire – i.e. how many items are available on that day (‘Qty Avail’) and of those that are not available, how many are actually on confirmed contracts (Qty On Hire’) vs those on unconfirmed contracts (‘Qty Booked’).  Hence the system treats unconfirmed contracts equally with confirmed contracts when determining the available stock, but shows the breakdown between the two.  Note however, that if an unconfirmed contract is marked as ‘Quote’ as the type of contract, then it is not included in these figures.

The final four columns indicate the ‘worst case scenario’ for the duration of the hire – i.e. the system will look through each day of the hire (up to the first 3 months), and find the lowest availability across this date range.  The date of the lowest availability is displayed in the final column

The screen highlights in red any items where the quantity available is below the quantity required – i.e. where there are insufficient items available for the hire.  It also highlights in yellow those items where this hire will leave a quantity remaining on or below the threshold set by the system option.

If no planned end date is entered, then the system checks the availability on the start date only, and the remaining columns are left blank:

Contract Entry - Validation.

Irrespective of whether or not you have pressed the ‘Bulk Availability’ button during contract entry, the system will check the availability prior to printing and confirming the contract.  If any items fall on or below the threshold, a message is issued:

..and if you say no, then the print/confirm is cancelled, and the focus is switched to the ‘Bulk Availability’ button ready for you to press the button to show the availability breakdown if required.

Planned End Date on Open Contracts.

The ‘Qty On Hire’ numbers reflect the planned end date (that may optionally be set on a contract) using the same rules as the ‘Stock Bookings’ screen.  If the planned end date is in the future, then the system assumes the contract will be off hired on that date.  If the planned end date is the past, then the behaviour is determined by the ‘Use Planned End Date if Expired?’ system option on the ‘Bookings Screen’ tab of Syrinx Configuration\System Options\Hire Contracts:

If the option is set to ‘No’ (the default) then the planned end date on the contract is ignored if it has elapsed.

 

Hire Stock List.

The ‘Bulk Availability’ button is also available on the ‘Hire Stock List’ screen, to show the availability of the bulk items belonging to the selected category:

This shows the availability at the given from date, and for the period up to the given to date.  These dates default to the current week ahead, but can be amended if required.
