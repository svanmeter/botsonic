Schließen eines Stundenzähler-Artikels

 

Wenn Sie einen Vertrag schließen, der einen Stundenzähler-Artikel enthält, werden Sie gefragt, wie hoch der Zählerstand ist, wenn der Artikel zurückkommt.


Schließen eines Artikels mit Stundenzähler

 

1. Schließen Sie den Vertrag, indem Sie in Counter System (Expert) auf Vertrag schließen gehen oder auf Transaktionen (linke Menüleiste) > Vertrag schließen (Elite).

 

2. Nachdem Sie die Artikel geschlossen haben, erscheint das folgende Fenster mit der Aufforderung, den Zählerstand des zurückerhalten Artikels einzugeben.




Wenn die gezählten Stunden größer sind als die in Ihren Parametern festgelegte Zeit, erhalten Sie folgende Meldung.

Beispiel: Wenn Sie an einem Tag 10 Zählerstunden zulassen und der Zählerstand 11 Stunden beträgt, erhalten Sie diese Meldung.



Eine weitere Meldung kann erscheinen, wenn der Zählerstand die tatsächliche Zeit überschreitet. Beispiel: Eine eintägige (24-stündige) Miete mit einem Zählerstand von 25 Stunden.



Dies ist physikalisch unmöglich, und Sie erhalten eine Warnung, in der Sie gefragt werden, ob Sie den Zählerstand erneut eingeben möchten (das Programm geht davon aus, dass Sie einen Fehler gemacht haben).



Wenn Sie sich gegen eine erneute Eingabe entscheiden, haben Sie die Möglichkeit, den Zählerstand oder die tatsächliche vermietete Zeit zur Berechnung der Gebühren zu verwenden.



Wenn Sie GPS(AEMP) auf diesem Gerät installiert haben, haben Sie die Möglichkeit, den Stundenzählerstand vom GPS-Gerät abzurufen.
