Inventory Count

 

This is a ‘global enabled’ feature that allows a user to assign the counting of items to other users to be completed in the POROne app. It requires Single Sign On (SSO) credentials to be used at login to Launchpad.

 

Security

 

Launchpad: 

  • Display of Inventory Count option is controlled with Left Menu security.  
  • The ability to ‘Add’ a new Inventory Count has two controls:
    • Launchpad Settings - ‘Insert Inventory Count’
    • Global Licensing: Inventory Count
  • The ability to modify an existing Inventory Count is controlled by Launchpad Settings: ‘Update Inventory Count’.
  • The ability to assign an Inventory Count to a user: the user must have ‘Item Count’ in Global Admin permissions.

POROne

  • For a user to have the option to create an Item Count, the company must first have the license for ‘Item Count’ turned on. Then, at the User in GAdmin, they must have ‘Item Count’ turned on.

Viewing Inventory Counts

  • Select the Inventory Count option from Left Menu
    • If an error is displayed when the Dashboard is opened, it could be that SSO credentials were not used to login. SSO is required for this feature.
  • From here you can search and filter the list of counts. You can also set ‘Saved Search Views’ as desired.
  • Highlighting one in the grid will display Summary data on the left side.

Creating an Inventory Count

  • Select the ‘+’ option from the Action ribbon. This will open an Inventory Count form.
    • Security and global permissions required to see option.
  • To add items to be counted, select the ‘Add Existing’ option.
    • From here you can search items to be added.
    • Tips for creating a count:
      • Hide ‘Shared Inventory’ so only serialized items are selected.
      • You only have one search opportunity to create a count. Once Submitted, additional items cannot be added.
      • You can select/deselect using the checkboxes on the left column.
  • Assigned To
    • Only users that have Global permissions for ‘Item Count’ will be options
  • Location
    • Select the location you expect the count to be performed at.
  • Expected Count Date
    • Select the date you wish the items to be counted by. You can filter and sort by this date in the list of counts.
  • Description
    • This field can be a more detailed description of the count.
  • Exclude Current Qty - currently not an active feature - future release.
  • Allow Additions - currently not an active feature - future release.
  • Submit to create new Inventory Count. 

Counting items on POROne

 

Count lists will be displayed on the Task page for the assigned user on POROne. That user will simply select the task and enter the current count of items being counted. The quantity displayed is real-time.

 

Finalizing Inventory Counts

 

A completed Inventory Count can be viewed and Finalized (Check mark option in Action ribbon). The Finalized process is the same as doing this from the legacy ‘Update Inventory Count’ option. The process adjusts the current quantity using the ‘Count Dif' value.

 

When viewing an Inventory count there is helpful information to help you understand the results and effect of Finalizing the count.

 

Qty Variance: Net number of items found/missing.

 

Value Variance: Naet value of found or missing items.

 

Counted Date: Allows you to see items that have been counted vs those that have not (no date).

 

Before Finalizing, you can ask employees to go double check and look again. They can go back into a count and update the counted values if needed. Once a count has been Finalized, you would need to create a new count for any items you want a re-count on.
