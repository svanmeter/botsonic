Elite | Accounting Sync Service




This article will show you how to see if it is running via the Database Locks Table. The Accounting Sync Service allows the accounting integrations like Sage Intacct, Great Plains, Sage 50, and more, to run and sync with your dedicated accounting software.

 

Accounting Sync Service

As of version 2023.05, we've placed the Accounting Sync Service in the Database Locks Table to indicate whether it is running.

If it is not running, it will need to be turned back on via Services Application in Windows. DO NOT clear any locks from this area.
