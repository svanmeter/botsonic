Syrinx® - Logging Customer Calls

 



 

Overview: This guide is designed to help you with logging customer calls.

 
 

There are a few ways to add calls into the system. One way would be to go into the customer record via CRM and choose to amend or query customer. You will then see a calls tab.
 

To add a call, you need to click Add Call at the bottom. You will then be presented with this screen.
 

You can enter the date and time, Contact (press insert if they don’t exist), and a Call Type (these can be added in the configuration). You then just need to enter the details of the call. At the bottom you can use the action after. This will remind you to call them back. You can set a priority and a call outcome. You can also assign the call to a different Syrinx user.

 

Alternatively, you can enter calls by going into CRM and then clicking on Calls Awaiting Action. There is also a keyboard shortcut of F11.
 

To add a call, you would need to click on the New Call button. You are prompted to search for the customer and then add the call information as detailed above.

 

All existing outstanding calls appear on this screen. You can change the user (permission based) at the bottom and see other users calls awaiting action. To close a call reminder, you can either edit the call and remove the action date or you can add a Follow Up Call. When you close the follow up call, you will be prompted as shown below. This will remove any action dates and make this call the only one that needs actioning.
 

 All outstanding calls can be viewed from the Calls Awaiting Action screen or via the User Alerts.
 

There are also call reports that can be run.
