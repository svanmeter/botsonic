Daily Analysis 


 

This report shows the income generated by number of days rented.  This is useful to determine whether an increase in a particular rental period such as a month will have any effect on the item income.  The report will look like the Hourly Analysis Report except it will group everything rented less than a day in one grouping. You can select 24,36,48 or 60 Months to be displayed.
