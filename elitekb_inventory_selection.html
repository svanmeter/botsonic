Bestandsauswahl

 

Überblick: Auf diese Weise können Sie anhand bestimmter Kriterien nach Inventargegenständen suchen. Zugriff über das Programmmenü und über das Counter System.

 

1. Suchfelder: Geben Sie in das Suchfeld für eine allgemeine Suche nach einem Artikel ein Schlüsselwort ein oder geben Sie den Artikelnamen, die Teilenummer, die Artikelnummer oder den Artikelschlüssel ein

 

2. Feldsuche - Ermöglicht dem Benutzer die Suche in den folgenden Feldern, die in einem Artikeldatensatz enthalten sind: Schlüssel, Name, Teilenummer, Suche, Hersteller, Modellnummer, Lieferantennummer, Gruppe, Lagerplatz, Barcode, Seriennummer, Lizenznummer, Stile und GPS-Gerätenummer.

3. Kategoriesuche - Ermöglicht dem Benutzer die Suche nach einem Artikel über eine einzelne Kategorie.

4. Elemente anzeigen - Mit dieser Option können Sie auswählen, welche Arten von Elementen durchsucht werden sollen. Sie können wählen, ob Sie nur nach Mietgegenständen oder gleichzeitig nach Miet- und Verkaufsgegenständen suchen möchten. Wenn Sie nach allen Typen suchen möchten, doppelklicken Sie auf das Wort "Elemente anzeigen". Alle Felder werden ausgewählt.

5. Stationen - Verwenden Sie das Dropdown-Menü, um Alle Stationen oder einen einzelnen Standort zu durchsuchen.

6.. Im Feld auf der rechten Seite können Sie die Multifunktionsleiste anpassen.

7. Mit der obigen Schaltfläche oben können Sie Ihre Multifunktionsleistenauswahl und Ihre Kontrollkästchenauswahl speichern.

 

Kopfzeile

In der blauen Kopfzeile können Sie aus einer Liste von Optionen auswählen, wie Sie die Informationen im Bestandsauswahlbildschirm anzeigen möchten. Klicken Sie mit der rechten Maustaste auf die blaue Kopfzeile, um auf zusätzliche Optionen zuzugreifen.


 

Wenn Sie das Kontrollkästchen aktivieren, wird eine neue Spalte zu Ihrem Bestandsauswahlbildschirm hinzugefügt. Sie können die Elemente nach oben oder unten verschieben, um Ihre Spalten neu zu positionieren. Durch Klicken auf die Schaltfläche Speichern wird die Bildschirmeinstellung für Sie gespeichert. Wenn Sie das nächste Mal mit Ihrer ID und Ihrem Passwort auf den Bestandsauswahlbildschirm zugreifen, werden die gespeicherten Einstellungen angezeigt. Einstellungen können entweder von einem bestimmten Benutzer, einer bestimmten Firma oder einer bestimmten Gruppe gespeichert werden. Wählen Sie einfach die entsprechende Einstellung aus dem Dropdown-Menü. Suchen Sie regelmäßig nach neuen Aktualisierungsspalten, die der Liste hinzugefügt wurden.
