Troubleshooting a Network Printer

 

This article covers common printer problem scenarios and suggests possible fixes for them.  For information on adding a new printer see the Network Printer Configuration Guide



Topics included in this article
 

  • Suddenly none of my users can print to a specific printer
  • Everyone prints to the correct tray except for one terminal

 

Suddenly None of My Users Can Print to a Specific Printer

 

1. Verify the printer is on.

2. Verify the network cable is plugged in (or WIFI is configured correctly).

3. Check the printer display for any errors such as a paper jam.
 

Everyone Prints to the Correct Tray Except for One Terminal
 

1. Go to the terminal with the problem.

2. Go to Devices and Printers (either from Start Menu or Control Panel).

3. Right-click on the Printers with the issue and click Printer Properties for Tray 1.

4. From the General Tab, click Preferences.

5. Find where it says which drawer to automatically print from, select Tray 1.

6. Click Ok, and exit out of the printer settings.

7. Repeat the same process for Tray 2, except change to automatically print to Tray 2.
