AEMP Set to Task Runner

 

 


Overview

This article will show you how to set a task to automatically update the AEMP files from your provider. Your scheduled tasks update the AEMP files from the provider to your Point of Rental system. Tasks can be set up to run after business hours.


How to get there:
Expert: Program Menu > Configuration > System Configurations > Configure Task Runner
Elite: Gear icon > App Configurations > System Configurations > Configure Task Runner


Setting a Task

 

1. Select either Add or Edit task. 

 

2. Enter the following user information in the appropriate fields listed below. The Start Time can be different depending on when you want the AEMP to happen. The drive letter for the File to Run and Start in Path text boxes can be different, too, depending on the drive letter the Point of Rental program is installed on.

 

Name - AEMP GPS
 

Description - AEMP GPS Task
 

File to Run - (PORDRIVE):\POR\PROGRAM\GPSTELEMETRICS.EXE
 

Start in Path - PORDRIVE:\POR\USERS\
 

Command Arguments - Auto
 

Recurrence - Recommended Minimum 2 Hours. Can be set to 1 Hour but if this user has lots of items and takes more than an hour to download it will not update that next hour, which will then be 2 hours.
 

Interval - Recommended Minimum 2
 

Priority - Low

 

3. Click the Save button when done.
