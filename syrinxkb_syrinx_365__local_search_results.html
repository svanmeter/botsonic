Search Results


When you open a Local Search screen for an entity, for example, Search Customers or Search Hire Contracts, you will be presented either with results or no results depending on your user settings. See Syrinx 365 | Local Search User Settings for more information.

There is a search field where you can type in something to search for i.e. stock number, customer name, or document number depending on what search you are in, and then will be presented with the results that match.





Using Filters


If you want to refine the results in the grid or define what you are searching for then you can use the Filters at the top.

When you click the dropdown in the More field you can choose what action you want to do.



Add Filter - This will start the process of adding a filter to the search.

Clear All - This will remove all the filters on that search.

Reset Filters - This will retain the filters but remove the values within the filters.

When adding a filter you can scroll to find the required filter name and check the box next to the name or you can use the Search field to find the desired filter.



The type of filter selected will determine the available criteria for the filter. When you have entered your search criteria, click Apply for it to be applied.

If you want to clear the criteria but keep the filter, then click Reset.



When you have filters enabled, they will be displayed along the top to display how the results have been returned.

You can add, remove, or click into the filter to amend its' criteria.




Grid Columns

There are a default set of columns displayed in the grid as part of the System View.

You can resize columns by dragging the edge of the header making it longer or shorter. You can also rearrange columns by dragging the header and dropping it into whichever position you would like to view it in.

For example, if a column over to the right is one you want to see on the left, then you'll be able to rearrange them.

If you want to add or hide columns, use the Hamburger Menu located at the top right of the grid and then select Actions > Select Columns.



You can then check to add or uncheck to hide columns by finding the column in the list. There is also a search field that allows you to quickly find the column.

When you're done making changes, click Apply to save your changes.
 
If you want to back out of the column selection without saving our changes, click Cancel.



 

Pagination

At the bottom right of the grid will be the pagination information, which will show how many items are being displayed per page and which results are being displayed. There are left and right toggles to move through pages.




Page Views

There will always be a System View page that contains "default" columns. Your user may have other saved page views that you have created or an admin user has created and assigned to you.

Please refer to the article Syrinx 365 | Local Search User Page Views for more details.
