Syrinx® - Countersign





Overview: This guide is designed to help you with using Countersign.

 

 

 

From the Print ribbon, you need to select Countersign.

 

 

You will then be asked how you have confirmed the customer’s identity. You can add additional customer identity types to the configuration.

 

The system will then push the document to the device.

 

 

The customer then needs to read the contract and swipe down to the signature box.

 

 

Once the signature has been captured, tap Sign Document. You will then be presented with the confirmation screen. The customer needs to agree with the conditions and tick the I agree box and then tap Confirm Signature (or Redo Signature if required).

 

 

Once completed, the user can view the completed document by going to the details tab of the unconfirmed contract.

 

 

The user can also access the document via the query contract screen.
