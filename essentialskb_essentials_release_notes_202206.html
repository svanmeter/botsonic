5/2/2022 - Version 2022.06


  • Delivery Vehicles - Can I get an Amen, a Hooray, or at least a Clap for our Development team? Months of hard work has now provided this landmark feature. 
    • Create all your delivery vehicles
    • Assign them to transactions (Yes, you can assign different delivery and pickup vehicles for each transaction)
    • Delivery/Pickup Information shows Assigned Vehicle
    • Create filters in your workbench to see individual vehicles' upcoming orders
    • Daily Deliveries and Pickups report now has a selectable column for Assigned Vehicle
 
  • Review of Workbench Features - Since we added another workbench feature for Delivery Vehicles, we thought it would be a good time to review some of the workbench features we have recently added. 
    • We have added a pickup icon to applicable contracts as well as an event icon on the workbench.
    • Event Name will now show on each contract on the workbench.
    • You can now create a filter for specific events, or you can simply use the search function on the workbench.
    • Great news! If you would prefer to start in the workbench instead of having to go through the dashboard, you can go to User settings and set workbench as your default.
    • We have added a shared filter for those of you that use the storefront to filter only web orders on the workbench. A great feature for you to be able to see those orders that were entered via the storefront.
    • It used to be that when you scrolled down in the workbench, the header would disappear. This caused people to get confused and lost, and some even had to phone a friend (us) to guide them back safely. So now, the header locks in place when you scroll down. As an added bonus, we added this same feature to other parts of the program, including inventory and customers.
    • If you create a non-shared filter in Inventory/Workbench/Customers, you can pin it so that it will stay on that selection when you come back to it, and will hold that setting for your user.
