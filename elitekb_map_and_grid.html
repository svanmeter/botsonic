Map and Grid

This will show all routes scheduled for a vehicle on the map. By choosing the map and grid feature, it will add functionality like validating stop addresses, verifying on-time statues and the ability to visually check the routing.

 

After specifying the departure time, click the Map and Grid button. The dispatch center will validate each address.

* Note – Each stop will be represented by a flag and a pop up balloon with information about the stop.

 

Once a route has been saved, the route number will turn red if a modification happens. The user can click on the route number to see what changes took place.

 

Overweight and Oversized

Based on weights/cubic sizes set on items and trucks, dispatchcenter will warn the user if they will be overweight or oversize in the route screen. Initially, you will receive a warning message if the route will be oversize/overweight. Additional information is available in the ‘Max Weight’ column in the route view. Weight is added and subtracted as contracts are dropped off or picked up. If at any point the truck will be overweight, it will turn that stop in the column red.

 

Independent of the weight, if at any time a stop sends the route oversize, that stop will turn yellow.
