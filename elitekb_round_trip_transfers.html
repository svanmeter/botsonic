Round Trip Transfers




Overview
 

Round trip transfers are when the same contract is used to transfer items to another store and then back to the original store. This will temporarily add the rental items into the other store’s inventory.

When using round trip transfers, it is recommended to answer No to the prompt that asks you to create a return transfer. If this feature will be consistently used, uncheck the Parameter setting "Ask for Return Transfer" in the Setup tab to stop the prompt from appearing. If yes is selected, a separate return transfer contract will be created. This can cause inventory issues as double the quantities will be transferred back to the original store.



Here's how to get there
Expert: Program Menu > Counter System > New Contract > Transfer Contract

Elite: Left Menu > Transactions > New Contract > Transfer Contract 



Transfer Contracts follow the contract Lifecycle. They can begin as a Reservation and then can be converted to Open, and Closed. Follow the below steps to transfer your inventory. 
 

1. Once you have selected Transfer contract, select the store your items are being transferred to.
 

2. Are you transferring now? Prompt – Select Yes, if transferring now. This option will create an open status transfer contract. Select No, if you are creating a transfer for a future date and select the date the transfer is going out. This option will create an reservation transfer contract.
 

3. Select the items to be transferred. Verify the hours match the time frame the items will be at the other store’s location. Two lines will appear, one negative quantity to remove from the original store and one positive quantity to add to the second store.

4. Manually check the Delivery and Pickup boxes. Verify the pickup date is the day the items are to return to the original store.

5. Send out the transfer. This step is only applicable if you selected No in step 2.

  • To send out the transfer, inquire the transfer contract and select Tools (in the top toolbar), then select Sent Out Reservation. The item quantities will show in the second store. 


7. When the items are returned to original store, close the transfer contract.

 

If you're using the Dispatch Center, the transfer contract will only appear in the original store to be routed.
