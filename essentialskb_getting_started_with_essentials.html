Grids

 

You interact with the system via grids.  They all work the same, so it’s easy to get started.

Help

 

Help is available throughout the application and is available on many fields. Just hover over the i symbol.
 

 

 

Bulk Actions

Filters

 


All the grids allow relevant filters.  Click to filter the grid. This will place the name of the filter above the grid.  Click that name to remove.

 

Edit a filter that you created: 


View the details of a filter someone else created (you can save your own copy):


Creating or editing a filter is easy.  You can add multiple criteria and only records that meet all the criteria are displayed.
 


If Share Filter is on, other users in your company can use your filter. You can remove a criteria by clicking: X


 

Search

 

Simply type into the search box to find something in the grid.
 


It searches the whole record including data that isn’t displayed in the grid.

 

Dropdowns


All dropdown inputs allow you to quickly filter to find the selection you’re looking for.


Dates

 

Anywhere a date is required, you can select from a calendar.
 


Changing Data

 

If you make a change to the information in a form, the system will prompt you to save it.
