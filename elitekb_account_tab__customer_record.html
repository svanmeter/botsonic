Konto Tab – Kundendatensatz

  

Übersicht

 

In diesem Dokument werden die Funktionen und Felder des Konto-Tabs in einer Kundendatei erläutert.


Rechnung an – Hier können Sie angeben, wohin die Rechnungsinformationen gesendet werden sollen. Sie können die folgenden Informationen eingeben (Kontakt, Telefon, Adresse, Stadt, Bundesland, Postleitzahl).

 

Auftragnehmer-Lizenz – Ermöglicht die Eingabe von Informationen zum Auftragnehmer (Lizenznummer, Ablaufdatum)

 

Sonstige Info – Ermöglicht Ihnen die Eingabe weiterer Kundeninformationen über die benutzerdefinierten Felder.

 

API Link – Dieser Link wird für die Salesforce-Integration verwendet.

 

Vertrag – Hier können Sie festlegen, wie Kunden ihre Rechnungen erhalten sollen. Rechnungen werden am Ende des Tages verschickt.

 

Format – Ermöglicht Ihnen die Auswahl verschiedener Rechnungsformate für den Versand an den Kunden.

 

Massensendung an – Hier können Sie auswählen, wie der Kunde seine Rechnungen erhalten soll. (Ausgedruckt, E-Mail, Fax).

 

Hinweis: Eine dieser Optionen muss ausgewählt werden. Wenn Sie die Rechnung per E-Mail versenden, muss in der oberen Hälfte des Kundendatensatzes eine E-Mail-Adresse oder eine Faxnummer angegeben sein. Wenn keine Option ausgewählt ist, wird die Rechnung standardmäßig ausgedruckt.

 

Sprache – Ermöglicht es Ihnen, Rechnungen in verschiedenen Sprachen zu erhalten (Englisch, Französisch, Spanisch) sowie zwei nicht definierte Felder.

 

Währung – Ermöglicht es Ihnen, eine Standardwährung festzulegen, wenn der Kunde eine andere Währung als die Standardwährung der Station verwendet.

 

Abrechnungen – Hier können Sie festlegen, wie Ihre Kunden ihre Abrechnungen erhalten sollen. Die Abrechnungen können täglich oder monatlich verschickt werden.

 

 

Gehört von – Ermöglicht es Ihnen, aus einem Dropdown-Menü auszuwählen, wie ein Kunde von Ihrem Unternehmen erfahren hat. Wählen Sie im Programmmenü die Option Marketingberichte, dann können Sie einen „Gehört von“-Bericht erstellen.


Zugewiesen an – Ermöglicht es Ihnen, einem Kunden einen Bediener zuzuweisen. Dies ist hilfreich, wenn Sie einen bestimmten Mitarbeiter mit dem Kunden beauftragen möchten.

 

Beim Öffnen erzwingen – Durch Ankreuzen der unten stehenden Kästchen wird eine Anzeige auf dem Bildschirm erzwungen, um zusätzliche Informationen einzugeben.

 

Vertragsinformationen – Wenn Sie dieses Kästchen aktivieren, wird auf dem offenen Vertrag eine Anzeige eingeblendet, in der Sie Vertragsinformationen hinzufügen können.

 

Bestellnummer – Wenn Sie dieses Kästchen aktivieren, wird auf dem offenen Vertrag eine Anzeige eingeblendet, um eine Bestellnummer hinzuzufügen.

 

Auftragsidentifikation – Wenn Sie dieses Kästchen aktivieren, wird auf dem offenen Vertrag eine Anzeige eingeblendet, um eine Auftragsidentifikation hinzuzufügen. Beispiel. (Super Bowl, Expo).

 

Website – Wenn Sie dieses Kästchen aktivieren, werden Kundenname und Verträge auf der Website angezeigt.

 

E-Mail – Wenn Sie dieses Kästchen markieren, werden keine Marketing-E-Mails an den Kunden gesendet.

 

Kreditkarte – Wenn Sie dieses Kästchen aktivieren, werden keine Historie-Kreditkartendaten des Kunden gespeichert.
