Elite Browser Configurations




Overview
 

This article will discuss a couple of key configuration icons and features for the Elite browser. Configurations are the landing page where any parameter, configuration, or security setting option can be accessed. Settings apply at the Company, Store, User, or Feature level.


Prerequisites

This is for use with our Rental Elite software program.

 

Accessing Configurations



A ‘gear’ icon in the upper, right portion of the header in the browser will provide access to a user. You can limit access to Configurations by updating Configuration Area security named: Configuration.

 

Configuration Options
 

From this landing page, the options are set into Configuration areas on the left side. Each area will have options on the right side. You can control access to an entire area, remove options, or just limit/expose individual options through security.

 

Search
 

This feature will search all titles and descriptions on the right side. There may be multiple areas that have setting options related to what you search. All searches will be displayed.

 

Navigation
 

When accessing Configuration, it opens in a Dashboard tab. This allows the user to navigate to other areas of the application and easily come back to it. Some Configuration setting forms will also open a separate Dashboard tab.
