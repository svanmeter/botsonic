Customer Income Report




Overview
 

The Customer Accrual Income Report prints the customer’s key, number, name, phone number, rent amount, sales amount, damage waiver amount, and the tax amount for the period selected.


How to get there:
Expert: Program Menu > Customer Reports > Customer Income Reports > Customer Income
Elite: Right Menu > Customer Income 

 

When you select Customer Income the following screen will be displayed:
 

 

Report Date - This allows you to select a date range to run your report.
 

Group By - This option allows you to select how the customers are grouped together.
 

Store - This option allows you to select a single store or all stores when running the report.
