Syrinx® - Create Hire Category
 



 

Overview: This guide is designed to help you create a new Hire Category.
 

 

A category is used to group similar items within the hire fleet. Select the Fleet tab and under the setup section of the Ribbon select Categories and Add New.

 

 

 

You can optionally set up standard hire rates for your hire equipment.  These rates are then pulled in automatically when creating a new contract but may be modified on individual contracts if required.

When entering new hire stock items to the hire fleet, you can then select the hire rate that the new item is to be charged out at, from the list of rates that you set up here.

Press the Show Detail button with the category selected.  The Amend Hire Category screen is shown and the hire rates may be entered on the Hire Rates tab. Start typing on the next line to create multiple rates for the category selected.

You can then link these rates with individual assets by amending the fleet item.
