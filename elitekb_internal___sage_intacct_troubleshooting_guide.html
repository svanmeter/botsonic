INTERNAL | Sage Intacct Troubleshooting Guide



Point of Rental Software offers integration with Sage Intacct. This means the software will automatically upload journal entries into Sage Intacct software. Customers must be on Rental Elite software for this integration. This article will walk through troubleshooting this.

 

The integration allows for transactions to be sent (based on Accounting Method) immediately, at the end of day batch (i.e., Cost of Goods Sold), and for Purchase Orders, it pushes over as a Vendor Invoice. 

 


The integration consists of three moving parts:
 

1. Setting up the General Ledger numbers in POR.

2. The Intacct Configuration setup.

3. Going into Intacct software to set up.  

 


Troubleshooting options
 

  • On the server, there will be a service running called PORAccountingSyncService. This service needs to be always running as it updates every 1 minute. If there is a failure to push, the sync will try to push 11 times before stopping. 
 
  • The first step to setting up the integration is to go to Configuration > App Configuration > Accounting Configurations > Account Numbers. Make sure the Export Format is set to API - Intacct. 
 
  • There is a character limit of 20 characters for the Accounting link in the Intacct Dimension Data section
 
  • There is a character limit of 100 for the title in the Intacct Dimension Data section. 

If you are not able to resolve the issue presented with the above information, please document/screenshot all information and escalate your ticket.
