Rental Production Analysis

 

The Rental Production Analysis lists the Delivery Count, Will-Call Count, Item Revenue, and various other totals for each day of the week for the selected week.  The report will look like the following:

 

 

Prep TimeOut/PrepTimeIn: Item User Defined 1 = ‘PrepTimeOut” or Item User Defined 2 = ‘PrepTimeIn’. In the item record, you can enter a number representing the prep time (usually in minutes) necessary for both out and in. This is very useful in projecting labor requirements by the user of the Rent Production Analysis Report (in Reservation Reports
