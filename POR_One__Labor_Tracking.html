POR One | Labor Tracking



This article covers prerequisites and setup for Labor Tracking. With Labor Tracking, you can track labor time and log it on a contract. The time logged may be billed.


Topics in this article include
 

  • Prerequisites
  • Enable Billable Contract Labor
  • Creating a Labor Item
  • Adding a Labor Entry
  • Labor Updates


 

Prerequisites

 

To use Labor Tracking, please make sure that you have the following:
 

  • the latest version of the Elite Software installed on your server
  • the latest version of POR One Mobile application installed on your mobile device
  • You or an Operator must be enabled for Single Sign-On (SSO).
  • You must have Labor Tracking licensed at an Organizational AND User Level. Labor Tracking must be licensed within Mobile Permissions. This will enable the feature in the Task Menu in POR One.
    • This can also be done within Global User Admin (GAdmin
  • Have an item created in your File Maintenance classified as a ‘Sales-Labor’ Item
  • Have Billable Contract Labor Security enabled 


    Enable Billable Contract Labor

    Billable Contract Labor is a security setting that will allow you to add labor items to Internal Repairs Orders, Internal Maintenance Orders, and Service Calls (IRO, IMO, ISO) through POR One. This will not apply to Work Order contracts.


    1. To enable this security setting, go to Configuration Gear > System > Security. In the search bar search Labor.

    2. Set to yes for either anyone in the Company, Group, or employee.

     


     

    3. You may need to log out and log back in.

    4. Go back to the Configuration Gear > Transactions > Billable Contract Labor will be visible.

    5. Next, check the box to set it and then press the submit button. 

     



    Creating a Labor Item

    Before you can begin tracking labor, you must create a labor item. Once created, you will need to add the item to the contract. This section will go over creating a labor item. See the next section Adding a Labor Entry to learn how to track labor.
     

    1. Go to the Configuration Gear > File Maintenance > Item File > Add Item. 

    2. Enter item key and name. 

    3. You can enter your labor rate in the sell price field.

    4. Set the item's quantity to 9999.

    5. Under classifications, set the item type to Sales - Labor item.

    6. Add inputs to additional fields as necessary. Once finished, press OK.
     



     

    Adding a Labor Entry


    Labor entries can be added to Internal Work Orders, Internal Repair Orders (IRO), Internal Maintenance Orders (IMO), and Internal Service Orders (ISO).  To use this feature you must be given permission using Mobile Permissions or GAdmin. 
     

    1. There are a few ways to create Labor Tracking entries. They are listed below.

    • From the Task menu, select the ‘+’ option then select Labor Tracking.
    • From Inventory search, search and select the item then expand the Work Order option and select the Work Order. The Labor Tracking option will be displayed.
    • From Work Order search, search and select the Work Order and the option will be displayed.


    2.  Once here, select Create.
     



    3. Type in the Work Order or Repair Contract information. You must have a match in order to proceed!



    5. Clock in to start the time tracking.



    6. Once you're ready to Clock Out, add notes, select a Labor item, and then adjust Actual Hours as needed. Actual Hours defaults to the calculated time from the Clock Out/In. This feature allows you to enter a different time value in case you or another user forgot to clock in, maybe they were pulled away from the job or they just need to communicate that the Clock In/Out period is not accurate. Once a Clock In, Clock Out, and Labor item are added the Complete option will be available.



    7. When you're ready select the Back option to clear the task from the list and send the data. After this option is selected, any adjustments to times must be made in Elite. The labor item is automatically added to the contract. This will only add labor as a line item to Internal Repairs Orders, Internal Maintenance Orders, and Service Calls (IRO, IMO, ISO). Labor will have to be manually added to Work Order contracts(WO). 



    Labor Updates

    You can view your Labor Notes and Status in the Labor Updates section in the Elite Browser. To get there, go to Left Menu > Tools > Labor Updates.
