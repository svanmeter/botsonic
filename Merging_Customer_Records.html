Merging Customer Records

Overview: There can be times when a customer has been entered into the system twice. You should never delete a record because you would lose an integral part of the transaction history. By merging two customer records you can combine the transaction history into one and not lose any data.

1. Security must be set to allow this. Be sure you are out of file maintenance prior to making this change. Once this is set you won’t have to do it again.

a. Select #10 Security from the Program Menu.

b. Chose File and Field Security.

c. When it asks for a password enter the required level (usually owners or regional manager). If you do not have the required level please see your supervisor or manager for access.

d. From the File Menu choose the Customer File.

e. The File Maintenance Security shows all the password levels and what each level can do.

f. Merge can be added, from the drop-down menu, to Store Manager or higher.

Merging Customers.

a. Select #6 File Maintenance from the Program Menu.

b. Choose Customer File.

c. Select the customer record that you want to merge into another record. This record will disappear.

d. From the customer record click on the “functions” drop down menu at the top left-hand corner and select “Merge into different customer”. If you do not see this option you are not in file maintenance with the appropriate password. Close out of file maintenance and try again with a higher password.

e. When selected you will see a warning: “This customer will be deleted at the completion of the merge; do you want to continue?” Answer ‘Yes’.

f. You will now need to select the customer that you want to merge this history into. The customer's record you select will be saved.

g. When you click OK you'll see: “Do you really want to delete (the name on the first record) and move their history to (the name of the second record)?” Yes or No

h. This is the last chance you'll get to change your mind.

i. Click “Yes” and the merge will be complete.

j. Select F5 on your keyboard to refresh your screen old customer will no longer show.

Note: There is no way to undo a customer merge. Please set your data file security appropriately to avoid any unnecessary grief.
