In Essentials, you have different statuses that you can set up for your business. This article goes over the different options for configurable dropdowns to allow more customization for your Essentials Software. To begin go to Configuration > Configurable Dropdowns. 


Topics in this article include

  • Maintenance Status
  • Unavailability Types
  • Missed Rental Reasons
  • Transaction Substates Configuration

 
 

Maintenance Status
 

This area will allow you to create different statuses in your maintenance area to figure out the current status of the maintenance.

 

 

 
 

If you go to Inventory > Maintenance, it will translate the existing statuses that are there.

 

 

 
 

Unavailability Types
 

This area will allow you to customize your unavailability types for your business, for example, end of the season.


 

Go to Inventory > Inventory > Unavailable Periods and click on New Unavailable Period. From the dropdown next to type, you can choose the status.

 

 

 
 

Missed Rental Reasons
 

This area will allow you to customize the different reasons for your missed rental. If you still want to use the existing ones, you can click the Add Default Reasons and it will add them to the list.
