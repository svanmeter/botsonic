Inspection app Setup




This article will go over security and setup for the Inspection App. 


Topics in this article include
           

  • Point of Rental - Security
  • Inspection App Configuration
  • Accessing Inspection App Outside of Secure Network

 

Point of Rental - Security

Anyone who needs to access the Inspection App, either to manage the app or to do the inspection, will require a Point of Rental Operator ID and Password and the appropriate security settings enabled. You can set up the Security Settings for the Employee Group Level, or Employee Level. To get there, go to the following locations below. 

 

Expert 

 Employee Group Level

Program Menu > Security > System Security > Security Defaults

This is where you can set up the default security settings for the company or the employee group.


 

Employee Level

Program Menu > Security > Employee Setup

This is where you can assign operators with security permissions so you can assign them to a security group.
 


 

Elite

Inspection App Security should be set by the Employee Group or by the Employee.

Security
 

Top Function Bar > Configuration Gear > System > Security

You can assign your operators to groups in addition to their assignment to the company.

Inspection app Configuration

The Inspection app can be used on a desktop or mobile device. 


Desktop Configuration

Here's how to get there
Expert: Program Menu > Item Availability > Inspection App
Elite: Left Menu > Tools > Inspection App


Follow the steps below to configure the app on your desktop:

  1. From the main login screen of the app, press the  icon, normally found in the bottom right-hand corner of the app. You can also press Settings within the main menu.
 
  1. Enter the Localhost IP address and port number.
 
  1. Once the connection is successful, you should see a green checkmark next to the server address after you click Save.



Mobile Device Configuration

The Inspection App can also be accessed from a mobile device via the Apple App Store or Google Play Store. 
 

Apple App Store: https://apps.apple.com/us/app/por-inspection/id1367236907

Google Play Store: https://play.google.com/store/apps/details?id=com.pointofrental.inspectionapp




Once the Mobile App is installed, the app will have to be configured to work with your system. Follow the below steps:
 

  1. From the main login screen of the app, press the  icon, normally found in the bottom right-hand corner of the app. You can also press Settings within the main menu.
 
  1. You will need to enter the name or IP Address of the Point of Rental Server as well as the Port Number. This information is provided by Point of Rental Support during the setup process.
 
  1. Once the connection is successful, you should see a green checkmark next to the server address after you click Save.

 

  1. After this is completed, you will be able to log in to your Inspection app with your local Operator ID and Password.

 


Accessing the Inspection app from outside of the Point of Rental Network

Warning! Opening ports for inbound network access can put you in a non-PCI-compliant state. It is recommended to have a VPN client access to internal resources from outside the network. Contact your Network Administrator for advice.
 

1. In order to access the Inspection app outside of the Point of Rental network, you'll need your Server's Local IP Address (for LAN and VPN access) or Public IP Address (for Port Forwarding or Hosted Solutions). You can find your Public IP address by opening your Point of Rental Server and going to https://www.whatismyip.com.

2. The Port Number was given to you during the setup of the Inspection app. If you do not recall it please contact Point of Rental Product Support for that information.

3. When you have completed steps 1 and 2, from the main login screen of the app, press the  icon, normally found in the bottom right-hand corner of the app. You will need to enter the name or IP Address of the Point of Rental Server as well as the Port Number. Once this is complete, you should see a green checkmark next to the server address after you click Save.

 

 

  1. After this is completed, you will be able to log in to your Inspection app with your local Operator ID and Password.
