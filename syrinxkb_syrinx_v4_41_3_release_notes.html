Option to Limit Entry of Timesheet Days up to Planned Offhire Date - A new system option, "Show only T/S days up to planned offhire date?",has been added to the Timesheets tab of Syrinx SP Configuration\System Options\Operated Hires.  When enabled, the system will limit the entry of timesheet days up to a hire contract's planned end date.  Dates after the contract's planned end date will not be listed on the timesheet and the user will not be able to enter hours for these dates.
 Display Mobile Job Status for Misc Jobs in Workshop Scheduler - The mobile job status will now be displayed for "Misc" jobs on the job entry and on the job properties in the workshop scheduler:
 Customer Contacts Sorted Alphabetically When Adding New Calls in the SalesApp - Customer contact names will now be sorted alphabetically in the contact dropdown when adding a new customer call in the SalesApp.
 List items on Syrinx Driver App load and unload jobs by their contract sequence -Items on the Job Summary and Item List pages for load and unload jobs on the Driver App will now be grouped as per their type (e.g. Hire or Sales) and in hire/sales sequence order to more closely match how those items are displayed on the printed hire or sales documentation.
Support for Sage 200 Professional 2023 R1 - The Sage 200C transfer tool now supports Sage 200 Professional 2023 R1.
Support for SQL Server 2022 - SQL Server 2022 is now officially supported for Syrinx databases.
Support for Windows Server 2022 - Windows Server 2022 is now officially supported.
Task Scheduler Archive Task to Handle Operator App Signature Image PNG Files - The archive process in the Task Scheduler has been updated to handle archiving .png signature images that are uploaded to the DB from the POR One Operator app.
Task Scheduler Archive Task to Scale Date/Time Lat/Long to the Resolution of the Image - The task scheduler mobile image task will now scale the Date/Time Lat/Long stamps (if enabled) that it applies to the archived images as per the size of the image.
Update Suppliers API POST and GET endpoints - A "currencyCode" field has been added to the output and as a parameter to the Suppliers GET endpoint.  Extra fields such a "vatNumber", "cisRegNumber" and user properties 1-10 have also been made available to set/update via the Suppliers POST endpoint.
New Invoices/Sales GET Endpoint added to SyrinxWebApi - A new Invoices/Sales GET endpoint has been added to SyrinxWebApi which allows for the retrieval of invoices from the Syrinx database.  This endpoint differs from the existing Invoices GET endpoint as it allows for the retrieval of any invoice in the system whereas the previous Invoices endpoint would only output sales invoices that required posting.  
Updated Labels on Get Invoices and PurchaseInvoices Endpoints - The descriptive labels for the GET Invoices and PurchaseInvoices endpoints on the Swagger page for SyrinxWebApi have been updated to reflect that they only output unposted invoices.
