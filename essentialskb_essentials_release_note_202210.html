7/25/2022 - Version 2022.10

 
  • Country Code Selector and Phone Number Validation - Essentials now features a County Code selector and validation on phone number fields. By default, the country will be the same as where the rental store is located but can easily be changed using a drop down menu. The phone numbers entered will also be validated for accuracy of length and area code. Some of the fields where you’ll find this enhancement are:
    • Customer Contact 
    • Employee records
    • Inventory Log 
    • Items
    • Purchase Order 
    • Stock 
    • Stock Instance 
    • Stock Inventory Log 
    • Transaction Payment
    • Vendors
  • Additional Rows for BOL Handling Unit - The handling unit section on the Bill of Lading (BOL) document was limited to 5 rows. You will now be able to add (or remove) rows if there are more than 5 rows of Handling Units needed. Each field on the added row can be intentionally left blank or filled by text input from the user just like the default rows. Feel free to add as many rows as needed, the BOL document will continue to expand onto additional pages. 
  • Amount to Collect “Pay in Full” more prominent - This one is a small tweak but the option to Collect a Payment in full by clicking the “Amount to Collect” line is more prominent to help counter employees know that the field is clickable to Pay in Full. 
Storefront Required Delivery Address - A new configuration has been added to  Customer Storefront Delivery Options that allows for the address to always be required to be entered by a Storefront customer rather than default to a pre-populated address.
