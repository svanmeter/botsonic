Hours Override Report

This selection produces a report of all hour overrides sorted by employee.  This is not an uncommon function; however, if one employee has a significant number of them especially to a customer it could signal a problem.  Note Your setup whether hour overrides are tracked under Security in Parameters.  If you have your parameters set to Never for tracking Hour Overrides, then no report will be generated since no tracking is being done.  If you have Hour Override tracking set to Close, then only hour changes done when closing the contract will be reported.
