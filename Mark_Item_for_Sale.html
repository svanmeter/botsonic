Mark Item for Sale

 

This option will mark this item for sale. The item will be displayed in the Items for Sale Report. The price listed on the report will be the selling price in the item record. Once the item has been marked for sale the following message will appear.

Here's how to get there:
Expert: Program Menu > File Maintenance > Item File > Tools
Elite: Gear Icon > File Maintenance > Item File > Tools




Enter the value of the item you are marking for sale.
 


 

Once the value has been entered, you will be prompted to set the manufacture, model number, and model year.
