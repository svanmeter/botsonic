Personal Data Removal


Overview

The Personal Data Removal tool allows you to purge personal data that the system holds about individuals on customers who are no longer flagged as current.  When run any information is replaced with the text ‘Purged Data’ All links to historical documents Contracts, Advice notes are maintained however the personal information is unrecognisable. This tool is to assist you satisfying any applicable data protection regulations.

Purge or Remove Customer Contract Information 

Log in to the Syrinx SP Configuration program, and select the System Administration/Bulk Update menu option.

Then select the Purge Personal data tab from the list of available bulk update options:

The process can be run for “All contacts” and removes all names, addresses, site and contact info for non current customers marked as “individual” and purges only contact information held against non current customers that are not marked as “individual”.  The process can also be run for a selected contact on a specific customer record.

Pressing OK will run the Purge.   Note this cannot be reversed.
