Create Service Contracts




Overview
 

This feature allows you to create service contracts making them accessible through Day at a Glance under the service tab.
 

Note: You will need to have the Service Module.



Here's how to get there
Expert: Program Menu > Counter System > Tools > Create Service Contracts
Elite: Transactions > Create Service Contracts

 


 

1. Select the Store Location.
 

2. Select the frequency for the service (Daily, Weekly, Monthly).
 

3. Select the number of days you want the contracts created for.

4. Press Continue. This will look for services created through the contracts, within the assigned time and create service contracts. You will see them under the Day at a Glance service tab. If you do not use this feature, Point of Rental will automatically create the service contracts two days from the current date when End of Day is run.
