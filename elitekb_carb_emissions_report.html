CARB Emissions Report

 

 


 

Overview
 

CARB stands for the California Air Resource Board. This report is compiled from information entered into the item record under the Engine tab. Please keep in mind that the information about engines is only one component of the reporting requirements.

 

Topics included in this article:

 

  • Engine tab under the item record
  • Running the CARB emissions report

 

 

Engine tab under the item record

 

To access the Engine tab, select the Asset tab from within the item record. Then select the Engine tab in the lower half of the window.

 

 

 

Running the CARB emissions report

 

The CARB Emissions Report can be accessed by going to Program Menu > Inventory Reports > CARB Emissions Report.
