Customer Dashboard

 

 

 

Overview

 

This article discusses the Customer Dashboard and its various capabilities. 

 

 

Topics include:

  • Customer Selection
  • Customer Lookup Columns
  • Field Search
  • Customer Dashboard
  • Tasks
  • Tools
  • Reports

 

 

Here's how to get there

Expert: Program Menu>Counter System>Customer Dashboard

Elite: select the Tools icon and then Customer Dashboard.

 

 

Customer Selection

 

After selecting Customer Dashboard the Customer Selection screen appears, where you can enter the customer’s name.

 

 

 

Customer Lookup Columns

 

 

 

These can be customized to each user. Right-click on the green header bar. A list of columns will be displayed. Check the ones you would like displayed when viewing customer look-up. Click the save button, and when you log in again, your settings will be saved.

 

Note: You should check regularly for updates to the Customer Dashboard Columns. 

Recently added Delivery City and Delivery Zip. 

 

Move Up - This allows you to arrange the columns up.

 

Move Down - This allows you to arrange the columns down.

 

Hide - This allows you to hide the columns from the customer dashboard.

 

Reset - This allows you to reset the boxes to the default settings.

 

Specific User - This allows you to save the setting for a Specific User based on Employee ID.

 

Store Default - This allows you to save the settings for a Store Location.

 

Company Default - Allow you to save the settings based on Company.

 

Groups - New groups are routinely added. 

 

 

 

 

Field Search

 

This allows you to search customers by the following criteria:

 

 

Containing - This field allows you to search customers within selected criteria. For example, if you enter phone in the search and type 972 in the containing field, it will search for any part of the number or word entered.

 

Customer Name - This allows you to enter a customer's name.

 

Customer Key - This allows you to look for customers by customer key.

 

Note: this would be a unique number you set up when creating the customer.

 

DL No - This allows you to look up a customer by driver’s license number.

 

Customer Number - This allows you to look up customers by customer record. This would be the number to the far right in the customer record. Point of Rental automatically assigns customer numbers when the customer is created.

 

Only Account Customers - With this box checked, Point of Rental will only search for account customers. Unchecking it will only search for cash customers.

 

Show Closed Accounts - Checking this box will show you closed customer accounts, unchecking it will only show open accounts.

 

All Types - This allows you to select the type of customer (Homeowner, Caterer, etc.)

 

All Customers - This allows you to select customers by aging (30,60,90,120 days).

 

Add Customers - This allows you to add a new customer to Point of Rental.

 

From License - Allows you to add a new customer based on their driver’s license. Point of Rental will populate the information from the driver’s license and auto add to the correct fields in Point of Rental.

 

 

 

Customer Dashboard

 

Displays a quick summary of all the active contracts that the customer has, along with any balance due on each contract. At the bottom of the screen, an aging summary is displayed for the customer.

 

 

 

Tasks

 

Create Reminder - From the reminder page, you will be able to create a reminder using Point of Renal Internal Task List, Microsoft Outlook, Microsoft Calendar, or Google Calendar. The reminder can be created for yourself, for an employee, employee group, or anyone with an email account. You can create reminders from within a customer’s record, the Customer Balance screen, or from Day at a Glance.

 

Tasks Assigned - From this option you can view any tasks that have been assigned to contracts dealing with this customer.

 

New Contract - Clicking this button allows you to create a new contract for this customer.

 

Print - Clicking this button will print the current list of contracts as viewed on the screen.

 

Notes: A contract can be modified by clicking on the contract number.

A customer record can be displayed by clicking on the customer key or name. If you are working on overdue accounts, you can click on the customer’s key to bring up the customer record.

A More screen listing customers can be displayed by clicking on the headings “key” or “name”. The More screen will be sorted in the order of the heading clicked.

The balance displayed may not match that on the customer’s statement. For example, if you bill only Closed contracts on the statement, the customer’s statement only shows Closed contracts even if the customer has some Open or Reservation contracts. On the Customers Balance screen, you may check or uncheck any of the Contract Status boxes in the lower left-hand corner of the screen to display certain status contracts you are interested in, including All (active) Contracts for this customer.

 

 

 

Tools

 

Make Account Payment - This allows you to make an account payment from this screen and apply payments to a customer's account. If you are using Elite, this security is tied to the ‘Account Payments’ option under the Left Menu security section.

 

View Customer Record - This allows you to view the customer record.

 

New Contract - This allows you to create a contract for the customer.

 

 

Reports

 

Open Items - This report will display all the rental items this customer has out on rent.

 

Open Items by Job - This report will display all the rental items this customer has out on rent. The items will be grouped together by Job Site/Venue.

 

Aged Balance - This report will list all the contracts for this customer with the age date of each contract.

 

Payment Report - This report lists all the payments received from this customer.

 

Call Logs - This report lists all the call logs entered for this customer.

 

Account Summary - Brings up a report to show all information in the customer record including notes and any contracts/authorized users.

 

Print All Contracts - This will print all contracts displayed in Customer Balance for the selected customer.

 

Export to Spreadsheet - This allows you to export information to an excel spreadsheet.
