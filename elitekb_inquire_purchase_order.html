Inquire Purchase Order

 

Allows you to view on the screen and/or reprint an active purchase order. No changes will be able to be made from this option.

 

Program Menu>Purchase Order

 

1. Click on the purchase order you would like to view, an option box will appear, select Inquire.

2. The purchase order will display on the screen. Click the OK button when done looking at the purchase order or click on PRINT if you want to print a copy of the purchase order out.
