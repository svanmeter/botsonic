Overview: This guide is designed to help you with querying hire contracts.

 



 

To query a Hire Contract, go to the Hire tab and the Query section of the ribbon. You can search for the Hire Contract using any of the blue fields in the query window.

 

 

 

You can further search by any of the blue fields in the tabs across the top.

 

By selecting Query, you can only view the details within the Hire Contract. You will not be able to amend them unless you click on the Amend button at the bottom of the window (not applicable to Unconfirmed, Closed or Cancelled Hire Contracts).

 

 

A useful Hire Contract query search tool is Log Entries. This will allow a search of a particular action within a Hire Contract function. It is particularly useful if you recall something was done some time back but do not have the details at hand. You can search by any of the blue fields and then select one of the items in the list on the left-hand side. The item is selected when it is highlighted in black. For example, should you want to see all Confirmed Exchanges for a period, select the date range and Exchange Confirmed and click Query.
