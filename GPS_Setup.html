GPS Setup [GPS Setup]

GPS Setup Item Record

 

How to get there: Program Menu>File Maintenance>Item Record>Asset Tab

 

1. From Functions, select Modify

2. Enter the GPS Equipment ID Number, in the GPS No field. 

3. You will need to exit and go back into the item record to save.

 

* Note -  Once you go back in, you will see at the top of the screen the display "GPS Reading" on the serialized unit.

4. Click on the GPS Reading, and a GPS Telemetric box will appear. Once linked, the GPS Telemetric box will populate the latitude and longitude along with the last time unit was updated. If the item has previously been updated, then you will receive a message showing the last time the meter hours were read.

 

 * Note - This option will only show up on serialized assets that are set up with a GPS and linked to an AEMP account. When selected you will see the Equipment information, location, and last updated meter reading.
