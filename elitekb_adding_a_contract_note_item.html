Hinzufügen einer Vertragsnotiz

 

 

 

Übersicht

 

Dieser Artikel führt Sie durch den Prozess des Hinzufügens einer Vertragsnotiz. Das Hinzufügen einer Vertragsnotiz ermöglicht die Eingabe von Notizen zum Vertrag in verschiedenen Schriftarten. Während Point of Rental schon immer über die Funktion Objektkommentare zum Drucken von Notizen auf Verträgen verfügte, haben wir kürzlich eine neue Methode zum Hinzufügen einer Notiz zu einem Vertrag eingebaut, die es ermöglicht, die Notiz in einer größeren Schrift, in Fettschrift und auf Wunsch auch in Farbe zu drucken. Mit der Notizfunktion können Sie das Aussehen Ihres Vertrags und die Kommunikation mit Ihrem Kunden verbessern.

 

 

In diesem Artikel enthaltene Themen

 

  • Erstellen eines Notiz-Elements
  • Hinzufügen einer Notiz zu Verträgen

 

 

So gelangen Sie dorthin:


Expert: Programmmenü > Dateiverwaltung > Artikeldatei

Elite: Zahnradsymbol > Dateiverwaltung > Artikeldatei


 

Erstellen eines Notiz-Elements



1. Sie müssen einen Artikel erstellen. Wählen Sie „Artikel hinzufügen“.

 

2. Die Taste muss mit NOTE oder NOTER beginnen.

 

3. Geben Sie in das Feld „Anzahl“ 99999 ein.

 

4. Setzen Sie den Artikeltyp auf Miete – Verschiedenes oder Verkauf – Verschiedenes. Der Typ bestimmt, wo die Notiz auf dem Vertrag erscheint. Eine Mietnotiz erscheint bei den Mietartikeln und die Verkaufsnotiz wird bei den Verkaufsartikeln gedruckt.

 

5. Wählen Sie eine Kategorie.

 

6. Lassen Sie die Mietpreise oder den Verkaufspreis leer oder auf Null gesetzt.

 

7. Geben Sie im Feld „Gruppe“ Ihre bevorzugte Farbe ein. Die Notiz wird in dieser Farbe auf dem Vertrag erscheinen, wenn sie gedruckt oder per E-Mail verschickt wird (vorausgesetzt, Sie haben einen Farbdrucker).

 

8. Aktivieren Sie auf der Registerkarte „Optionen“ das Kontrollkästchen für „Verfügb. Nicht prüfen“.


Hinweis: Wenn Sie vor kurzem Point of Rental erworben oder aktualisiert haben, sehen Sie, dass bereits eine Vermietungs- und eine Verkaufsnotiz im System eingerichtet sind.

 

 

Hinzufügen einer Notiz zu Verträgen

 

So gelangen Sie dorthin:

Expert: Programmmenü > Counter System

Elite: Transaktionen > Neuer Vertrag/Transaktion > Vertrag ändern



1. Erstellen Sie entweder einen neuen Vertrag oder wählen Sie Ändern, um einen bestehenden Vertrag zu ergänzen.

 

2. Wählen Sie die Schaltfläche „Element hinzufügen“.

 

3. Wählen Sie Ihr Notiz-Element.

 

4. Geben Sie eine Menge von 1 ein, und setzen Sie den Preis auf Null. Geben Sie dann die Notiz so ein, wie sie erscheinen soll, wenn das System Sie auffordert, den Artikelnamen einzugeben. Ihr Hinweis kann bis zu 30 Zeichen lang sein.

 

5. Wenn Sie alle erforderlichen Ergänzungen oder Anpassungen vorgenommen haben, wählen Sie OK.

 

6. Sie können auswählen, ob Sie den Vertrag drucken, per E-Mail oder Fax versenden oder anzeigen möchten.

 

7. Nachdem Sie den Vertrag angesehen, gedruckt oder per E-Mail verschickt haben, kehren Sie automatisch zum Hauptmenü des Counter Systems zurück.

 

8. Wenn Sie sich entscheiden, keine der oben genannten Maßnahmen zu ergreifen, werden Ihnen die folgenden Optionen auf dem Bildschirm unten angezeigt:
