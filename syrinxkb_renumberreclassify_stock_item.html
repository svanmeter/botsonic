If you wish to change an item's stock number or move an item in to a different hire category, you will need to use the "Renumber" option on the "Fleet" tab of Syrinx SP as these details cannot be amended directly on the stock item record. You simply need to enter in the stock number and then press tab or return and then either enter in a new stock number for the item in the "New Stock No." field or select a new hire category from the "New Category:" dropdown and then press "OK" to complete the process. If you have a large number of items to renumber/reclassify this can be accomplished via the "Stock Renumber/Reclassify" option in Syrinx SP Configuration\Administration\Import Data.
