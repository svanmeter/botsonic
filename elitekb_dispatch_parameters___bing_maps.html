Dispatch Parameters | Bing Maps

 

 

 

This article will cover Bing Maps in Dispatch Center. We have completed the overhaul of the Dispatch Center to utilize an internet-based mapping source, Bing Maps. Key benefits are up-to-date maps that provide more accurate routing and a noticeable increase in route processing speed.


 

Topics included in this article

  • Prerequisites
  • Getting there
  • Licensing Bing maps




 

Prerequisites

Map Point end-of-life: As of January 1, 2015, Microsoft stopped updating our original mapping source Map Point. However, we will support anyone who has Map Point installed on our 2017 Dispatch Center. Be aware that Microsoft no longer supports Map Point and you can no longer purchase or download the Map Point installation software. If you move servers or need to re-install Map Point, your only source will be the CD/DVD provided to you by Microsoft. For your convenience, safeguard your Map Point installer. You also have the option to upgrade to Bing Maps.

You can preview the Bing Maps feature for 10 days by going to Options > Parameters > Map Options and pressing the “Try Bing Maps” button. From here you can also request a Bing license from OnTerra. Doing so will result in a purchase agreement being emailed to you. Upon purchase, you can return to this screen to install the license keys. 

In order to take advantage of Bing maps, each customer must purchase a valid license key from OnTerra Systems. OnTerra is an authorized Bing reseller and creator of route enhancement software, RouteSavvy, which provides route optimization.

These are annual subscription services that must be maintained to ensure access to the mapping sources. The price point is similar to Map Point. We considered using Google's services, but the cost to obtain a license by our customers would have been considerably larger. This can be confusing as Google and Bing maps are free and accessible to everyone through a web browser. However, any commercial usage requires commercial license and cost.



Getting there

Here's how to get there
Expert: Program Menu > Counter System > Dispatch Center > Options > Parameters
Elite: Left Menu > Tools > Dispatch Center > Options > Parameters
 

 

 

Licensing Bing maps
 

1. Open the Dispatch Center.
 

2. Choose the Options menu from the top of the screen.
 

3. Choose Parameters.
 

4. Select the Map Options tab.
 

5. Double-click the Bing Key license in the bottom left of the screen.

 


 

6. Copy and paste your new Bing Key, that you have gotten from On-Terra, into the dialog box.

7. Click OK to save.

 


8. Exit the Dispatch Center and restart.
