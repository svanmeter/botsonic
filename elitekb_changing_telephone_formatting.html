Changing Telephone Formatting



 

Overview

You can define the telephone format you want to show on your contracts. This document will walk you through how to do this. Whether you want your phone format to contain spaces, periods, dashes, you now can define this easily.


Here's how to get there
Expert: Program Menu > Configuration > System Configurations > Language Translations
Elite: Configuration Gear > App Configurations > System Configurations > Language Translations

 

1. Scroll until you find the row for the Phone format.

2. Under the Undefined field, enter the format you would like to use, for example, XXX.XXX.XXXX or XXX-XXX-XXXX.
 

 

3. Next navigate to Configuration > System Configurations > Parameters > Printing tab. Change the printing language by selecting "Undefined" from the dropdown labeled Default Printing Language. Select OK. 
 

4.  When you view a contract, you will see the new format listed.
