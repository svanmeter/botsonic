Suggested Items Reordering. You can reorder items in suggested items now. Simply edit the sort order number, change your number, save and it will reorder based on the new number. 
 

 

Reorder Points. Unserialized items have 2 new fields listed below. Reorder point is if the item gets at or below that point you want to know when to reorder. Reorder amount is the number you want to order when you reorder.
 


 

Stock Take Report for Reorder Points. This report has been enhanced to include only showing reorder only items.
 

 

When you run the report, you have 3 new columns you can add. Reorder Point, Reorder Amount, Order Amount. Note: Item only shows up if at or below reorder point. The order amount is the reorder amount minus the available quantity.
 

 

Last Minute Reservation. In configuration/date time options, you have a new setting Maximum period to be considered a last minute reservation. For example, if you have it at 24h, a notification will be sent if a reservation is made for less than 24 hours from now. In order to turn it on you will need to go to notifications, and mark it Yes

 

 

Receive contract options. In config/printing options, you now have the ability to determine what selections are available when closing a contract
