Edit Zip Code

 

The Point-of-Rental™ System comes with all the United States ZIP Codes already loaded into the database.  However, new ZIP Codes are added and the names of cities are changed.  Anytime a ZIP Code that does not exist is entered on a customer’s record, you will be given the opportunity to enter the city and state for that ZIP Code.  That information is then added to the ZIP Code database and will not need to be entered again.  If the counter person entered the wrong City, State or the name of the city has changed, then you will need to come here to modify the ZIP Code record. 

 

Type in the zip code that needs to be changed, click the SEARCH button. The City, State that corresponds with the ZIP Code will be displayed. Make the appropriate changes then click the OK button.

Select the add button to add a new zip code. You can assign a tax jurisdiction to one zip code or a range of zip codes. Select from the drop-down menu, which tax jurisdiction is to be assigned to the zip code.

Jurisdiction - This would be the same number issued by the Tax Authority that you would add to the Tax File under Jurisdiction. This would usually be imported, not manually entered.
