This feature allows you to break out the Purchase Order into separate sequences. The status of the Purchase Order is irrelevant.

From the Purchase Order Dashboard, find the Purchase Order needing to be broken out. By right-clicking, this box will appear:

Once you select the option, this message will appear (Selecting No will return you to the Purchase Order Dashboard):

After selecting Yes, you can select which items will be broken out into the next sequence:

Once you have identified which items need to be broken out, select the Breakout button at the bottom right-hand corner of the screen (In Red):

This message will appear letting you know the Purchase Order has been successfully broken out:

Select Cancel. This will bring you back to the Purchase Order Dashboard where you can see Sequence 1 and 2 of this example Purchase Order:
