1. Hire

There is now a “Suspense Days By Customer” option in Syrinx SP Configuration\Administration\Import Data which can be used to import customer suspense days in bulk.

Document Text templates linked to confirmed hire contracts can now be amended in the header of the Amend Hire Contract screen.  Previously, it was only possible to amend the Document Text template linked to a hire contract while it was unconfirmed.

User can now set an “Hourly” rate against cross hire supplier agreed rates:

A "Documents" button has been added to the "Off Hire Contract" and "Partially Off Hire Contract" screens that when pressed will open the underlying hire contract's documents folder so that users can easily view documents linked to the contract and also drag and drop any documents linked to the offhire into the contract's documents folder.
 

2. Invoicing

Summary: When emailing copies of invoices to a customer via the “Query Invoice” screen or via the "Email Invoices" screen, any scanned timesheets linked to the invoice will now be attached to the email along with the PDF copy of the invoice.  When emailing from the “Query Invoice” screen this behaviour is controlled by the “Include additional attachments” check box on the send email dialog which will be enabled by default if the system is configured to print time sheets with invoices.  As per emailing scanned timesheets with invoices from an invoice batch, the customer must also have the “One File Per Invoice?” option enabled on the “Invoice Delivery” tab of their customer record.

 

3. Purchasing

The “Documents” button on the “Query Purchase Invoice” and “Amend Purchase Invoice” screens can now be used to open a folder specific to that purchase invoice so that users can drag and drop documents related to the purchase invoice into that folder.  Previously, the “Documents” button on these screens was only used to display scanned purchase invoices.  In order to use the new feature a root folder for purchase invoices must be defined in Syrinx SP Configuration\Documents\Root Folders.

 

4. Accounts

A new view “qCardTransactions” has been added that can be used to report on card transactions initiated by the SagePay module.

A new “Add Headers” option has been added to the “Opera II” tab of the “Configuration Settings” screen for the Transfer to Opera tool.  Enabling this option forces the transfer tool to produce the posting files that it uses for the Opera import in a different format.   This format contains column headers for each column of data and can be useful when importing into an Opera III SQL database which has had modifications to its table structure.

Quickbooks 2019 Desktop version is supported.  Users will still need to select the “Quickbooks 2008” option when installing the transfer tool for Quickbooks.

The Transfer to Sage Line 50 tool will now recognize and transfer any department codes that have been manually entered against purchase invoice lines via the “Enter Purchase Invoice” screen.

The Transfer to Exchequer tool should now always prompt the user to provide an exchequer password in order to support the logon methods in the latest version of Exchequer.

6. Schedulers

A “Mobile Job” button has been added to the properties pane of jobs in the Driver, Workshop and Operator Schedulers which can be used to open the mobile job linked to that item in Syrinx.

The date/time started values set against Service Reminder jobs will now be set to the scheduled start date/time that is used when scheduling the job via the Workshop Scheduler.  Previously, the system was using the current date/time when the user was physically scheduling the job in the Workshop Scheduler.

A quantity column has been added to the lower “unallocated” section of the driver scheduler which will display the quantity value linked to the first hire item on the contract.  This option is enabled via the “Show Unallocated Qty?” option on the “Driver Scheduler” tab of Syrinx SP Configuration\System Options\General.

 

7. Android Apps

The travel and work times entered against jobs on the Workshop App can now be entered in 5 minute intervals.  Previously, these entries were limited to 15 minute intervals.

 

8. Syrinx CWA/CWS/CWO

The “Equipment On Hire” screen in SyrinxCWS can now be set to exclude items that are on unconfirmed offhires\awaiting collection.  This behavior can be turned on by disabling the “Show Items on Collection in Equipment On Hire?” option on the “Syrinx WS” tab of Syrinx SP Configuration\System Options\Syrinx WS.

When clicking on the “Insp. Cert” or “Service Cert” buttons on the “Equipment On Hire” or “Previous Hires” pages in SyrinxCWS, the user will be presented with a list of available documents and can elect to print one or all of those documents.

Mandatory fields are now highlighted on the Registration page of the SyrinxCWO online ordering site.

“Site” labels against address fields in the online ordering site, SyrinxCWO, have been changed to “Delivery” where applicable.
