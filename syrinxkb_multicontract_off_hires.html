Multi-Contract Off Hires

 

Overview

When off hiring equipment, each off hire note is typically tied to one particular contract.  You can also configure Syrinx to allow a single off hire note to incorporate all the equipment at a particular site, regardless of how many open contracts there are for that site.  This option may be useful if you hire scaffolding-type equipment where the same type of equipment is delivered many times to the same site over several days or weeks, and you want to off hire the equipment quickly even though the equipment may be spread over many contracts.  It may also be useful if you require the ability to do a single collection of all the equipment from a particular site at the end of a project.

With the option set, the behaviour of the ‘Off Hire Contract’ screen remains unchanged (i.e. the complete off hire of one particular contract), but the ‘Partially Off Hire Contract’ screen effectively becomes a ‘Partial/Site Off Hire’ screen.  If you select the ‘Partially Off Hire Contract’ option and pick a contract, the note can include all equipment from the selected contract’s site.

Configuration.

To turn the feature on, use the ‘Multi-Contract Off Hires?’ option on the Off Hires(2) tab of the Syrinx SP Configuration\System Options\Hire Contracts screen:

You can choose between enabling the feature for all account customers, or for just selected account customers.  If the latter option is chosen, then use the ‘Account’ tab on the ‘Amend Customers’ screen to turn the feature on for a selected customer:

Creating a multi-contract off hire.

As stated above, if you chose the ‘Partially Off Hire Contract’ option, and there is more one contract for the selected contract’s site that can be included on the same off hire note, then the system will ask if they are to be included:

Note that a contract can only be combined with other contracts on the same note if it is not 

  • already on an open off hire or exchange note
  • an operated or cross hire contract
  • marked to be invoiced at off hire rather than in batch

Both the title bar and the ‘Contract No’ field make it clear that it is a multiple contract off hire, and the button to the right of the contract number opens the ‘Query Hire Contract’ screen for the set of contracts included on the note:

The Order No displayed is the order number for the contract initially selected when the note was created; however all contracts from the site are included, regardless of the order numbers on each contract.

If there are any contracts raised by other depots at the same customer site, then these will also be included on the off hire note, and each time you enter the partial off hire screen you will get a warning to this effect:

At the bottom of the screen there are ‘Collect All’ and ‘Rerurn All’ buttons:

If pressed, all items on the off hire note are marked for collection or return.
