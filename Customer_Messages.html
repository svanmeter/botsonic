Customer Messages




Overview

Customer Messages allows you to display a message every time you open a contract for this customer.


Topic included in this article:

  • How to Add or Edit Messages

 

Here's how to get there
Expert: Program Menu > Configurations > Customer Configurations > Customer Messages
Elite: Gear icon > App Configurations > Customer Configurations > Customer Messages



When you add a new message, the system will automatically assign a message number.
 




How to Add or Edit Messages
 

1. Click on the description you desire to change or click the Add button to add an additional method. In either case, the following screen will be displayed:
 


 

2. If you want to force the counter person to acknowledge the message, then check the “Force Acknowledge” box. 

 

3. Once the message is added, enter the message number into the customer’s message # field in the Account tab.
