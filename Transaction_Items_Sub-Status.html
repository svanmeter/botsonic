Transaction Sub-Status

 

Transaction Items Sub-status can be set up allowing you to have a sub-status for each line item on the contract.  The sub-status can be anything you desire such as Counted, Counting, Loading, etc.  You can update the line item sub-status through Day at a Glance.

You can add a new Transaction Item Sub-status by selecting the add button.

Description - Name of Sub-Status

Inactive - Check this box to mark the sub-status as inactive
