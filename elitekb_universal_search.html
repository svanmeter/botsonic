Universal Search

 

This powerful ‘hub’ in Elite allows for both search and creation of things like Contracts, Operators and more.
 

 

When an option (other than All) is selected, the user can type search criteria and they will see immediate results they can select from.  If they select <enter>, that will display a ‘Search tab’ on the dashboard for the option selected.

 

Search in Elite is more ‘google like’.  We are searching against several fields using ‘contains’ logic so the search returns tend to be broader than Core search.
