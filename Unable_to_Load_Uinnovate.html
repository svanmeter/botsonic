Unable to load uInnovate

If a customer gets the "What Happened? message from loading Uinnovate:


 
  1. The customer needs to be on the latest version of the program. 
  2. The server time needs to be within +/- 2 minutes of the correct time. 
    1. Restart the program and test when the time is corrected. 
    2. Time can only be corrected as an admin user.
