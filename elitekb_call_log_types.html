Anrufprotokolltypen


Übersicht

 

Mit den Anrufprotokolltypen können Sie die Art der getätigten oder empfangenen Anrufe verfolgen und darüber berichten. Sie können neue Typen hinzufügen und die Standard-Anrufprotokolltypen bearbeiten, mit Ausnahme von System und Systembrief. Diese beiden Typen sind innerhalb des Systems einmalig.

 

In diesem Artikel behandelte Themen:

 

  • Einrichten von Anrufprotokolltypen
  • Anrufprotokolltypen für Mietabmeldungen

Einrichten von Anrufprotokolltypen

  Um die von Ihnen gewünschte Art der Anrufprotokollierung einzurichten, gehen Sie bitte zu Programmmenü > Systemkonfigurationen > Transaktionskonfiguration > Anrufprotokolltypen. Sie werden aufgefordert, Ihr Passwort einzugeben, um das folgende Fenster zu sehen:



Anrufprotokolltypen für Mietabmeldungen

 

Fügen Sie die von Ihnen bevorzugte Art von „Mietabmeldungsprotokoll“ oder „Mietabmeldung“ hinzu und speichern Sie dies. Bitte stellen Sie sicher, dass Sie ein neues Counter System öffnen, damit dies implementiert werden kann.

 

1. Wenn Sie einen Vertrag abschließen und Mietabmeldung wählen, wird die Protokollart automatisch ausgewählt.

 

2. Ein Vermerk mit der Nummer und dem Zeitpunkt der Mietabbrechung wird zu den Auftragspositionskommentaren für alle in der Mietabmeldung enthaltenen Artikeln hinzugefügt.

 

Wenn Sie einen neuen Anrufprotokolltyp „Mietabmeldung“ oder „Mietabmeldungsprotokoll“ erstellen, wird dieser Anrufprotokolltyp automatisch ausgewählt, wenn Sie beim Schließen eines Vertrags eine Mietabmeldung durchführen. Auf diese Weise können Sie Berichte über alle Mietabmeldungen erstellen und Systembenachrichtigungen so einstellen, dass der Disponent sofort benachrichtigt wird, wenn bei einem Vertrag eine Mietabmeldung vorliegt.

 

Darüber hinaus können Sie das Anrufprotokoll als „Mietabmeldungsprotokoll“ erstellen, und es wird automatisch der folgende Text zu den Auftragspositionskommentaren jedes abgerufenen Artikels hinzugefügt: „Abgemeldet # xxx am TT/MM/JJJJ für TT/MM/JJJJ“. Diese Information wird auf den Vertrag gedruckt.

 

Mietabmeldung - Wenn dieser Anrufprotokolltyp vorhanden ist, wird er für das Anrufprotokoll verwendet, das aus dem Mietabmelde-Verfahren erstellt wird. Dieser Anrufprotokolltyp wird auch für das Mietabmelde-Verfahren von Mobile Workforce verwendet.

 

Mietabmeldungsprotokoll - Wenn dieser Protokolltyp vorhanden ist, wird er für das Anrufprotokoll verwendet, das aus dem Mietabmelde-Verfahren erstellt wird, und es fügt dem Auftragspositionskommentaren für jeden abgerufenen Artikel die Mietabmeldenummer sowie Datum und Uhrzeit (bevorzugt) hinzu.
