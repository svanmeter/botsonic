Passwords



 

You must enter the owner or a manager’s level password to be able to modify or inquire the current passwords.  You will only be able to see and modify the level of password you entered and lower.  Type the new password and then click OK.  If you don’t want to save the changes, click CANCEL.


 

 

* Note: If we set the minimum password to display to a specific password level you will not be able to view the account summary report if you are trying to access the customer record through customer dashboard.
