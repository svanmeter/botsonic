3/21/2022 - Version 2022.04

 
  • Undo Receive - We told mom once that it’s better to ask forgiveness than to ask for permission. That didn’t work out so well. But with our software, we forgive you with no punishment. If you have received an item or items by mistake, you can now un-receive them. Not only will it show the item as still out, but it will change the contract status back to order if the contract is in received status.
  •   Edit Hour Meter Time - “Hey Cooter, I thought you said the meter reading was 125.5 but it’s really 135.5!!” Oops. Now you can correct it by just editing as long as the transaction is still in order status.
  • Workbench Enhancement part 55 - OK, part 55 is a little ambitious, but we have been on a roll with the workbench. We have added a pickup icon to applicable contracts as well as an event icon.
