Ordering custom questions - You now have the ability to predetermine the order of your custom questions so they show up in the proper order on the storefront. In Configure Custom Questions, you can now see this new field called Sort Order

 



When you are editing one of the custom questions or adding a new one, you can choose the order # you want it to go in.

 

 

Address Autocomplete - In order to take advantage of this new feature, you will need to obtain a google maps API key that will require you to create an account. Once you have your API key, you can put it in company configuration/address display. 

 

 

Anytime you add a customer and begin typing the address, it will suggest addresses for you to choose from, and it will populate address/city/state/zip when you select

 

 

Rental Periods on Invoices - If you do automatic billing, when you select “Invoice” as the Payment Method, you will see a new selection at the bottom to show rental period dates on invoice. If you select that, you are also able to choose if billing ends on the invoice date, which is the default, or if you bill in advance.

 

 

Item History - In each item, you now have a tab called Item History. It will show you previous contracts that the item was rented on. Very useful for you to quickly see who had the item last in case it was damaged
