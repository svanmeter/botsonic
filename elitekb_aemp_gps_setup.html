AEMP GPS Setup




This article will explain the AEMP Telematics Integration Menu, as well as how to set up a new manufacturer, and how to import the files from your Manufacturer/GPS Provider to your Point of Rental system. AEMP GPS Setup allows you to get the hour meter reading as well as the location of items in your fleet. To enhance this, your items will be polled at least once a day and the current meter will be automatically updated in the inventory record. This ensures your maintenance due and service due is up to the minute all the time. So, if your customer is running your equipment 24/7, you will know right away that you need to go out and service the equipment. This will allow you to make informed decisions about necessary maintenance and repair services. 


Topics in this article include

  • Getting there

Getting there

How to get there:
Expert: Program Menu > Configuration > System Configurations > AEMP GPS Setup
Elite: Gear Icon > App Configurations > System Configurations > AEMP GPS Setup



AEMP Telematics Integration Menu
 

Web Service URL - This is the URL for the manufacturing website.
 

Username - Enter Username.
 

Password - Enter Password.
 

Last Imported - This shows the date items were last imported.
 

Records Imported - Number of records imported.
 

Raw Import - This is the raw data received from the manufacturer.
 

View Data - This is the current data showing the links for the part and serial numbers to the GPS units. This will let you know if any unit is missing information and may still need to be linked.
 



Setting New Manufacturer
 

1. Call your provider to obtain/confirm any required login information (username, password, client secret, etc.). Point of Rental does not keep records of this information. Enter the website under the Web Service URL from the manufacturer or GPS provider.
 

2. Enter the username and password for the manufacturer or GPS provider.
 

3. If the GPS provider requires a 2.0 integration select that radio button and the list of options underneath the Auth type will expand to include API Key, Oauth 1, and Oauth 2.
 

4. Click Save. You will receive this prompt to enter the name of your AEMP provider.
 

5. If the provider requires AEMP 2.0, select which Auth type is required and then select the Configure Advanced Auth button that will appear.  Enter the required information in the screen that appears to retrieve the token and save this screen.
 

Note: This will add the Manufacturer to the drop-down list. Once created, you can interface with any manufacturer or GPS provider that is compatible with the AEMP standard.
 

6. Enter Fleet Description by entering the AEMP Provider Name.
 

 


To Import
 

1. Select Manufacturer or GPS Provider from the drop-down menu. The Web Service URL, Username, and Password will populate the appropriate information.
 

2. Click Update.
 


Note: In the box on the right side of the screen, file import information will populate.

 

3. Files will be imported from the manufacturer. You will see in the bottom right corner, next to the update button, the number of files that have been imported and the last import date.
 


Note: You need to set the GPS # in each of your serialized item records. Go to Program Menu > File Maintenance > Asset Tab.
