Customer does not have integration but wants details

Salesforce information


When the customer gets Merchant ID, we normally get a copy of the Name, Site ID and Key.
 


We will still need to get the API Key for it to connect to our system. You will have to email partnerconcierge@cayan.com to get that information.

 

In about an hour you will get an email from Concierge giving you the API Key.
 


 

Set up on customer Rental Essentials (V3.0+)

 

1. Log in to Billing Profile of Essentials.


2. Impersonate the Customer.


3. Go to Accounting > Credit Card Processing.
 



4. Under Update Merchant Credentials, click on the Cayan Button.
 



5. Plug the Cayan Credentials underneath the area “Cayan Credentials."

6. Click Save when you are finished.
 


 

Adding a certificate in Windows

 
For Cayan to work, we will need to install a Certificate on their computer.

 
1. Remote into the customer's computer.
 

2. Bring over the Security Certificate from https://drive.google.com/drive/folders/1CpbLLcrtBFO43yzlaMasnrEX-LtGTUN_?usp=sharing

 

3. Double click on the Certificate.

4. Select Install Certificate.

 

 

5. Under the Store location, click on Local Machine.

6. Click Next.
 

 

7. Under the Store, click on Place All Certificates in the following Store.

8. Select browse and navigate to the “Trusted Root Certification Authorities.”
 

 

9. Click Next and Finished.

10. Restart all browsers for the Certificate to take effect.

 

 

Adding a certificate in macOS

 
For Cayan to work we will need to install a Certificate on their computer.


1. Remote into the customer's computer.

2. Bring over the Security Certificate from https://drive.google.com/drive/folders/1CpbLLcrtBFO43yzlaMasnrEX-LtGTUN_?usp=sharing

3. Open Keychain Access (you can do that from Spotlight, from TV Connection click Windows Key + spacebar).

4. Go to the System area and drag and drop the Cayan certificate in that area.
 

 



5. Double click on the Cayan Certificate.

6. Ensure all dropdown settings are turned to ‘Always Trust.'



7. Close that window out.

8. You will be asked to type in an Admin Password. Enter it in that field.




9. Have the customer test Cayan and Rental Essentials.
 

If you get an error for “Getting a Credit Card - Invalid Transport Key”, follow the instructions in that section of this document.


 

Connecting a terminal to Essentials

 

Note: Current credit card devices that we support at this current time are the MX915, M400, and e285. You will need to set the device up with Static IP (an IP address that doesn't change). See the owner’s manual on your device to set that up before following these steps.
 

1. Go to Stations > Registers, or Registers.

 


 

2. Click the pencil next to the Register you are connecting the Terminal to.
 


 

3. Click the (+) underneath Register Configurations.

 


 




4. Click save when you are done filling in the information.

5. Click the Checkbox to associate the register with the terminal.

6. Click Save.
 

Name - You will need to enter a name for the register (you can name this whatever you would like).

Cayan Terminal Number - Although this field is not required, it will be the number that is used to identify your terminal with MasterCard, if supported. If you don’t have one, then leave it blank.

IP Address/URL for your Cayan Terminal - This is the Terminal IP Address that you have to get the customer to access the terminal.
 
  • On the keypad, type in 000.
  • The terminal will ask for a password: 9416557
  • Have customer give you their IP Address.
  • On their computer, do a Ping Test to make sure that you can read it.
    • You can go to Command Prompt and type “Ping <IP Address> and Enter.
    • If you get a response, then you are good with the terminal.
      • If you don’t, then have them contact their Hardware Tech to make sure that it is connecting to the internet correctly.
 


 

Multiple registers

 

If a customer has multiple registers they use, they will have to use their Register URL that is available in the Stations > Register module. If they do not, then anything that is associated to that register will not work and will cause complications, since it will default to the main register.
 


 

Getting a Credit Card - Invalid Transport Key


 If you get this error, you are going to have to reinstall the Certificate as a Local Machine (PC) System (Mac). Then open up a browser and type in, IPAddressofTerminal:8443/v2/pos?Action=Status&Format=JSON.
 
  • Example: https://192.168.1.20:8443/v2/pos?Action=Status&Format=JSON

It will show something like this.
 


Have the customer test the Certificate.


 

Connecting Genius Mini to Rental Essentials (Android)

Customers now can take payment on their tablets or phones with the Genius Mini! To take payment on the mobile app, you have to plug in the Genius Mini CHB20.
 

1. Download the Genius Mini App from the Google Play Store.

https://play.google.com/store/apps/details?id=com.bbpos.android.tsys&hl=en_US&gl=US 
 


2. Follow Genius App and Instructions to set up your device.
 

3. Download the Rental Essentials application through the Google Play Store.

Google Play Store: https://play.google.com/store/apps/details?id=com.pointofrental.essentials.app&hl=en _US&gl=US

 


 

4. Open the application.

5. Select Take Payment.

 



6. Type in the desired Contract Number or Customer Name.

7. Select the Customer.
 

 

8. Enter the information needed and take the payment.

 


9. This will take you to the Genius App so you can finish the process of the payment.

 

Note: The Genius Terminal Serial Number MUST be connected to their account. If it is not, then you will get an invalid Transport Key.
