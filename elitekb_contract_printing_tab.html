Contract Printing Tab



Overview

This article will discuss how to get to the contract printing tab, and the various printer configuration options available to you within the menu.



Topics included in this article:
  • Contract Printing Tab


Here's how to get there
Expert: Program Menu > Configuration > System Configurations > Printer Selection
Elite: Configuration Gear > App Configurations > System Configurations > Printer Selection


Contract Printing Tab
 


 

Laser Contract Printer - All laser printed contracts will be printed to this printer.
 

Laser Contract Copy 2 - If a printer is selected, then every time a laser contract is printed, a second copy will be printed to this printer. This can be used to handle multi-color printers.
 

Laser Contract Copy 3 - If a printer is selected, then every time a laser contract is printed, a second copy will be printed to this printer. This can be used to handle multi-color printers.
 

Quote Contract Printer - If a printer is selected, then every time a Quote contract is printed, this printer will be the default.
 

Reservation Contract Printer - If a printer is selected, then every time a Reservation contract is printed, this printer will be the default.
 

Closed Contract Printer - If a printer is selected, then every time a Closed contract is printed, this printer will be default.
 

Dot Matrix Contract Printer - All form printed contracts will be printed to this printer.
 

Receipt Printer - This is the default printer for sales only contracts. This printer is also used for printing bank deposit slips as well as disbursement records.
 

Cash Drawer - This is the printer that the cash drawer is connected to. Usually this would be your receipt printer.
 

Deposit Drawer - This is the printer that the deposit drawer is connected to. Usually this would be the same as your cash drawer printer.
