If you have our kiosk add on feature, this will show you how to set up your kiosk devices and how the kiosk feature works, giving you a better understanding of how to enable and configure the Kiosk for customer operation.  



Topics in this article include:

  • Setup steps
  • Notes regarding the Kiosk use

 




Setup steps
 

1. From the home screen, select the Stations tab, then choose Kiosks from the drop-down menu.
 

 

 

2. Click the New Kiosk button.



3. Enter the desired name of the Kiosk Station. If this will be the main or only Kiosk, click the Default Kiosk? box below the Machine Name field. Be sure to click Save in the lower right corner.
 

 

4. To configure the Kiosk settings, select the Pencil icon next to the Kiosk name.
 

 

5. To edit the default configurations, select the Pencil icon in the Kiosk Configurations section.
 

 

6. Complete the Name, Welcome Page, Signature Page, and Final Page fields. If desired, add an image to the Splash Screen Photo and select any Required Fields for Customer Registration. Click Save at the bottom right corner of the page. Ensure the associated box is checked for the created configurations and click Save again. The Kiosk is now enabled.

 

 

7. Test the Kiosk device by entering the Kiosk URL found on the Kiosk menu.
 

 

8. The Splash Screen Photo should load automatically. Click anywhere on the screen and the Welcome Page will appear.
 

 

9. The customer will follow the prompts to enter their personal information. These are the required fields that were designated in the configuration settings.
 

 

10. When the customer clicks Save and Continue they will be redirected to the Signature Page. These are the Terms & Conditions that were entered in the configuration settings.

 

 

11. After the customer signs and clicks Accept, a message loads with the Final Page Text that was entered in the configuration settings.
 


 

Notes regarding the Kiosk use

 
  • If the customer is not in the system, a customer file will be created automatically. If the customer has previously been entered, their information will be uploaded. 
 
  • If the customer does not have a current Quote or Reservation, a transaction will be created and set to Reservation status with no Rental Items or other charges listed. If the customer does have a Quote or Reservation, the transaction will be set in Reservation status with a digital signature automatically uploaded into the transaction.
 
  • Configuration settings will allow a notification email to be sent with each customer sign-in into the Kiosk.
