Bestellung ändern

Wenn Sie die Bestellung aus dem Programmmenü auswählen, werden Sie automatisch zum Anzeigebildschirm geleitet. Sie haben die Möglichkeit, alle Ihre Bestellungen (Offen, Ausstehend, Guthaben, Geschlossen, Gebucht, AP-Export, Untervermietung) auf einen Blick zu sehen. Wenn Sie einen Vertrag ändern möchten, klicken Sie auf den zu ändernden Vertrag.

 

Sie können die gewünschte Bestellung auswählen, indem Sie auf die Bestellzeile klicken. Wenn Sie auf das Pluszeichen klicken, werden alle Artikel der Bestellung angezeigt.

 

Sie haben mehrere Möglichkeiten (Anfrage, Drucken, Ändern, Empfangen von Artikeln aus der Bestellung, Aufgliedern von Einzelposten, Klonen, Stornieren oder Exportieren in eine Tabelle). 

 

Sobald der Bestellbildschirm angezeigt wird, können Sie jedes der Felder ändern.

 

Optionen

 

Lieferung an Station: Unter dieser Option können Sie als Versandadresse ganz einfach die einer anderen Station nennen. Natürlich können diese Felder auch manuell ausgefüllt werden.

Hauptbucheinträge: Wenn Sie ein integriertes Abrechnungspaket verwenden, wird die Abrechnungswarteschlange der Bestellung angezeigt.

Bestellung aufheben: Wenn eine Bestellung bereits gebucht wurde, aber weitere Korrekturen erforderlich sind, kann der Benutzer dieses Feld verwenden, um die Bestellung aufzuheben. Sie muss vor dem erneuten Export manuell aus der Buchhaltungssoftware entfernt werden. Dies ist abgesichert durch Bestellungen im Sicherheitsraster auf Unternehmens-/Gruppen-/Mitarbeiterebene.

Station der Bestellung ändern: Dies ermöglicht dem Benutzer, die Station der Bestellung zu ändern. Die Sicherheit hierfür ist die gleiche, die mit der Änderung der Vertragsstation in Counter System verbunden ist.

Bestellung bearbeiten: Hier wird eine Listen-/PDF-Ansicht aller Änderungen einer Bestellung angezeigt.

Anhänge: Alle für die Bestellung relevanten Dokumente/Fotos können hier verlinkt werden.

EDI-Import: Bei großen Einkäufen kann eine CSV-Datei von Artikeln/Mengen importiert werden, anstatt jeden Artikel einzeln hinzuzufügen. Weitere Informationen finden Sie im EDI-Importdokument.

 

Offen - Diese Registerkartenoption zeigt alle offenen Bestellungen an.

Ausstehend - Diese Registerkartenoption zeigt alle ausstehenden Bestellungen an. (Nicht beendet)

Kredit - Diese Registerkartenoption zeigt alle Kreditbestellungen an.

Geschlossen - Diese Registerkartenoption zeigt alle geschlossenen Bestellungen an.

Gebucht - Diese Registerkartenoption zeigt alle geschlossenen Bestellungen an, die in der Kreditorenbuchhaltung gebucht wurden.

AP-Export - Ermöglicht den Export von Bestellungen in eine iif-Datei, die in Ihre Buchhaltungssoftware importiert werden kann. Es wird eine Importdatei mit dem Namen AP yyyy.mm.dd.batch#.iif für QuickBooks und yyyy.mm.dd batch#.Ap.csv für durch Kommas getrennte Dateien erstellt. Die Datei wird in dem unter Programmmenü definierten Exportverzeichnis abgelegt: Konfigurationen: Kontonummern.

Untermiete ausstehend - Diese Option zeigt alle ausstehenden Untermietkaufaufträge an.

Anfragen - Mit dieser Option können Sie alle Bestellungen für eine bestimmte Bestellung einsehen

Drucken - Mit dieser Option können Sie eine Bestellung ausdrucken.

Ändern - Mit dieser Option können Sie Änderungen an einer bestimmten Bestellung vornehmen.

Artikel aus der Bestellung erhalten – Mit dieser Option können Sie Artikel in einer bestimmten Bestellung erhalten.

Bestellung klonen - Mit dieser Option können Sie eine Kopie einer bestimmten Bestellung erstellen.

Ungültig - Diese Option zeigt alle ungültigen Bestellungen an.

In Tabellenkalkulation exportieren - Ermöglicht es Ihnen, eine Bestellung in eine Excel-Tabelle zu exportieren

 

Wenn Sie das Auswahlfeld für mehrere Artikel aktivieren, erhalten Sie ein neues Rechtsklickmenü, mit dem Sie eine Überweisung erstellen oder mehrere Artikel gleichzeitig erhalten können.
