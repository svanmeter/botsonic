Setting Security Permissions with the POR One App and Global User Admin

 

1. To access these permissions, an administrator must go to Configuration Gear > System Global User Admin. This is located in the middle part of the screen.

 

 

2. This will open your default web browser window where admin credentials can be used to log in.

 

 

3.  On the User List Page, this is where you can set permissions for parts of the POR One Mobile App. You can also give other employees access to View and Edit Permissions from here as well. To give them access click the check box under the appropriate column.

 


Viewing or Editing Permissions
 

Edit Permissions - A user with this option selected can update permissions for other users.

View Permissions - A user with this option selected can log in and view the settings but cannot make any changes.
