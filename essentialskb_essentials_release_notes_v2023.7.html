Release Notes for v2023.7 (5/15/2023)




 

 


 
  • Price List Report - Sure you can get individual prices for items several different ways. But now you can get a list of ALL your item prices to give to your customers. Then you can view it, email it, or save it. You can even save it, then print it. And if you are really stuck in the 90's, you can save it, print it, and then fax it.
 
  • Statements option to show all contracts owed - We recently added functionality for Batch Statements. You didn't think we were going to just stop did you? We now have a setting that gives you the ability to not only bill for contracts that have never had statements run on them, but also for the old ones that the customers haven't paid yet. And just as cool, it shows the previous statement for each of those 'rebilled' contracts. It might as well just say 'Hey Customer, this one that says it was already sent to you, we asked you to pay already, so pay up.'
 
  • Exclude Overdue Contracts on Workbench - If you have someone that manages overdue contracts, and some of your users don't need to see them on the workbench, there is now a filter to exclude overdue contracts
 
  • Storefront Time Windows enhancements - On the storefront, your customers could only choose 30 minute windows by default (9:00, 9:30...etc), but you now have a setting that you can define them. Starting at 1 hour, down to 30 minutes, then 20, 15, 10, or even 5. So if you are so awesome at having things ready and doing deliveries at 2:05pm, go for it.
 
  • Mobile Version Workbench Redesign - If you use the mobile version, you may notice we have changed the look and feel. We have aligned all rows, and made things a lot easier to read. In short, we made it better and more user friendly.
