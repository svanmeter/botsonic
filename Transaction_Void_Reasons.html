Transaction Void Reasons

 

Transaction Void Reasons allows users to select a reason for voiding a contract. When voiding the contract, the user will be able to select from a list of voided reasons.

 

From Program Menu>Configurations>Transaction Configurations>Transaction Void Reason

 * Note: Under the inactive column, True = Reason is marked as Inactive False = Reason is Active


For Steps on Voiding a Contract click the following link: Void Reservation Contract
