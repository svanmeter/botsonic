If you need to define depot specific reorder levels for stock items, you first need to make sure that the system option "Set Re-order Levels / Order Qtys by Depot?" on the consumables tab of System Options\Sales has been enabled. With that option enabled, there will be a "Depots" tab on the Amend Sales Stock Items screen on which you can define depot specific values for "Reorder at Level" and "Quantity to Reorder"
