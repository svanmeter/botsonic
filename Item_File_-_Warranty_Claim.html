Warranty Claim

 

Overview: Tracking warranty claims in POR. When you write an Internal Repair Order, if the item has a current warranty you will be notified. In addition, you can create a warranty claim tracking from within the IRO. Once you create a new warranty claim, you can define the vendor for the claim, the date submitted, the claim number, the contact name, credit received, and any notes. As the claim is processed, you can update the claim status.

 

Adding a Warranty to an Item:

By defining item warranties, the system will notify you that an item may be covered by warranty when a repair contract is created. Modify the item in File Maintenance.

 

Select the Warranty tab, right click in the gray area at the bottom, and select Add.

 

The following box will appear. Enter in the appropriate information. You can add as many warranties as necessary.

You can either click to look up or select the Vendor Lookup button to look up the Vendors. Type in the Vendor’s name or select All Vendors.

 

 

Creating a Warranty Claim:

 

From an Internal Repair, Order selects the Tools to drop-down menu and select Create Warranty Claim.

*Note: When manually creating a Service Ticket for an item, the system will check whether the item is still under warranty. If so, you will be given notice of the warranty information.

 

You will see the following box. Click the Get Vendor button to select the vendor. You can also enter a claim number, contact, credit invoice number, credit received, and credit amount. The claim date and entered by fields will be automatically populated.

 

To update an existing warranty claim, from the Tools drop-down menu in the Counter System select Warranty Claims.

 

 A list of warranty claims will be displayed. Selecting a warranty claim will open it for editing.
