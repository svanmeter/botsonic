Calls Tab

 

 

 

Overview  


This article goes over the calls tab in the customer record. It explains the various fields and options related to the call log.

 

 

Calls tab

 

Left-clicking on the Calls tab in the customer record will display the fields in that tab.

 

 

Call Logs are used to document communication between you and your customer. By logging calls, you can preserve valuable information while making it visible to everyone.

 

Show All Calls - With this box checked, all calls for this customer will be shown. When it’s unchecked, only calls that don’t have a particular contract number will be shown.

 

Print - This button will print all of the calls to the printer.

 

New Call - This button allows you to create new calls logs for the selected customer. Calls can also be created within contracts.

 

﻿

 

Force acknowledgment - You can check this box to force a message to appear on the screen when a contract is created for a customer.

 

Reminders - This gives you the option to create a reminder, to create a reminder with a call log, and to view reminders in the task list.

 

Modify - This allows you to add, delete, or modify a call log. Any modification will be followed by a date stamp. Security levels can be set not to allow employees to modify the call logs.

 

Add Note - This allows you to add notes to your call log.
