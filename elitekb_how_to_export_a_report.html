How to Export a Report

 

Overview: Almost all reports generated with the point of rental can be exported to other programs. You have four different methods of exporting any report.

 

Exporting: Generate a report. From the toolbar at the top of the screen, there are several icons.

 

1. Exporting a report to Microsoft Excel or any spreadsheet you install on your server.

 

2. Exporting a report to a PDF document

 

3. Exporting a report to Crystal Reports where the report can be manipulated

 

4. Export to a destination of your choice, where you can choose the file type. Use this option if you would like to export the drilled down report. Once you have opened the report and drilled down into it, click the icon below to export, then change the file type from Crystal to xls.
