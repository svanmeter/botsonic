Adding Sort Order Field

 



Overview
 

Adding to the Sort Order Field allows you to group items together to create a visual group for your customer or operation. 


Here's how to get there:
Expert: Program Menu> Counter System> Enter user ID and Password> Modify Contract
Elite: Left Menu> Transactions> Modify Contract.



Adding sort order field:

1. Select a customer.

2. Use your Ctrl key and left-click your mouse to multi-select items, then right-click one of the items. This will allow you to change the sort field for the selected items or add to the sort field. (i.e. sort field was ‘And you add ‘10’ to the sort field; the result is ‘10A’.

 

 

 

Note: The items marked as '10' in the sort field group together, then the items within group10 still sort from A-Z; this is true in group '20' also.


 

In this example, we used the 'NOTER' (See Contract> Contract Note Item) item to create labels within the line items on the printed contract.
 

Using the Add to Sort Order Field we have created a group of items for the Front yard and a group for the Backyard. This type of view makes it much easier for your customer or employee to understand the breakout of each area or group.
