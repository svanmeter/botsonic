Syrinx 365 | Search Configuration System Options



This article will cover where to find and use the search function for Configuration System Options. This search screen is designed to help the process of finding the system option you are looking for within the many menu options and tabs within those.


Topics included in this article
 
  • Switching to Configuration
  • Opening System Options Search
  • System Options Search
  • System Options Columns




Switching to Configuration


If your user role has Configuration Settings then, upon clicking the User Settings button at the top right, 
will display a toggle to switch between the Main Program and Configuration Program. 



When you toggle to Config then the left-hand menu options will populate according to your user role.

 


Opening System Options Search

From the left-hand menu select System Options.



Then select Search System options.


   
System Options Search

The search screen works as other Local Search screens do, in that you can use the search field and you can add filters. All of the columns are displayed as default.

  
   

System Options Columns


Name - This column contains the name of the option within the actual database table. This can be useful if you are looking for an option within the SQL table, especially as some options are abbreviated.

Menu - This column displays which left-hand menu option the system option is in, i.e. the screenshot above would be found in the Accounts menu option.

Description - This contains the tab name that the system option is on within the menu item and a description of what the option is. In the example above, those options all sit in the tab of Accounts in the menu item of Accounts. In some cases, the description in this search grid, may not match 100% the description within Syrinx as some have been extended to be more descriptive. Where the system option has a dropdown selection, the options have been included in the description.

Value - This will display the value that is set on that option, which could be a 1 or 0 for True or False, or an actual value. This may help to easily identify what that system option is currently set to without the need to open the form.

To open up the form for a System Option you can click the Name on the row, click the ellipsis (the three dots) on the left-hand side of the row then the Open button, or click the row so that the Summary Panel is populated and click the Breakout button.
