Item Pictures

 

Overview:  This explains how to add pictures to item records for use in Item Availability, Price Lists, Tear Sheets, and an integrated Website.

 

*Note: It is recommended that this procedure be done from the server. If done from a terminal the image quality could be degraded. The system will resize the image (all images are sized with the same width of 640 pixels) for uniformity and optimized for web viewing. In Elite, images larger than 4MB will warn the user, but allow them to proceed.  MakeWeb has a max size of 4MB for a single image to upload to an integrated website. 
 

1.   Find your image. Going to the manufacturer’s web site or doing an image search will usually find good results.

a.   Make certain you have permission to use copyrighted images. Point of Rental cannot be held responsible for any unauthorized use of copyrighted material.

2.   If you are selecting a WEB image right-click the image:

a.   First, select properties and verify that the width of the image is 250 pixels or larger.

b.   Second, select Copy.

3.   If you are selecting an image file from your camera or a saved file from your computer, right-click the file:

a.   First, select properties, click on the Details tab, and verify the width of the image is 250 pixels or larger

b.   Second, select Copy.

4.   Open the Point-of-Rental Program

a.   To add a picture to an item: From the Counter, System Menu chooses #11 Item Availability. Select the item you want to add a picture to.

b.   To add a picture to an item category: From the Configuration Menu select Item Categories. Select the category you want to add a picture to.

5.   In the Picture or Web Picture field, right-click and select PASTE, as shown below.

6.   You may be asked for your ID.  POR will mark that you added the picture.

 

*Internet Browsing: As a reminder, you should always have anti-virus software installed and kept current on any computer accessing the internet.

*Note: Adding a picture to a HEADER record will automatically update the serialized items.


*Drag and Drop:  SQL database users can also drag and drop images from file explorer into their intended item record.
