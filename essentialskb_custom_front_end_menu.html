This article goes over the Custom Front End Menu feature in Essentials.


Topics in this article include:
  • How to get there
  • Instructions
  • Menu Type Options
  • How to sort items
  • How to edit and remove items
 


 

How to get there

You can get there from Configurations > Front End Custom Menu Setup.

 



Instructions
 

1. Once you are there, click "Generate Custom Front End Menu" to get started.
 


2. Once it loads, it will place your current categories under the Top Level.
 


 

3. Categories cannot have subcategories. You will need to create a new hierarchy and put the list of items under it. To do this, add a Child Menu Node to the Top Level, click on the green + next to the level you want to add it on.
 


4. To create a New Top-Level Node, click on + New Top-Level Node.
 


5. Whether you are creating a Child Menu or Top-Level node, you will see the below configuration window appear. This is where you can enter a name and select the Menu Type. To view the definitions of each menu item type, please see the Menu Type Options section of this article.
 


6. You can also choose to display your menu either horizontally or vertically.
 


7. When you are finished, press Add Node to save.


 

Menu Type Options
 

  • Menu Category - This allows for more stacking of subcategories.
     

  • Custom Content - This allows you to place custom content on the storefront.
     

  • Internal Link - This allows you to preview the internal portal as your customer. This will help you determine what you want your customers to look at.
     

  • External Link - This allows you to enter a website outside of your storefront. Must include HTTPS:// or HTTP://
     

  • Item Container - This allows you to select a category/items that you want to include.
     

  • Contact Page - This allows your customer to send you a question or send you a message through a menu item.

 

How to sort items

To sort items, you can use the Up and Down arrows on the right of the menu topic.

 



How to edit and remove items

To Edit and Remove an item you can use the Pencil icon to edit and the Trashcan icon to remove it.

 


If you edit one of your items, you will then have the ability to set it as the homepage, as well as add a custom header and footer for that item.
