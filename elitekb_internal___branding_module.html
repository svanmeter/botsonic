Point of Rental Setup
 

Point of Rental Setup Part 1
 

This will go over how to set up Point of Rental.
 

Setup Transaction Operation
 

Here's how to get there
Expert: Program Menu > Configuration > Transaction Configurations > Transaction Types
Elite: Configuration Gear > App Configurations > Transaction Configurations > Transaction Types

 

1. Click Add.
 

2. In the “Name” field enter in the Branding Store Name.
 

3. For “Status” change the type to “Regular Contract”.
 

4. Click “OK”.
 

5. Find the one you created.
 

6. Make a note of the number in the Number column.
 

  • Brand_Location:___________________
     
  • Branding Database Setup
     

This will go over how to set up a blank database with the information you have gathered and import that into the customer database
 

1. Copy ParameterFileBrandBlank.mdb to your local side. This is located in our Download Folder.
 

2. Open the database and if applicable click “Enable Edit”.
 

3. Click on ParameterFileBrand on the left and enter in the correct information into the correct fields.
 

4. For the store logo open the image in paint, select the entire image (CTRL +A), copy image (CTRL + C). Then in the database right click on the field and select “Paste”.
 

5. Once done, close out of the database.
 

6. Copy database over to the customer's server.
 

7. Go to <PORDrive>:\POR\Program\ImportTable.exe.
 

8. Select “Import From” to be the ParameterFileBrandBlank.mdb database.
 

9. This doesn’t change if they are on SQL or JET. The same process happens.
 

10. Select “Import To” database to be the appropriate database.
 

11. In “Tables to Import”, select ParameterFileBrand.
 

12. In “Options” select “Append New Data”.
 

13. Click “Start Import”.


 

Point of Rental Setup Part 2

This will go over testing the Branding Module and validating that the logo and store information is correct.
 

1. Validate that the branding information is correct.
 

2.  Open Point of Rental to get to a new contract. 
 

Here's how to get there
Expert: Program Menu > Counter System > New Contract
Elite: Left Menu > Transactions > New Contract
 

3. Write up a test contract and in the “Information Tab" choose the appropriate Transaction Operation.
 

4. View the contract on the screen and validate that the information is correct.
 

5. Update the ParameterFileBrand information.
 

6. If any of the data is incorrect, you will need to do a database edit to update the information. This doesn’t apply to the store logo. If the store logo is incorrect you will need to go back to the ParameterFileBlank.mdb and replace the image. Once the ParameterFileBlank.mdb is updated you will copy the database back over to the customer server and using ImportTable.exe in the “Options” choose “Delete Existing Data”, then click “Start Import”.



 

Troubleshooting
 

Q: Why is the store information not coming over?
 

A: The main reason is that either the Transaction Operation was not selected on the contract or the Transaction Operation is not set properly in ParameterFileBrand.
