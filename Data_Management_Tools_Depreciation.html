Data Management Tools - Depreciation Setup

 

Select # 13  - Depreciation Setup from the Menu.

 

Make your selection using the drop-down menus - Selections if applicable:  Division, Item Category, Purchase Price Greater than (enter amount), and Select by Capitalized or Non capitalized.

 

 

Enter the Values to update. -- Enter Salvage Value,  Enter the Number of Years to Depreciate, Select the Schedule you want (Federal is the main used for Reporting but other can be set up if you have multiple ways you would like to report on).

 

Enter the Equipment GL Acct and the Accumulated Depreciation GL Account.  

 

Select your Method of Depreciation from the drop-down menu.  You check the box mark the items as Capitalized as well.

 

 

Select Go and the records will be updated with the new depreciation schedules.  A window will pop up when letting you know the process is done.
