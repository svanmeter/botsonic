Feature License Refresh

 

 

This Configuration option is used to refresh the local system with any license updates made in Global Admin.  For instance, if a user recently purchased a feature where the access is controlled with a global license, this option would be used to refresh the local system with that license permission.

 

Once the license permission is set in Global Admin, click on the ‘refresh’ button - a success notification should display.  Then logout and back in and the newly licensed feature will be available.
