POR One | Offline Functionality




You may will lose internet or cellular connectivity, but you can still access some parts of the POR One Program. Here is what you can view and edit in the application. You can continue to use the POR One app while offline or you do not have a network connection. Some functions of the app will be limited. You will be notified via a ‘toast’ notification that it has gone offline and again when network connectivity is restored, and is online again.
 



Topics in this article include

  • Search Pages
  • Details Pages
  • My Tasks
  • Messages
  • Support Requests




 

Search Pages
 

By default, the search results pages will show a list of anything recently viewed. This means that if you have searched for a contract and then viewed the Details page, it will be cached when offline and shown on the recently viewed list. Searching is not available offline. The following pages support caching of the recently viewed list:
 

  • Contracts
  • Customers
  • Inventory
  • Work Orders


 

Details Pages
 

The Details pages for any search results that have been cached in the Recently Viewed list will be cached: additional images, links to any listed contracts, stock items, or completed tasks will not be available offline. Each detail page has a list of cards or accordion-style drop downs and this list details the offline functionality for all:
 

  • Contracts
    • Information card - Details are shown Offline.
    • Images - Not Supported
    • Delivery & Pickup - Details are shown Offline.
    • Contracts - List shown Offline but links to details pages are not supported
    • Line Items – The list is shown Offline but links to details pages are not supported.
    • Related Tasks - Not Supported
    • Creation of new Tasks = Not Supported
    • E-signature - Not Supported
 
  • Customers
    • Information card - Details are shown Offline.
    • Images - Not Supported
    • Contact Info - Details are shown Offline.
    • Contracts - List shown Offline but links to details pages not supported
    • Related Tasks - Not Supported
 
  • Inventory
    • Information card - Details are shown Offline.
    • Images - Not Supported
    • Contracts - List shown Offline but links to details pages are not supported
    • Work Orders - List shown Offline but links to details pages are not supported.
 
  • Work Orders
    • Images - Not Supported
    • Line Items - List shown Offline but links to details pages are not supported
    • Related Tasks - Not Supported



 

My Tasks
 

Tasks created by you or associated with your user, and are either New or In Progress will be cached to the device. There is no limit to the number of tasks that can be pending completion. All functions of the task are available offline (e.g., take condition photos, log meter readings, and collect signatures).

Tasks that can be modified or queued for completion offline are:
 

  • Customer Pickup* 
  • Customer Drop Off
  • Delivery by Driver*
  • Pickup by Driver
  • Labor Tracking
  • Item Count
  • Inventory count

 

*Note: These tasks also support a selection of Serialized Items. The list of the available items will be cached from the time the task was created or the last time the selection list was viewed.


 

 

Messages
 

You will be able to view any messages that have been sent or received within the last 30 days. The Create and Send buttons for new messages will be disabled (gray) while offline.


 

Support Requests


You will be able to create support requests while you are offline. These requests will capture a screenshot and the same details as when you are online. Once you are back online, the request will be sent to the POR Support Queue.
