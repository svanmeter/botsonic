Region

There are different regions for Rental Essentials. Please make sure you are signing the customer up in the right region. Also, there are specialty sites for Enterprise Customers. Please reach out for assistance with those accounts.

 

US: pointofrentalcloud.com

UK: rentalessentials.co.uk

AU: porcloud.com

 

 

Creating the Account


In this article, we are going to work on the Point of Rental Essentials US Prod. The steps will be similar for the other regions.

 

1. Go to pointofrentalcloud.com/signup and fill out the customer information. This will work best in your browser in Incognito or Private Mode.

 


 

2. Under Package Information choose a package for the customer. We are going to change this when we get to the next step.
 


 

3. Agree to all the Terms and Conditions and do the reCAPTCHA and click on get started. This is going to create the account for the customer so they can sign in and get started. Billing will not be done during this process. We will have to send an invoice to the customer after we finalize the billing for the customer.

 

4. After this process is done close this tab and open up the billing site.


 

Billing Site - Cancelling Free Trial
 

1. When you log in to the Essentials Billing Site it is going to create a contract with a free trial. This will need to be cancelled. To do that click on the contract from the Dashboard.

 


 

2. From the Contract Screen click on Cancel Transaction from the blue bar on the top.
 


 

3. When it asks you if you are sure that you want to do this, click on Cancel Transaction.
 


 

4. The contract is now canceled and it will take you to the Workbench where you can see other contracts and customer invoices in the Essentials program.

 

 

Set up Current Features
 

For this next step, you will need to have the purchase documentation next to you to make sure that you are assigning the right information and package to the customer. You might also want to have the Pricing Guide up if you have questions about pricing.

 

1. From the Essentials Billing site go to People > Companies.
 


 

2. Locate the customer and then click the pencil next to the company you are creating the billing for.
 


 

3. For this part of the billing setup, you will need to make sure that it matches with the billing site. Scroll down to the Current Live Company Features. This is where you will be working at.
 


 

4. From the Add Feature you will add the features that the customer is supposed to have. For example, they will get 3 users, storefront, eSign, SMS, sales, countersign, and iteminstancephotoupload. When you add the feature it will replace what is currently up on the top after you refresh the screen to the correct amount. Please keep the bottom line as well. This is for Custom Development if the customer would like that.

 


 

5. Refresh your screen to make sure that everything is correct. Keyboard Shortcut: Windows: ctrl + r; Mac: cmd + r

 

6. After you refresh you will see the updated information.

 

 

 

Set up Billing

For this next step, you will need to have the purchase documentation next to you to make sure that you are assigning the right information and package to the customer. You might also want to have the Pricing Guide up if you have questions about pricing.

 

1. From the Essentials Billing site go to People > Companies.

 


 

2. Locate the customer and then click the pencil next to the company you are creating the billing for.

 




3. For This part of the billing site you will need to make sure that it matches with the billing site. Scroll down to the Current Company Packages (For Next Billing Period). This is where you will be working.
 


 

4. Add the package that you will need to add. In this example, we will be adding the Standard Plan (Accounting Billed), and 2 Additional User (Accounting Billed). Click Add Package when you are adding the line.
 


 

5. Refresh your screen to make sure that everything is correct. Keyboard Shortcut: Windows: ctrl + r; Mac: cmd + r
 

 

6. Once refreshed you will see the items that you have added.
 

 7. Now lets create the contract. Under the Billing Period you can choose Monthly, Quarterly, or Annually.
 


 

8. Then click on N/A - Create Quote for Customer under the Current Contract area.

 


 

9. This is going to take you to the Contract Creation Screen Enter in the Start Date and the End Date will automatically populate. Click on Create Reservation when finished.

 


 

10. Once you are on the contract screen, you will have to add the items on the purchase documentation again to line up what you placed earlier. To do this, use the Search Bar in the middle of the screen.

 


 

11. This is going to also add other items that are needed for the account, called Suggested Items in Essentials. You can leave these be.

 


 

12. After you add all those items to the contract you can invoice the customer. If the customer is going to be billed using the Essentials Application, click on Invoicing and create a new invoice.
 


 

Single Store to Multistore Process


To set up a Multi Store location you will need to set up a Parent Location, Headquarters (HQ) and any additional stores in Essentials. The Parent Location is responsible for grouping the stores and HQ. The HQ is responsible for holding inventory for the company.
 

1. To create a new company go to People > Companies.

 


 

2. Click on +New Company.
 


 

3. Fill out the required information. For the company name, you will want to place the Company Name plus the word parent. For Example ‘Scooterbug Parent.’ Fill out all the other required fields and then click save.
 


 

4. Once saved, you will have to edit that company and enter in the information again.

5. Repeat steps 1-4 for the HQ Store and any additional stores purchased.

6. Next, you will link the stores and HQ to the Parent company that was just created. You will go into the stores and under the Parent Company field, select the Parent company.
 


 

7. If all is correct it will look like this example below. All will be grouped under the parent company.
 

 
 

Billing Information
 

  • For new customers, billing will need to be done on the parent company.

  • For existing customers, billing will be done on the first existing store.
