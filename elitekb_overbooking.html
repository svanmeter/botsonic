Überbuchung

Überblick: Verfahren zur Bearbeitung von Überbuchungen von Mietartikeln für das Counter-System.

Überbuchungen werden am besten über den Bildschirm „Artikelverfügbarkeit“ bearbeitet, wo Sie Zugriff auf alle notwendigen Tools haben, um sie korrekt zu verarbeiten und abzuwickeln. Obwohl es im Abschnitt „Reservierungsbericht“ einen Überbuchungsbericht gibt, ist dieser für eine breite Datumsbereichsansicht konzipiert, um Kaufentscheidungen zu treffen, und nicht unbedingt, um täglich Überbuchungen zu bearbeiten.

 

Wählen Sie auf dem Bildschirm „Artikelverfügbarkeit“ das Dropdown-Menü „Bestand“ und wählen Sie „Alle Überbuchungen prüfen“.



Nachstehend finden Sie die Liste der überbuchten Artikel.



*Hinweis: Sie müssen nicht mehr warten, bis alle Artikel durchsucht wurden. Der Bildschirm zeigt jeden überbuchten Artikel so an, wie er ihn findet. Mithilfe der Esc-Taste können Sie den laufenden Vorgang unterbrechen. Das ist praktisch, wenn Sie die anzuzeigende Station oder die zu durchsuchenden Daten ändern möchten, damit Sie nicht warten müssen, bis Ihnen alle gewünschten Informationen angezeigt werden.

  • Standardmäßig ist das Datum auf Sonntag der aktuellen Woche eingestellt. Sie können dieses Datum auf einen beliebigen Tag ändern, an dem Sie auf Wunsch eine Wochenendansicht erhalten möchten. Sie können Ihre Ansicht auch auf einen ganzen Monat (28 Tage) erweitern.
  • Wenn Sie ein System mit mehreren Stationen sind, können Sie einzelne Stationen, alle Stationen oder Stationsgruppen anzeigen.

Wählen Sie den Artikel, mit dem Sie arbeiten möchten, indem Sie ihn einmal mit der linken Maustaste anklicken. Wenn Sie mit der rechten Maustaste auf einen Artikel klicken, wird der Bildschirm „Artikelvertragsdetails“ direkt geöffnet, sodass Sie die nachstehende Kalenderansicht überspringen können.



• Wenn Sie auf einen der Tage klicken, an denen eine Überbuchung vorliegt (durch eine rote negative Zahl gekennzeichnet), wird die folgende Vertragsdetailliste angezeigt.

• Wenn Sie mit der rechten Maustaste auf einen Vertrag in dieser Liste klicken, können Sie Anfragen, Ändern, Stornieren, Drucken, Versenden, Untermiete anfordern, sodass Sie mit einem Vertrag interagieren oder direkt von diesem Punkt aus eine Untermietanfrage stellen können.



Aus dieser Sicht können wir deutlich erkennen, dass eine Überbuchung vorliegt. Ein Vertrag kann geändert werden, um die Menge zu reduzieren oder einen neuen Artikel zu ersetzen (natürlich mit Zustimmung Ihres Kunden).

Drücken Sie die Taste OK, um zum Bildschirm „Artikelverfügbarkeit“ zurückzukehren.

Wenn Sie das Dropdown-Menü „Bestand“ auswählen, können Sie Bestellartikel, Untermiete erstellen oder Stationstransfers auswählen.



Wenn Sie Bestellartikel auswählen, wird eine ausstehende Bestellung erstellt (Sie müssen über das Bestellmodul verfügen). Sie werden nach der zu bestellende Menge, dem Händler, von dem Sie kaufen möchten, wenn Sie mehrere Händler festgelegt haben, und, wenn Sie mehr als eine ausstehende Bestellung für diesen Händler haben, nach der Bestellung gefragt, zu der diese Bestellung hinzugefügt werden soll. (Weitere Informationen finden Sie im Hilfeabschnitt zu „Bestellungen“)

Wenn Sie „Untermiete erstellen“ auswählen, wird ein neuer Untermietvertrag gestartet. Sie wählen einen Untermieter, Daten und Artikel aus. (Siehe Hilfeabschnitt zum Untervermietungsprozess)

Multi-Station:

Wenn Sie „Stations-Transfers“ auswählen, können Sie einen Artikel gleich überführen oder einen zukünftigen Transfer planen. (Weitere Informationen finden Sie im Hilfeabschnitt zu Stationstransfers.)



Wenn eine Bestellung erstellt wurde, sieht die Verfügbarkeit so aus.



*Hinweis: Die Bestellmenge in der oberen rechten Ecke. Beim Anlegen wurde die Bestellung mit dem Eingangsdatum 28.10.2012 gekennzeichnet. In der 28-Tage-Kalenderansicht zeigt der 28.10.2012 29 verfügbare Artikel und eine Erhöhung um 15 gegenüber dem Vortag an.

*Hinweis: Weitere Einzelheiten finden Sie im Hilfeabschnitt zu „Bestellungen“.

Sobald ein „Untermietvertrag“ oder eine „Bestellung“ erstellt wurde, sieht die Verfügbarkeit wie folgt aus.

*Hinweis: Weitere Informationen finden Sie im Hilfeabschnitt zum Untervermietungsprozess.

Da überbuchte Artikel bearbeitet werden, werden sie aus der Liste der überbuchten Artikel entfernt.

Multi-Station: In einer Umgebung mit mehreren Stationen ist es wichtig, die Überbuchungen des Unternehmens zuerst anzuzeigen und zu bearbeiten und dann auf Stationsebene zu arbeiten, wenn das Ziel darin besteht, Überbuchungen durch Transfers zu beseitigen. Wenn das Unternehmen überbucht ist und die Überbuchung entweder durch einen Kauf oder eine Untermiete gelöst wird, wird standardmäßig auch die Überbuchung der Station gelöst. Wenn auf Stationsebene noch eine Überbuchung vorliegt, sollte ein Transfer die Überbuchung ausgleichen.

Kritische Überbuchung: Wenn ein Artikel nicht ganz überbucht ist, sich jedoch auf einem von Ihnen als kritisch niedrig eingestuften Niveau befindet, können Sie mithilfe des „Berichts zu kritischen Überbuchungen“ proaktiv statt reaktiv handeln. Normalerweise wird dieser Bericht am Montag für das nächste Wochenende ausgeführt. Artikel, die für den Versand vorgesehen sind, aber unter Ihrem kritischen Wert liegen, werden gemeldet, sodass Sie die Möglichkeit haben, die Bestandszählungen lange im Voraus zu überprüfen und alle erforderlichen Anpassungen vorzunehmen. Programmmenü – Reservierungsberichte

• Die Mindestanzahl wird auf Artikelebene im Feld „Mindestanzahl“ definiert; es wird entweder eine Zahl oder ein Prozentsatz angegeben. Wenn Sie gewarnt werden möchten, sobald die verfügbare Menge eines Artikels zwischen 0 und 20 liegt, geben Sie 20 in das Feld „Mindestanzahl“ ein; Geben Sie dagegen ,20 für eine prozentuale Warnung von 0 - 20 % ein. Wenn die Verfügbarkeit eines Artikels unter Null sinkt, ist er standardmäßig überbucht.

In „Artikelverfügbarkeit“ ist die Überbuchung nur für Mietartikel vorgesehen.

Sicherheit für die Sichtbarkeit der Buchung und Artikelsuche

Für Benutzer, die die Struktur Station/Bezirk/Region verwenden, besteht die Möglichkeit, die Anzeige des Inventars aus anderen Stationen/Bezirken/Regionen und die Buchung dieses Inventar über die folgenden Sicherheitseinstellungen zu steuern.

Launchpad-Einstellungen:

Artikelsuche – Die Ansicht ermöglicht/beschränkt die Möglichkeit eines Benutzers, Inventar auf Stations-, Bezirks-, Regions- oder Unternehmensebene anzuzeigen. Ein Benutzer hat möglicherweise Einblick in das Inventar, sodass eine Sicherheitsbuchung möglicherweise eine Überbuchung nicht zulässt. Das Inventar kann auf Mitarbeiter-, Gruppen- oder Unternehmensebene zugewiesen werden.



Transaktionen:

Eine Sicherheitsbuchung ermöglicht/beschränkt die Möglichkeit eines Benutzers, das Inventar eines anderen Standorts auf Stations-, Bezirks-, Regions- oder Unternehmensebene zu buchen. Es kann auf Mitarbeiter-, Gruppen- oder Firmenebene zugewiesen werden.



Beim Abfassen eines Vertrags beinhaltet der Überbuchungsbildschirm die gleichen Informationen, ungeachtet dessen, ob ein Vertrag im Launchpad oder im Core-Programm geschrieben wird, auch wenn sie sich optisch unterscheiden. 



Die „Maximale sichere Menge“ bezieht sich auf das, was in meiner Station verfügbar ist, aber je nachdem, wie abhängig ich von meiner Sicherheit bin, kann ich möglicherweise mehr buchen (wenn ich beispielsweise in meinem Bezirk buchen kann). Wenn dies der Fall ist, kann der Benutzer einfach unten auf die Schaltfläche „Fertig“ klicken, und die angeforderte Menge wird auf dem Vertrag überbucht. Wenn ich nicht die Sicherheit habe, so viele zu buchen, wird auf der Schaltfläche „Überschreiben“ und nicht „Fertig“ angezeigt. An dieser Stelle kann jemand mit höherer Sicherheit die Anfrage überschreiben, oder der Benutzer wird gezwungen, die Menge erneut einzugeben, wenn er keine Freigabe hat.
