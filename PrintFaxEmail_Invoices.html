Print\Fax\Email Invoices





This article will discuss the various options for delivering invoices to customers. These options include print, fax, and email. You can generate all the closed contracts at the End of the Day and send it to your customer by printing for, faxing, or emailing to the customer, depending on which option is set up in their customer record. You can preview invoices prior to sending them out.


Topics in this article include

  • Generate Invoices
  • Automatic Method at End of Day
  • Troubleshooting

 

How to get there
Expert: Program Menu > Accounts Receivable > Print\Fax\Email Invoices
Elite: Accounting > Print\Fax\Email Invoices


Generate Invoices
 


 

Last Run Date - This shows the last time the invoices were generated. This will include the date and the last batch number.
 

Payment Type - This will allow you to choose Account, Cash, or both types of customers.
 

Include Invoices Previously Billed - With this checked, all invoices previously billed in the specified date range will be listed.
 

Include Credits - With this box checked, all invoices with credits will be listed.
 

Store to Invoice - This option selects which store to print invoices from.
 

Print All Invoices - With this option selected, all invoices will print.
 

Send via Email - Checking this box will include any customers that are marked for email through their customer record.
 

Send via Fax - Checking this box will include any customers that are marked for fax through their customer record.
 

Print - Checking this box will include any customers that are marked for print or as default in their customer record.
 

Settings - This allows you to set how the invoices will print out at the end of the day. Your settings include Standard Messages, Email Set Up, Include Finance Charges, or Auto Generate at End of Day, and you can enter a message to be sent with your invoices.
 

Preview - This allows you to preview the invoices prior to sending them out. Once you have previewed them, select generate to send out the invoices.
 

Note: You can send invoices to multiple individuals by adding Accounts Payable to the Department field under the contacts tab, in the customer record.



 

Automatic Method at the End of Day
 

You must have the Mass Fax/Email Module, otherwise you will not be able to automatically print/email/fax invoices at the End of Day. (Call for a quote)
 

You will need to collect email addresses and fax numbers.
 

Each customer record will need to be set to None, Email, Fax, or Print in the Contract Mass Send To field.
(Refer to the How to Guide: Acct. Mass Print-Fax-Email Invoice Setup.pdf for additional information)
 

The recommended setup is to create a default customer and set the Contract Mass Send To field to print or email as desired.
 

Note: A default customer record is a template for all new customers created. If the field is set to print in the default customer record, then any new customer added to the system will be set to print automatically.


Next, use Database Mass Change to set ALL customers to print or email.
 

Note: If you are not familiar with this process you should not attempt this step on your own, call support for help.

 

Once your customers are set up correctly, running your End of Day procedure will automatically generate invoices by the method selected for each customer. Only contracts that were closed since your last End of Day and have a balance due at the time of running your End of Day will generate.
 

Note: Invoices use a #10 windowed envelope for invoices and contracts.




Troubleshooting
 

Not all invoices are not being sent?
 

This is only applicable if you are licensed for the Mass/Fax/Email module. When sending an invoice, Point of Rental generates a PDF. This process may take a few seconds. If your mass/fax/email settings do not allow enough time for the PDF to generate, the invoice may not be sent. To increase the amount of time, follow these steps.
 

Expert: Program Menu > Configuration > Reports and Notifications > Mass Fax/Email Settings
Elite: Gear icon > App Configurations > Reports and Notifications > Mass Fax/Email Settings
 

1. Update the email delay to 5 seconds. 



How can I confirm which invoices have been generated? 

Option 1

Expert: Program Menu > Accounts Receivable > Print/Fax/Email Invoices
Elite: Accounting > Print/Fax/Email Invoices 
 



1. Select your invoice settings.

2. Check the box for Include invoices previously billed.

3. Select Preview.



Option 2

Expert: Program Menu > Configuration > Reports and Notifications > Mass Fax/Email Settings
Elite: Gear icon > App Configurations > Reports and Notifications > Mass Fax/Email Settings


1. Select Email Setup.

2. Enter a BCC email.

 


I have confirmed not all of the invoices were sent. Do I have to resend all of the invoices to resend the ones that did not send?
 

No. You can mark the invoices as "not sent". This will prompt Point of Rental to resend the invoices at the end of the day or the next time you generate invoices. To do this follow the below steps.


Start with one to make sure you receive the expected result then move forward with the others.
 

Expert: Program Menu > File Maintenance > Transaction File
Elite: Gear icon > App Configurations > File Maintenance > Transaction File
 

1. Find the contract the invoice was generated from.
 


 

 

2. Select the record.
 

 

3. Select Functions > Modify. This option may require a high password level. If you are unable to perform this action, please contact your Point of Rental administrator. 

 


 

4. Remove the billed date that is displayed and then Press Enter.



5. Press Okay.


6. Regenerate invoices manually, or wait for the end of the day (only applicable if you are licensed for Mass/Fax).
