PORCloud v2 | Connecting to Shared Folders


 

In this article, we are going to go over how to connect to a shared folder. If you are on PORCloud v2 you are able to access some items from your local computer. Here are the steps on how to access those folders from your computer. To setup a shared folder please contact our IT Support Team at Support@pointofrental.com. 

 
 

Connecting to a Shared Folder


1. Open File Explorer (folder icon).

2. Right click on This PC from the left menu.

3. Select the Add a network location.

4. Select Next.

 


5. Select Choose a custom network location.

 


 

6. Select Next.

7. Type https://[EnterServerIndentifier].porcloud.com/[ShareName]  into the ‘Internet or network address:’ field.
 


8. Select Next.

9. Enter your domain username and password. The username will have ‘PORCLOUD\’ in front of it.



10. Select OK.

11. You can use the default name for the shared folder or create a new one.



12. Select Next.

13. Select Finished.

14. Your folder will appear in the left menu.
 


 

15. You can create a desktop shortcut by:
 

a. Right click the folder on the left menu.

b. Select Send to.

c. Select Desktop (create shortcut).
 

 

16. The QB, Pictures and Attachments folder will use the same schema of https://[EnterServerIndentifier].porcloud.com/[ShareName], replacing [sharename] with qb, pictures, or attachments as necessary.

17. The physical location of each share is below:


a. Attachments - C:\POR\Attachments

b. Pictures - C:\POR\Pictures

c. QB - C:\POR\QB

d. SharedFolder - E:\Shared_User_Files
