Syrinx® - Linked Documents




Overview: This article explains how to add documents to root folders and then link them to items in Syrinx.

 

Topics included:

  • Adding documents to Syrinx root folders
  • Adding linked documents to stock items



Adding documents to Syrinx root folders

If you have safety leaflets / operating instructions in electronic (PDF) format, these can be loaded into the system. Staff can then be prompted automatically to print the appropriate leaflets/docs according to the hired equipment.

In the Syrinx Configuration program, use the ‘Reference Folders’ tab on the ‘Document Root Folders’ screen to enter the location of the top-level folder for your safety leaflets. These folders would need to be set up in a central location on the server.
 

Your PDF files would need to be placed into one of these folders on the server, so they can be accessed by all users.

 Then use the ‘Reference Documents‘ screen to load the names of all the PDF files beneath the given root folder.

Select the root folder where the PDF file was saved to and then click on ‘Load Files’. This will load the files into Syrinx associated with the root folder.

These are now ready to be linked to the corresponding fleet items/categories and automatically printed with each new hire. Note that this feature requires Acrobat Reader to be installed on each client's PC.

You can set when these are to be printed with hire contracts and/or prompt a user to print them using the ‘Print Docs with Hire/Sales' option. This is set in Syrinx Configuration program > Documents.
 

 

Adding linked documents to stock items

To link a document to a stock item, open the stock item record and click on the Linked Documents tab.

Then click on the ‘Add’ button to add a linked document. Select the root folder where the document was saved from the drop-down menu. Then select the relevant PDF file from the list of files in that folder.

This will now link that file with this stock item. Depending on the settings selected in the ‘Print Documents with Hire/Sales’, whenever this item is included on a contract/advice note, then the document will print automatically or the user will be prompted to print or not.
