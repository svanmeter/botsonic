Convert to Base Rate


This tool allows the user to convert large numbers of items to a base rate at one time. Items can be selected by division, category, and/or item type.  ﻿

  • In the rate engine drop down, select the new base rate engine. If you want to update the rate engine on the items to this rate, you must check this box to update them, otherwise they will keep the previous rate engine assigned to them.  
  • In the Base Period, select which rental period rate you would like to set for the base rate on the items. This is most commonly the 24 hour rate. In other words, the current 24 hour rate on the item will be copied over into the base rate field on the options tab and be used in calculating rates using the base rate engine. 
  • Hit Preview. 
  • The user can then check the box to ‘Select All’ and update all items, or choose individual items to update.  
  • When selection is complete, hit ‘Go’ to update items.
