Tax Rate Change

 

    In order to change tax rates, a new replacement jurisdiction must be created.

 

1. Right-click on the tax jurisdiction you want to change and select Tax Rate Change.

 

     2. A Date Box will appear – Enter the expiration date of the old tax jurisdiction.

   

     3. After selecting the expiration date, a new tax jurisdiction will appear with a new jurisdiction number and Begin Date. Make the necessary changes to the Tax Jurisdiction. The Description and Max Tax Amount will be carried over from the original jurisdiction if no changes are made.

* Note -  You can breakdown the tax jurisdiction by city, state, county tax. This will also show as a new format on your contract listing each individual tax.

     4. Select OK.

     5. In the list of Tax jurisdictions, locate the original and right-click "Edit Tax Jurisdiction".

     6. In the top left corner, check the box "Inactive".

     7. A prompt will appear asking if you would like to update all items, customers, and job sites to the new tax jurisdiction. Select Yes.

Note* - This will not update the store default. To update the store default tax go to #9 Configurations> System Configurations> Parameters> Accounting Tab> In the drop-down, select the correct tax jurisdiction.

 

 

If viewing the original tax jurisdiction, it will show the selected end date and the new jurisdiction number in the replacement jurisdiction.

 

**** ANY PREVIOUS CONTRACT WRITTEN WILL DISPLAY A MESSAGE, LETTING YOU KNOW THE TAX RATE HAS CHANGED. **** You will have the option to keep the old tax rate on the contract or use the new tax rate. The only time the message will not be displayed is when doing auto continuation. Any contracts auto continued will continue with the new tax jurisdiction.
