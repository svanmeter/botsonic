This article will show you how to create and edit delivery, event, and general locations.


Topics in this article include:

  • Adding a new general location

  • Editing inventory's location

  • Adding a new delivery location

  • Adding a delivery location to a contract

  • Adding a new event location

  • Adding an event location to a contract



 

 

Adding a new general location
 

1. To see all locations in Essentials, go to Inventory and click on Locations.

 

 

2. Click + New Location at the top and select general location as the location type. Then, enter a name for the location. In this case, it can be Warehouse-Main Street or something similar, so you know that it is the warehouse location. You can enter an expiration date for that location as well as additional notes that will allow you to enter information about that location, for example, a gate code or additional information.

 

 

3. After you have added the location name, complete the address fields, and click Save. Address fields for general locations are not required but they are helpful for you to keep track of the physical location. You can also add an expiration date so that the location will become deactivated automatically.

 

 

 

4. Repeat for any other general locations you may have. 

 

 

Editing inventory’s location

Once all general locations are entered, go to your inventory list and edit the item that is stored at one of the general locations. From the Basic Information screen of the inventory item, select the location where this item is stored from the drop-down under Administrative Information. 

 

 

Note: If you have an off-site warehouse where you want to keep track of the inventory stored in it and you have some of the same inventory at the store, you will need two separate inventory items created as only one location can be defined per inventory item. Please refer to our Adding New Inventory Items video to walk through the process of adding new inventory items.

 

 

Adding a new delivery location
 

1. Go to Inventory > Locations. Click + New Location at the top and select Delivery Location as the location type. Then, enter a name for the location. You can enter an expiration date for that location as well as additional notes that will allow you to enter information about that location, for example, a gate code or additional information.

 

 

2. After you have added the location name, complete the address fields and click Save. Address fields for delivery locations are not required but they are helpful for you to keep track of the physical location.

 

 

 

Adding a delivery location to a contract
 

When delivery locations have been added, you have the option to select a saved delivery address when creating a new rental.

 

1. To do so, choose the delivery option and select Use Saved Location Address.

 

       

 

2. Then select the correct delivery address from the Deliver To drop-down menu that appears and continue with your new rental.

 

 

 

3. Once the delivery location is selected and you have gotten to the next step of the transaction, you will see a Delivery/Pickup Information section. When you expand this section, you will see the delivery location along with the address, if entered.

 

 

  

 

Adding a new event location
 

1. Go to Inventory > Locations and click + New Location at the top. Select Event Location as the location type and then enter a name for the location. You can enter an expiration date for that location as well as additional notes that will allow you to enter information about that location, for example, a gate code or additional information.

 

 

2. Once you have added the location name, complete the address fields and click Save. Address fields for event locations are not required but they are helpful for you to keep track of the physical location.

 

 


 

Adding an event location to a contract
 

Once all event locations are entered, you now have the option to select a saved event address when creating a new rental.

 

1. To do so, select For Event and then Use Saved Location Address.

 

 
     

2. Then select the correct address from the Event Address drop-down menu that appears and continue with your new rental.

 

 

 

3. Once the event location is selected and you have gotten to the next step of the transaction, you will see a Delivery/Pickup Information section. When you expand this section, you will see the event location along with the address, if entered.
