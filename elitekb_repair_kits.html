Repair Kits


 

This option allows you to view or select the items that make up a repair kit. A repair kit is a group of items that can be used on an Internal Repair Order. This would most likely be used with items such as air filters, oil filters, etc. When you setup a repair kit, you simply select the items you want to see when doing an IRO. You can optionally set the quantity for each item in the repair kit. The Repair Kits screen looks like the following:


 

Repair kits can be edited by right clicking on the item or anywhere in the gray box (if you are adding a new item).


 

The selection criteria presented to the operator can be managed in the type drop down.

*Note: If you don’t have the repair items in your item database, you can select a miscellaneous sales item. You will then be asked the item name. When entering the name, be sure to also enter the part number. Next it will ask the cost of the item. This amount will be charged against the Repair Cost of the item.

Repair kits can be setup on both the header and the serialized unit. When the serialized item is added to the repair order, both kits will present themselves so that the operator may choose the required items.
