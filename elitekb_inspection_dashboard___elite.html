Launchpad – Inspektions-Dashboard

Counter System>Dashboards>Inspektions-Dashboard

Dieses Dashboard ist für unsere Inspection App-Benutzer verfügbar, um den Werkstatt-Managern die Organisation und Überwachung von Inspektionen zu erleichtern. Dieses Dashboard kann nach Station gefiltert werden (oben rechts) und hat auch verschiedene Gruppierungsoptionen für die aufgelisteten Inspektionen (unten links). Mit einem Rechtsklick auf die blaue Spaltenüberschrift kann der Benutzer die Reihenfolge und Sichtbarkeit der verfügbaren Spaltenoptionen anpassen.

*Hinweis: Die Spalten für Offene Verträge, IRAs und IWAs zeigen die Anzahl der Verträge und nicht eine Vertragsnummer an. Wenn ein Gerät beispielsweise auf drei verschiedenen Wartungsaufträgen steht, wird in dieser Spalte die Zahl „3“ angezeigt.

 

Inspektionsaufträge sind durch Registerkarten getrennt, die ihrem Status entsprechen: „Nicht gestartet“, „In Bearbeitung“, „Zur Überprüfung“, „Abgeschlossen“ und „Alle“. Die Registerkarten „Abgeschlossen“ und „Alle“ enthalten auch einen Datumsfilter für eine einfachere Suche.

 

Durch Rechtsklick auf eine Inspektion kann der Benutzer sie einem Bediener, einer Gruppe oder einer Station zuweisen. Wenn sie einem Bediener zugewiesen ist, kann außerdem eine Nachricht an diesen Bediener gesendet werden. Es kann eine numerische Priorität von 0-4 eingestellt werden, wobei 4 die höchste Priorität ist. Diese entsprechen den Prioritäten innerhalb der App.
