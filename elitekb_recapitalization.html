Recapitalization



Recapitalization allows you to add value to an existing serialized rental item and then separately capitalize the increase. You can manually add the improvement line in the Item Record under Purchase Details or you can write an IRO for the unit. By writing an IRO, the cost of the improvement will post to the General Ledger and adjust COGS and the appropriate sales asset account.

For Elite Only

Once a repair order is closed, users can add the repair cost to the item's value. If the repair cost is added as an improvement, a purchase line for the value of the repair will be added to the rental asset. It can then be depreciated. Within Security, under the ‘Transactions’ area of the security grid, controls who can and cannot do this.


For Expert Only

For repairs valued at $1000 or higher, users can add the repair cost to the item's value. This option appears after the repair order is closed. If the repair cost is added as an improvement, a purchase line for the value of the repair will be added to the rental asset. It can then be depreciated.
