Resetting Point Static IP Address

 

Configure Point Credit Card Static IP Address for terminals for POR Enterprise:

 

Go to Counter System>Tools, in the upper left corner of the screen.

Select Printer Set Up

Go to Advance Tab – Make sure there is a terminal assigned under Verifone Terminals.

 

Check to see if there is a Static IP address assigned.

 

Go to Security>Verifone Terminals.

This will display the terminal and IP Address (if assigned) if not go to:

 

Go to  Start bottom left corner

Use the command prompt and ping to find an unused static LAN; make sure it is not in the DHCP range.

 

From the credit card terminal device.

 

Press 1+5+9, then 166832 to login.

Navigate to Administration > Communication > Network > Ethernet.

Click ‘Mode’ and change to ‘StaticIP’ and click OK.

Configure the static IP address, netmask, gateway, DNS1, and DNS2 addresses.

Click the Apply button in the upper right.

Click the ‘Home’ link in the upper left then click the ‘Reboot’ button. Run all POR updates

 

We should be able to get the IP FROM POR.
