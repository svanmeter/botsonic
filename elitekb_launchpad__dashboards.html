Launchpad - Dashboards

From the home screen of your Launchpad software, you can assign different tabs add widgets to the tabs summarizing information for viewing such as (Income, Expenses, Sales, Inventory, Transactions).

 

Selecting the black + box will open a screen allowing you to select how you would like to add a dashboard.

From Dashboard settings, select Edit Dashboard from the drop down menu.

The Dashboard tab will be highlighted, allowing you to change the name of the tab. Select Save at the bottom right corner of the screen.

To add widgets to a dashboard, click here to see document on Adding Widgets

 

Search

Different dashboard will have extended features to help define the information you are looking for. ex. Search box, Chart Flows, etc.

In the example below, you can see in the customer dashboard, you have an Advanced Search feature available when selecting the orange arrow.

When selecting the Advance Search, a new screen will appear allowing you to redefine your search by ( Key, Status, Type, Restrictions, Cust Name, Cust Number or Drivers Lic).
