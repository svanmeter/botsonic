Creating an RDP
 

To create a Remote Desktop Connection, follow these steps:
 

1. Type remote in the taskbar search box.

2. Click on the individual search result for the Remote Desktop Connection app.

3. Click the Show Options button.

4. Click the Save As button.

5. Choose a location where you want to save the remote desktop connection.

6. Enter a name and click the Save button.



Configure an RDP


The following list outlines the options utilized for the recommended configuration of the Microsoft Terminal Server Client. Remote stores connecting to the server over VPN (virtual private network) or through Port Forwarding/Triggering are limited by the internet Upload and Download speeds at both the store with the server and the store with the PC. The visual experience of the recommended settings outlined will not resolve unstable internet service or faulty equipment but can allow stores with bandwidth constraints to possibly have a better user experience.

 

1. Right-click on your remote full desktop icon.
 

2. Click Edit.

3. This will bring up the remote desktop connection settings window.



4. Click Show Options to bring up all available settings for the icon. 



5. Click the Display tab.



6. Under Colors select High Color (16 bit). 



7. Click the Local Resources tab.



8. Under Keyboard click the drop-down menu and select On this computer. 

9. Under Local Devices and resources uncheck Printers. 
 

10. Click the Experience tab.



11. Under Performance click the drop-down menu and select Low-speed broadband (256kbps – 2 Mbps).



12. Click the General tab and click Save or Save As if you would like to save this icon somewhere else.
