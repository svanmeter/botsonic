Custom Filters

This article explains how to create and use custom filters in the system.

Topics included in this article

  • Creating custom filters
  • Applying custom filters
  • Managing custom filters

Creating custom filters

Custom filters allow you to personalize your data views according to specific criteria.

  1. Navigate to the section where you want to apply a filter, such as Inventory or Transactions.
  2. Click on the Filters button and select Manage Filters.
  3. Choose Add New Filter, then specify your filter criteria and name the filter for easy recognition.

Applying custom filters

Once a custom filter is created, it can be applied to view the data as per the set criteria.

  1. Go to the relevant section where your custom filter is applicable.
  2. Click on the Filters button and select your custom filter from the list.
  3. The data will be displayed according to the criteria specified in your filter.

Managing custom filters

You can edit or delete custom filters as needed.

  1. Access the Manage Filters option from the Filters button.
  2. To edit a filter, select it from the list and make the necessary changes.
