Overview

The below process outlines how you can upgrade the Syrinx Mobile Web Service to the latest version.  This procedure is only applicable if you are running Syrinx version 4.35 or later.  If you are running an earlier version of Syrinx please contact Syrinx support on 0118 956 9577 or email uksupport@pointofrental.com.  You can find your Syrinx version by clicking on the symbol in the upper left corner of the Syrinx client:

Download and Run the Syrinx Android Web Service Installer

 

The current version of the Syrinx Mobile Web Service can be downloaded from the following location:

http://www.syrinxusers.co.uk:443/Syrinx%20Mobile%20Service/SyrinxMobileServiceInstaller.exe

Once downloaded, run the installer SyrinxMobileServiceInstaller.exe and press the “Install” button when prompted and then press “Next” on the subsequent installation screen:

 

 

The installer should detect that there is an existing installation of the Syrinx Mobile Web Service.  Press “Next” to begin the upgrade process:

 

 

Next, verify the connection details for the Syrinx database:

 

 

Finally, click the “Next” button to begin the installation:

 

 

The installer will make the appropriate system changes and advise when finished.  Click the “Finish” button to complete the installation:

 

 

Note: The Syrinx Mobile Web Service Installer can only upgrade the main instance of the syrinx mobile web service site.  If multiple versions of the site exist on the web server, the files in those sites may have to be manually updated (i.e. copy the non-site specific program files from the site that was successfully upgraded to these additional copies of the site)

Test Internal Connectivity to the Web Service

 

Now test the web-service out and see if it works.

 

First you need the IP address of the machine you installed the web-service on.

 

Go to:

  Start Menu

  Type CMD

  Select Command Prompt 

  Type ipconfig 

  Note down the IPv4 address      

 

Now open Internet Explorer (or whatever browser you are using) and type the following address into the address bar (replacing the red text with your IP address):

 

http://your_IP_address/SyrinxAndroidService/api/Syrinx/Checkconnection

**Alternatively, you can use the URL http://localhost/SyrinxAndroidService/api/Syrinx/Checkconnection but this connection string will only work on the web server and cannot be used for external connections.

 

If you get the following pages up (Depending on browser), it means your web-service is running correctly:

Test External Connectivity to the Web Service

 

Test the connection to the upgraded Web Service from one of the Syrinx Mobile apps on a device.  If the users are not able to connect and are receiving the following message on their devices it may be necessary to restart the web service’s application pool:

 

 

You can restart the application pool via Start\Control Panel\Administrative Tools\Internet Information Services (IIS) Manager.  You simply need to select the “Application Pools” entry and then select the “Syrinx Android” application pool on the right side of the screen:  

 

 

Then select the “Recycle” option in the “Actions” pane of IIS:

 

 

If users are still unable to connect please contact Syrinx support on 0118 956 9577 or email uksupport@pointofrental.com .
