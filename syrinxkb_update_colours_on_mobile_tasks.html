Syrinx® - Mobile Tasks Colour Changes

 

 

 

 

Overview: This guide is designed to help you with customising the colours of the Mobile Tasks in the Driver & Workshop Schedulers.


 

Here's how to get there:

Syrinx Configuration > General > Mobile > Mobile Task Types > Colours  

 

 

Using the little arrows at the bottom of the window, select the Job type you would like to change the colour of. For example, Record 5 of 31 is Job Type Hire Contract Delivery.

 

From the below link, select the colour required and populate the App & Scheduler – Task Appearance fields by typing in the colour name exactly as it is presented in the colour table. Note: Not all mobile devices recognise the colours from the HTML table and may stay the same even if set to change.

 

https://www.w3schools.com/colors/colors_groups.asp
