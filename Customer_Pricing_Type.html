Customer Pricing Type




Overview

The Customer Pricing Types allow you to group customers based on discounts given.


Topics in this article include

  • Creating Pricing Types
  • Assigning Pricing Types

 

Here's how to get there
Expert: Program Menu > Configuration > Customer Configurations > Customer Pricing Types
Elite: Gear icon > App Configurations > Customer Configurations > Customer Pricing Types


Creating Pricing Types
 

1. Select Add.

2. Use the dropdown menu to select a Commission Level.

3. Select OK.

 



Assigning Pricing Types
 

You can set up generic types such as Best Customer, Good Customer, or Linen Discount. Once the Customer Pricing Type has been defined, simply select the desired type from the drop-down box in the customer’s record. You can then set a special price on an item record and use the appropriate Pricing Type. Any customers set to that Pricing Type will then automatically get that special rate when renting the item.


To set the commission level for the customer type, select the description line, and from the drop-down menu, you can select the type of commission.
 

Note: If you want to remove a Customer Type, click on the item description, and mark the box as inactive.
