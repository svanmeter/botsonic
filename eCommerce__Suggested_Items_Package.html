Overview

 

Suggested Items Package Records are helpful in several ways:
 

  • To create a sleek and sophisticated website, making it easy for your customers to find what they are looking for and submit their quote request.
 
  • To save time when you receive new inventory: if you have 50 items that have the same suggested items, you could create 1 suggested items package, then assign the suggested items package to the 50 items as a package. 6 months from now when you want to add a new suggested item to those 50 items, all you need to do is add it to 1 record instead of all 50.
 
  • To help your staff: they can be beneficial to your counter staff when searching for inventory and extremely beneficial to new employees learning your inventory.


Step-by-Step

 

1. Create Suggested Items Package Record
 

  • We recommend that you create a new category in which to create these new package records. This will help you keep things clean, and once all your records are created, the categories can be renamed. (Configuration -> Item Configurations -> Item Categories)
     

  • The item type should be ‘Rental – Package’.
     

  • Assign a commonsense key, like “ORANGELINENSPKG”.
     

  • Determine whether you want this item to show in the inventory selection screen when searching for items in POR. If you do not want it to show, mark the item ‘No Print On Contract’.
     

  • Assign 9,999 to the Quantity field to indicate that this is not an actual inventory item.
     

  • Be sure that the Header Key field is blank.
     

  • If you are using Global Headers set the Home and Current Location to ‘000’. Otherwise, create this record for each store. (Global Headers make this process much easier).
     

  • Assign a category to this item. (Linens)

    Example: Entering a Suggested Items Package Record


 

2. Build the suggested items list for the package item
 

  • Access the new Suggested Items Package record via File Maintenance --> Item File.
     

  • Select TOOLS at the top of the screen, then select SUGGESTED ITEMS.
     

  • Right click in the window and select New.
     

  • Starting with the first line, right click or double click the grey box in the item key column.
     

  • If you use Global Headers insert the Global Header. If you use Store Headers insert the Store Header. If you do not use Headers insert the actual item.
     

  • Enter all items required to build this suggested item package. Be consistent in the order of the items selected. (i.e. Round, Square, Rectangle etc…), as the order you enter them is the same order used to list them on the website.
     

  • Make sure the Special Box is not checked.
     

  • Extend the number of suggested (packaged) items if necessary:
     

  • If you have more than 17 items to add:

    • Assign the first 16 individual records to the main package record’s Suggested Items list.

    • Proceed to Step 3.
       

  • If you have 17 or fewer items to add, add them all and skip Step 3.

    Example: Orange Linens




    3. Determine if it is necessary to hide items on your website
     

  • If you do not have an integrated website you can skip this step.
     

  • If you do not want the individual items to list on your website on the category page, and only want the packages to show, you must do 1 of two things:

    • Mark each actual item ‘Hide on Website.’

    • Or, during the web update process you can choose to not upload the category where the individual items reside, if it’s different from the category holding the package records.

    • The Ecommerce (formerly known as RentalHosting) Team can help you determine the best way to accomplish this.
       

Remember, this is one example of creating Suggested Items Packages. Depending on your setup and situation, slight adjustments to these instructions may be necessary. This procedure can be modified to accommodate most situations. Contact your Web Design Team for more information.
