Task List & Reminders



This article will discuss the task list and reminders within Point of Rental's software. The Point of Rental system offers useful management tools such as task management and reminders. These tools are especially useful for instances when a team member needs to remember a task/piece of information or remind another team member of a task/piece of information. 


 

Topics in this article include
 

  • Getting there
  • Reminders
  • Reminder definitions
  • Tasks




Getting there

Here's how to get there
There are many places to get access to reminders. 

Expert: Counter System > Task List/New Reminder
Program Menu > Task List
Expert/Elite: Contract Screen > Actions > Reminder
Expert/Elite: Customer Dashboard > Task >  Create Reminder
Expert/Elite: File Maintenance > Customer File > Tasks > Create Reminder

Reminders

From the Reminder page, you can create a reminder using Point of Rental's internal task list, Microsoft Outlook, Microsoft Calendar, or Google Calendar. The reminder can be created yourself, for an employee, an employee group, or for anyone with an email account. When you select reminders, you will get the following screen:



Reminder definitions

Reminder Type - This is the method you will use to send the reminder. These are Task List, Outlook Calendar, Microsoft Calendar, and Google Calendar.

Date/Time - This is the date and time you want the recipient to receive the reminder.

Send To - This is where you choose the reminder recipient: Yourself, Selected Employees, Another Employee, Employee Group, and An email address. Another Employee and Employee Group are both pull-down menus. When using the Selected Employee option, after you select an employee and click on the OK button, a list of employees for you to choose from will appear.

Reminder - This is the subject line and the body of the reminder.


Tasks

The To Do Task feature allows tasks to be assigned, modified, and completed by yourself or other employees. The tasks can be for one time or periodically such as daily, weekly, monthly, or yearly.



Group - You can select different group types (Office, Drivers, etc.). You can also select an individual or select individuals to send the reminder to.

Store - From the drop-down menu, you will have the option of selecting all stores or an individual store.



Contract - Here you can enter the number for the contract associated with this task.

Description - Here you can enter a brief title or subject line for the task.

Due Date - You can manually enter or use the calendar icon to select the date on which you would like the task completed.

Notify when completed - Checking this box will allow you to be informed via a Point of Rental internal email message upon the completion of a task.

Frequency - This dropdown allows you to set the recurrence rate for this task.

Task - Here you can enter in-depth information about the task within the text field.

Once all of these fields are completed, select OK and the task will be created.


Note: When you have a task or reminder that is due, a BOLD RED link will appear in the bottom right corner of the Counter System Menu (Expert only).
