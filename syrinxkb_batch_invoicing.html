Syrinx® -  Creating a Batch of Invoices

 

 

 

 

Overview: This guide is designed to help you with creating a batch of sales invoices for account customers.

 

 

 

 

To create a batch, you will need to go into Invoicing and then click on Account.

 

You will then be faced with the following screen.

 

You need to establish when you want to invoice up to and the physical invoice date. You then need to decide what you want to invoice by ticking the relevant boxes. You have the option to include or exclude customers.

 

• The notes box can be used for invoicing one contract. You would type in their Contract: 02-1234 and then the system would only invoice that one contract.

• Bottom Title is for notes that appear on every invoice, for example, “Merry Christmas.”

• Email File - this can be used when invoices are being sent via email. If you wanted to send an attachment with all the invoices you would do the link here. This can also be set up in the configuration if you send the same attachment each time.

• Sales - you can run the batch on a sales person. If this is left blank you can still find the information by running an invoice report by sales person.

 

Once you have made your selections you will need to click Confirm.

 

This has now generated a batch of invoices. You will see the record count below. We would normally advise you to click on the report button at the bottom. This will give you a batch report detailing all of the invoices in this batch. Once it has printed you can close out of the batch screen and go through the report.

 

If you need to make any changes to the invoices, we would advise that you cancel the invoice batch, go and amend the hire/sale and then re-do your batch. There is an edit button where you can manually make changes. These changes will not reflect in the contract.

 

You can also exclude an invoice from the batch. Doing this will remove it completely and will reappear on the next batch.

You can manually cancel a line from an invoice as well. Please note if you do cancel a line it will NEVER be invoiced.

 

Once you have made your alterations and you are ready to print/email your invoices, click on Review/Print.

 

Your batch will open again. You will need to click Print followed by Email Invoices (there are separate instructions for setting the customer up to receive invoices via email).

 

Once you have printed and emailed your batch is ready to be confirmed. To confirm the batch you need to go into invoicing and then Confirm Batch.

Once a batch has been confirmed the invoices can be posted to your accounts. Invoices cannot be changed only credited.
