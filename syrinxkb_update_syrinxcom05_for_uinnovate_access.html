Introduction

Syrinx Users will no longer be able to access the uInnovate ideas board via their own web browsers from Dec. 6th 2022 when single sign-on is enabled and the ideas board is made private.  From this date, uInnovate will only be accessible from the dedicated launch button in Syrinx clients running version 4.40.6 or later:

User running Syrinx versions 4.39.4 through 4.40.5 can follow the below procedure to add this functionality to these earlier Syrinx versions if they are having issues connecting to uInnovate.  Users running Syrinx version 4.39.3 or lower should upgrade to the latest version if they wish to continue accessing the uInnovate ideas board.

Update SyrinxCOM05 dll Files via UpdateCOM05.EXE Utility

Users can download the below utility, UpdateCOM05.EXE, that will replace the Syrinx program files used when connecting to uInnovate:

 

http://www.syrinxusers.co.uk:443/download/Syrinx%20Fixes/UpdateCOM05.EXE

Username: Download Password: c0nc3pt 

 

**Should you be unable to download the .exe version of the file due to restrictions in your web browser, a zipped

version is available for download at the below link:

 

http://www.syrinxusers.co.uk:443/download/Syrinx%20Fixes/UpdateCOM05.zip

Username: Download Password: c0nc3pt 

 

After downloading the executable, close any running Syrinx v4 clients on the PC and then run the UpdateCOM05.EXE file and reply “Yes” to the prompt to overwrite the Syrinx COM05 dlls:

 

Manually Update SryinxCOM05 dll Files

Should there be any issues with the UpdateCOM05.EXE executable such as it being blocked/quarantined by the PC’s anti-virus software.  Copies of the Syrinx COM05 dlls are also available on the download site:

 

http://www.syrinxusers.co.uk:443/download/Syrinx%20Fixes/SyrinxCom05.zip

Username: Download Password: C0nc3pt 

 

After downloading the zip file and prior to extracting the COM05 dll files from it, right click on the .zip file and select “Properties”.  If there is an “Unblock” option on the “General” tab of the file’s properties window, this should be ticked to prevent the O/S from blocking the execution of those files:

 

After unblocking the files, close any running Syrinx v4 clients on the PC and then extract the four dll files held in SyrinxCOM05.zip and copy them to C:\ProgramData\Higher Concept\SyrinxSP v4\Com\Com05 and overwrite the existing files in that directory.

 

Verify uInnovate and Esign Functionality

After replacing the Syrinx COM05 dlls, users should verify that the Syrinx client shortcut opens the uInnovate site correctly within the POR Browser.  Esign/Countersign users should also test that functionality on the PC as the Syrinx COM05 dll also controls Syrinx’s interaction with these add-on modules.
