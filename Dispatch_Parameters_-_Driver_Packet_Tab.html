Driver Packet Tab

Here's how to get there
Expert: Program Menu > Counter System > Dispatch Center > Options > Parameters > Driver Packet
Elite: Dispatch Center > Options > Parameters > Driver Packet


 

Route Packet Options

This allows you to set the criteria for how the driver’s packets will be viewed and printed through the dispatch center. Checking the appropriate boxes will activate this feature.

 

Trip Sheets

You can select the information to be printed out on the driver’s trip sheet. From the drop-down menu, you can select either by Stops, Stops & Contracts, or Stops & Contracts & Items.

 

Contract Formats

There are many different contract formats to choose from. You can set different formats for delivery, pickup, and service contracts and set the number of copies to be printed. 
 


 

Manage Pre-Packets

This allows you to add a PDF or RTF file attachment to the Driver Packet when printed. When you select the button, you be taken to the POR Drive:\POR\Map\PrePacket folder. To add a file, copy and paste the file into the folder you want to attach.
 


 

 

When printing the driver packet, you will have the option to include the pre-packet documents.
