Item Rate Variance

 

This report will show you the difference between the price that you charged on a contract and the published rates for that item. You can set up 3 groupings to generate this report, including Customer Name or Salesman.

This is a drill-down report so using your mouse you can double-click on the Status in blue to get more detailed information.
