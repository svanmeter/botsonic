Income by Zip Code Chart

 

The Income by ZIP Code Chart prints a Pie Chart showing the breakdown of customer income by ZIP Code for the highest income ZIP codes.  The percentage of income by ZIP Code is also displayed.  You can select Current Year-to-Date, Last Year, or Lifetime income.  Note:  The Last Year Income field was added with Revision 7.0 in July 2003.  Until you go through a fiscal yearend, there won’t be any values in Last Year Income.
