This article will go over creating and updating variant items. Variants are different versions of the same item that have different attributes, such as size, color, or style. You can add variants to items in Rental Essentials to offer more options to your customers and track your inventory more accurately.

If you are updating an item that already had stock assigned, this stock will either need to be sold or removed from the item record prior to assigning Variant Properties.



Topics in this article include

  • Adding Variant Properties
  • Adding Stock


 

Adding Variant Properties
 

1.  Go to the Inventory page and select the item you want to add variants to by clicking on the edit Icon.
 

 

2. Click on the Variants tab.
 




3.  Click on the Make Item Variant button, if applicable then Create variant property.
 



4. Enter a name for the variant and choose an attribute from the drop-down menu. You can also add a new attribute by typing it in the field and clicking on the Add button.
 



5. Enter a value for the attribute and click on the Add button. You can add more attributes and values by repeating this step.

6. Click on the Save Changes button at the bottom of the page to apply your changes.


 

Adding Stock


When you are adding stock to the item you will be able to choose the variant property that you would like that stock item to go into.
