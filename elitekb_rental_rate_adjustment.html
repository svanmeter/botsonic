Einrichten von Mietpreisanpassungen

 

 

 

Übersicht

 

Mit der Mietpreisanpassung können Sie Mietpreisanpassungen für Mietartikel einrichten und planen.

Enthaltene Themen in diesem Artikel:

 

  • Mietpreisanpassung
  • Filtern von Daten durch Importieren und Exportieren von CSV-Dateien

 

 

Voraussetzungen

 

Die Mietpreisanpassung ist für Massenänderungen von Mietpreisen gedacht. Wenn Sie den Tarif für einen einzelnen Artikel anpassen möchten, können Sie dies über die Artikeldatei in der Dateipflege tun. 

 

Mietpreisanpassung

 

1. Gehen Sie zu Datenbankverwaltung > Datenverwaltungstools > Mietpreisanpassung. Sie werden aufgefordert, Ihre ID und Ihr Passwort einzugeben, wenn Sie Datenverwaltungstools aus dem Menü auswählen. 



2. Wählen Sie unter Auswahl die Station und die Kategorie(n) aus, für die Sie Preisänderungen planen möchten.

 

Wählen Sie alle Stationen aus (im Gegensatz zu einer einzelnen Station):

Wenn Sie alle Stationen auswählen, wird der Sammelartikel mit dem neuen Preis aktualisiert. Jeder Datensatz, der keinen Sammelartikel hat, wird ebenfalls mit dem neuen Preis aktualisiert. 

Wenn Sie eine bestimmte Station auswählen::

Wenn Sie den Preis im Sammelartikel aktualisieren, wird für die betreffende Station ein neuer Sonderpreis in den Sonderpreisen der Sammelartikel erstellt. Wenn Sie einen Artikel haben, der keinen Sammelartikel hat, wird der Artikeldatei ein Sonderpreis hinzugefügt (der Preis des Artikels wird nicht aktualisiert).

 

3. Sobald Sie diese Auswahl getroffen haben, klicken Sie auf die Schaltfläche „Daten laden“. Es wird eine Liste von Artikeln angezeigt. Markieren Sie das Kästchen neben den Artikeln, die Sie ändern möchten. Durch einen Doppelklick auf „Auswählen“ können Sie alle Kästchen entweder an- oder abwählen.



4. Als nächstes geben Sie den Prozentsatz der Preiserhöhung ein und ob er gerundet werden soll (z. B. auf den nächsten Euro).

 

5. Legen Sie das Datum fest, an dem die Preisänderung in Kraft treten soll. Wenn Sie nicht möchten, dass sich die Preise für zukünftige Reservierungen ändern, markieren Sie auch dieses Kästchen. Für diese Reservierungen wird ein spezieller Preis festgelegt, um zu verhindern, dass sich die Preise aufgrund der neuen Preise ändern. Wenn Sie dieses Kästchen nicht ankreuzen, werden die Reservierungen bei der nächsten Änderung aktualisiert.

 

Wenn Sie sich für einen Upload für ein zukünftiges Datum entscheiden: 


Nur die Preise des Sammelartikels (Station 000) werden in der Nacht vor dem geplanten Datum nach der Tagesendverarbeitung in der Artikeldatei aktualisiert. Alle artikelspezifischen Preise bleiben als Sonderpreise erhalten und werden nicht in die Artikeldatei geladen. Diese neuen Preise werden in der Tabelle der Sonderpreise unter der Artikelnummer gespeichert. 


6. Wenn es sich nicht um eine dauerhafte Preisänderung handelt, deaktivieren Sie das entsprechende Kontrollkästchen und legen Sie ein Enddatum fest. 


7. Wählen Sie „Preise aktualisieren“. Bei den aktualisierten Artikeln wird nun „Aktualisiert“ angezeigt. 




8. Wenn der Benutzer eine zweite Mietpreisänderung für denselben Artikel und denselben Zeitraum vornimmt, hat die zweite eingegebene Mietpreisänderung wird den Vorrang übernehmen. 

 

Filtern von Daten durch Importieren und Exportieren von CSV-Dateien 

Es gibt auch eine Import-/Exportfunktion. Der Benutzer kann die Daten filtern, die er anpassen möchte, und sie dann in eine Tabellenkalkulation exportieren. 



Die Preise können in der Kalkulationstabelle bearbeitet und gespeichert werden. Sobald der Benutzer „CSV importieren“ wählt, erhält er eine Liste, aus der er auswählen kann.



Geben Sie nach dem Laden an, wann die Preisänderung erfolgen soll und klicken Sie auf „Preise aktualisieren“.
