Depreciation Set Up

 

 

Setting Up Depreciation

 

From Program Menu> System Configurations> Parameters> Accounting Tab

a. Capitalize Extra Charges: If they want to capitalize on Extra Charges, this box will need to be selected. If they use the PO Module, the extra charges are Shipping, Tax, and Misc which will be charged to the asset account and not expenses (Meaning, the extra charges are capitalized with the purchase price). If they don’t use the PO Module, the extra charges will need to be manually entered.

b. Included Extra Charges: If they want to include the extra charges into Inventory Value, this box will need to be selected.

c. Depreciation %: This is only used if the Depletion method is chosen.

 

From Program Menu> File Maintenance> Item File> Asset Tab

a. Capitalized box will need to be checked to capture depreciation.

b. Select the Depreciation button.

c. The first schedule is the only one that will be part of the General Ledger entry.

d. From the drop-down menu, select the depreciation method.

e. Salvage Value: How much the item will depreciate too. If it’s depreciating to $0, leave it alone.

f. Additional Depreciation: Holds extra depreciation taken outside of the normal depreciation. i.e. accelerated depreciation (Section 179).

g. Equipment GL Accounts: Equipment Inventory Account outside of the general Equipment Asset Account.

h. Current Month Depreciation: The amount the item will depreciation that month.

i. Total Depreciation Accrued: The total amount the item has depreciated.

j. Book Value: The value of the asset.

k. Accumulated Depreciation acct: The Depreciation account outside of the general Depreciation account if it is broken out into more than one account.

l. Recompute: Will re-compute the depreciation based on any changes that were made to the item’s purchase detail and/or depreciation schedule. (This is not normally done. Re-computing will create changes in your GL accounts for previous periods.)

 

- Depreciation will need to be set for each serialized item. You can set the schedule on the Header record but that is only for reference.

- Improvement: This feature adds more value to the item and depreciates the expense. This can be done through an IRO if the total repair cost is more than $1,000.01 (Tools> Add as Improvement) or by right-clicking anywhere below the purchase details within the Purchases tab of the serialized item then selecting Add Improvement.
