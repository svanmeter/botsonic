Migrating from Mozy to Carbonite for Offsite Data Backup

 

Carbonite, an offsite data backup service provider, acquired Mozy, Inc. from Dell Technologies in May of 2018. What does this mean to current Mozy customers who use Mozy to back up their most critical Rental Expert data offsite? It means transitioning to Carbonite is mandatory to maintain valid backups after March 30th, 2019. That’s when Mozy will cease to backup selected files within the Mozy client application installed on servers and computers. If you never installed Mozy then you can disregard this article.

 

For more details on the acquisition see the following link, https://www.carbonite.com/news/article/2018/03/carbonite-closes-acquisition-of-mozy-inc.-from-dell-technologies.

Carbonite’s transition plan from Mozy includes:

1Automatically installing Carbonite on the same server or computer Mozy is installed on (does not include automatically selecting proper files to backup)
2Requiring the Carbonite account owner (same owner as of the Mozy account) to register with Carbonite
3Requiring the Carbonite account owner to manually select which files to backup
4Requiring the Carbonite account owner to schedule the desired backup routine
5Requiring the Carbonite account owner to uninstall the Mozy client. Note: the Mozy backups should remain available for restore for 30+days. Contact Mozy support for further details
 .

 

Automatically Installing Carbonite on the Same Server or Computer Mozy is Already Installed on

In the system tray on the taskbar, you should find both the Mozy icon along the new Carbonite icon. If you see both the Mozy and Carbonite icons then you should perform the steps detailed in this article

Note: The Carbonite backup client installed is considered the “computer” version rather than the “server” version. The server version requires a Java client and consist of a file explorer window to select which files to backup. For more information, contact Carbonite support.

 

Requiring the Carbonite Account Owner Register with Carbonite

Double-click the Carbonite icon in the system tray. A Carbonite window opens. Click the My Account link in the upper right, then click the Update your account information link. A web browser for a Carbonite login appears. Log in using the same credentials used for your Mozy account. Continue through the Carbonite account activation process.

 

 

 

 

 

Requiring the Carbonite Account Owner to Manually Select Which Files to Backup

Mozy customers will find Carbonite automatically installed a client, but it requires you to manually select files to backup, i.e. the critical Rental Expert files. By default, Carbonite backs up commonly known files that exclude Rental Expert/Elite data such as databases, attachments, statements, etc. We recommend unselecting the default files and folders that Carbonite automatically selects. To do so open any Windows File Explorer windows and navigate to Computer or This PC

 

 

Next, all Carbonite installs require a manual selection of Rental Expert files to backup. See the article named Which Point of Rental Files to Backup for a list of the directories we recommend for backup.

 

Requiring the Carbonite Account Owner to Schedule the Desired Backup Schedule

By default, Carbonite backs up data continuously. Depending on the size of data being backed up and the speed of the Internet a different backup schedule may be desired. Failure to do so could result in slow performance during open hours.

To set the backup schedule, launch Carbonite by double-clicking the icon in the system tray. Click the Settings & controls link. Under the ‘Schedule’ section configure the desired backup routine, i.e. Back up once a day starting at 11:30 PM.

 

 

Requiring the Carbonite Account Owner to Uninstall the Mozy Client

Uninstall the Mozy client like any other application on a Windows server or computer. Please note, the Mozy account is still active and be logged into at https://mozy.com where backups can still be restored. Technically, this is not required but the Mozy client will cease to backup as of March 30th, 2019.

In Windows 7 PCs and Windows 2008 R2 servers navigate to Control Panel > Programs and Features. Next, select MozyPro from the list of programs and click Uninstall.

In Windows 8 PCs and Windows 2012 R1 servers, navigate to Apps & Features. Next, click on MozyPro and click Uninstall. Follow the uninstall prompts to complete the uninstall.
