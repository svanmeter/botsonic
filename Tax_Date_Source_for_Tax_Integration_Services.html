Tax Date Source for Tax Integration Services



 

Overview

Rental Elite allows you to change the tax date source both when getting a quote and when posting tax for tax integrations. You have the option to select the contract open date or the current date. This article will walk you through how to change these settings under the Parameters Accounting tab. 

 

1. From the Elite Configuration menu, go to App Configurations > System Configurations > Parameters > Select Store 1 > Accounting Tab.

 

2. From the drop-down menu labeled “Sales Tax Calculation Method,” confirm the correct accounting integration has been selected.

 

3. Select the square button to the left of the drop-down menu. This will display the window for Sales Tax Service Configuration.

 

 

4. In the Tax Date Source section, navigate to the drop-down menu labeled “When Getting Quote.” Select either Contract Open Date or Current Date.

 

 

5. From the drop-down menu labeled “When Posting Tax,” select either Contract Open Date or Current Date.

 

 

6. Select OK.
