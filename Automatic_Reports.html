Wie Sie Automatische Berichte einrichten und hinzufügen

Navigieren Sie zu Programm-Menü > Konfigurationen > Berichte und Benachrichtigungen > Automatische Berichte.  

 

Hinweis: Sie benötigen das Massen-/Fax-/Email-Modul, um Automatische Berichte erstellen zu können.

 

Zusätzlich zu den voreingestellten Tagesendberichten können Sie jeden beliebigen Bericht auswählen, der in den von Ihnen gewählten Intervallen gedruckt werden soll. Das Programm enthält zusätzliche Berichte, die unterhalb der voreingestellten Berichte aufgelistet sind. Sie können jedoch Berichte hinzufügen, kopieren oder löschen, indem Sie mit der rechten Maustaste auf die Liste der Berichte und dann mit der linken Maustaste auf die entsprechende Auswahl klicken.




Wenn Sie einen der Berichte in der Liste auswählen, erhalten Sie das folgende Menü:



Testbericht: Sie können auf diese Option klicken, um den Bericht auf dem Bildschirm anzuzeigen.

Parameter neu laden: Konfiguriert die Berichtsparameter neu.

Beschreibung: Dies ist der Name des Berichts.

Station: Der Standort, für den der Bericht erstellt wird.

Dateiname: Dies ist der Name der Datei in Crystal Reports.

Benutzerdefiniert: Wenn Sie diesen Bericht in Crystal Reports erstellt haben, dann markieren Sie dieses Feld.

Drucken: Wählen Sie aus, wie oft der Bericht gedruckt werden soll. Sie haben die Wahl zwischen Nie, Täglich, Wöchentlich, Monatlich, Vierteljährlich und Jährlich.

Drucker: Wählen Sie einen Standarddrucker aus, auf dem der Bericht ausgedruckt werden soll.

Datei speichern: Wählen Sie aus, wie oft der Bericht als PDF-Datei gespeichert werden soll. Sie haben die Wahl zwischen Nie, Täglich, Wöchentlich, Monatlich, Vierteljährlich, Jährlich, Zweimal im Monat und Bei Bedarf.

Dateiname: In den meisten Fällen können Sie dieses Feld leer lassen; die Datei wird in Ihrem Ordner [x]:\por\End of Period\ gespeichert. Sie können den Speicherpfad und den Dateityp festlegen, indem Sie den Dateinamen mit einer entsprechenden Erweiterung angeben.

Beispiele: pdf, csv, xls, htm, xml, doc, rtf, rpt, tab, txt, iff. Googeln Sie eine Erweiterung, um weitere Informationen über den Dateityp zu erhalten.

Ein Dateiname ohne Erweiterung wird standardmäßig als pdf angezeigt.

Dateiname.pdf (erzwingt den Dateityp, unter dem gespeichert werden soll) [x]:\meine dokumente\pdf-dateien\dateiname.pdf (wenn Sie einen Pfad definieren, müssen Sie auch den Dateinamen und die Erweiterung festlegen). Wenn Sie den Bericht ohne die Berichtsnummer und das Datum im Dateinamen speichern möchten, geben Sie außerdem einen senkrechten Strich zwischen dem Berichtsnamen und dem Format ein, in dem Sie den Bericht speichern möchten, z.B.: Kontozusammenfassung|pdf 

E-Mail: Wählen Sie aus, wie oft der Bericht per E-Mail verschickt werden soll. Sie haben die Wahl zwischen Nie, Täglich, Wöchentlich, Monatlich, Vierteljährlich, Jährlich, Zweimal im Monat und Bei Bedarf.

E-Mail Adresse: Geben Sie die E-Mail-Adresse ein, an die Sie den Bericht per E-Mail erhalten möchten.

Fax: Wählen Sie aus, wie oft der Bericht gefaxt werden soll. Sie haben die Wahl zwischen Nie, Täglich, Wöchentlich, Monatlich, Vierteljährlich, Jährlich, Zweimal im Monat und Bei Bedarf.

Faxnummer: Geben Sie die Faxnummer ein, an die der Bericht gefaxt werden soll.

Die Optionen E-Mail und Fax sind nur verfügbar, wenn Sie das Modul Massen-Fax/E-Mail erworben haben.

Zweimal im Monat werden Berichte am 1. und 15. erstellt.

 

Speichern eines Berichts als .csv-Datei

 

Geben Sie im PDF-Speicherbereich den Namen des Berichts gefolgt von a.csv.txt ein. Dadurch wird der Bericht korrekt formatiert, so dass er in ein Buchhaltungsprogramm importiert werden kann.



Berichte auf Abruf

 

Damit ein Bericht „auf Abruf“ ausgeführt werden kann, muss eine Aufgabe im Task-Runner eingerichtet werden. Wenn die Aufgabe ausgeführt wird, findet sie alle Berichte, die „auf Abruf“ eingerichtet wurden, und führt die entsprechende Aktion aus (Drucken, E-Mailen, Speichern oder Faxen).

 

Navigieren Sie zu Konfigurationen>Systemkonfigurationen>Task Runner konfigurieren>Aufgabe hinzufügen.

 

Hier sehen Sie ein Beispiel für eine Aufgabe mit den entsprechenden Einstellungen. Es sollte immer das Programm PrintEOMReports.exe auslösen und in Ihrem Benutzerordner starten. Die Eingabe der Parameter sollten wie folgt lauten: die Stationsnummer, für die Sie sie ausführen möchten, 128 (dies bedeutet „auf Abruf“ und wird sich nie ändern), und die Stapelnummer. Wenn Sie den Wert auf Null setzen, wird kein bestimmter Stapel geprüft.
