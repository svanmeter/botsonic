Data Management Tools - Change Case

 

 

 

Select # 1 - Change Case From the list.

 

Select the Table that you want to clean up from the Table Drop-down Menu

 

 

Select the field within that Table from the Field Drop Down Menu. In the Example Below we have selected the Table - Customer file and now we have highlighted the Field of City.

 

 

Now we can select the operation that we would like performed.  In this case, we are choosing to change that field to Proper Case.

 

 

Click the go box in the bottom right-hand corner.  The system will now change that field in that table.  When it is complete you will see a box that says Done.  Click ok To complete.
