Detail Job Report by Sales Rep

 

This report prints all reservations that are pending with the delivery dates and item details.  It will show all the open items on the job site with the expected return date as well as all the Accounts Receivable that haven’t been paid.  If you have the Mass Fax/Email module, you can set this report to be automatically emailed to the sales rep each day.
