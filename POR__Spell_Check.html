POR Spell Check

 

Point of Rental will continuously spell check while you are writing. Any unrecognized or misspelled words will display a red wave-like line underneath.

When right clicking the misspelled or unrecognized word, a drop down list will appear displaying a list of suggested words and options of Ignore All, Add to Dictionary, and Spell.

Ignore All: When selected, this will ignore and remove all of the red wave-like lines from the chosen words.

Add to Dictionary: When selected, this will add the chosen word to the dictionary. This will also remove the red wave-like lines.

Spell: This option can only be opened by highlighting and right clicking the misspelled or unrecognized word. A box will appear where you can select a suggested word, Ignore, Ignore All, Change, Change All, Add and request more or different suggestions by selecting Suggest

 



To remove words that have been added to the dictionary, follow the below path to file Custom.dct.

Path - :\POR\Program\Custom.dct

*Note: Custom.dct must be opened with Notepad



When opened, all words that have been added will be displayed. Deleting words from this list will remove them from the dictionary.



 

Save the file and exit.

For the changes to take affect Point of Rental must be closed and reopened.
