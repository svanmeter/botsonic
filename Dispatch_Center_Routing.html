Dispatch Center Routing


 

Allows the dispatcher to assign contracts to routes and calculate on-site time.

 

From Program Menu>Configuration> Counter System>Dispatch Center

 

Once you are in the dispatch center, you will automatically be on the current date to start routing. If you are routing for a different date, you can either select from the up and down arrows or use the calendar to select the appropriate date.

 

 

Select All Trucks – You will see a list of all trucks and trips for the selected day.

 

 

Select a specific Truck – Will show all contracts and trips for that truck only.

 

Not on Truck – will show all un-routed and pre-routed contracts.
