Intacct Setup to Enable Shipping Misc as Subtotal Line

The below will allow shipping/misc to come over to Intacct as Subtotal Lines rather than individual line items on the vendor invoice.

1. In Intacct, go to Purchasing > Setup > Transaction Definitions

 

Select ‘Edit’ on Vendor Invoice

In the Accounting Section, check the box Enable Subtotals. Add a line to the table for Shipping and Misc. Spelling is important here. Use those exact keywords in the Description field.
