Customer Labels




Overview
 

The Customer Label routine allows you to print labels for your customers. The labels are printed 3 across and 10 down on standard Avery 5160 labels.


Here's how to get there:
Expert: Program Menu > Customer Reports > Customer Labels
Elite: Right Menu > Customer Labels

 

When you select Customer Labels, you will get the following screen:
 

 

Customers to Report - Choose the selection criteria you want to use to print your labels. Your choices are All customers, customers by Customer Type, or by Customer Restriction.
 

Payment Type to Report - Set the criteria for Account, Cash, or Account & Cash customers.
 

Print Customer Number - This option sets whether the customer’s number is printed on the label.
 

Yearly Income - You can run these labels based on the customer’s yearly income. Select Greater Than, Equal to, or Less Than and type in your desired dollar amount.
 

Lifetime income - You can run these labels based on the customer’s lifetime income. Select Greater Than, Equal to, or Less Than and type in your desired dollar amount.
 

Active Date - Set an Active Date to run the labels for. Example: After 01/01/08 and Before 01/01/09 would get you all active customers for the 2008 year.
 

Printer Select - This button defines the output printer for the report.
 

Select Customers Manually - This option allows you to select each customer individually.  You will be given the standard customer lookup screen where you can select customers by key, name, number, or driver’s license number.
 

Specify - This takes you into the customer file in Report Generator where you can be more specific on the customers you want to print labels for.
 

Cancel - This button cancels your selection and returns you to the Program Menu.
 

Generate - This button displays the customer file selection options if “Select Customers Manually” is not checked and displays the customer lookup screen if it is checked.  Once your selections have been made the labels will be displayed on the screen.  Click the printer icon at the top of the screen to print.
