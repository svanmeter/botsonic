Time Clock Report 

 

Selecting Time Clock Report you can generate a report based on an individual employee, a group, a store, or all employees listing their clock-ins and clock outs for the selected date range. When using the Mobile Workforce, make sure the Start and End date ranges cover a time where a mobile user clocked in

Employees to report – This option will allow you to select all employees or a particular employee.  Click the down arrow for a listing of all employees and select the desired employee.

Group to report – This option allows you to select a group of employees to report.  Click the down arrow for a listing of all the groups and select the desired group.

Store to report – This multi-store option allows you to select only a certain store to report.  Click the down arrow for a listing of all the stores and select the desired store.

Start date – This date determines the starting date for the report.  Click the down arrow for a calendar to select the desired date.  The default date will be the first day of the last full workweek as defined in your store parameters.

End date – This date determines the ending date for the report.  Click the down arrow for a calendar to select the desired date.  The default date will be the last day of the last full workweek.

Print detail – This option allows you to print each clock in and out for the selected employee(s).  If this box is unchecked, then only the daily hours will be displayed for each employee.  In summary mode, you will be unable to modify time clock entries.

Include cleared entries – This option determines whether entries that have already been cleared are printed on the report.  If you are printing historical timesheets, then check this box.  If you are only interested in current entries, leave this box unchecked.

Print employee signature line – This option allows you to print a signature line for each employee.  This is used if you have your employees sign their timesheets before turning them in.  If this option is selected, each employee will be printed on his or her own page.

 

Select Printer – This button defines the printer that the report will print to.

 

Cancel– This button cancels your selections and returns you to the Program Menu.

Generate – This button generates the report.

Note: If using the Mobile Workforce, you will see times entered in green, the geo-location in the ‘Clock In Note’ and ‘Clock Out Note’ columns. If a mobile user clocks in/out and the geo-location doesn’t appear then the mobile user’s phone has location verification disabled and should enable it

  

 

To see the mapped site of the clock/in  clock/out just click the green Clock In/Clock Out entry and a map will pop-up showing the location
