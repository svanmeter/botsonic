Routing a Truck

 

Pre Routing Process

 

You have a few options available when starting the routing process. You can choose to Select all trucks, specific truck or not on the truck.

 

Select All Trucks – You will see a list of all trucks and trips for the selected day.

Select a specific Truck – Will show all contracts and trips for that truck only.

Not on Truck – will show all un-routed and pre-routed contracts.

 

Geographical Process

 

You will need to geo-code or set trip names for each contract so we can group them by geographical areas.

 

1. Click on the trip field to edit and set the Trip Name.

 

            a. Review the customer's address and determine the geographical location.  

 

 

2.  Click in the Trip Field and enter an alpha/numeric description for the area this is going to.

 

 

            Ex. We use “ARL to denote the city of Arlington. Using a short code eliminates misspellings and saves time.

                        * Note – This Field is not designated for the driver’s name.

 

3.   Determine the unload/load time or on-site time.

4.   Click the Items button in the bottom left button.

            a. Will show a list of items on the contract to help you determine the unload time (delivery) or load time (pick up)

 

 

5.  Enter the on-site time in man-minutes in the unload time. (Ex). If you would normally send two people and you would expect it to take 40 minutes, the man-minutes would be 80 minutes.

            * Note – In your item records the average setup time can be set. The system will use the item setup time to offer the user a suggested Unload time for the contract.

 

6.  Review the customer's address to determine the Trip name ( Ex. If the customer city is Arlington, you could call the Trip ARL.

 

 

            a. If the Trip name cannot be determined from the address, use the Map It button to see the stop location on a map.

 

            b.  Using a web-based map the location is displayed allowing for an accurate Trip assignment.

            c.  When finished click the red X in the top right corner of the map.

 

7.  Once you have entered the number of crew and the man minutes, you will see in the route now has been assigned a route number and is in the planning stages.

 

 

8.  Click on the trip Field to set the Trip name for each unassigned stop.

 

9. Move a Trip to a different track by simply dragging the Trip and dropping it on the appropriate truck.

            *Note – You can Merge Trips together by dragging and dropping them into each other.

 

 

*Note – You can also move stops from one trip to another, Simply hold the Ctrl key, select the stops you wanted to move, and drag them into the trip you want to move them to.

 

Each truck has now been verified by location, crew, and man minutes. You will see that the Route Number has been assigned to the trip and the Route status has changed from Undefined to Planning. Under the trip column, the abbreviated city name should appear.  You can drag each route to its appropriate truck, once done the truck number will appear under the truck column, letting you know each route has been reviewed and assigned to a truck.
