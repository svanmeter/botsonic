This article goes over how to connect Cayan and Cayan terminals in Essentials. POR Pay is a new micro application that integrates natively with Expert/Elite/Rental Hosting, Syrinx, and Essentials. The following are the currently supported setup and migration processes: Not all processors support all of POR Pay’s features. Also, processors only support some countries and currencies.

 
 

Topics in this article include

  • Cayan: New Essentials Implementation
  • Cayan: Existing Cayan Customers in Essentials - POR Pay Migration
  • (Optional) Cayan: Terminal Order and Setup (new or existing customer, post migration)
  • Processing Payment with POR Pay

 

Cayan: New Essentials Implementation

 
1. Go to Accounting → Credit Card Processing.


 
2. In the ‘Update Merchant Credentials’ Section, select Payment App, and click Save.
 
 
3. Go back to Accounting → Credit Card Processing.


 
4. In the Card Processing section, click Connect with Point of Rental.
 
 
5. The POR Payments interface will display (pictured below). If not displayed, refresh the page.
 
 
6. From the POR Payments interface, expand the Organization Info accordion, fill out the organization info, and click Save.
 
 
7. Expand the Payment Processors accordion and click Add Payment Processors.
 
 
8. In Processor Selection, choose ‘Cayan’ from the dropdown and click Select.
 
 
9. Give your processor a Label (anything works to identify the processor and multiple processors are supported), select the appropriate Currency, enter in the Merchant Name, Site ID, Cayan Key and Cayan Web API Key and click Save.
 
 
10. When the Cayan process is complete, the processor on the list will be gray and show a status of ‘Credentials are valid.’
 
 
11. The setup process is complete. You will now see the option to use the configuration in all application functions where payments are supported (counter, storefront, email, etc.). When multiple processors are set up, you will have a dropdown to select the appropriate processor.


 

Cayan: Existing Cayan Customers in Essentials - POR Pay Migration


The migration script is in beta, with alpha testing approved. The worst-case scenario in running the script is existing tokens no longer being valid and some CC reporting issues. These issues can be manually hot-fixed by development if reported.
 
  1. To initiate the process, create a Tier 2 support ticket with the following information:
    1. Title - Enter the title "POR Pay Essentials Setup for Client XXX"
    2. Description -
      1. Server or Location 
      2. Company ID (company details)
 
2. Advise customer we will coordinate on a migration date/time after hours.
 
3. Push ticket from Salesforce to Azure DevOps (AZD).
 
4. Hop in the ‘#por-pay’ Slack channel chat and include @Ben Mohorc. If you don’t have access, Kirk Bennett can provide.
 
5. Development will facilitate the migration. The customer can order the terminal in preparation for the support.


 

(Optional) Cayan: Terminal Order and Setup (new or existing customer, post migration)


The client has access to execute this process. Please note that the Cayan Terminal must be connected to the Internet from the same network running Essentials. Cayan Certificate will also have to be run as well on those computers in order for the Terminals to run Properly. The Client will need to also be on a Chromium Base Browser for this to work. 
 
 
1. Log in to your Essentials rental management platform.
 
2. Navigate to Accounting → Credit Card Processing.
 
 
3. Find your processor from the list and click Add Terminal.
 

4. Enter a Label (anything you want to identify later), the IP Address from the terminal, and click Save.


5. Navigate to Stations → Registers.
 
 
6. Click the pencil icon for the register you want to add the terminal to.
 
7. From the ‘Register Configurations’ section → click the plus icon .
 
8. Enter the Name you want to identify the terminal by → select your terminal from the dropdown and click Save.
 
 
9. Back in the Stations → Register section, check the 'Associated’ box for the terminal you just set up and click Save, completing the terminal setup process.
 
 
10. A Cayan Certificate is needed if the customer is going to be using Credit Card Terminals. Please see our documentation on Connect Cayan to Rental Essentials on how to install the Certificate on the computer. Click here to get the Certificate from Global Payment Integrations. 
 

 

  Processing Payment with POR Pay 

 
Once setup is complete, you can leverage POR Pay anywhere in Essentials where payments are collected (counter, storefront, auto billing, pre authorization, refunds, and automatic billing).
 
1. For this example, we will collect a payment from Workbench → Click an existing Reservation → Collect Payment.
 
 
2. Select ‘Card’ as the payment method, then select how to charge the card. Select ‘Manually Enter Card,’ for new card form (other options are ‘Terminal’, if set up, and card's on file, if card was previously run with POR Pay), enter the amount, and click Save.
 

3. Enter the card information, verify the customer information, scroll to the bottom and click Process. 
 
4. You will then be redirected to the contract upon payment success.
