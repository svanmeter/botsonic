  •   Display the Price List for this Item on Contracts and Invoices - You now have the ability to display rates on your contracts! If enabled, your contracts and invoices will display the current Daily, Weekly, and Monthly rates. You can find this setting by going to the Configuration icon > Company Configuration > Printing Options. By displaying item rates it can help reduce questions and confusion your customers may have about pricing. Please note, that rates will not appear on Receipts.
  • Performance - We optimized searches across the application for performance.
  • Availability List Report - Added the ability to filter the Availability List report by Unavailability type.
  • Inventory Reports - We've updated the names of inventory reports for consistency.
  • Dashboards - We have enhanced the dashboard by adding the ability to drag and drop things where you want them. You can hide things you don't want to see. You can customize your dashboard to your needs.
  • New Vendor - Added clarifying text when adding a vendor to make it clear that you're creating a new vendor.
  • Fixed issue - In which CC Fee is applied when paying via eSign, and Pay was not applied on the contract, resulting in a refund amount.
  • Fixed issue - In which delivery and pickup fees were double charged.

Note on Storefront Changes

We are constantly working to improve the user experience, performance, and visual layout of the Point of Rental Storefront service. As part of our ongoing efforts to improve the user experience, we have identified necessary structural changes to the Storefront HTML that include a high chance of impacting Custom CSS layouts. If you have Custom CSS, please follow the steps below to make sure you are not impacted.
 

  • Review your Custom CSS
  • Please check your storefront to confirm there are no conflicts with your custom CSS
  • Make any changes needed to ensure that your designs still function as intended
  • If you need to remove any custom CSS, please find the instructions here.
