How to Enable Pre-Authorization

 

To enable pre-authorizations:

  1. Navigate to Program Menu > Configuration > Store Configurations > Parameters
  2. Click the Drawer tab
  3. Check-box Allow Pre-authorization
  4. Click the OK button to exit
  5. Exit the program and re-open each workstation

You will now get a P – CC Pre-authorization payment type option on the contract payment screen under the Deposit amount drop-down.
