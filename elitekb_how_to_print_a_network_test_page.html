How to Print a Network Test Page

 

For Dell 2350dn

Menu > Network/Ports > Network Menu > Network Setup > Reports > Print Setup Page

 

For Dell B2360dn

Menu > Network/Ports > Network [x] > Network [x] Setup > Reports > Print Setup Page

 

How to manually change IP Settings:

 

•  View or change the current TCP/IP Address

Menu > Network/Ports > Network [x] > Network [x] Setup > TCP/IP > IP Address

•  View or change the current TCP/IP Subnet Mask

Menu > Network/Ports > Network [x] > Network [x] Setup > TCP/IP > Netmask

• View or change the current TCP/IP Default Gateway

Menu > Network/Ports > Network [x] > Network [x] Setup > TCP/IP > Gateway
