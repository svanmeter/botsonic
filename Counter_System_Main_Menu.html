Overview
 

This article will discuss the Counter System, Counter System processing, and some of the special features associated with the Counter System. The Counter System program is used to write rental contracts, reservations, quotes, and work orders, as well as modify contracts, close contracts, and convert quote and reservation contracts into rental contracts. Other features include the ability to view customer balances, inventory availability, send or view inter-office e-mail, make cash disbursements, record 'missed rentals,' view items due for maintenance, and record cash deposits into and withdrawals out of the cash drawer.

Here's how to get there
Expert: Program Menu > Counter System

Elite: Transactions (left menu) > Quick Buttons
 

After the program has been started, you will be asked to enter your unique employee ID (identification code) followed by the ENTER key. If you enter an invalid code, the program will redisplay the Enter ID screen.

 

Note: Items may be selected from any menu using any of several options. The menu item number may be entered followed by the ENTER key. Or, the underlined letter may be typed followed by the ENTER key, i.e. type O for Open Contract. You can also use the Window short cut key by holding down the ALT key and pressing the underlined letter. This method does not require the ENTER key. Or you can left click the mouse whenever the cursor is positioned over your desired selection.

Special Features

From the main Counter System screen, you will see messages appear in each corner of the Counter Menu screen. These messages are sent to you, requiring your attention. Once the message has been addressed, it will disappear from the screen.
 

Group Tasks pending - Located in the bottom right corner of the Counter System Menu. This is an alert or task assigned to members in a group that requires attention.

 

Task Pending - Located in the bottom right corner of the Counter System Menu. This is an alert or task assigned to you that requires attention.
