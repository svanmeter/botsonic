Elite Cloud | File Transfer




This article will go over how to import and export files from your local computer to the Elite Cloud Server. 




Topics included in this article
 

  • Exporting
  • Importing




Exporting

 

Due to limitations in the copy/paste functionality on a cloud server, when users export something, they are given the option to open it within the server or to save it locally.  ‘Open to server’ will open it in the same server session.  


‘Export to client’ will further prompt the user at the top right of the screen and allow the user to save the exported document locally.  Once the download is confirmed, a file explorer save screen will open allowing the user to navigate to the folder/drive they would like to save the file to.

 

 
 

Importing

 

To import files, users will now select the ‘up’ arrow in the top right bar of the screen.  A file selection box will appear.  Select the file for upload.  The user will be given the option to open the file or open the folder the file exists in. Multiple files can also be uploaded at once with a single prompt. Once uploaded, the file will appear in the ‘Upload’ folder under the POR directory.  The subfolder will be labeled for the user that did the upload. If the user wishes to upload duplicate files for ones that already exist in this folder, they will have to be done individually, not en masse.
