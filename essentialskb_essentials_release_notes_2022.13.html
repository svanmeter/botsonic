9/27/2022 - Version 2022.13


  • Delivery Window Enhancements - We told you we were on a mission to improve this functionality. We have ‘Delivered’ and added 2 nice features to this.
 
  • Addition to Storefront - You want your customers to be able to choose their delivery times, but only certain delivery times. You might want to allow them to choose between 8am-12am, or 1pm-5pm, but you don’t want them to put in shorter delivery windows, like an 8am-9am if that time frame has to be scheduled over the phone or in person.
 
  • Delivery Window Fee - If you charge more for certain delivery times, you can now do that automagically. For example, if 8-12 and 12-4 delivery times are part of the standard delivery fees, yet a tight window like 8-9 costs more, then you can set that time to be an additional $25 or whatever you want it to be so it automatically charges that amount more. No remembering or using post it notes.
 
  • Totals on Workbench - We had some requests to show the totals/amounts paid on the workbench for each transaction. That sounds great as you could easily see amounts owed/paid for each upcoming transaction, but that could turn the workbench into clutter. So in order to keep workbench from looking like an episode of hoarders, yet still give you that great new functionality, our great development team created a way for you to hover over each transaction and see all the totals. 
 
  • Multiple Payments correction - When you went to a customer record that owed on multiple contracts and tried to make payments on multiple contracts (we are sure that you already knew you can do that), there was a dreaded error. We fixed it.
 
  • Outh long email user enhancement - For those of you with super duper long email addresses, you can now log in using OAuth. Per our customers request that reported this, we emailed this release note to them at H.Wolfeschlegelsteinhausenbergerdorff@MyEmailAddressIsRidiculouslyLong.ugh.aargh.shh.com

Design Enhancement on Storefront - The storefront header, which includes the company logo, the date-picker, user navigation (login, language select, and cart), EMP view, event details, and multi-store selector, have been redesigned to better suit the available space. EMP view, event details, and multi-store selector now float to the right side of the screen as a separate row underneath the user-nav and date-picker.
