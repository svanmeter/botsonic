Date/Time Out Override Report

 

This is not an uncommon function; however, if one employee has a significant number of them especially to a customer it could signal a problem.  Note Your setup whether date/time out overrides are tracked under Security in Parameters.  If you have your parameters set to Never for tracking Date/Time Out Overrides, then no report will be generated since no tracking is being done.  If you have Date/Time Out Override tracking set to Close, then only changes done when closing the contract will be reported.
