Signature Pad (Topaz/ClipGem) Installation & Setup





This article is going to show you how to install the ClipGem (Topaz) Signature Pads for the Expert and Elite software. The Topaz Signature Pads are compatible with USB 2.0 and 3.0.



Topics in this article include
 

  • Signature Pad Setup

  • Enabling Picture Saving

  • POR Software Setup for OCR and Signature Pad Files




What's the difference between a Signature pad and CounterSign?


Signature pads are small devices used to capture signatures. Once setup, you can request and capture a  signature. Signatures are stored as PDFs within a contract's Transaction Edit report. 


CounterSign is similar to a signature pad in that it also uses a device to capture signatures. However, with Countersign your customers can view, initial, and sign the contract on the device. Signatures are stored as a PDF and can be found within a contract's Transaction Edit report.






Signature Pad Setup
 

1. Download and run the Signature Capture Pad (ClipGem) Installer from www.porhelp.com.

i. If you are running Windows 7 through Windows 11 you will need to save the program to the Downloads folder and then run the program as an administrator.

ii. To run the program as an administrator, go to the Downloads folder and right-click on SigCapInstaller, then left-click on Run as Administrator.   

 

2. When asked if you want to allow the program to make changes to the computer, click Yes.

 

 

3. When you are told that you have to run the program as an Administrator, click OK. 

 

 

4. Click Next.

 


 

5. Click Next.

 


 

6. Click Install to start the installation.

 

 

7.  You will be given an error message such as the one below. Click Ignore to continue the installation.

 


 

8. Click on Agree to continue.

 

 

9. Click on Finish to complete the installation.

 

 

10. Plug the signature pad into a USB port and wait for the drivers to be installed. The computer will display a message when the drivers are done installing.

 

 

11. After the drivers are installed, open the Signature Pad application. If the application does not appear, left-click on the Show hidden icons box in the lower right of the taskbar.

 


 

i. Right-click on the Electronic Clipboard.
 

 
 

ii. Left-click on Show. This will open the Signature Pad window. This will open the signature pad application.
 




Enabling Picture Saving

In order to save to the server's Pictures folder, you will need to make sure that the folder has read and write permissions. You will also need to be sure that the ‘taskrunner’ user is created in the server and that it is saved in the Credentials Manager on the local PC.
 

1. With the Signature Pad window open, left-click on Options, then left-click on Set File Transfer Mode. This will allow you to set where the signatures will be stored.

 

 

You will get the following screen.

 

 

2. To save signatures to a server, left-click the Server Share radio button, then left-click the Set Path box.

 


 

3. You will get the following message. Click on OK.

 


 

4. In the Address Bar, navigate to the server POR\Pictures folder. You can also type the IP address of the server in the address bar if you prefer to use that method. For example, if your server’s IP address is 192.168.3.123, you would type: \\192.168.3.123\Pictures then press Enter on the keyboard.

 


 

5. If you are prompted for credentials, use the rental\taskrunner credentials. Be sure to check Remember my credentials. Left-click OK.

 


 

6. Left-click on NewTextDocument, then left-click Open.

 

 

If there is not a text file called ‘NewTextDocument,’ create a new one. Right-clicking inside the Pictures folder, highlight New, left-click Text Document. Then complete Step #6.

 

7. The file path should be seen under Set Path. Click on OK at the bottom of the screen.

 



 

8. Test the pad by attaching a blank contract or piece of paper. If you use a blank contract, you will write on the signature field. On a piece of paper, it will be 1 inch up from the bottom of the pad and 1 inch right from the left of the pad.
 

9.  If successful, you will see the signature written in the window and will see it saved in the folder where you set the path.

 

 


 





POR Software Setup for OCR and Signature Pad Files


1. Open Point of Rental Software takes note of which store you are on.

 

2. Open the Parameters in your Point of Rental software.

 

Here's how to get there
Expert: Program Menu > Configuration > System Configuration > Parameters
Elite: Configuration Gear > App Configuration > System Configuration > Parameters


3. Press Parameters and login.
 

4. Select which Store.
 


 

5. Press on the Setup tab and underneath the signature section, check the options for what you want POR to ask a signature for.
 


6. Press OK. Continue for all other stores if needed.
 

7. Open Printer Selection.

Here's how to get there
Expert: Counter System > Help > Print Setup
Elite: Left Menu > User Tools > Operator Information > Print Setup

8. Go to the Advanced tab and then click on Modify and login.



9. Set the Scan, OCR, and Signature Filenames to point to the appropriate location (should be C:\POR\Pictures), and the appropriate files.
 

Note: Make sure that the Scan Filename ends with an * as seen in the picture above.


10. Click OK.
