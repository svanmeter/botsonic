Syrinx 365 | Local Search User Settings



This article will cover the user settings that apply to Local Search screens, as these will determine how the screens work.


Topics included in this article
 
  • User Settings
  • Default Search Result Size
  • Show Real Search Data on Open
  • Return Sorted Search Data




User Settings


If you click the user settings option, at the top right of the screen, then you can configure the items displayed below.

These determine how the search screen will display the data.




Default Search Result Size


This slider will set the maximum number of records that will be returned when using a Local Search screen.



Show Real Search Data on Open

If this option is unchecked when you open a Local Search screen, results will not be displayed until you start using the search field or defining filters.

If it is checked when opening a Local Search screen, results will be displayed up to the result size set on the slider.

  

Return Sorted Search Data

If this option is checked, the results returned will be sorted by an order set on that search.

If it is unchecked, then the system won't apply any sorting and results will be displayed by how SQL sorted it, which may not be the same for each search.
