POR Browser Settings

 

 

 

Overview

 

This article will discuss different browsers relating to use with Point of Rental's Expert and Elite software. A system can be updated to use different browsers when accessing internal websites from the system.

 

Options are:

 

  • Chromium Browser (typically system default)  
  • .Net Browser
  • OS System Browser

 

Changing the default browser is done by double-clicking on the PORBrowser.exe file located in the Program folder.  The changes are done in a loop through the three options, allowing you to either remain on the existing setting or update to the next one.  

Most systems are set to use the ‘Chromium Browser’ by default so these instructions start from that point.  However, if the system is already set to another browser, you may start at a different point.

 

 

Changing from Chromium browser

 

Double clicking on PORBrowser when the system is set to Chromium will display this option.  Selecting Yes makes no changes.  

 

Selecting No updates the system browser to .Net Browser.

 

When a browser window is opened from the application (such as Net Help) and .Net Browser is the system default, you will see ‘i~’ in the page title. 

 

 

 

 

Changing from .Net browser

 

Double clicking on PORBrowser when the system is set to .Net Browser will display this option.  Selecting Yes makes no changes.  

 

 

Selecting No updates the system browser to OS Default Browser.

 

When a browser window is opened from the application (such as Net Help) and OS Default Browser is the system default, you will see ‘#~’ in the page title. 

 

 

 

 

Changing from OS Default Browser

 

Double clicking on PORBrowser when the system is set to OS Default Browser will display this option.  Selecting Yes makes no changes.  

 

 

Selecting No updates the system browser to the Chromium Browser.

 

When a browser window is opened from the application (such as Net Help) and Chromium Browser is the system default, you will see ‘#~’ in the page title. 

 

 

 


 

Double-clicking again will put you back to the top and the loop starts over.
