POR Pay | Global POR Credit Card Processing



 

Point of Rental offers its own credit card integration through which you can choose a variety of processors (CAYAN, Stripe, Peach Payments, etc.). This opens up many options globally that have been unavailable through our previous integrations. Like other integration options, this allows you to take and reconcile payments within POR, keeping end of day processes seamless and easy.

 

Topics in this article include

  • Set Up

  • Terminals

  • Payments

  • Refunds




 

Set Up

 

This integration is set up on the drawer tab of Parameters (Configurations > System Configurations). 
 

1. Select ‘POR’ from the combo box of options.

2. Then select ‘Global Setup’ to configure the credit card setup.  

  


3. Select ‘Add Payment Processors’ to configure the processor of your choice. 

4. Follow the prompts (these will be slightly different depending on the processor), to enter credentials. For example, CAYAN will provide credentials for users to enter, while Stripe users will need to setup or login to their account.

5. Once the credentials are saved, take the processor ID and enter it in the appropriate space on the drawer tab. This will be used to determine which processor to use if stores have unique processors set up.

 


Once this is done, terminals can be added. If using an onscreen terminal, rather than a physical terminal, no further setup is needed.  

 
 

Terminals

 

You will need to input whatever label they have designated to the terminal and the terminal’s IP address. This can be obtained from the settings on the terminal itself. After this information is saved, the terminal label and its system generated terminal ID will appear under payment terminals. The terminal ID will be needed to set up the terminal in security so that it can be assigned to a user profile. 

 

1. To complete the setup of a physical terminal, open Security from Program Menu and go to Verifone Terminals. 

 

2. Add a terminal. You will need the terminal ID from configurations. 

 

3. Label the terminal and select POR Payment from the dropdown.  

 

 

4. This can now be assigned to a user profile. These can be accessed through User Profiles (the option right above Verifone terminals in the security menu).

 

5. Navigate to the advanced tab of the selected profile and select the modify button. 

 

6. Choose the correct terminal from the dropdown at the bottom.

 

 
 

Payments

 

When a payment is taken, you will have the option of using either the physical or the onscreen terminal. If a token exists, an existing card can be used, or a new card can be entered. Selecting the physical terminal will direct your customer to interact with the terminal. By selecting the onscreen terminal, it will appear on your screen.

 

 

Onscreen Terminal:

 


After completing the payment, you will be guided through the usual contract processes to print/email the contract or return to another screen.

 
 

Refunds

 

Whether you are able to do an independent or dependent refund will depend on the processor. Those who support independent refunds, will be able to refund any amount to a particular card. Otherwise, you can only refund up to the amount paid on any particular card. If a contract has two $10 payments on two different cards, then $10 has to be refunded to each card, rather than being able to refund $20 to one card.
