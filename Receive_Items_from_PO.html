Artikel aus PO erhalten


Wenn Sie Artikel von einer PO erhalten, wird die Menge des Inventurartikels um die erhaltene Anzahl erhöht. Außerdem wird die Kaufhistorie der Artikel mit Kaufdatum, Bestellnummer, Bestellmenge, Kaufpreis und Zusatzkosten aktualisiert.

 

Programm-Menü>PO

Sie gelangen in die PO-Anzeige, klicken auf die ausgewählte PO und es erscheint der folgende Bildschirm.



Wählen Sie „Artikel aus PO erhalten“, um die ausgewählte PO zu öffnen.



Sie haben die Möglichkeit, alle Artikel auszuwählen, indem Sie mit „Ja“ antworten, oder Sie können einzelne Artikel auswählen, die Sie erhalten möchten, indem Sie „Nein“ wählen. Wenn Sie „Nein“ wählen, haben Sie anschließend die Möglichkeit, einzelne Artikel durch Doppelklick auf die Spalte „Erhalten“ auszuwählen. Sie können den vollen Betrag oder einen Teilbetrag erhalten.

•Hinweis: Ein Sammelartikel ist ein Platzhalter, um die Gesamtmenge eines Miet- oder Verkaufsartikels anzugeben, die man besitzt.

 

Sammelartikel erhalten  - Wenn Sie einen Miet- oder Verkaufssammelartikel in die PO aufgenommen haben, wird beim Eingang der Artikel eine doppelte Kopie des Sammelartikels der Artikeldatei mit dem Symbol # oder - erstellt, was anzeigt, dass der neue Artikeldatensatz entweder ein serialisierter Artikel oder ein Stationsstandort sein wird.

Ausnahme Wenn Ihr Sammelartikel WIDGET heißt, wird der serialisierte Datensatz WIDGET #01 oder WiDGET-1 heißen

# - Serialisierte Artikel * Hinweis: Wenn mehr als eine Menge empfangen wird, werden die serialisierten Datensätze in der Reihenfolge #01, #02, #03 erstellt.

-1 Stationsstandort  * Hinweis: Wenn es mehr als einen Stationsstandort gibt, werden Sie gefragt, an welchem Stationsstandort Sie den Bestand eingeben möchten -1, -2, -3 (bei Multistore)

*** Sie können die Versandkosten, Steuern und sonstigen Gebühren auf die Artikel der PO umlegen, indem Sie das Kontrollkästchen in „Konfiguration“> „Systemkonfigurationen“> „Parameter“> „Buchhaltung“> „Zusatzkosten in den Bestand einbeziehen“ und/oder „Zusatzgebühren aktivieren“ anklicken. Mit diesen Einstellungen werden diese Gebühren auf alle Artikel der PO verteilt, also auch auf Nicht-Bestandsartikel. Wenn Sie nicht möchten, dass die Zusatzgebühren den Nicht-Bestandsartikeln zugeordnet werden, müssen Sie die Bestandsartikel empfangen und alle Zusatzgebühren hinzufügen, speichern, und dann die PO bearbeiten und die Nicht-Bestandsartikel hinzufügen.

 

Transfer erhaltener Artikel

Erhaltene Artikel können in eine andere Station übertragen werden. Sie kann aufgerufen werden, wenn der Artikel in der Bestellung einen verknüpften Vertrag von einer anderen Station und eine erhaltene Menge hat. Es überträgt die Menge, die Sie erhalten haben, von einer Filiale auf die andere.



Der Benutzer wird aufgefordert, die Filiale auszuwählen, und es wird ein Transfervertrag erstellt. Die eingegangene Menge wird dem Transferticket hinzugefügt und aus der PO entfernt.
