The Syrinx database and the Syrinx client are not the same version. If this is a new Syrinx client install on to an existing system, you will need to download/install a version of the Syrinx program that is compatible with your database version. If users receive this message following an upgrade to the Syrinx database, the Syrinx client has not been updated to the latest version. Please consult the upgrade guide for instructions on updating the Syrinx client software on the individual PCs.
