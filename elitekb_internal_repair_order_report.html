Internal Repair Order Report

 

This report shows all rental items that are currently in for repair.  It will show each item on any repair ticket and the total cost that has been added to the Repair cost for each item.  When you close out an Internal Repair Order, you will be prompted if the “maintenance hours” should be reset.  If you answer yes then the maintenance hours but not the frequency for that item will be reset to zero.
