Syrinx® -  Sending Invoices via Email

 

 

 

 

Overview: This guide is designed to help you with setting customers up to receive invoices via email.

 

 

 

Topics included:

  • Setting Up the Customer
  • User Settings

 

 

 

Setting Up the Customer

 

You need to go into CRM and then Amend Customer.

 

Then find the customer that you are trying to set up.

 

 The first step is to make sure the contact person is in the system, if you click on the Contacts tab.

 

If they are in there, double-click on them to see if they have an email address, if not then enter one.

 

If the contact you want isn’t in there click Add at the bottom, adding a name and email address. The more you can enter here the better.

 

The next step is to set the customer up, if you go to the Invoice Delivery tab.

 

To enable email invoicing, you need to tick the Print Batch Invoices to PDF? If you do not want a paper copy tick the box, then select the contact you would like to send the invoices to. You can also CC someone. By default, if there are 10 invoices to be sent it will be one attachment with all 10 invoices inside. If the customer would like 10 attachments, then tick the one file per invoice box.

 

 Note: This can be imported via Excel if you have multiple customers to update.

 

 

 

 

User Settings

There are a number of email settings which are located in the main Syrinx program that the user themselves must configure to ensure emailing runs smoothly.

  1. Login to Syrinx SP
  2. Select User 
  3. Select User Options (Ctrl+F7)

 

The first tab you need is called Email. Go to this tab. Here you can define how your emails will be sent, either via Outlook or via CDO. There is also a path here to override the default PDF overlay file for emailing out documents if you need to override it.

 

 

The second tab, which is for invoice delivery, is called Batch Invoice PDFs. On this tab, you can select the delivery method for your batch invoices and also the save folder for the PDF files. Syrinx saves the PDF files prior to sending, and you can also set Syrinx to automatically delete the PDF files after the email process is complete.

 

 

There are 4 options here for Send Emails and they depend on what you selected on the Email tab. These options are

 

  • Outlook – Send Immediately
  • Outlook – Preview
  • CDO – Send Immediately
  • Create PDF Only
