Übersicht

 

Es gibt Fälle, in denen Sie jemandem eine kostenlose Vermietung für einen gesperrten Zeitraum in Verbindung mit einem anderen Mietartikel gewähren möchten. Ein Beispiel: Jemand kauft Zement und Sie bieten ihm den Anhänger für die ersten zwei Stunden kostenlos.

 

Die in diesem Artikel behandelten Themen umfassen:

 

·       Einrichten von Dateidatensätzen

·       Erstellen des Vertrages

 

 

Einrichten von Dateidatensätzen

 

 

Bevor Sie den Vertrag erstellen, müssen Sie die folgenden Dateidatensätze einrichten:

 

Hauptdatensatz: Dies ist der eigentliche Miet- oder Verkaufsartikel auf dem Vertrag.

 

Sekundärer Datensatz: Der kostenlose Mietartikel, der für einen bestimmten Zeitraum gewährt wird.

 

Ermäßigter Gutschein: Damit wird der Mietpreis, der im Sekundären Datensatz steht, ermäßigt.

 

1. Gehen Sie zum Hauptdatensatz (in diesem Beispiel Zement).

 

2. Erstellen Sie den Sekundären Datensatz (zum Beispiel Anhänger).

 

3. Erstellen Sie einen neuen Datensatz und nennen Sie ihn „Rabattgutschein“. Vergewissern Sie sich, dass Sie Miet-Gutschein als Artikelart auswählen.  

 

4. Gehen Sie zum Hauptdatensatz und geben Sie den Anhänger als vorgeschlagenen Artikel ein

 

5. Gehen Sie zum Sekundären Datensatz. Erstellen Sie einen Paketartikel und fügen Sie den Rabattgutschein als Teil des Pakets hinzu. Hinweis: Der Gutschein wird als Mietartikel mit einer Menge von 99.999 und einem Verkaufspreis von einem negativen Betrag erstellt (legen Sie den Betrag so hoch an, dass Sie ihn niemals überschreiten würden).

 

 

Erstellen des Vertrages

 

1. Geben Sie den Hauptartikel des Vertrages ein.

 

2. Das Feld für den vorgeschlagenen Artikel wird angezeigt. Geben Sie die Menge ein, die Sie hinzufügen möchten.

 

3. Wählen Sie Serienartikel.

 

4. Bei der Eingabe der Menge geben Sie einen negativen Betrag ein. Hinweis: Dies ist der gleiche Betrag wie der Stundensatz, den Sie kostenlos gewähren möchten. Wenn Sie z. B. einen Anhänger für zwei Stunden kostenlos zur Verfügung stellen und der normale Stundensatz für zwei Stunden 35,00 Euro beträgt, dann geben Sie einen negativen Betrag von 35,00 Euro für den Gutschein ein, um den Preis auszugleichen.



Der Gutschein wird dem Vertrag hinzugefügt, um die Anhängermiete auszugleichen.
