Registerkarte „Kommentare hinzufügen“


Übersicht

In diesem Artikel erfahren Sie, wie Sie über die Registerkarte „Kommentare hinzufügen“ Kommentare oder Notizen zu Verträgen hinzufügen. Die Kommentare werden auf dem Vertrag unter dem letzten Mietartikel gedruckt. Wenn der Platz für die Kommentare auf einer Seite nicht ausreicht, werden die Kommentare auf einer zweiten Seite unter Verwendung des gleichen Sammelartikels fortgesetzt.

So gelangen Sie dorthin:

Expert: Programmmenü > Counter System > Vertrag ändern

Elite: Transaktionen > Vertrag ändern

Hinzufügen von Kommentaren

1. Wählen Sie den gewünschten Vertrag auf dem Auswahlbildschirm aus.



2. Das Fenster zur Vertragsbearbeitung wird angezeigt. Wählen Sie die Registerkarte „Kommentare hinzufügen“. Wenn bereits Kommentare vorhanden sind, wird die Registerkarte als „Kommentare anzeigen“ angezeigt.




3. Fügen Sie Ihre Kommentare hinzu. Bitte denken Sie daran, dass die Kommentare auf dem Vertrag unter dem letzten Mietartikel gedruckt werden. Wenn der Platz für die Kommentare auf einer Seite nicht ausreicht, werden die Kommentare auf der zweiten Seite im gleichen Sammelartikel fortgesetzt.
