Logging in and out of Syrinx 365



This article will cover logging into Syrinx 365 and the various methods of doing so, including Local Connection, Windows Authentication and Global Logon


Topics included in this article
  • Logging in using a Local Connection
  • Logging in using a Global Login
  • Logging in using Windows Authentication
  • Logging out of the Syrinx365


Logging in using a Local Connection

Click the ‘Use Local Connection’ box, the 1st time of logging in will need the Server Name and Database Name to be populated on the right under ‘Enter Connection Details’.
Enter the Username and Password into those fields and click ‘Sign In’
If you click 'Remember Me' then when you return to this login screen the Username will be populated and you will need to enter the Password


Logging in using a Global Login

'Local Connection' should be unticked to login using a Global Login
Enter the email address that the Global User has associated with it in GADMIN
Enter the Password and click 'Sign In'
If you click 'Remember Me' then when you return to this login screen the Email will be populated and you will need to enter the Password

If you have forgotten the Password then click 'Forgot Password' and enter the email address to receive a Reset Code

Logging in using Windows Authentication

Click the ‘Use Local Connection’ box, the 1st time of logging in will need the Server Name and Database Name to be populated on the right under ‘Enter Connection Details’.
Tick 'Use Windows Authentication' and your username will be displayed and greyed out
Click 'Sign In'
Logging out of Syrinx365

In the top right hand corner click the dropdown next to the Username and click Log out
