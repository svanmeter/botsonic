Syrinx 365 | Local Search Summary Panels



This article will cover the summary panels displayed within Local Search screens.


Topics included in this article
 
  • Location of Summary Panels
  • Populating Summary Panels
  • Using the Breakout buttons
  • Expanding Summary Panels




Location of Summary Panels

When you are in a Local Search screen there will be summary panels to the left of the search grid.



The summary panels will be relevant to the actual search screen which is open, i.e. the above panels are related to a Purchase Order search.


Populating Summary Panels

When you click on a row in the search results then the summary panels will be populated with the data from that selection.




Using the Breakout buttons
Some of the summary panels will have a breakout button , this will either be in the header of that panel or against the line level item.

Clicking the breakout button will open the corresponding record.

For example, clicking this breakout button will open the Purchase Order.



Clicking this one will open the Supplier record.



 

Expanding Summary Panels

Some summary panels have an expanded view option so that more columns can be viewed 
on this Purchase Order example. The Line Details has the option to expand the view as you might want to see more data about the items on the order than you can see in the summary panel.




Clicking the expand button will open a larger grid with more detail.
