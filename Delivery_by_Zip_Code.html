Delivery by Zip Code

There are two ways of doing this. The first requires the user to set up an item for each zip code in order to track revenue or frequency. The second allows the user to assign zip code charges based on zones but does not provide reporting for individual zip codes.

 

Tracking Zip Codes Individually:

Allows you to charge a set price for delivery/pickups and track the revenue by zip code.

 

Program Menu>File Maintenance>Item File

1. Add New Item

2. In the Key enter the zip code for the delivery/pickup. * Note: make sure you put a pipe symbol | in front of the zip code.

3. Enter the Name, Description, and part No (if any)

4. Enter a Quantity of 99999

5. Select the type, category, and department. The type would be Sales Item, Category: Delivery

* Note – If you do not have a Category created called Delivery. You would need to create one from Configurations, Item Categories.

6. Enter Sell Price – This would be the amount you would charge for the delivery pick up.

You can track the revenue of the delivery/pickup by Month to Date, Year to Date, or Lifetime to Date. You will also be able to track how many times this delivery/pickup was entered on a contract in a Month, Year, Lifetime under the times out column.

If there are any notes you or instructions you would like to have printed on the contract. You can enter them under the Print Tab. Located under the Details Tab

Assigning Zone Charges By Zip Code:

Program Menu>File Maintenance>Item File

1. Add New Item

2. In the Key enter, the zone for the delivery/pick up. * Note: make sure you put a pipe symbol | in front of the zone.

3. Enter the Name, Description, and part No (if any)

4. Enter a Quantity of 99999

5. Select the type, category, and department. The type would be Sales Item, Category: Delivery

* Note – If you do not have a Category created called Delivery. You would need to create one from Configurations, Item Categories.

6. Enter Sell Price – This would be the amount you would charge for the delivery pick up.

7. Enter the zip codes to be associated with this item in the ‘Notes’ tab.

8. Whenever one of these zip codes are used in the delivery address on the contract, it will automatically pull the correct delivery zone item. If that zip code is on more than one delivery zone, a list will be presented to choose from.

 

Changing Contract’s Store With Delivery Item

There may be times when a user would like to go ahead and write a contract in their store but would like to send it out of another store. This can be done by selecting the appropriate delivery zone item when scheduling a delivery. These items can be set up just as the zones above, however, the home store needs to be set as ‘000’ and the current store should be set as ‘001’, ‘002’, etc….

If all of these delivery zones have the same zip code in the notes, the user will get a list of all of the applicable items after the delivery zip is entered.
