Images for the website (NOT item images!)

 

Use your own images, if you have good-quality original photos.

 
  • Manufacturer’s websites - be sure the images don't have a watermark on them
  • Google image search - again, be sure the images don't have a watermark on them
  • Copyright-free image websites: Pixabay and Wikimedia Commons.
  • Websites where you can purchase images: iStock, Dreamstime, Shutterstock, etc.  Please consult with the Web Team prior to purchasing any images!
  • Copyright infringement is important to avoid. Point of Rental and Rental Hosting cannot be held responsible for any unauthorized use of copyrighted material.
  • Slideshow images must be at least 1920px wide.
  • We usually need images for the homepage, about us page, etc. to be at least 600px in width
  • It's best not to use photos with customer, employee, or event guest faces in them.  You would need to get permission from each person and privacy issues could arise.
  • Keep in mind that dimly lit showroom pictures and dark, stormy, cluttered, and dirty exterior shots don't present well on websites.  For best results, hire a professional to get a few good images.
  • We will also need a clean, original logo file, preferably a transparent version.

POR data
 

If you do not have images inserted in all of your item records, we recommend you get started on that project asap.  You can refer to this article for tips to make them look the best on the website:


https://por.my.site.com/uLearn/s/article/eCommerce-Item-and-Category-Images


If your website will have multiple catalogs (equipment rentals, party rentals, equipment sales, and/or supplies) we will need to create a Store Division (Configuration -> Store Configurations) for each catalog.


Then you will need to assign each Category (Configuration -> Item Configurations -> Item Categories) that will be included on the website to its appropriate Division.  The Web Team can help you with this if needed.


This is a good start.  After we send you the preview of your new website we can work on the details!  Of course, if you have questions, we can address those anytime!
