Labor Updates

 

Labor Updates is a Labor Tracking feature.  It is used to make updates to labor entries from one place without having to access each individual contract the labor entry is associated with.

 

Security

A user only needs access to the feature to have the ability to modify labor entries.  The security to expose the option is in the Left Menu area.  The option is accessed from the Tools section in the Left Menu.

 

How it works…

Selecting the Labor Updates option from Left Menu will open up a new dashboard tab.  Existing labor entries will be displayed in a standard search grid.  Like other grids in Elite, you can create saved search views for specific filtering you use on a regular basis. 

 

To edit labor entries there are two options.  One is to use the ‘Edit’ option from the Action options menu.  The other is to double click on the row.  These will activate field outlines on those that can be updated:

 
  • Item - this is the Sales - Labor Item that can be related to the labor entry.
  • Clock In and Clock Out dates/times - these are the actual dates and times the mobile user clocked in and out.
    • Calculated time is derived from the Clock in and Clock out.
  • Actual Hours - this is a value the mobile user can add to communicate when the calculated time does not accurately reflect the time they spent.
  • Billable hours: This value, along with having a Labor item, will set a line item on the work order.
  • Notes: this value is entered by the mobile user.
 

You can Tab through the fields to make changes.  If you hit Enter, this will save those changes and close the editing feature for that row.

 

You can also select to remove an entry from the Action options.  Any work order line item created as a result of that entry will also be removed.
