Default Contract Status

 

 

At a company level, you can determine in what status a new contract is started in. You have the options of None, Quote or Reservation.

The configuration setting is located in the Transaction area of Configuration. There is security that controls if a user can see/access it.

 

None

 

This opens a new contract in no defined status. The status option will be Required. The user must select an option to create a new contract.

 

Quote

 

This opens a new contract in Quote status.

 

Reservation

 

This opens the contract in Reservation status.
