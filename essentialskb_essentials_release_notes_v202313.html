Release Notes for v2023.13 (9/19/2023)

 

 

Our development team has been working hard and working overtime to come up with some really great features that have been asked for by you. We love our customers, so we have made the changes you have asked for. We have marked those features so that you know they came from you.


Once the release is complete, just refresh your browser to get the new update. Hit Ctrl-F5 in Firefox and IE, or Ctrl-Shift-R in Chrome.

  • Different Due Dates per item - We now have a due back field for each item so if an item is due back at a different time than the others, you can set it and the price will calculate correctly.

 

  •   Rental Dates on Invoices - In the past, the current end date for each item was the transaction due date. We have now made this the end date of each line item for more clarity to your customers when billing.
 
  •   Menu Pulldown Enhancement - It used to be that when you hovered over the menu at the top, that menu item stayed open if you didn't select anything and moved the mouse away. Now it will close automagically for you.
 
  •   Prepared for Sendout Filter - If you have marked a contract as 'Prepared for Sendout', you now can filter all your contracts on the workbench to see all of them prepared, or all of them not prepared.
 
  • Contact Us - Have questions and need contact information? There is now a Contact Us link at the bottom of the page.
 
  •   Multi-Store Manage Maintenance from HQ Store - If you are multi-store, you now have the ability to manage maintenance schedules from the HQ store. If there is not a maintenance schedule in an individual store, it will then look at your HQ store to see if you have set up global maintenance on the item.

 
Storefront Enhancements
 
  • We have heard your comments and are expanding the image size for images on the Product Detail Page!  In the recent past, they were 220 x 220. With this release image sizes now support up to 1000 x 1000.  The new standard image size will be 600 x 600 pixels and if the image loaded is larger a magnifying glass will appear on the image indicating that a larger image is available to zoom in on.
 
  • For mobile devices, the images are also enlarged and effectively go edge to edge on most screens (with appropriate padding).
 
  • In addition, we've also added a carousel of images for the thumbnails, so you can load more images/videos.
 
  • The upcoming changes to the Product Detail Page layout will make these upgrades really stand out!
 
  • We've made some tweaks to the Product Listing Page as we dial in our new styling, and improve the loading of the page.
 
  • Change the "Calculate Best Rate" and "Best Rate" display to read "Calculate Rental Price" and display "Rental Price" after it's calculated.
​​​​​​​
  • Centered the display of information within the box, for better readability on desktop and mobile.
