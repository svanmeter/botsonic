Metered Items on Varying Hour Shifts





This article will walk you through configuring your rate engine to accommodate metered items to run on varying shift lengths (8 hours, 16 hours, 24 hours, etc..). To do this, you'll need to create multiple rate engines. Once created, it's recommended to assign them to either a jobsite or contract.

Creating a Rate Engine
 

As mentioned above, you'll need to create multiple rate engines; one for each shift length.

How to get there

Expert: Program Menu > Configurations > System Configurations > Rate Engine
Elite: Gear icon > App configurations > System Configurations > Rate Engine



 

1. Select the Add button.
 

2. Enter a name.
 

3. Configure the Overtime and Clock Overtime area. For for information, see our article Standard Rate Engine.

 

4. Set up the Meter Overtime area:

  1.  In the Hours in a day field, enter the amount of hours for the shift before it goes to overtime (8 hours, 16 hours, 24 hours, etc..).
  2. Enter the Days in a week and Weeks in a month.
  3. Enter a "% charge for a meter overtime". This sets the percentage of hour meter overtime charged to the bill. An entry of 50% would only charge half of the normal overtime rate for hour meter usage.
 

5. Press OK to save.

 

6. Repeat steps 1-5 until a rate engine is created for each shift.

Applying the Rate Engine to a Jobsite

Here's how to get there
Expert: Program Menu > File Maintenance > Job Site
Elite: Gear icon > File Maintenance > Job Site
 

1. If you need to create a new job site, select the add button. Otherwise, select the desired job site. 

2. Select the Pricing tab. There is a drop-down menu for rate engine on the right side of the window.  


3. Select the desired rate engine from the drop-down.
 

 




 

   Applying the Rate Engine to a Contract (Whole Contract)

 

1. To change the contract Rate Engine, go to the Information Tab on the contract.

 

 

2. Change the Rate Engine to the appropriate time frame that you need.
 


 

Applying the Rate Engine to a Line Item on the Contract
 

1. To change the rate on the line item, you can do that by right-clicking on the line item of the contract.
2. Click on Set Special Rate.
 

 

 

3. Under the Special Rate area, you can choose the Rate Engine that you need.


4. Press OK when you are done.
 



 

 Verification by Counter Staff (when applying Rate Engine to contract)

It is recommended that you create a contract question to remind counter people that they will need to verify the Rate Engine.

 

How to get there

Expert: Program Menu > File Maintenance > Item File > Functions > Contract Questions

Elite: Configuration Gear > File Maintenance > Item File > Functions > Contract Questions
 

 

Enter the question that you want to verify with your counter staff, then press OK.
