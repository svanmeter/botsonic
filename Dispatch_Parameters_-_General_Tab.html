Dispatch Parameters

 

Overview: You can set different criteria within the dispatcher center on how information is viewed, printed, assigned, sent, and received. When creating and sending out daily routes.

 

Allows you to define overweight warnings, Driver Packets, Map Options, GPS settings, and both Customer and Dispatcher Notifications.

 

General Tab

 

Warning for Overweight -  Use the up/down arrows to change the level at which you will be warned about overweight loads. This is separate for pickups and deliveries

Late Stop Grace Period – Amount of time used as a buffer in determining if a stop is on time, late or early. Using the default of the time entered in the grace period.

Minimum On-Site Time – Allows you to set the minimum amount of time a driver and crew are to be on-site.

Warn prior to showing this many stops –  This prompts a warning message when viewing the maximum number of stops allowed.

Refresh Transaction Data Interval – Refreshes the contract information in set intervals. Ex. Enter 5, refreshes every five minutes.

User Define Alias  Rules -  You can define the minimum number of acceptances and minimum match rate percentage required to lock in an alias. The default is 5 accepts and 65% match rate.

Locked Route Admin Level - This allows you to select the security level to lock routes. You can select from (Owner, Regional Manager, Manager, Assistant, Counter, None).

Allow Date Shifting of Deliveries – Checking the box allows you to shift the date of the deliveries

Allow Date Shifting of Pickups – Checking the box allows you to shift the date of the pickups.

Allow Date Shifting of Service – Checking the box allows you to shift the date of any service route.

Always Show Missed Events for Today – Checking the box will display missed routes for the day.

Optimize for a Service Company

Sort Truck Tree Ascending – Check will sort the trucks in ascending orders.

Use Metric Standards for weight/distance – Checking the box will use metric standards when calculating the weight and distance for the vehicles.

Maintain trip modes between days – checking the box will allow you to maintain trip mode from one day to the next.

Auto Close old Logistic Tickets – Checking the box will allow Point of Rental to auto-close any old logistic tickets.

Contract Grid Item - You can select the item's name and quantity shown when looking at the contracts grid but not showing all item rows.

First Item - the first item on the contract

High Cost - The highest purchase priced item

High Quantity - The item with the highest quantity on that contract

This allows you to decide the one line that would be most helpful in your decision making without requiring the display of all contract lines.
