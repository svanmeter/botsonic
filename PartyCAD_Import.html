PartyCAD Import



This article will discuss PartyCAD Import. This option allows you to import the items on a PartyCAD drawing directly into a quote. Before exporting items from PartyCAD, you must enter item keys into PartyCAD so that it knows which item to select for the quote.

To do so, export your items from the PartyCAD drawing into an export file in the \POR\Pictures\Party CAD\directory. Once you select this option, you will see all the export files in that directory. Select the desired file and click OK. A quote will then be shown with all the items on the quote. This is a good way to check for availability quickly without having to manually type all the items into the quotes.

For more information about exporting your design items from PartyCad, visit this link: http://www.hufsoft.net/NetF/Help/index.html?export_list_of_design_items.htm

 



Show Imported - This will show you a list of files that have been imported.
