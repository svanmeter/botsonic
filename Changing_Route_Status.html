Changing Route Status [Changing Route Status]

Ändern des Routenstatus


Übersicht


In diesem Artikel wird erklärt, wie Sie den Status einer Route im Modul Dispo-Zentrale ändern können.

 

Die in diesem Artikel enthaltenen Themen:

 

·       Ändern des Routenstatus

·       Beibehalten der Stoppabfolge

 

 

Ändern des Routenstatus

 

1. Sobald der Auftrag einem Lieferwagen zugewiesen wurde, müssen Sie zu diesem Lieferwagen gehen und den Status der Route ändern. Suchen Sie im linken Fensterbereich der Dispo-Zentrale den Lieferwagen, dem Sie die Route zugewiesen haben. Unter der Lieferwagen-Nummer wird Ihnen die zugewiesene Route angezeigt.  



2. Klicken Sie mit der rechten Maustaste auf die Route unter der Lieferwagen-Nummer. Wählen Sie Routenansicht.



3. Hier können Sie den Routenstatus über das Dropdown-Menü ändern. Wenn Sie fertig sind, haben Sie die Möglichkeit zu speichern und zu schließen, nur Raster oder Karte und Raster anzuzeigen. Wählen Sie abschließend Speichern.


4. Wenn der Status geändert wurde, können Sie den Vertrag aus dieser Route in eine andere Route ziehen und ablegen.

 

 

Beibehalten der Stoppabfolge

 

Wenn Sie mit der rechten Maustaste auf eine Route klicken, haben Sie die Möglichkeit, einen Vertrag auf den nächsten Werktag zu verschieben. Sie können dann die Option wählen, die ursprüngliche Stoppabfolge beizubehalten, wenn Sie einen Vertrag oder eine Route auf einen anderen Tag verschieben. Dadurch werden die Stopps in der Reihenfolge beibehalten oder ein vorrangiger Stopp wird an den Anfang der neuen Route gesetzt.
