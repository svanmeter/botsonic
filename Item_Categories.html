Item Categories



 

Item Categories are used for sorting and summarizing data in reports. It's very uncommon to use more than one item category in GL. You need to define each category that you desire to summarize, for example, Automotive, Delivery, or Lawn & Garden. 


Getting there

Here's how to get there
Expert: Program Menu > Configuration > Item Configurations > Item Categories
Elite: Configuration Gear > App Configurations > Item Configurations > Item Categories

Once you select Item Categories, you will see the following screen:
 

 

Division - The Division groups categories together. For example the Equipment and Party divisions.

Print - This option will print a listing of all item categories.

Add - This option will allow you to add a new item category. 

Delete Unused - This option will delete categories that don’t have any items.
 

To modify an existing item category, click on the desired category. Whether adding or modifying, the Edit Item Category screen will be displayed.

 

Edit Item Category screen
 


Styles - With the new Styles fields in the Record Details screen, you can now organize certain item record groups by up to three predefined styles. For instance, your linen records can now be organized by Linen Fabric (Style #1), Color (Style#2), and Size (Style #3). Or you can now have your Tent/Canopy category item records organized by Tent/Canopy Style (Style #1), e.g. Frame Tents, Pole Tents, and Canopies. In addition, the Item Category Edit screen will allow you to name your own three styles per category. If you have questions on how to use this feature on your integrated website, please contact our Web Design Team.

Max Discount - Each category can have a maximum discount percentage allowed. This value will override a discount put on a contract. For example, you could enter a 10% maximum discount for the Delivery category. If someone gave a customer a 20% discount, it would automatically decrease the discount on the delivery charge to 10%.

﻿Web Picture - If you are selecting a WEB image, right-click the image. Make sure you verify the width of the image, it needs to be 250 pixels or higher.
 

General Ledger Number - You can define a General Ledger number for each category. If you define a G/L Account number then any income amounts in that category will be broken out on the General Ledger Entries sheet. If no G/L Account is defined, then all the income in the category will stay in the summary accounts of Rental or Sales income.
 

Note: For multi-store systems, you can put “###”, “##”, or “#” anywhere within the GL Number to be replaced with the store number upon General Ledger Entry creation. For example, a GL Number of 1100-## will print 1100-01 for store #1 income and1100-02 for store #2’s income.
