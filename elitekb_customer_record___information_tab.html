Customer Record | Information tab




This article will go over the Information tab within a customer record. The record can be accessed through file maintenance. This tab contains the customer's status, type, credit limit, and other receivable information. As a reminder, the customer record fields can be hidden from users with a specific password level or lower. 


Topics in this article include
 

  • Record Type
  • Account Information  
  • Paid
  • Current Account
  • Notes






Record Type

In the Personal Information field, you will be able to specify the record type that you would like. The business record type will display a name field so you can enter the name of the business, whereas the individual type will show you an option to enter the customer's first and last name.



In the configuration area in Elite, you can set the way you want to display the names of the individual's record type. Under Customer Configuration > Name Format you can change the display type to First Name Last Name, or Last Name, First Name.



Account Information


Customer No - This is the number assigned by Point of Rental when a customer record is created.

Opened - This is the date the customer record was created.

Last Active - This is the date when customers placed their last order.

Previous Contract - This displays last contract number customer placed.

Status - This allows you to select the status of customer (Cash or Account).

Options -  This allows you to select different options about the customers (No Checks, Pay on Return, Pay on Open, Credit Hold). You can also enter alert options (Wanted, Undesirable). When creating a contract a message will appear, to let you know the status of the customer.

Group -  This allows you to further define customers by a particular group (Referrals, Internet, etc.).

Type -  This allows you to select a type of customer (Homeowner, Corporate, School).

Sales Rep - This allows you to assign a sales rep to the customer.

Commission - This allows you to set the type of commission assigned to the customer (5-year Account, House Account, Corporate Account).


 

Paid


Year to Date -  This displays the total income for the current year.

Life to Date - This displays the total income since the customer was created.

Last Year -  This displays the total income for the previous year.

Life No of Contracts -  This displays the total number of contracts, the customer has placed since they were active.


 

Current Account 

 

Quantity Out - This displays the total number of rental items on all contracts currently in an open status.

Last Paid Amount - This displays the amount the customer last paid.

Last Paid Date - This displays the date the customer last paid.

High Balance - This displays the highest balance the customer has on their account.

Credit Limit - This displays the customer's credit limit. To change the displayed limit, modify this field.

Current Aging - This displays the number of days the customer has a balance due.

Current Balance - This displays the amount the customer currently has outstanding.


  
Notes

Customers marked as Undesirable or Wanted will display in red within this area.
