Creating and Assigning Certifications

 

 

 

 

Overview
 

This article explains how to add new certification types for employee certifications. The certifications tab in the employee's info screen allows you to keep a record of any certifications an employee has received. You can enter how long the certification is valid for along with who provided the training and any cost associated with obtaining the certification.

 

 

Adding a certification type and Assigning Certifications

 

1. In Expert, go to Security > Employee Setup and enter your password. If you are in Elite, go to Gear Icon > Core Options > Security > Employee Setup.

 

2. Select an employee and enter your password when prompted.

 

3. From within the Edit Employee’s Info window, click certification types and then click Add. 

 

 

4. Fill in the required information in the Edit Certification Type window. 

 

 

5. Once the type has been added, select OK. A list of the certification types will be displayed.

 

 

6. Underneath the Certifications tab in the employee’s record, you will now be able to select the newly added certification type from the drop-down menu.
