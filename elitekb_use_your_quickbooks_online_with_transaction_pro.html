Use Your QuickBooks Online with Transaction Pro


 

This article explains how to transfer your general ledger information from the Point of Rental software to your QuickBooks Online accounting software, using a third-party software Transaction Pro (Required).
 

     
    Topics in this article include

    • Prerequisites
    • Connecting Your Transaction Pro Account to QuickBooks Online
    • Importing Your General Ledger to QuickBooks Online



     

    Prerequisites
     

    • Point of Rental accounts will need to be set up in the Account Number Configurations as well as QuickBooks.
    • Account Names should not have ANY special characters (#,*, etc.) as these do not import correctly. See our article on Account Numbers for more information.
    • A Transaction Pro account: This will require a subscription. 

     

    Connecting Your Transaction Pro Account to QuickBooks Online
     

    If you are using the online-based Transaction Pro (recommended) then please see this article on Connecting QBO to Transaction Pro. Click here.
     


    Importing Your General Ledger to QuickBooks Online

     

    Transaction Pro has an article on this which you can read by clicking here. 

     

    1. In Transaction Pro, click on Import File.
     

     

    2. Then click on Journal Entries.

     

     

    3. Import the General Ledger File.
     

    Note: This will have to be a .csv, .xls, or .xlsx format. Please see our Account Numbers article to make sure you choose the right format. 

     

    4. When you choose the file and click next you will need to map the columns to the right information.

     


     

    Reference numbers can be DOC Num or Trans Type. Map other columns as shown, and set the adjustment default value to true.

    5. Once set, save mapping options, give it a name (POR IMPORT, etc.), and save. This will save the settings for the next import. 
     

    6. Select Preview and import. This will show the data to be imported.
     

    7. Select Import, and it will ask you if you are ready to import. If so, say yes and if the data goes in, it will show imported data. If not, it will highlight the fields unable to import in red. Correct the data and try again.
