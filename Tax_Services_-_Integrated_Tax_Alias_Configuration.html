Tax services - Integrated Tax Alias configuration.  Tax details on contracts.

 

Tax integrations such as CCH or Vertex return underlying tax details that some users or their customers may find useful.  Things like a breakdown of specific underlying taxes or special taxes based on a location, item type or rental value.  While we summarize those in the right column totals, we allow for these tax details to be included on the contract form.

 

Integrated Tax Alias configuration

 

Located in the System area of configurations, this feature has multiple uses:

 

Set tax names you want printed on contracts

 

The option labeled: All tax names will print on contracts.  Add an Alias to change what prints.
 

 

This option, when checked, will default all current and future tax names to print on contracts.  As new and different tax names are sent from the tax service, they are added to the database and will be set to always print on the contracts.

 

When this option is not checked, you can set tax names individually to print on contracts using the ‘Show on Contract’ checkbox on each line.  These boxes are checked and inactive while the main configuration is checked.

 

Set an Alias for tax names

 

Another feature to this configuration is the ability to change what prints on the contract.  Instead of, say, Texas Emissions Reduction Plan, you could set an alias of ‘TERP’.  

 

These aliases allow for more tax details to print on the contract in a more readable format.

 

You can also search for tax names (includes the alias) as well as filter to only those that have an alias set.
