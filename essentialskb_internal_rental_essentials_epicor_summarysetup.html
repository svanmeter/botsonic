Epicor P21 into Rental Essentials is programmed completely by Epicor. We do not currently have a test environment for their product.

 

When they request a new setup (it should come from someone with an email address ending in  epicor.com), follow the steps below. The things required from them are:

  1. Company name
  2. Number of stores
  3. Number of users
  4. Number of users to allocate at each store (if more than one store)
  5. Company address/phone
  6. At least 1 user for the company with email address

 


Steps for Adding Company

 

  1. Login to epicorrentalessentials.com (if you don’t have creds, ask Joe W, Evan, or Robert F to provide for you).
  2. Go to Add company, and add Parent company. Note: Each setup must be set up as multi-store regardless of only 1 store or more…Parent…HQ…Store1..Store2…etc.
  3. Add the rest of the stores as you would any other multi-store (HQ, Store 1, more…etc.).
