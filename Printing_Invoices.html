Printing Invoices





This document will walk you through printing invoices. There are three ways you can print invoices: manually as you close the contract, manually for an entire day, or automatically at the end of the day. The automatic method removes the human element, ultimately eliminating errors (invoices not being sent to a customer). This expedites getting an invoice into your customer’s hands, you getting paid, and reducing accounts receivable aging. We will cover the last two methods in this guide.


Topics in this article include

  • Manual Method for Specified Date Range
  • Automatic Method for End of Day
  • Troubleshooting
 

How to get there
Expert: Program Menu > Accounts Receivable > Print Invoices
Elite: Right Menu > Searchbar > Print Invoices
 

 

 Manual Method for Specified Date Range
 


 

1. Select the beginning date for the invoices.
 

2. Select the ending date for the invoices.
 

3. Choose whether to include either Account, Cash, or both types of customers.
 

Note: It is recommended to include both Account and Cash.
 

4. Choose to include customers who have been marked as Email, Fax, or Print.
 

Note: This is generally only used to regenerate invoices due to a printer or email failure. If you do not have the Mass Fax/Email module this section will be grayed out.
 

5. Select the invoice format to use. The system defaults to the Invoice Format.
 

6. Select which store, or all stores, the invoice will generate for.
 

7. Select where you want the invoices to print from.

8. Click Generate to create the invoices.
 

Note: As with most reports, they generate to the screen where you can review them before printing.


 

Automatic Method for End of Day
 

You must have the Mass Fax/Email Module for automatic invoices. If you do not have this module you will not be able to automatically print/email/fax invoices at the End of Day. (Call for a quote)
 

You will need to take the time to collect email addresses and fax numbers.
 

Each customer record will need to be set to None, Email, Fax, or Print in the Contract Mass Send To field.
 

The recommended setup is to create a default customer and set the Contract Mass Send To field to Print or Email as desired. 
 

Note: A default customer record is a template for all new customers created. If the field is set to Print in the default customer record, then any new customer added to the system will be set to Print automatically.


Next, using Database Mass Change, you can set ALL customers to Print or Email.
 

Note: If you are not familiar with this process you should not attempt this step on your own, call support for help.

 

Once your customers are set up correctly, run your End of Day procedure and invoices will automatically generate based on the method selected for each customer. Only contracts that were closed since your last End of Day and have a balance due at the time of running your End of Day will generate.
 

Note: Invoices use a #10 windowed envelope for invoices and contracts.

 

 

Troubleshooting
 

Not all invoices are being sent?

This is only applicable if you are licensed for the Mass Fax/Email module. When sending an invoice, Point of Rental generates a PDF. This process may take a few seconds. If your Mass Fax/Email settings do not allow enough time for the PDF to generate, the invoice may not be sent. To increase the amount of time, follow these steps.
 

Expert: Program Menu > Configuration > Reports and Notifications > Mass Fax/Email Settings
Elite: Gear icon > App Configurations > Reports and Notifications > Mass Fax/Email Settings
 

1. Update the email delay to 5 seconds. 



How can I confirm which invoices have been generated?

Run the Print/Email/Fax Invoices report

Expert: Program Menu > Accounts Receivable > Print/Fax/Email Invoices
Elite: Accounting > Print/Fax/Email Invoices
 

 

1. Select your invoice settings.

2. Check 'Include invoices previously billed.'

3. Select Preview.

 
 

Add a BCC email to Mass Fax/Email Settings
 

Expert: Program Menu > Configuration > Reports and Notifications > Mass Fax/Email Settings
Elite: Gear icon > App Configurations > Reports and Notifications > Mass Fax/Email Settings
 

1. Select Email Setup.

2. Enter BCC address.



I have confirmed that not all of the invoices were sent. Do I have to resend all of the invoices to send out the ones that were not sent?
 

No. You can mark the invoices as "not sent". This will prompt Point of Rental to resend the invoices at the end of the day or the next time you generate invoices. To do this, follow the steps below:


Start with one to make sure you receive the expected result then move forward with the others.

Expert: Program Menu > File Maintenance > Transaction File
Elite: Gear icon > App Configurations > File Maintenance > Transaction File
 

1. Find the contract the invoice was generated from.



2. Select the record.

3. Select Functions > Modify. This option may require a high password level. 
 

Note: If you are unable to perform this action, please contact your Point of Rental administrator. 


4. Remove the billed date that is displayed and then Press Enter.


5. Press Okay.

6. Regenerate invoices manually, or wait for the end of the day (only applicable if you are licensed for Mass/Fax).
