Reports within a Contract


 

Transaction Edit

This report allows you to see each edit made on a contract. It displays the operator (who made the edit), date, store, action, status, Amt Changed, Paid, Payment Meth, Deposit, Modification #, and note.

 

 

Related Contracts

 Once a Quote has been converted to a Reservation, or a contract has been continuation billed, or a Service Call has been created from an Open contract, the related contracts will be stored with each of these examples. To observe them from within the contract, you can click the Related Contracts button to see all related contracts. Click on a record to inquire the selected contract.

 

 

Print Instructions

Allows you to print any instruction file in relation to the items on the contract.

 

Posted to Totals Report

This report is based on the Accounting Method that you have set in Parameters. It will break down the contact you are viewing by rental, sale, damage waiver, asset sale, finance charge, other, percentage, and sales tax.
