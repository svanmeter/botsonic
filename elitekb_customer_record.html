Customer Record

 

Security Options

Security options for the Customer record are found in the Customer area in Security.  There are multiple options for many of the fields.  

  • View - when set to Yes, these fields will be displayed
  • Edit - when set to Yes, a user can update the content of the field in Modify mode.
  • Required - when set to Yes, a value will be required to be added/selected.  This is indicated by a ‘Required’ label and an accompanying alert ‘badge’ on the tab.
 

Customer records can be saved as ‘incomplete’ to allow a user to continue with the rental process as opposed to setting erroneous values.  Alert badges will display while in ‘readonly’ to let the user know there are unsatisfied fields that need correction.  In modify mode, those fields will be identified.

 

 

Duplicate Customers

As existing customers are updated or new records created, there are 6 fields that will be checking the database for duplicate values.  Key, DL Number, Phone, Mobile, Email and, for Business Type only, Name.  

 

A hint will be displayed below the field when a duplicate value has been entered.  Simultaneously, a ‘Potential Duplicates’ button will be displayed in the footer of the form.  Selecting this button will display the customer(s) that have those duplicate values.  Selecting one from the list allows you to view and use that record.  Or, you can close that record and continue with the original one.

 

Records with duplicate values can be saved, other than a duplicate to Key.  Key is required to be unique for each Customer record.



 

Related records

 

 

On the upper right side of the Customer record form are icons for related records.  These can be filtered and sorted to view and find them as the user prefers.  Records from the list can be selected where there is a hyperlink symbol.  This will create a ‘breadcrumb’ link to the Customer record.  Below is an example of going from viewing a Customer record, then selecting a Call record from the related records list to then selecting a Contract link from the Call record.
