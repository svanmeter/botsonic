Bestellung stornieren

Wird verwendet, um eine bestehende Bestellung zu stornieren. Dies würde zur Anwendung kommen, wenn eine bereits ins System eingegebene Bestellung nicht benötigt wird. Die Option zur Stornierung löscht alle Auftragspositionen der Bestellung und ändert den Status zu „Storniert“. Wenn Sie der stornierten Bestellung später wieder Artikel hinzufügen, ändert sich der Status der Bestellung zu „Offen“ oder „Abgeschlossen“. Um eine Bestellung zu stornieren, müssen Sie die gewünschte Bestellung auswählen.


Klicken Sie mit der rechten Maustaste auf den Vertrag, den Sie stornieren möchten.

Der Vertragsstatus wechselt von grün auf rot und die Mengen werden auf Null gesetzt. Der Status der Bestellung wird auf STORNIERT geändert.


Klicken Sie auf die Schaltfläche „Anwenden“, die Artikelzeile wird ROT markiert.

Wählen Sie „OK“

Sobald der Vertrag storniert wurde, verschiebt er sich auf dem Hauptbildschirm vom Tab „Offen“ zum Tab „Storniert“.
