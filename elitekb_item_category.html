Artikelkategorie

Artikelkategorien werden zum Sortieren und Zusammenfassen von Daten in Berichten verwendet. Sie müssen jede Kategorie definieren, die Sie zusammenfassen möchten, z. B. „Automotive“, „Lieferung“ oder „Rasen und Garten“. Nach der Auswahl von „Artikelkategorien“ wird der folgende Bildschirm angezeigt:

Sparte - Die Sparte fasst Kategorien zusammen, z. B. Ausrüstung und Party-Sparte.

Unbenutzte löschen - Mit dieser Option werden alle Kategorien gelöscht, die keine Artikel enthalten.

Drucken - Mit dieser Option wird eine Liste aller Artikelkategorien gedruckt.

Hinzufügen - Fügt eine neue Artikelkategorie hinzu.

 

Um eine vorhandene Artikelkategorie zu ändern, klicken Sie auf die gewünschte Kategorie. Unabhängig davon, ob Sie eine Kategorie hinzufügen oder ändern, wird der Bildschirm „Artikelkategorie bearbeiten“ angezeigt.

 

 

Sachkontonummer - Sie können für jede Kategorie eine Sachkontonummer definieren. Wenn Sie eine Sachkontonummer definieren, werden alle Einnahmen in dieser Kategorie auf dem Blatt „Sachkonto-Einträge“ aufgeteilt. Wenn kein Sachkonto definiert ist, verbleiben alle Einnahmen in der Kategorie in den Summenkonten der Miet- oder Verkaufseinnahmen. Hinweis: Bei Multi-Station-Systemen können Sie „###“, „##“ oder „#“ an beliebiger Stelle in die Sachkontonummer eingeben, die dann bei der Erstellung der Sachkontobuchung durch die Stationsnummer ersetzt wird. Zum Beispiel wird eine Sachkontonummer von 1100-## 1100-01 für die Einnahmen von Station 1 und 1100-02 für die Einnahmen von Station 2 gedruckt.

 

Max. Rabatt - Jede Kategorie kann einen maximal zulässigen Rabattprozentsatz haben. Dieser Wert setzt einen auf einen Vertrag gesetzten Rabatt außer Kraft. Sie könnten zum Beispiel einen maximalen Rabatt von 10 % für die Kategorie „Lieferung“ eingeben. Wenn einem Kunden ein Rabatt von 20 % gewährt wird, würde der Rabatt auf die Lieferkosten automatisch auf 10 % reduziert.

 

Stile - Mit den neuen Feldern „Stile“ im Bildschirm „Datensatzdetails“ können Sie jetzt bestimmte Artikeldatensatzgruppen nach bis zu drei vordefinierten Stilen organisieren. Zum Beispiel können Ihre Leinen-Datensätze jetzt nach Leinen-Stoff (Stil #1), Farbe (Stil #2) und Größe (Stil #3) geordnet werden. Oder Sie können Ihre Datensätze der Kategorie Zelte/Deckensegel jetzt nach Zelt/Deckensegel-Stil (Stil #1) ordnen, z. B. Rahmenzelte, Stangenzelte und Deckensegel. Außerdem können Sie im Bildschirm „Artikelkategorie bearbeiten“ (unter Konfiguration; Artikelkonfigurationen; Menü „Artikelkategorien“) Ihre eigenen drei Stile pro Kategorie benennen. Wenn Sie Fragen zur Verwendung dieser Funktion auf Ihrer integrierten Website haben, wenden Sie sich bitte an unser Webdesign-Team.

 

Web-Bild - Wenn Sie ein WEB-Bild auswählen, klicken Sie mit der rechten Maustaste auf das Bild: Stellen Sie sicher, dass Sie die Breite des Bildes überprüfen, sie muss mindestens 250 Pixel betragen.
