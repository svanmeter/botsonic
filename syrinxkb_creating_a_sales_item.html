Syrinx® - Creating a Sales Item

 

 

 

 

Overview: This guide is to help with creating a sales item in Syrinx.

 

 

Sales stock items are categorized by a set of Sales Categories, similar to the way Hire Categories are used to group the different hire stock items.  Before adding the sales stock items to the system you must first decide how the sales stock is to be organized and create an appropriate set of Sales Categories. 

 

From the stock tab, select the menu option Stock > Setup > Categories. Press the Add New button that is displayed. The Add New Sales Category will be displayed.

 

 

Enter a code for the sales category. Note that the code must be unique across all sales and hire categories. It is recommended that you use eight characters or less.

 

Enter a name for the category.

 

If you want the system to allocate stock numbers automatically as new sales stock items are entered enter a stock number prefix.

 

If sales of items in this category are not subject to the default Tax Rate, then select a different Tax Code from the list.

 

You can optionally set a default profit margin percentage against the category. If set, then when you amend the purchase price of an item, the system will automatically increase the sales price (if required) to match the default margin.

 

The Sales Stock Items screen is used both to enter new sales stock items and to look up and modify information on existing sales stock items.

 

To add a new sales stock item: Select the menu option Stock > Sales Stock > Stock List.

 

Select the category that the new item is to be added against, from the drop-down list, then press the Add button.

 

 

To confirm the new stock item select OK.

 

Sales items can also be added to the system whilst entering a hire contract, advice note, or purchase order/receipt, by pressing the Insert key when positioned on the stock number field.
