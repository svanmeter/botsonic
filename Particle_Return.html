Particle Return [Particle Return]

Teilweise Rückgabe


Mit der Teilrückgabe können Sie eine gesamte Auftragsposition oder eine Teilmenge einer Auftragsposition zurückgeben. Sie legen fest, welche Auftragspositionen zurückgegeben werden und wie viele. Wird nicht die gesamte Menge zurückgegeben, wird dem Kontrakt eine neue Auftragsposition mit der Menge hinzugefügt, die der zurückzusendenden Nummer entspricht. Die ursprüngliche Auftragspositionsmenge wird um den gleichen Betrag verringert. Für diese Art des Abschlusses müssen Sie weniger als 200 Auftragspositionen auf dem Vertrag haben.

 

Artikel auswählen



Geben Sie die zurückgegebene Menge ein



Sie werden sehen, dass eine neue Auftragsposition erstellt wird, die die Anzahl der zurückgegebenen Artikel anzeigt. Die ursprüngliche Auftragsposition, die zeigt, was noch vermietet ist, bleibt bestehen.



Dies kann auch für Verkaufsartikel erfolgen. Wenn etwas in einer Station verkauft und in einer anderen zurückgegeben wurde, werden diese Verkaufsartikel zu den Massenverkaufsartikel des aktuellen Artikels hinzugefügt, nicht zu den Verkaufsartikeln der Station.

 

Wenn dies bei einem vorausberechneten Fortsetzungsvertrag geschieht, hat der Benutzer die Möglichkeit, den Preis auf Null zu setzen.
