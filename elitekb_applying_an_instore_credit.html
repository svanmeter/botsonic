Anwendung eines Laden-Guthabens


 

Übersicht

Die Anwendung eines Laden-Guthabens zeigt Ihnen, wie Sie über den Zahlungsbildschirm ein Guthaben auf einen geschuldeten Betrag für einen Vertrag anwenden können.

So gelangen Sie dorthin:

Expert: Programmmenü > Debitoren > Debitorenmanager

Elite: Buchhaltung > Dashboard Debitoren


 

1. Wenn der Kunde ein Guthaben auf seinem Konto hat, kann er es auf der Zahlungsseite eines jeden Vertrags als eine seiner Zahlungsarten auswählen. Wird das Guthaben nicht vollständig aufgebraucht, bleibt der Restbetrag auf dem Konto des Kunden erhalten.

2. Wenn der Vertrag durch die Inanspruchnahme von Laden-Guthaben nicht vollständig bezahlt wird, können Sie den Restbetrag bezahlen oder gegebenenfalls eine Anzahlung annehmen.
