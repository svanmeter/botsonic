Item Income Trend by Store Report

 

This report allows you to select four time periods for income analysis.  This is useful for comparing income for the first quarter for the last four years or comparing the fourth quarter, third quarter, second quarter, and first quarter of the current year.  This report can be printed in detail where each item’s detail shows or in summary where just the category totals are shown.

 

When you select this option, you will get the following menu:

 

Category – Use this button if you want to select a single category for the report.

Store to Report – This pull-down menu determines if all or only one store is to be reported.  It is only available to multi-store systems.

Rental Items – Check this box to include rental items in the report.

Sale Items – Check this box to include sale items in the report.

Print Details – Check this box to list each item along with its income.  With this box unchecked, only the category totals will be reported.

Beginning Month – Select the beginning month for the report.

Beginning Year – Select the beginning year for the report

Ending Month – Select the ending month for the report.

Ending Year – Select the ending year for the report.  Note: the ending month and year must be later than the beginning month and year.

Same Period Each Year – This button will set the same beginning and ending months for periods 2 through 4 to be the same as period 1.  It will also set the years to be the previous four years.

Printer Select – This button allows you to change the default printer for the report.
