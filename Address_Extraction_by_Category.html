Adressenextraktion nach Kategorie


Übersicht

Mit diesem Bericht können Sie eine Liste der Kundenadressen exportieren, indem Sie die gewünschte Miet- oder Verkaufsartikelkategorie auswählen und alle Kunden auflisten, die etwas aus dieser Kategorie gekauft oder gemietet haben.

In diesem Artikel enthaltene Themen

  • Adressenextraktion
  • Adressextraktions-Export


 

So gelangen Sie dort hin:

Expert: Programmmenü > Marketingberichte > Adressenextraktion nach Kategorie

Elite: Rechtes Menü > Berichte-Symbol > Adressenextraktion nach Kategorie


Adressenextraktion


1. Sie werden das untenstehende Menü sehen. Wählen Sie die gewünschte Artikelkategorie. 



2. Sobald Sie die gewünschte Kategorie ausgewählt haben, werden Sie aufgefordert, ein Datum auszuwählen. Der Bericht zeigt die Objekte in dieser Kategorie an, die seit dem ausgewählten Datum vermietet wurden. 



Adressextraktions-Export


1. Klicken Sie bei geöffnetem Bericht auf das zweite Symbol in der oberen linken Multifunktionsleiste. 



2. Ihr Bericht wird exportiert und in Excel geöffnet.


Hinweis: Wenn Sie keinen Zugang zu Excel haben, können Sie Ihren Bericht trotzdem exportieren.

 

(Zum Exportieren ohne Excel)


3. Klicken Sie bei geöffnetem Bericht auf das sechste Symbol in der oberen linken Multifunktionsleiste. 



4. Geben Sie einen Dateinamen ein, wählen Sie den Dateityp und den Speicherort aus.
