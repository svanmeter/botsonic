Quote Analysis Report

This analysis report shows the number of quotes written each month and the dollar amount of the quotes (normal and work order quotes). It will also show the number of quotes canceled and a number of quotes converted to reservations.

In this report, drill down to view contract detail. For example, if you double-click the Quotes value highlighted below in blue, you'll be able to see additional information.
