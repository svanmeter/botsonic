  • Tax Rate Name on Invoice - Previously, the contract had the tax rate name, while the invoice pretty much just said ‘tax’. We made them consistent. Now the Invoice shows the tax rate name.
 
  • Tooltip for Item History - In each item record, there is an item history tab. You all knew that right? It caused a little confusion because it only gives the last 6 months of data. Its design is really to show you the last few rentals in case you need to see the last few people that rented it in case you found damage afterward. However, we weren’t very clear on that last 6 months thing. So we added a tooltip that explains that at the top, and also a link to the transaction list report in case you want a longer set of data.

Note on Storefront Changes

We are constantly working to improve the user experience, performance, and visual layout of the Point of Rental Storefront service. As part of our ongoing efforts to improve the user experience, we have identified necessary structural changes to the Storefront HTML that include a high chance of impacting Custom CSS layouts. If you have Custom CSS, please follow the steps below to make sure you are not impacted.
 

  • Review your Custom CSS
  • Please check your storefront to confirm there are no conflicts with your custom CSS
  • Make any changes needed to ensure that your designs still function as intended
  • If you need to remove any custom CSS, please find the instructions here.
