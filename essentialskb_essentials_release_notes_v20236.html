Release Notes for v2023.6 (4/24/2023)

 


 

 
 
 
  • Transaction Type - We have added functionality so that you can define the types of Transactions and assign them to each contract. Things like 'Annual Event', 'Company Party', 'Government Bid'....etc. Note: These can be used by any Rental Company, Party/Event/General Rental/Heavy Equipment, whatever you want to use it for. For your benefit, we have also added this as a filter, and a field, as a selectable column in the reports listed below so you can report off of it.
    • Transaction History
    • Transaction List
    • Transaction List by Current Closed Date
    • Transaction List by Item ID
    • Transaction List by Invoice Date
 
  • Editing Deposit for Quotes - We have now added the ability to change the deposit amount in Quote Status. Previously you could only do this when taking payment in a reservation.
 
  • Delivery Vehicle on Picklist - A while back, we added a very well received feature called Delivery Vehicles. Very cool stuff from our development team. Then we added them to Transactions. Then we added them to the delivery/pickup reports. Then we added it as a filter on the workbench. Now you want them on Picklists. You got it.
