Employee Email/Logs

The feature permits anyone with a valid operator ID to send and receive emails to or from anyone else on the system with a valid operator ID. * Note: This is a private Email system within Point-of-Rental software. It should not be confused with Internet Email.

 

If Employee email/Logs is selected the following options will be displayed:

Send Message

Read Interoffice Email

Add Employee Log

View Employee Log
