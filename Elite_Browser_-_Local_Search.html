Local Search (Elite browser interface)

 

When searching for anything like an Item, Contract, Customer, etc., you will always start in the upper left Search box.  This search will produce a list of returns based on the thing you are searching for and the value you put in the search box.

 

This will display a new Dashboard showing the results.  From here, you can do what is called ‘Local Search’.  You can filter these broad results into more specific results by adding additional filters.

 

There are ‘pre-loaded’ options for each thing you may be searching for.  At this point you have several options:

 
  1. Select one of the pre-loaded options and add a value to filter your results by.  For instance here is a Contract search.  I could further filter these results by the Status:

 
  1. You can continue to add filters to the ‘pre-loaded’ options, or you can add a new filter by selecting the ‘New Filter’ option.  This will produce a searchable list of all options you can filter by:

 
  1. A ‘down arrow’ indicates an option(s) has been selected in the filter and a ‘+’ indicates there is no filter option selected:
 
  1. You can also completely modify the default filters and settings and save your own.  These selections will appear, by default, whenever you trigger a result for that object.
 
  • There are four default system views: Quotes, Reservations, Open Contracts, and Internal Work Orders. The view filters can be manipulated, but the original settings will always persist.  If the user tries to save a manipulated system view, it will be saved as a customer user view and be labeled ‘Reservations-COPY’(for example).  It can then be further edited as a saved user view if so desired.  System views will be separated from user views and will be at the top of the drop down.


  • You can also designate one of them to be your ‘Default’ view.  This means when a new search grid is opened, that one will be selected automatically. This is selected by checking the green arrow on the right of the selection box.
  • Making any changes to a Saved Search will prompt the Save option to display so you can make those changes permanent.  Or, select cancel if it is a temporary change.
  • Clear Page View - If you are currently using a Saved Page View and decide you want to start from scratch and have no filters set, select this option.  You can always go back and re-load your Default view.
  • Delete Page View - This will bring up a list of your Saved Views.  From here you can select one or many of them.  And Delete them.
 

Each View you create is unique to the thing you are searching for.  A Contract search view is unique and only available from a Contract search.
