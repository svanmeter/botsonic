Tools

 

1. Show item- opens the item file

2. Deliveries & Pickups- changes the 4-week calendar to display the number of contracts with a delivery/pick up an item on it

3. Linked Item- will display an item that has been linked in the Options tab of the current item record

4. Show Web Page- opens the web page associated with this item

5. Change User- allows you to change user

6. Open Contract- opens a new contract for this item

7. Open Reservation- opens a new reservation for this item

8. Open Quote- opens a quote for this item

9. Open Internal Repair Order- opens an IRO for this item

10. Open Internal Maintenance Order- opens an IMO for this item
