Web Reservation Verification

Overview: Enables a rental store customer to confirm a reservation or request changes electronically through your integrated website. This functionality must be turned on throughwww.rentalhosting.com to activate the verification link.
 

  • With this feature enabled, the customer receives the emailed PDF reservation with the additional Verification link at the bottom of the email
  • The customer will review the emailed PDF reservation (attached in the email) – then if they click on the Verification link:
    • They will see a web page similar to this where they can confirm the reservation as is by selecting “I have read my reservation and agree to its terms and conditions” or request changes.



Adding the web verification link to an email message: Navigate to Program Menu > Configuration > Store Configuration > Email Letters > Select or create a new message to include your web verification link. Position your cursor in the message where you want the link to be placed (typically toward the bottom of the email), then use the Special Fields button to select {WebVerificationLink}. This tag will be replaced with a clickable web link to your website. You can setup varying messages for different situations, such as Will-Call or Delivery, so an appropriate message is delivered to your customer.

Sending an email with a verification link: Anywhere you can email a contract to a customer, including Day at a Glance for bulk emailing (bulk emailing does require the Mass Fax/Email Module), you can select your email letter containing the verification link. You can modify the message or send it as is.

If changes are requested by your customer, an email will be sent to the email address set up at www.rentalhosting.com. 

If the reservation is confirmed, a call log is written to the contract and the contract is marked as confirmed (Confirm Reservation or Delivery Confirmed accordingly). Verifying a contract turns the appropriate date fields red in the Day at a Glance screen for easy identification.

Using the Day at a Glance bulk email feature (Mass Fax/Email module required): When you send bulk emails for a specific day or range of days, reservations that have previously been sent will only be sent again if a change has been made to them since the last verification email was sent. 

You need to run Web Quote Import or Update Website, which runs every night, for the information to be brought from the Rental Hosting’s Server to the Rental Store’s Server 

Configuring the Web Verification Link: Below is an example message. Please change according to your business standards.
