Canceled Contracts Report


Location: Launchpad>Right Side>Reports Icon>Canceled Contracts Report


Overview: This selection produces a report of all contracts that were canceled before giving a contract number.  This is not an uncommon function; however, an employee could act like they were writing a contract then cancel it if a customer didn’t want a receipt.
 

Instructions:

1. First, you must enter the beginning date
2. Enter the end date.
