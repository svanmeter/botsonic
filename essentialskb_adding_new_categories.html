Topics included in this article:

 

  • Creating a category
  • Creating a category from the inventory page
  • Adding inventory to a category

 

 

Creating a category

 

1. Go to Inventory > Categories and select + New Category.

 

2. Add the name of the category, description (if desired), and picture (if desired). You can even enter an Income Account Type as well. 

 

 

 

3. Click Save and you now have a new category or group.

 

 

Creating a category from the inventory page

1. To create a new category from the Inventory page, go to the Item Information and click the down arrow for Category. Click Add New.

 

 

2. Enter the name for the category that you would like this item to be in. When you are done press Save on the item.

 

 

3. You can add more information about the category by going to Inventory > Categories and filling in the description (if desired) and picture (if desired). You can even enter an Income Account Type as well.

 

 

Adding inventory to a category

Once these categories have been created and saved, you can edit your inventory items and select the appropriate category from your drop-down menu. Click Save. 

 

 

Now, when you search for items within a certain category or group, either when creating a new transaction or on your storefront, you can enter the category name and everything in that category will appear. This will narrow down your search results by including only those items within the category.

 

Note:  You can change or rearrange the order in which your categories appear by clicking on a category and holding the mouse while dragging it to the position you want.
