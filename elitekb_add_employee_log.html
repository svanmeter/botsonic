Add Employee Log

 

 


Overview

Add Employee Log allows you to enter notes about your employees. The employee log can be used to keep track of any warnings, reprimands, concerns, or positive recognition for specific employees. 


 Here's how to get there: 
Expert: Program Menu > Counter System > Employee Email/Logs > Add Employee Log
Elite: User Tools > Employee Email and Logs > Add Employee Log

 

 

1. Select an employee. A new window will appear with a populated list of all your employees registered within the software. Press Ok once selected.

2. Select Log Type.

3. Select Password Level.

4. Use the dropdown to select the type of log: general notes, recognition, reprimand, tardy, and verbal warnings.

5. Use the dropdown to select the password level required to view notes: anyone, the manager only, or the owner only.
 

6. Press Ok.
