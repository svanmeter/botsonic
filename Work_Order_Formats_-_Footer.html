Work Order Formats - Footer
 

Location: Program Menu>>Configurations>.System Configurations>Work Order


Purpose: The footer tab allows the user to change the look of the information located at the bottom of the Work Order by selecting different available options. T

Key Features:  Key functions of the footer tab include:

  • 16 different footer types of footers.
  • Work order totals that separate labor and parts. 
  • You also can turn on or off features like the owner’s signature or the customer signature. Or
  • Turn off the totals section or the footer entirely.
