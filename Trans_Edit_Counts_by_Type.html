Trans Edit Counts by Type

 

This report will break down each type of contract by Open, Close, Modify, Open Reservation, Cancel Reservation, and Send Out Reservation.  The count of each type as well as the total rental, sales, damage waiver, and total amount will also be printed.  For each type, the maximum amount of each contract and the average amount of each contract is computed.  This report groups each type of contract by the type.  The report will look like the following:
