Overview: This guide is to help you in creating kits and inclusions for fleet.

 

 

Inclusions are where you can link fleet items to other fleet items where they may be regularly used or supplied with that specific fleet. You can add inclusions to the category when amending the category within the Incl Hire tab.

 

 

You can make the selections have no charge by selecting the no charge flag, pre-selected and making the inclusion mandatory.

 

The same process can be made against individual fleet items by amending the fleet item and updating the Inc Hire tab of that specific fleet item.
 

 

Kits are where there is a package of items that make up the hire. The package or kit would be created as a bulk item in Syrinx, and the individual fleet items can be added to it as inclusions.

 

Create a category for the intended package that has the individual items flag unselected, and then create the bulk item for the package in bulk items.
 

 

Now amend the newly created package, and include the hire assets which would make up the package.

 

 

When selecting the individual items within the package, you can remove the asset number so it can be selected from the hire contract at the point of generating the hire.

 

You can also allocate the hire revenue to the included items within the package by selecting the flag on that screen. Any item that is to have revenue allocated within the package will need to have a default hire rate set against it.

 

When creating the contract for the package, you would then select the individual items based on availability.
