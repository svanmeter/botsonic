eCommerce | Troubleshooting: Web Quote Import Not Working/Website Not Updating

 

 

 

When web quotes are not populating the Web Quote Import window, or a website is not updating, first try to run Update Website in POR (Configurations -> Other Updates) See Updating Your Website in Auto Mode .


If it is successful, skip down to the POR Scheduled Task section in this article. Otherwise, continue with the IP Address section in this article.

 

 

 

 

IP Address

 

Reasons IP addresses change:

  • changing IS providers
  • moving your server
  • installing a new server or updates
  • IS providers are also known to change non-static IP addresses at random.
  • etc.

We always recommend that the customer ask their provider for a static IP. This will prevent the IP from changing and causing trouble!

 

To get things back up and running properly, we need to enter your current IP address into our firewall.


To find your correct, current IP address, please go to porhelp.com and click on the green “Show My IP Address” button, then send us the resulting IP address.
 

 

We will then enter the new IP address into our firewall and have you run the Update Website process to be sure it's working properly.

 

 

 

 

POR Scheduled Task

 

If you are able to update your website manually, but it doesn't automatically update overnight, the culprit most likely is the Scheduled Task set up in POR.

Check the date/time stamp in the bottom right corner of your website, and if it is not up-to-date with today's date, please contact support.
