Suppress Fulfillment

You can now set an item to suppress fulfillment so it does not show in the fulfillment process. This is useful for items like delivery, discount items, coupon items, or items that are not actually touched in the operation. The update needs to be set up in the item file, under the options tab.

 

Check the box for Suppress fulfillment
