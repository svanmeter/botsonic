Forgotten Password

This article provides guidance on how to handle situations where a user has forgotten their password.

Topics included in this article

  • Password reset process
  • Troubleshooting password reset issues
  • Preventing future password issues

Password reset process

If a user forgets their password, they can easily reset it using the system’s password reset feature.

  1. On the login screen, click on the ‘Forgot Password’ link.
  2. Enter the user’s email address associated with their account.
  3. The user will receive an email with a link to reset their password. This link is valid for a limited time.

Troubleshooting password reset issues

In case the user encounters issues during the password reset process, follow these troubleshooting steps.

  1. Ensure the user is entering the correct email address associated with their account.
  2. Check the spam or junk folder for the password reset email.
  3. If the reset link has expired, repeat the password reset process to receive a new link.

Preventing future password issues

To prevent future password-related issues, encourage users to take the following steps.

  1. Use a strong, memorable password or a trusted password manager.
  2. Regularly update their password for enhanced security.
  3. Keep their email address up to date in the system.
