Labels

 

Overview: Point of Rental allows you to print different size barcoded labels, using a zebra label printer. You can use either the Wasp WWS500i or Honeywell MS9540 barcode scanners.

 

Note: You would need to purchase the Zebra Label Printer and barcode guns separately.
