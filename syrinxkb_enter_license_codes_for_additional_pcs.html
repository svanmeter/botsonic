Entering License Codes



In order to enter license codes to add additional PCs to the Syrinx system, you will need to open the “Registered PCs” menu option on the “Administration” tab of Syrinx SP Configuration: 

 

 

Then click the “Change” button next to the current number of PCs licensed:

 

 

You simply need to enter the new “total” number of PCs licensed and then enter the license code which was provided to you.  The code needs to be entered in all CAPS.  After you do this, there will be additional licenses added to the system which will then be picked up by the first unlicensed Syrinx clients that connect to the Syrinx database with the main Syrinx program.
