Analysis Report Menu



 

The Analysis Report Menu provides an assortment of reports to help you analyze how your rental store is performing. Elite does not have any menu or report groupings like Expert. You will have to search for reports individually. 


Here's how to get there: 
Expert: Program Menu > Analysis Reports
Elite: Right Menu > Reports icon > Search bar
