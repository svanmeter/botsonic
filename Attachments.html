Attachments




With the Attachments function, you can add supplementary files to Point of Rental's records. This document will walk you through that process as well as the Attachment menu. This feature has recently been redesigned. Attachments are no longer held as a shortcut file on the hard drive. They are now held as a link within the database allowing for much faster access. In addition, you can set the security level required to view attachments. This means you could hold the original purchase invoice for a piece of equipment and set the security where only users with a Regional Manager's level can view it. This is also true of attachments for Customers, Transactions, Job Site/Venue, and Purchase Orders.


Topics in this article include
 

  • Getting there
  • Adding Attachments to Records
  • Deactivating Attachments
  • Attachment Menu

         



Getting there
     
You can access attachments from any transactions (contracts), items, customers, jobsites, vendors, and purchase orders. Once you're in a record, you'll see an Attachments option in the top menu toolbar.
 


Adding Attachments to Records
                             

1. Attachments can be added to transactions (contracts), items, customers, jobsites, vendors, and purchase orders. Open a record of your choosing.

2. In the record, go to the top toolbar. Find and select Attachments. Below is an example of Attachments within a transaction. 
 


3. Once selected, this screen will appear. Select Add.
 


4. The screen below will appear. Enter a description, set the file/URL location, and assign a password level from the dropdown menu. Using a URL location allows you to link documents found online to your record.
 


5.  Web Settings Section - This section is only applicable if your website is integrated with Point of Rental. Enter or select a group and enter a sort order. These options will allow you to choose how your attachments will be displayed on your website. Or you can check the Hide on Website box to hide the attachments from being visible on your website. 

 

     
Deactivating Attachments

When creating a new attachment, or editing a preexisting one, you have the option to deactivate the attachment.
 

1. To edit a preexisting attachment simply right-click and select Edit. 

2. Simply check the inactive box within the attachment and it will be deactivated. 
 

 



Attachment Menu

The attachment menu allows you to add and edit your attachments. 



File tab
 

1. You can refresh the menu.

2. You can export your attachments to Excel.


Options Tab
 

1. You can also filter your attachments.

2. You can show details that reveal additional data columns for your attachments.

3. You can highlight and unhighlight invalid attachment files.

4. You can show and hide your inactive attachments.                           



Multi-Select:  When using the ‘Set File’ option, users can select multiple files by holding down the control key and clicking on each one.  They will be added as separate attachments under the record.

POR One Images (Elite only): Images imported from the POR One app will appear in the attachments grid as a list of uneditable URLs. Clicking on the URL will open the image.
