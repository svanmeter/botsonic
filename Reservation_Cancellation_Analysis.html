Reservation Cancellation Analysis

 

This report will show you how much money was canceled in pending reservations for the time you select and will list the amount of time before the event that the reservation was canceled. The report will also display any money collected.

 

Group By:

Allows you to select from the drop-down menu different criteria when running the report.

* Note - Groups should be checked routinely for additional groups.
