Workforce Edits Report

 

Location: Program Menu>Override Reports>WorkForce Edit Listing

 

Overview: This selection produces a report to track all transactions made while signed into the mobile workforce. The report will list the type of transaction. Examples of these transactions include but not limited to:

  • Arrived,
  • Departed,
  • Call Log,
  • Close Service
Any performing tasks in the mobile workforce module and provides the timestamp with GPS data.

Pin Icon - Will display a map to show the location of where the driver was when the task was performed. This is useful if you are tracking your trucks, to confirm your crew is on-site or if the equipment was damaged on site. You would have a map of the location.
