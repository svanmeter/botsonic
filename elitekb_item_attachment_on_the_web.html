Item Attachment on the Web

 

You can now select which item attachments that you want loaded and be visible on your website. One of the three group "types" (Video, Literature or Sites) will need to be entered. .

ex. This would be useful if you have instruction manuals and other customers facing document readily available to your customers.

 

The attachments themselves will need to be links on the web. The web based program will NOT upload any local PDF files the POR user will enter in the attachment field. The attachment links can point to other manufacturers’ websites PDF files or the POR user can create their own google drive or online shared drive, copy the manufacturer’s PDF files to their google/online shared drive, and copy the google/online shared drive link in the attachment field.
