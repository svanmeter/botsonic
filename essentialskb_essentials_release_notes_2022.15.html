11/7/2022 - Version 2022.15

 
  • Expiration Date for Locations - You can add as many locations as you want. Event or Delivery or General. Matter of fact, it’s so much fun, you can keep adding them until they are a really long unmanageable list to choose from, even if you don't use them again. But now, you can put an expiration date on that job site or event location and it will automagically disappear when that date passes. And if you need to reactivate it later, you can even do that.
 
  • Transaction Substatus enhancement - We noticed the new Transaction Substatus feature is a hit. We have made an upgrade. You can now set a default for each status. So if you have Ready to Pull and Completely pulled as your reservation substatuses, you can set Ready to Pull as the default.
 
  • Transaction List Report Enhancement - If you use Variants, you will like the change to this report. We have added the variant type as a column to the Transaction List and other associated Transaction reports to make it easier to find what color linen or bicycle frame type was on each transaction.
