Initial WebPay Instructions

 

This document will walk you through getting started with WebPay. There are 3 steps; the first is to gather information from your Point of Rental Software and enter it into this document. The second step is to fill out a questionnaire, and the third is to return this document, filled out, to the Web team. Please note that WebPay only supports payments for quotes and reservations. WebPay does not support payments for any other contract type like Open or Work Order contracts. 

 

Webpay Requirements

  • Point of Rental Software Version 2022 or later
  • A Cayan (TSYS) or Moneris account
  • Latest patches and updates for your Point of Rental Software


 

Step 1:  Find the Information within Point of Rental

 

What information does the WebPay team need? 

You’ll need to gather some information to pass along to the Web team. Some of this information may be found in your Point Of Rental database. Here are the four pieces of information related to your Point of Rental Cayan (TSYS) or Moneris account:

  1. Merchant #
  2. Site ID
  3. Merchant Key
  4. Web API Key
 

How do I find the information in my Point of Rental Software?

The steps below will make this as easy as possible and speed up the implementation process. Once you have found the information, enter it in the designated table below. To confirm you’ve located the correct information, reference the definitions below the table. 


 

Value Needed

Your Credentials

Merchant Name:
 

Site ID:
 

Merchant Key:
 

Web API Key:
 
 

Merchant Name:  This value is a store-name-like string. It is NOT an all-numeric value (which is the internal TSYS Merchant ID).  For example, “XYZ Rental LLC.”

Merchant Site Id: This value is an 8-character long ALPHANUMERIC string. For example, “ABC1N1Z9.” 

Merchant Key:  This value is a 29-character long ALPHANUMERIC string. For example, “ABC12-CWPCX-447J5-GKWKG12XYZ.”

Web API Key: This value is  a 16-character long ALPHANUMERIC string. For example,  “ZZOF8DE6CKQS7S3Z.”

 

The Steps:

  1. In this first step, you’ll locate your Merchant Name.
    Within the Elite interface, go to the Configuration Gear (top-right corner) > select App Configuration under Legacy Options > System Configuration > Parameters > and select the Drawer tab.
    Within the Expert interface, open the Program Menu > #9 Configuration > System Configurations > Parameters > and select the Drawer tab.
  2. Copy and paste the Value from the Field “Merchant #” into the designated area above next to “Merchant Name.” The Merchant # in Point of Rental is an alphabetic value. We can't use your TSYS merchant number.


 

 
  1. In this step, you’ll locate your Site ID and Merchant Key.
    Within the Elite interface, go to your Configuration Gear > Security under Legacy Options> and select Verifone Terminals.
    Within the Expert interface, open your Program Menu > #10 Security > Verifone Terminals.
  2. Select Store #1. Then copy and paste the values from the fields “Site ID” and “Merchant Key” into the designated table.
 

 
  1. In this step, you’ll find the Web API KEY.
    Once you retrieve the Merchant Name, Site ID, and Merchant Key from within Point of Rental, you can request the last credential needed from Cayan (TSYS). You can ask them to email your "Web API Key." Or you can call Cayan (TSYS) directly by dialing 800-941-6557, selecting Option 2 Client Support, then choosing Option 2 for Account Changes.
    If you have any issues with obtaining the Web API Key, please loop us in at webpay@rentalhosting.com.
