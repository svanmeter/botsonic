Rental Hosting Guide and Overview


What is Rental Hosting?
RentalHosting.com offers exclusive web interfaces that integrate with our Point of Rental Software software.

What does Rental Hosting do?
The website has full shopping cart functionality for online rental quotes that will be automatically imported into Point-of-Rental. The website allows customers to quickly search thousands of inventory items with the search bar feature.

What syncs to the Rental Hosting website?
The POR software integration causes the website to receive daily automatic uploads of item rates, photos, specifications and instructions
The stores parameter file is used for the contact information and location on the website, this includes the store hours from the calendar maintenance settings.
When they subscribe to have an Integrated Website Hosting, the website will automatically be updated every night.

If the customer also have the Website Import module then the customer will have quotes imported every hour in the counter system for them to convert to real contracts in Point of Rental.

Using the Website Import module the customer could also have a credit card integrated website using CAYAN/Tsys, this will import credit card payments directly into Point of Rental onto the contract or customers account with a re-usable token to charge or refund the customer later.


When we write a case for the Rental Hosting what’s involved?
If we can change data that affects their request, proceed with those changes before writing a second case for transfer to the Rental Hosting team.

Common Issues for Rental Hosting:
1) When a customer calls in saying that their website has not updated recently you should first try to run "Update Website" and see the response you get back. If it works from here and updates the website then you should check to make sure POR Task runner is working properly and that they have an active task for the website to be updated. If it’s determined that they changed something on their network giving them a different IP address you should now go to PORHELP.com and click on “Show My IP Address” with this new IP address information you would make a case with it and send it over to Rental Hosting to update their firewall.

2) Anything looks-wise goes straight to Rental Hosting.  If the customer is asking about changing how the website looks or functions this would go to rental hosting.

Then if needed, tell the customer that the other help he needs will require a callback. In this second case and before the transfer, be specific in the notes. Assign your second case to the Rental Hosting Queue.
