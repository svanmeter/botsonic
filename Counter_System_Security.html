Counter System Security



 

This article will discuss security options through the Counter System. Specific options can be found under New Contract, Close Contract, Return Menu, and Dashboards that can be password protected or entirely prohibited. In these cases, the user will have to log in as an owner in order to access.


Here's how to get there
Expert: Program Menu > Counter System
Elite: Left Menu > Transactions > Quick Buttons
 


 

1. Once in counter, select ‘New Contract,’ ‘Close Contract,’ or ‘Dashboards.' Use New Contract here.

2. Right click on one of the options such as ‘Reservation.’ You will be prompted to enter a password.
 


 

3. Next, you will get a list of security options. Choose the level of password security required to access that option or disallow it entirely.
 


 

When a user tries to click on one of these options after the security has been set, they will either be prompted for the appropriate level of password or notified that a specific function has been disabled.
