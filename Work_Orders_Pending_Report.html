Work Orders Pending Report


Location: Program Menu > Maintenance Reports > Work Orders Pending Report



Overview: This report shows all pending work orders in the system. The work order report will consist of the following:


Definitions: 

NameName of Customer
WO TypeWork Order Number
Contract NumberList the Work Order Number
Contract QuestionsList responses to Work Order questions
DateDate of Work Order
PhoneCustomers Phone Number
PO NumberPurchase Order Number (if any) 
InformationInformation in relation to the Work Order
OperationWork Orders Operational Status from the Contract
