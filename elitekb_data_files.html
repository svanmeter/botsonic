Data Files

(Multi-Store)

 

Some of the data files are shared among all stores, while others are local to each store, as defined in the chart below. notice that each store has its own calendar file in case the business hours and or days are different. You should also be aware that each store has its own parameter file, so a change at one store does not automatically affect the other stores.

Important: Because all billing and end-of-month reporting is done at the main store, you should make sure that all stores shave the same accounting method defined in the parameter file (cash or accrual), and that all store s have the same end-of-month and end-of-year defined.
