eCommerce | Updating Your Website in Auto Mode



 

As you make data changes in your POR system, the automatic nightly update will refresh the website with your changes. If you wish to see your changes immediately, you will need to manually update the website.

 

To Update the Website

 

  • System Configuration Menu

    • Other Updates

      1. Update Website – click “OK” on any pop-up messages you see

      2. Click on the “Update Site Data” box

      3. The process will then run & update the website

      4. Click “Exit” in the final pop-up window

 

TIP:  If you have not inserted any new item images since the last update, check the “No Pics” box and the update will skip uploading the item images. Depending on the total number of item images you have, this can make the update run a bit faster.
