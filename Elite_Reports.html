Elite Reports

 

 

 

Most reports in the system can be accessed from the Right Menu Reports option.

Selecting this option will open up the Reports feature.

 

From here there are several options to choose from.

Custom

This option will open up the Custom Reports menu.

 

Search

When using this option, it will produce a search result from All reports regardless of the current view.

 

All

This is a list of all reports available to the user. Exposing these reports is controlled in the Report Engine security area. Deselecting here will remove the report for that user, Group or the Company.

 

Most Used

This list populates over time as reports are accessed.

 

Favorites

From the All or Most Used list you can select the ‘star’ on the right side and mark a report as a Favorite. These reports are displayed in the Favorites option.

 

Saved

These are ‘system generated’ reports that can be accessed from here. These change as new reports are created and old ones are replaced or removed. These are reports that have already run and only display the results - not the option to run the report again and set new parameters.

 

Reports

The report options are many. In most cases, a report configuration window is displayed so you can define Stores, Groupings and many other options to help you get the information you are looking for.
