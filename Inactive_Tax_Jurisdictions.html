Inactive Tax Jurisdictions 

 

How to get there: Program Menu>Configurations>Accounting Configurations>Tax Tables

To make the old one inactive, right-click on the old Tax Jurisdiction and select Make Inactive

1.  A message will appear asking if you want to make the tax rate inactive. Answer yes to the following dialog box. This will update your system with the new tax jurisdiction and move the old one to the inactive list.

 

You can view all your inactive Tax Jurisdiction by checking the Show Inactive Jurisdictions in the bottom left corner.

2.  Configuration, System Configuration, go to parameters, select store, select accounting tab, and change store tax code.

 

If the user does not manually make the jurisdiction inactive, a task will run at end of day that will mark all jurisdictions that have reached their expiration date as inactive.
