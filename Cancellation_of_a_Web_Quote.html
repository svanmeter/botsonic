Stornierung eines Web-Angebots

 

Übersicht

 

In diesem Artikel erfahren Sie, wie Sie ein Web-Angebot stornieren können. Sie können ein Web-Angebot nicht über den Importbildschirm löschen. Sie müssen das Angebot importieren und die Stornierung innerhalb des Programms vornehmen.

 

Stornierung eines Web-Angebots

 

1. Gehen Sie über das Programmmenü zum Counter System.

 

2. Wählen Sie „Tools“ in der oberen linken Ecke des Bildschirms und wählen Sie "Web-Angebot importieren“.



3. Machen Sie einen Doppelklick auf das Web-Angebot, das Sie stornieren möchten.




4. Sobald Sie die Option ausgewählt haben, wird der Bildschirm zur Vertragsbearbeitung angezeigt:




5. Wählen Sie innerhalb des Vertrags die Option Stornieren aus. Hinweis: Dadurch werden die Web-Angebote aus dem Feld „Web-Angebot importieren“ storniert.
