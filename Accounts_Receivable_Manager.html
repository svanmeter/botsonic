Accounts Receivable Manager



 

Overview 

The Accounts Receivable Manager will go over the proper accounts receivable procedures used within the Point of Rental Software.

Topics include:

  • Accounts Receivable Manager
  • Editing Customer A/R Details
  • Making Payments


 

How to get there:
Expert: Program Menu > Accounts Receivable > Accounts Receivable Manager
Elite: Accounting > Accounts Receivable Dashboard




 

Accounts Receivable Manager

1. You will be prompted to enter your user ID, before being met with the following screen:
 

 

Note - The top line in bold indicates the total dollar amount of receivables in that given time frame. (30, 31-60, 61-90, 91-120, 120 days overdue, and all total receivables).



2. You can filter the following fields to your specifications:
 

Days - You can enter the number of days the account is overdue. The system will show you increments of the number of days entered. (Ex. 5 days entered, will show 5, 10, 15 days. If entering 30 days, it will show 30, 60, 90, 120)

Payment Type - This field allows you to toggle between an account, cash, or all payment type view.

Group By - This allows you to select customers by (Type, Group, Sales Rep, or Job-site).

Closed - This allows you to select all closed contracts with money due on the contract.

Open - This allows you to select all open contracts with money due on the contract.

Reservation - This allows you to select all reservations with money due on the contract.

Show Credit Balance - This allows you to see all credit balances due. Checking this box will show customer credits.

Assigned to Me - This allows you to see all Customer Accounts assigned to you.

Show All - Shows all Accounts. 

Due - This column displays the number of days until the next call date.
 

    Note - Double-Clicking the name field opens the customer manager screen where you can access the customer’s information and telephone number.
 


 



 

Edit A/R Details

1. Right-click on a customer’s name and select Edit A/R Details to access the customer’s call frequency, enter internal accounts receivable notes, assign an Operator, enter a call log, or select the type of call. 


 


 

Next Call - This allows you to set the date for the next reminder or establish a frequency of calls to be made.
Ex: every 7 days.
 

A/R Notes - This allows you to enter internal notes for your Accounts Receivable employees.
 

Assigned A/R Operator - This allows you to assign an employee to a customer's account. This person would oversee contacting the customer for payments.
 

Export to Spreadsheet - This allows you to export the call logs to an excel spreadsheet.
 

Forced Acknowledgment - This allows the call log to populate when anyone looks up the customer record.  
 

Select Type - This allows you to create reports pertaining to the call logs.



2. Once you have finished your call logs, click save, you can now move on to the next customer that you need to call. You will notice in the dashboard the Assigned and A/R Notes have information in the appropriate columns, allowing you to see the customer has been contacted.

 




Making an Account Payment through Account Manager
 

1. You can make payments to a customer’s account through the Accounts Receivable Manager.  You can do this by right-clicking on either the key or the customer name, then selecting the Customer Dashboard.
 




2. To make a payment, select Tools, Make a Payment, and then enter your user password.
