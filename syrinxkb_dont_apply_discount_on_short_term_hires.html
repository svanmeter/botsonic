Overview

It is possible to set up discounts for customers, account and cash within Syrinx, but you may not want the system to discount the hire rate if the item is off-hired after only a short period i.e. one day.  However, if kept on hire for longer than 1 day then the discount will apply to the whole hire.  There is an option to set the system to not discount a hire if the duration is less than or equal to the specified number of days.

Configuration.

Contract Entry – Printed Contract.

There is a document option ‘Show Hire Discount%’ in Syrinx Configuration which, if set, will print the hire discount on the contract.  This system option needs consideration in that the printed contract given to your customer may indicate that a discount is to be applied, when (depending on the hire duration) this is not the case.

If you enter a planned end date on the hire contract, and if the projected duration of hire is within the new system option setting, then the discount% is not printed on the contract.

As the printing (and applicability) of the discount on any given contract is subject to the hire duration, the document option ‘Show Planned End Date’ should also be ticked, so that the planned end date is also printed on the contract.  Therefore, if the hirer queries why a discount has not been applied then the printed contract will show the start and end date, and it can be explained to them that the discount doesn’t apply for this hire duration.

Contract Entry – Cost button.

The projected cost of the hire in contract entry will reflect these settings, by pressing the button, and only apply the hire discount to the projected cost if the duration is more than the minimum set by the new option.

Off Hire Contract – Cost button.

The discount percentage shown on the Cost of Hire screen at off hire is normally fixed – i.e. it is the discount set on the contract/item.  With this option, however, the discount depends on the number of days to charge, which can be amended on the Cost screen.

Batched Invoices.

If the invoice is batched, rather than printed at off hire, then again the discount is only applied if the duration is past the minimum set by the option.
