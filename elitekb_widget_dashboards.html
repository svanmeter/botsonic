Widget Dashboards

 

Dashboards that contain “widgets” are sometimes referred to as “Bi-Dashboards”.  Bi means Business Intelligence.  The idea is to display data related to your business in a small, visual way that allows you to quickly see and act on it if necessary.  They are visual reports.

In Launchpad there are a number of widgets available covering a wide range of data points.  There are filters on the dashboard that will allow the user to filter to certain stores, Groups of stores or to Sales Reps.

 

Adding a New Dashboard

A brand new installation or user may not have any dashboards displayed.  In this case there are two options to add a dashboard:

 

  1. From the + symbol in the dashboard header or select the “Get Started” option.

     

 

Either one will display the same “Add a Dashboard menu:

 

Add a New Dashboard - This allows the user to create their very own dashboard.  Only they will see it unless they decide to share it.

  1. Select a Dashboard Type

    1. Customer and Executive will give access to the same widgets

    2. Sales will only allow the user to add certain widgets as they are designed to filter to Sales Reps.

  2. Add your dashboard title.  This is what will display in the Dashboard tab.

  3. Pick your widgets.  They are grouped into categories.

  4. From the list of widgets, you can choose to add one or, by checking the “Select Multiple” option you can add several at once.

     

  5. When you add one widget, you can change the Title on the widget.

  6. Select Add will place the widgets on your dashboard.

  7. If you want to add other widgets from other categories, prior to selecting Save, select “Add Widget” from the Dashboard Settings option on the right side.  This will take the user back to the Add a widget area.

  8. You can also click and drag the widgets into different positions.

  9. Selecting Save will set your dashboard into your Launchpad.  It will be there every time you log in.

 

Modifying existing Dashboards

If you have created the Dashboard, you have complete control over it.  If you are viewing a dashboard someone else has created and shared with you, there may be limitations depending on your security permissions.  You can also click and drag dashboards into the order you prefer.

 

Dashboard Security - Launchpad Settings area

  1. Edit Any Dashboard - This gives the user permissions to treat any dashboard like they created it.  All of the options will be available.  This would typically only be given to a small number of Admin users.

  2. Edit Company Default Dashboard - This allows the user to “force share” a dashboard onto all users in the company.  It also allows a user to make certain modifications to a dashboard that was ‘force shared’ with all users.  They can modify the widgets, move them around or Copy the dashboard.  They cannot change the Sharing properties, Hide or Delete it.

  3. Edit Group Default Dashboard - This allows the user to “force share” a dashboard onto a Group or individual user in the company.  It also allows a user to make certain modifications to a dashboard that was ‘force shared’ with a Group they are in or with them individually.  They can modify the widgets, move them around or Copy the dashboard.  They cannot change the Sharing properties, Hide or Delete it.

 

Shared Dashboards

Dashboards can be Shared or Force Shared.  They can be shared with all users in the company, a Group or an individual user.  A shared dashboard is identified with an orange dot on the dashboard tab, a force shared dashboard is identified by a blue dot.

Managing Dashboards displayed

Selecting the + symbol in the dashboard header will produce the Add a Dashboard menu.

  1. My Dashboards - This will display all dashboards you have created but not shared.

  2. Shared Dashboards - This will display all dashboards shared by you, to you individually, to your Group or to all users.  You can select and deselect from here to change your display.  If it is forced, you will not be able to uncheck it.

  3. You can also select the ‘X' that appears on hover in the dashboard tab.  It will always read that you are “Deleting” the dashboard.  If it is your dashboard it will be Deleted.  If it is not yours and you have “Edit Any Dashboard” permissions, it will delete it for you and everyone else.  Otherwise, it will just Hide it on your Launchpad.
