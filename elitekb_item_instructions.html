Item Instructions

 

Overview:  You may add instructions to individual items, allowing them to automatically print upon opening a contract or they can be manually printed when writing a reservation. These are usually safety instructions or “How to Operate” instructions.

 

1.   Modify an Item Record within File Maintenance. Instruction files should be assigned to the Header and Serialized records; You can assign different instruction files to each serialized record if necessary.

a.   When the instruct file field is added/modified in the Header record when saving the record, the system will prompt to update all serialized items.

b.   When a new serialized item is created from the Header record, the instruct file field will copy from the Header.

2.   Find the “Instruct” file on the left side.  You can type a file name that already exists in the c:\por\instruct folder or you can enter a new file name.  If you are creating a new file it will be WordPad or Word document (i.e. a 5-horsepower front tine merry tiller could be TIL5MT).

3.   Once you type in that name, tab, or enter out of the field.  You will now see a Lookup button.

* Note - This field can only contain up to 25 Characters

 

4.   Click the Lookup button.  You will be notified that the file does not exist. And asked if you would like to create it? Select Yes.

5.   Your default document program will open (usually WordPad).  Create your instructions in this document.  Once finished close the document and save.  Once a file exists you can use the same file name for other “like” items.

 

*Note: You may also use Adobe PDF files for this.  Save the PDF file in c:\por\instruct.  Keep the name short.

 

6..  To select a file that already exists double click the field in the Item Record a list of possible files will be displayed.  Select the file you would like to use.

7.    You must also define an instruction printer on all terminals you want instructions to print from.  This is done through configuration/printer selection/other tab and define your instruction printer.  This would normally be the plain paper tray on the front counter.  You would not normally do this for office terminals.

 

8.   You can also define the parameter to print instructions.  By default, will print instructions on an open contract if there are any items on the contract with instructions defined.  However, you may want to be asked if you want instruction sheets to print each time you open a contract.  This is done through configuration/parameters/printing/print letters.
