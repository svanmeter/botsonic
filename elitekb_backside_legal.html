Backside Legal


 

This article discusses the backside legal printout field. This option is for the full-page legalese that prints on the backside of the contract. 


 

Topics included in this article

  • Requirements and limitations
  • Backside legal upload
  • Two column import
  • Add the Backside legal to the Contract format


 

Requirements and limitations
 

  • All documents must be saved in file type .RTF (Rich Text Format).
 
  • All text must either be typed in WordPad or copied and pasted into WordPad with all fonts changed to any font family below (we suggest using Arial Font):﻿
 
  1. True Type Fonts (TTF)
  2. True Type Collection (TTC)
  3. Printer Font Metrics (PFM)
  4. Adobe Font Metrics (AFM)
  5. Printer Font Binary (PFB)
  6. Printer Font ASCII (PFA)
 
  • The size of the file is limited to 64,000 characters.
 
Note: About half those characters are used by formatting, so the less formatting you do, the more characters you can use.
 
  • As a general rule, use Arial Based Font, keep your document under 5 pages, and use WordPad to save your document.

 

 

Backside legal upload
 

1. You can upload the text by going to Configuration > System Configurations > Laser Contract Formats > Edit Contract Attachments. This option is for the full-page legalese that prints on the backside of the contract. It will need to be saved to your server prior to adding the documentation into Point of Rental. Once the documents have been added to the server, you can load the document from the file. 

 

2. Click on the three dots next to the load from file box. 

 

 

3. A Windows explorer window will open so that you can find your saved document. Once selected, click OK and then select load. You should see your document displayed in the preview window.

 

Note: The document should be entered as a rich text document. You can also copy and paste from a Word document. Depending on the length of your document, some legalese can show on more than one page.

 

 

4. Checking the backside legalese box in Laser Contract Formats will display the legalese when emailing a customer.

 

 

Two column import

 

It is not recommended to import a two-column document. Depending on the program used to create the document, this can cause the import to show differently in Point of Rental. The document will be imported as one long document with no columns.

 

 

Add the Backside legal to the Contract format
 

To add the backside legal to the contract format you can go to Configuration > System Configurations > Laser Contract Formats. Click Edit on the format that you are using.
 

1. From the Edit Laser Format window, click on Sections.

2. Check the box for Backside Legalese.
 

If you want it to print after the contract you can check that box as well.
