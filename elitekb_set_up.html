Set-Up Tab

 

The selections for the Set-Up tab are as followed:

 

General Settings:

Accept ENTER for Yes- checked means that the ENTER key will be accepted as a Yes response to most questions.  Unchecked means that a Yes response must be indicated by entering a Y and ENTER.

Require Operator Id –checked means that an Operator Id must be entered after every transaction in Counter System.  Unchecked means that once an Operator Id has been entered, the system will use that Operator Id until the Enter key is hit on the Counter System menu without any menu selection.

Select Items First on Open Contract – checked means that you select all the items first and then select the customer. Unchecked means that the customer information is entered first and then the items he wants to rent.

Item key begins with store number – checked means that the format of the item key begins with the store number. Unchecked means that the item key ends with the store number.  This option is only for multi-store systems.

Auto Select Store Bulk Item on Reservation – Will automatically select your store's bulk item on a reservation, instead of giving you the option of choosing between store locations.

Don't require Decimal Entry – With this box checked, you will not be required to enter a decimal in the amount field.

Allow account modifications on the counter – checked means that account customer records can be modified on the counter. Unchecked means that account customers can only be modified in File Maintenance.

Ask for Return Transfer - checked will allow an automatic return on transfer items. Uncheck will not create an automatic return.

Days to hold completed contract – Set this to the number of days to keep completed contracts in the active file until they are moved to the history file. A number around 3 should be sufficient. The larger this number, the more contracts will be in the active file which increases the time it takes to close for the day.

Maximum Deliveries per day –. Allows you to enter the maximum Deliveries in a day. * If you want unlimited, then enter 9999

Maximum Pickups per day– This allows you to enter the maximum Pickups in a day. * If you want unlimited, then enter 9999

    * Note: Under security, you can set the password level required to override the maximum number for items you want to add an additional delivery.

 

Signature:

Checked means the counter person will be requested to capture the customer’s signature for that payment method and transaction type.  Unchecked means the counter person will not be requested to capture the customer’s signature.  Note: You must purchase the optional signature capture device and software.

 

Force Settings:

These parameters allow you to force your counter people to enter contract information, deposit amounts, payment amounts, payment method, and others.

 Force entry of deposits on open if applicable– checked means that you must enter an amount in the deposit field at the payment screen when opening or sending out a contract.

 Force entry of rental/sale on open if applicable – checked means that you must enter an amount in the rental/sale field at the payment screen when opening or sending out a contract.

 Force entry of payment method – checked means that you must select a payment method (cash, check, etc.) from the payment screen when opening or sending out a contract.

 Force acknowledgment of Damage Waiver– checked means that upon opening a new contract, your counter people will be asked whether they want to charge damage waiver.

 Force serialized item selection – Checked means that you must select a serialized item when opening or sending out a contract.  Unchecked means that you do not have to select a serialized item until the item is closed.

 Force entry of contract information on open – Checked means that you must enter information into this field on the contract for every contract when opening or sending a contract out. Unchecked means this field is not required.

 Force entry of auto license on open – Checked means that you will be forced to enter information into this field on the open of a contract. Unchecked means this is not a required field.

 Force entry of ‘Ordered by’ – checked means that the counter person must select or enter something for the ‘ordered by’ field on the contract when creating the contract.

 Force ‘Picked up by’ for Account Customers – checked means that the counter person must select or enter something for the ‘picked up by’ field on the contract when opening a contract for an Account customer. Unchecked means that the ‘picked up by’ can still be entered, but it is not forced.

 Force ‘Picked up by’ for Cash Customers – checked means that the counter person must select or enter something for the ‘picked up by’ field on the contract when opening a contract for a Cash customer. Unchecked means that the ‘picked up by’ can still be entered, but it is not forced.

 Force salesman on open –checked means that you must select a salesperson whenever a contract is opened.

 Force ‘Customer heard about us’ question– checked means that you must select how the customer heard about you when adding a new customer.  Unchecked means that you aren’t required to select the method; however, you can manually select the method if desired.

 Force entry of call log type – Checking the box means that you must select a Call Log Type whenever writing a new Call Log.

 Force contract type on open – If you have set Contract Types under configuration, this box being checked means you must select a type for every contract when writing the contract. Unchecked means this is not required.

 Force delivery truck on new contract – If you have set the delivery truck on a new contract. With this box checked and you check the delivery box in a contract. Once you pass the Delivery and Pickup time, click ok. A message will prompt you to say you must select a delivery truck. You will be taken to the delivery tab where you can enter this information

 Force Pickup Truck–  If you have set pickup truck on a new contract. With this box checked and you check the pickup box in a contract. Once you pass the Delivery and Pickup time, click ok. A message will prompt you to say you must select a pickup truck. You will be taken to the pickup tab where you can enter this information.
