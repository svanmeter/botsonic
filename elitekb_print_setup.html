Print Setup

 

This selection allows you to define the names of printers and if applicable the cash drawer, to be used by this terminal.  Note: The printer names displayed are the names that you selected previously.  If you change the name of a printer or remove a printer from your system, you will need to reselect the correct printer or printer name before the system will once again print to that printer.  For example, if you originally had a printer named Okidata 320ML and you renamed it to Joe’s Okidata, you will need to reselect Joe’s Okidata for the contracts to print to that printer.  Once Printer Selection is selected, you will see the following menu:

 

Contract Printing

 

Laser Contract Printer – All laser printed contracts will be printed to this printer.

Laser Contract Copy 2 – If a printer is selected, then every time a laser contract is printed, a second copy will be printed to this printer.  This can be used to handle multi-color paper.

Laser Contract Copy 3 – If a printer is selected, then every time a laser contract is printed, a second copy will be printed to this printer.  This can be used to handle multi-color paper.

    * Note: Double clicking on the word Laser Contract Copy, will change the field from grayed to an active field.

Quote Contract Printer – If a printer is selected, then every time a Quote contract is printed, this printer will be the default.

Reservation Contract Printer – If a printer is selected, then every time a Reservation contract is printed, this printer will be the default.

Closed Contract Printer – If a printer is selected, then every time a Closed contract is printed, this printer will be the default.

Dot Matrix Contract Printer – All form printed contracts will be printed to this printer.

Receipt Printer – This is the default printer for sales only contracts.  This printer is also used for printing bank deposit slips as well as disbursement records.

Cash Drawer – This is the printer that the cash drawer is connected to.  Usually this would be your receipt printer name.

Deposit Drawer – This is the printer that the deposit drawer is connected to.  Usually this would be the same as your cash drawer printer.

 

Other Printing

Report Printer – All reports will be printed to this printer.

End of Day Printer – All the reports that prints during Close for Day Processing will be printed to this printer.

End of Month Printer – All the End of Month reports such as Item Income, Customer Income, and General Ledger Entries will be printed to this printer.

Instruction Printer – All item instruction sheets will be printed to this printer.

Load Slip Printer - The load slip is a form that prints from the printer assigned through the users printer selection. The load slip lists the contract number, customer name, date and item.

Attachments Printer – Any contract attachments will be printed to this printer.

Label Printer – This is the printer that item labels are printed.  The labels can be printed from Item Availability.

FAX – This is your FAX printer that uses the modem in your server.

 

Advanced

User – This is the user name of the current terminal.

Company – This is the company number in use.

Store – This is the store number that you are logged into.

Number of Stores – This is the number of stores that you are licensed for.

Store Code – This is the code used to signify this store.

Menu Code – This is the menu code used for this terminal.

Scan Filename – This is the path and filename of the customer’s picture from the scanner.

OCR Filename – This is the path and filename of the customer’s license information.

Signature Filename – This is the path and filename of the customer’s signature capture file.

**Sig Pad and OCR - Point-of-Rental ignores special characters in Windows User names. To get the correct username look at the bottom the Program Menu or the Counter System Menu on the terminal that the device is connected to. Insert an image.

Verifone IP: - If you utilize the VeriFone terminals for credit cards through Point-of-Rental, the IP address of the terminal that this computer is using would be entered in there.

Once you select all your printer names, click the OK button.  You might be asked whether to update the default printers.  If you answer Yes, then your current printer selections will be the printer selections for ALL terminals.  If you answer No, then only the printer selections for this terminal will be updated
