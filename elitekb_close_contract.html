Vertrag schließen

 

Übersicht

 

In diesem Artikel werden die Schritte zum Schließen eines Vertrags in Point of Rental sowie die verschiedenen Felder in den Fenstern „Vertrag schließen“ und „Vertrag schließen - Optionen“ erläutert.

 

Zu den Themen in diesem Artikel gehören:

 

·       Vertrag schließen

·       Vertrag schließen - Fenster

·       Vertrag schließen - Optionen

 

 

Vertrag schließen

 

1. Gehen Sie in Expert zu Programmmenü > Counter System > Vertrag schließen. In Elite navigieren Sie zum Symbol Transaktionen im linken Menü und wählen „Vertrag schließen“.

 

2. Verwenden Sie den Bildschirm „Vertrag schließen“, um den Vertrag zu suchen und auszuwählen, den Sie schließen möchten.


3. Nachdem Sie den gewünschten Vertrag ausgewählt haben, können Sie zwischen den verschiedenen Abschlussoptionen wählen.


4. Je nachdem, welche Option Sie gewählt haben, werden die Artikel im Vertragsbearbeitungsbildschirm als zurückgegeben angezeigt. Sie haben den Vertrag nun erfolgreich geschlossen.


Weitere Beschreibungen der verschiedenen Felder und Optionen finden Sie in den folgenden Abschnitten über die Fenster „Vertrag schließen“ und „Vertrag schließen - Optionen“.

 

 

Vertrag schließen - Fenster

 

Nachdem Sie die Option Vertrag schließen ausgewählt haben, gelangen Sie zu einem Fenster, in dem Sie den Vertrag, den Sie schließen möchten, suchen und auswählen können.



Sie können nach einem Vertrag suchen, indem Sie die Vertragsnummer, den Kundennamen oder Artikel eingeben oder eine der Suchoptionen aus dem Dropdown-Menü „Suchen nach“ verwenden. Unter „Feldsuche“ können Sie nach einem bestimmten Wort oder Satz innerhalb des im Dropdown-Menü ausgewählten Feldes suchen. Hinweis: Für jede Rückerstattung bei einem geschlossenen Vertrag kann eine Benachrichtigung hinzugefügt werden. Sie können den Mindestrückerstattungsbetrag festlegen, bevor Sie benachrichtigt werden. Weitere Einzelheiten finden Sie in unserem Artikel über Systembenachrichtigungen.

 

Sie können jeden der Verträge auswählen, indem Sie auf die Vertragsnummer klicken. Wenn Sie auf das Pluszeichen klicken, werden alle Artikel dieses Vertrags angezeigt. 

 

Status - Diese Kontrollkästchen ermöglichen die Auswahl nur von Verträgen, die einen bestimmten Status haben. Hinweis: Einige Status sind nicht verfügbar, wenn Sie bestimmte Funktionen ausführen. Der Status „Geschlossen“ ist zum Beispiel nicht verfügbar, wenn Sie Verträge schließen.

 

Anzuzeigende Station - Dieses Dropdown-Menü ermöglicht die Auswahl nur der Verträge für die gewünschte Station.

 

Zahlung - Diese Kontrollkästchen ermöglichen die Auswahl nur von Verträgen für Rechnungs- und/oder Vorkassekunden.

 

Suchen-Schaltfläche - Mit dieser Schaltfläche werden alle Verträge gesucht, die den Auswahlkriterien entsprechen, und zwar anhand der markierten Felder.

 

Hinweis: Alle Beträge, die von einem Rechnungskunden fällig sind, werden in grün angezeigt. Alle Beträge, die für einen abgeschlossenen Vertrag für einen Vorkassekunden fällig sind, werden in rot angezeigt. Bei allen Verträgen, die zur Lieferung oder Abholung markiert sind, wird der Vertragsstatus fett dargestellt.

Nachdem Sie einen Vertrag gesucht und ausgewählt haben, wird er zu Ihrer Überprüfung angezeigt. Wenn der Vertrag nicht gefunden wird oder Sie den Vertrag ablehnen, werden Sie erneut nach der Vertragsnummer gefragt. Wenn die von Ihnen eingegebene Vertragsnummer bereits geschlossen ist, werden Sie benachrichtigt und kehren zum Hauptmenü zurück.

 

 

Vertrag schließen - Optionen

 

Wenn Sie einen offenen Vertrag auswählen, wird das folgende Menü angezeigt.



Diese verschiedenen Optionen werden im Folgenden erläutert.

 

Abbrechen - Schließt lediglich das Optionsfenster, ohne weitere Auswirkungen.

 

Alle Artikel schließen - Hiermit werden automatisch alle offenen Mietartikel geschlossen. Das bedeutet, dass das Programm, wenn der Vertrag nicht zurückgegebene Mietartikel enthält, diese zurückgibt, die tatsächlichen Mietzeiten berechnet und die Standardpreise für die tatsächlichen Zeiten berechnet. Der Standardpreis bedeutet, dass dem Kunden der niedrigste Satz für den Zeitraum berechnet wird, in dem er den Artikel hatte. Wenn der Kunde den Artikel beispielsweise ursprünglich für einen Tag gemietet hat, ihn aber eine Woche behalten hat, wird ihm der Wochenpreis berechnet.

 

Alle Artikel schließen, keine Überziehung - Damit werden alle offenen Mietartikel automatisch geschlossen, ohne dass zusätzliche Gebühren berechnet werden, selbst wenn die Artikel nach dem fälligen Datum und der fälligen Zeit zurückgegeben wurden. Wurde ein Artikel beispielsweise ursprünglich für einen Tag gemietet, aber eine Woche lang behalten, werden keine zusätzlichen Gebühren berechnet. Hinweis: Je nachdem, wie Ihre Parameter definiert sind, kann für diese Option ein Passwort erforderlich sein. Wenn es sich um einen gelieferten Vertrag handelt, ist kein Passwort erforderlich. Wenn diese Option verwendet wird, wird eine Transaktionsbearbeitung geschrieben, die zeigt, dass der Vertrag ohne Überstunden abgeschlossen wurde.

 

Teilweise Rückgabe - Damit können Sie eine ganze Auftragsposition oder eine Teilmenge einer Auftragsposition zurückgeben. Sie geben an, welche Auftragsposition zurückgegeben wird und wie viele. Wenn nicht die gesamte Menge zurückgegeben wird, wird dem Vertrag eine neue Auftragsposition mit der Menge hinzugefügt, die der zurückgegebenen Menge entspricht. Die ursprüngliche Auftragspositionsmenge wird um dieselbe Anzahl verringert. Für diese Art des Schließens müssen Sie weniger als 200 Auftragspositionen im Vertrag haben.

 

Mietabmeldung - Damit können Sie die Mietzeit für Artikel stoppen, ohne den Vertrag oder die Artikel zu beenden. Auf diese Weise können Sie die Mietzeit für einen abgeholten Artikel anhalten, wenn der Kunde anruft. Wenn der Artikel dann in Ihrer Station zurückgegeben wird, schließen Sie das Ticket wie gewohnt ab. Das System ermöglicht es Ihnen, einen Eintrag im Anrufprotokoll vorzunehmen, wenn Sie die Zeit anhalten, um dem Kunden eine Rufnummer zur Überprüfung zu geben. Außerdem wird der Vertrag als Abholung gekennzeichnet, und es können Datum und Uhrzeit der Abholung festgelegt werden.

 

Alle schließen Test - Damit können Sie dem Kunden am Telefon ein endgültiges Preisangebot machen, ohne den Vertrag tatsächlich schließen. Sie werden aufgefordert, das voraussichtliche Datum und die Uhrzeit des Vertragsabschlusses einzugeben. Das Programm simuliert dann einen vollständigen Vertragsschluss und zeigt Ihnen den geschätzten fälligen Betrag an. Diese Transaktion wird am Ende des Tages nicht in der Transaktionsübersicht erscheinen.

 

Auftragspositionenabrechnung - Damit können Sie den Vertrag mit allen bereits geschlossenen Auftragspositionen dieses Vertrags schließen. Auf diese Weise können dem Kunden die bereits zurückgegebenen Artikel in Rechnung gestellt werden. Es wird ein neuer Vertrag mit einem Buchstaben hinter der Vertragsnummer erstellt, der alle offenen Auftragspositionen enthält. Hinweis: Diese Option ist nur verfügbar, wenn Sie zuvor Artikel im Vertrag geschlossen haben.

 

Fortsetzungsrechnung - Damit können Sie einen monatlichen Vertrag schließen und einen neuen Vertrag zu Abrechnungszwecken zu eröffnen. Sie werden aufgefordert, das Abschlussdatum einzugeben. Dies ist das Datum, an dem der Vertrag geschlossen und der neue Vertrag geöffnet wird. Alle Artikel werden für einen Monat geöffnet. Wenn Sie diesen Vertrag bereits weiter abgerechnet haben, ist diese Auswahl abgeblendet und nicht zugänglich. Wenn die Auswahl ausgeblendet ist, müssen Sie den letzten Vertrag auswählen, den Sie erneut in Rechnung stellen möchten (d. h. Vertrag 12345A statt 12345).

 

Einsatzorttransfer - Damit können Sie die Miete für den ursprünglichen Vertrag abmelden und als fällige Abholung kennzeichnen. Es wird ein neuer Vertrag für den neuen Standort geöffnet, der als Lieferung an die neue Adresse angezeigt wird.
