Syrinx® - Mobile Checklists (for Fleet Items)

 

 

 

 

Overview: This guide is designed to help you create Mobile Checklists for Fleet Items.

 

 

 

 

Mobile Checklists are simple checklists completed on the Driver App for equipment on or off-hired. This will allow the driver or yard person to ensure certain things are checked when equipment is delivered or returned at the off-hire.

Mobile Checklists are set against the Fleet Categories and different checklists can be created for on-hires and off-hires, allowing you to tailor your checklist to suit easily.

 

Creating Mobile Checklists are done from Syrinx Configuration > General > Mobile Checklists > Add New > Name the checklist > OK.

 

 

Enter a question for the checklist > Click OK.

 

 

Repeat the above steps until all questions required for the checklist have been created.

 

Once the checklist is completed, click Make Live.

 

A checklist will not be available for use until it has been made live.

 

**Do not make a checklist live until it has been completed**

 

 A default checklist can be selected in Configuration if the checklist will apply to most of the fleet items. Alternatively, the checklist can be set against the fleet category. To select a default checklist go to Configuration > System Options > Contract Entry > Defaults > Select a default checklist from the Checklist: dropdown menu > OK.

  

 

To set the checklist against the Fleet Category go to Syrinx > Fleet > Categories > Select the Category > Show Detail > Misc > Select the checklist from the dropdown menu under Contract Checklist: (for on hires) or Off hire Checklist: (for off hires), or both.

Once this has been set, the checklist will appear as part of the job in the Driver App. The driver will select the responses and add notes if required.

 

When going into the Mobile Job, there will now be a Checklist tab which will show all the results and comments entered from the Drivers App.
