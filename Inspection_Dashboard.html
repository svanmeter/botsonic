Inspection Dashboard

Counter System>Dashboards>Inspection Dashboard

This dashboard is available for our inspection app users to make it easy for shop managers to organize and monitor inspections. This dashboard can be filtered by store (Top Right) and also has various grouping options for the inspections listed (Bottom Left). Right clicking on the blue column header allows the user to customize the order and visibility of the available column options.


*Note: The columns for Open Contracts, IROs, and IMOs display the contract count rather than a contract number. For example, if a piece of equipment is on three different maintenance orders, the number ‘3’ will be displayed in this column.


Inspection Jobs are separated by tabs that correspond to their status into ‘Not Started’, ‘In Progress’, ‘For Review’, ‘Completed’, and ‘All’. The ‘Completed’ and ‘All’ tabs also contain a date filter for easier searching.

By right clicking on an inspection the user can assign it to an operator, group, or store. Additionally, when it is assigned to an operator, a message can be sent to that operator. A numeric priority can be set from 0-4 with 4 being the highest priority. These correspond to the priorities within the app.

Double clicking on the item name will pull up the item record. This is useful for referencing the inspection history of the item.
