Item File | Linked Item



This article will discuss linked items, including given examples. In the rental industry, there are times when items relate to other items. For example, the shampoo will probably be sold whenever a carpet cleaner is rented. The link item field is used to remind the counter person to sell the customer shampoo by automatically "pointing" to shampoo when a carpet cleaner is rented.



Topics in this article include
 

  • Getting there

  • How to use the Link Item field

  • Examples





Getting there

Here's how to get there
Expert: Program Menu > File Maintenance > Item File > Options tab
Elite: Configuration Gear > File Maintenance > Item File > Options tab




How to use the Link Item field


Select an Item

When selecting an item, avoid linking items to both a serialized and header item. Doing so will add the item twice to the contract. Instead, link the item to either a header OR its serialized item.


Item Link Setup

The Link Item or Fuel field must be correctly set up to get the desired action. Within an item record, there are two fields that can be used to link items: Link Item, and Fuel. The "link item" field is 10 characters and can be found in the Options tab. The "fuel" field is 5 characters and can be found in the Details tab. Ten additional fields may be defined in the item notes using the {} symbols. Using linked items is only recommended when adding to the serialized items and not the header. If you need to add a linked item to the header, you must use either package items or suggested items. 

 

  1. Configure the first character in the field. It should be one of the following letters: "O", "T", or "C". You can optionally use a lowercase “o”, “t”, or “c” to automatically add the item to the contract without prompting the counter person. If you're using both the Link Item and Fuel fields, be sure to only use one field with the letter O. The item cannot have two open linked items configured. It's common to have an "O" in the Link Item field while configuring the Fuel field with the letter "c".
 

 
  • The "O" stands for Open (opening of a contract)
  • The "T" stands for Open using the Third Item Lookup
  • The “C” stands for Close (closing of the item or contract)
 
  1. The next characters can be either the related item’s number, key, or partial key.



Examples

Below are different examples and configurations for the Link Item field. 
 

  • The item number for shampoo is 456. If you want to prompt for shampoo whenever the carpet cleaner is rented, enter O456 in the link field of the carpet cleaner. Whenever the carpet cleaner is rented or Opened, the counter person will be prompted for shampoo.

 

  • The diesel fuel’s key is DIE and its number is 1234. If you wanted to charge for diesel fuel whenever a backhoe is returned, you would enter cDIE or c1234 in the “link item” field of the backhoe. Whenever the backhoe is returned the counter person will be prompted for diesel fuel used.

 

  • Sandpaper’s keys are SAND-30, SAND-50, SAND-110, and SAND-220. If you want sandpaper prompted whenever the sander is rented, you would enter OSAND in the Link Item field of the sander and each of the sandpapers. As long as a key of SAND doesn’t exist, a "MORE" window will appear starting at SAND. The up and down arrow keys can then be used to pick the grit of sandpaper wanted. To exit the additional selection, press the ESC key.
     

  • Items may be chained together using the Link Item field of scaffold ends, enter "o" followed by the scaffold brace item number. In the scaffold brace's link item field enter "O" followed by the scaffold caster item number. In the scaffold caster's link item field, enter "O' followed by the scaffold board item number, etc.
 
  • Optionally you can set the quantity of the items linked to being the same. For example, if customers usually get the same number of knives, forks, and spoons, then under the knives 'link item" field enter "OFORK#*" and under the forks field enter "OSPOON#*". When you rent the knives and enter 89 for the quantity, you will be asked if you want forks. If you say YES then 89 forks will automatically be added. The same thing will happen with the spoons.
 
  • If a fixed fee is to be charged whenever an item is rented, enter the amount in the "fuel" field without any "O" or "C". If you want to charge $1.50 each time a lawnmower is rented, enter 150 in the "fuel" field.
