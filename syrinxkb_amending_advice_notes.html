Syrinx® - Amending Advice Notes




Overview: This guide is designed to help you with amending an Advice Note. Once an Advice Note has been confirmed, it may be amended or cancelled providing it has not been invoiced.



Topics included:

  • Searching for an Advice Note
  • Advice Note Overview


 

Searching for an Advice Note

The Advice Note can be searched by populating any of the blue fields.





Advice Note Overview

                             



Click on Add Item to add additional items to the Advice Note

Then select the additional items from the dropdown menus and complete the fields in the Add Advice Note Line window.

 

 

The Log will record all amendments made within the Advice Note.
