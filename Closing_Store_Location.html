Schließen eines Stationsstandorts

Übersicht

 

Es kann vorkommen, dass Sie eine Station schließen muss. Es ist wichtig zu wissen, was Sie mit Ihrem Bestand, Ihren Forderungen und Ihren Kundendaten tun müssen. Im Folgenden finden Sie einige Schritte, die Sie beachten sollten, wenn Sie Ihren Bestand von einer Station in eine andere verlagern.

 

Zu den Themen in diesem Artikel gehören:

 

·       Bestand

·       Verschiedene Möglichkeiten zur Übertragung von Bestandsposten

 

 

Bestand

 

Massenartikel - Wenn Sie eine Station mit Massenartikeln schließen, müssen Sie die Ausstattung von einer Station auf eine andere übertragen. Wenn der Transfervertrag abgeschlossen ist, wird der Bestand der auflösenden Station um die Massenartikel erweitert. Zu Berichtszwecken muss der Heimatstandort in der Station, die geschlossen wird, geändert werden.

 

Serialisierte Artikel - Wenn Sie eine Station mit serialisierten Artikeln schließen, müssen Sie die Ausstattung von einer Station auf die andere übertragen. Wenn es sich um serialisierte Artikel handelt, können Sie die Artikel ganz normal mit einem Transfervertrag übertragen. Da die serialisierten Datensätze denselben Sammelartikel verwenden, werden die serialisierten Nummern der Reihe nach vergeben.

 

 

Verschiedene Möglichkeiten zur Übertragung von Bestandsartikeln

 

Je nachdem, wie viel Bestand Sie von einer Station in eine andere übertragen, gibt es verschiedene Möglichkeiten, den Bestand zu übertragen, z. B. durch einen Transfervertrag, Artikelverfügbarkeit und Datenbankmanagement.

 

 

1.Transferverträge

 

Sie können einen Transfervertrag erstellen, indem Sie auf Programmmenü > Counter System > Neuer Vertrag > Transfervertrag gehen, wenn Sie sich in Expert befinden, oder Transaktionen > Neuer Vertrag > Transfervertrag, wenn Sie sich in Elite befinden.

 

a. Wählen Sie Transfervertrag.

 

b. Wählen Sie die Station aus, in die Sie die Ausstattung übertragen möchten. Wenn z. B. Station 2 geschlossen wird, sollten Sie die Ausstattung von Station 2 nach Station 1 übertragen.


c. Wenn Sie die Übertragung jetzt durchführen wollen, wählen Sie „Ja“.


d. Fügen Sie die Artikel dem Vertrag hinzu.

 

e. Versenden Sie den Transfervertrag. Sobald die empfangende Station den Transfervertrag schließt, werden die Artikel in ihren Bestand aufgenommen.

 

2. Artikelverfügbarkeit

 

a. Sie können Artikel von einer Station zu einer anderen übertragen, indem Sie auf Artikelverfügbarkeit gehen und den Artikel auswählen.

 

b. Gehen Sie oben auf dem Bildschirm auf Bestand > Stationstransfer > Transfer jetzt starten. Hinweis: Dies ist keine normale Methode, da Sie immer nur einen Artikel auf einmal übertragen können.



3. Datenübertragung

 

Sie können das Datenverwaltungsmodul erwerben, um Ihren Bestand aus einer Station zu exportieren, alle notwendigen Korrekturen vorzunehmen und in die andere Station zu importieren. Für diesen Service fällt eine monatliche Gebühr an. Sobald der gesamte Bestand aus der zu schließenden Station entfernt wurde, setzen Sie sich bitte mit Point of Rental in Verbindung, um weitere Bereiche zu klären, die möglicherweise übertragen werden müssen (Forderungen, Lieferwagen usw.), bevor die Station als inaktiv gekennzeichnet werden kann. Klicken Sie hier, um uns zu kontaktieren.
