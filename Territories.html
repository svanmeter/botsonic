Territories

 

 

 

Overview

 

Territories is a CRM feature designed to work with Sales Reps. Territories are geographical areas that are determined by related Zip/Postal Codes. Topics included in this article:

 

  • Security
  • Search, select, and view Territories
  • Adding/modifying Territories
  • Adding new or existing Zip/Postal Codes
  • Territory maps - US only

 

 

Security

 

There are 3 options to consider for Territories:

 

 

 

 

Search, select and view Territories

 

  • You can search and find Territories from the master search option in the header of Elite. A list of existing Territories will be displayed:

 

 

  • Viewing a Territory as shown above (highlighting the row) will display Summary info on the left.  
  • Selecting a Territory (clicking on the Name) will open the Territory form where it can be viewed and modified. A Territory is made up of General Info, Related Sales Reps and Zip Codes.

 

 

Adding /modifying Territories

 

General tab: 

 

  • Name - Short name that helps people understand the area.
  • Description - As much detail as you want to add to help people better understand the area.
  • Inactive - If you decide that a Territory is no longer valid to have, you could rename it and edit as needed. Or, you can set it to Inactive.

 

Related:

 

  • Sales Reps - You can have multiple Sales Reps assigned to a Territory. You may have an Aerial division and an Underground division with different Sales Reps covering the same area. 
  • Zip/Postal Codes - This is where you can add new or existing Zip/Postal codes to this Territory to define it.

 

 

Adding new or existing Zip/Postal Codes

 

Add New:  

 

  • Both a code and City are required to add a new Zip Code. The form will validate with existing zip codes and let you know if there is a match.
  • If you create a new one from this form or an existing one validates, on Submit this code will be added to the related Zip/Postal codes for this Territory.

 

 

Add Existing:

 

  • Selecting this option will open a new search window and display results based on the filters set.
  • A couple of ways to find a list of existing codes for a specific area are:
    • Set a filter to have the City, State contain a certain value: i.e., Dallas
    • Set a filter for the Zip Code to be ‘starts with’ the lowest known zip code for the area. In many areas, codes were created methodically and are similar by area.
    • If you set the filter for Territory Assigned to ‘Show,’ you will be able to see if specific codes are already on another Territory.
  • Once you have results, you use the ‘Select All’ or ‘Select Some’ feature using the check boxes on the left column:

 

 

  • If you select a code that is already assigned to a Territory, you will be alerted of this and asked if you want to re-assign to this current Territory.

 

 

 

Territory maps - US only

 

When you have created a new Territory, the assigned zip codes will draw a map into the Google map feature in the Summary panel. This will allow the user to review the boundaries and coverage of the Territory. This map can also be pushed out into a larger view if desired.
