Customer Heard About Us




Overview

With Customer Heard About Us, you can enter and manage the different ways customers are learning about your business. 

 

How to get there
Expert: Program Menu > Configuration > Customer Configurations > Customer Heard About Us
Elite: Configuration Gear > App Configurations > Customer Configurations > Customer Heard About Us. 



Adding and Editing Methods
 

In this table, you can manage your Customer Heard About Us options. These options can be customized to gather the information you want. This is a great tool to use from a marketing perspective.
    


 

1. Click on the method you desire to change or click the Add button to add an additional method. In either case, the following screen will be displayed:
 

 

Number - If you are adding a method, the number will be automatically assigned.

Name - Enter a description for the method.

Inactive - Check this box if this method is no longer available.
 

Note: Enter any notes you desire for this method.
 

2. Make the appropriate changes to the method and click the OK button.
