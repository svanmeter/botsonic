Reservierung versenden


Diese Auswahl wird verwendet, wenn Miet- und/oder Verkaufsartikel geändert werden sollen. Klicken Sie im Programmmenü auf „Counter System“, „Vertrag ändern“

Im Bildschirm „Transaktionsauswahl“ müssen Sie die Vertragsnummer eingeben oder den Vertrag über eine der Suchoptionen im Pulldown-Menü suchen, das durch einen Mausklick auf das Feld „Suchen nach“ aufgerufen wird. Unter „Feldsuche“ können Sie ein Feld nach einem bestimmten Wort oder Ausdruck an einer beliebigen Stelle des Feldes durchsuchen. Wenn Sie ENTER drücken, kehrt das Programm zum HAUPTMENÜ des COUNTER SYSTEM zurück.



Sie können jeden der Verträge auswählen, indem Sie auf die Vertragsnummer klicken. Wenn Sie auf das Pluszeichen klicken, werden alle Artikel dieses Vertrags angezeigt. Wenn Sie auf den Kundennamen klicken, wird der Kundendatensatz angezeigt. Alle Beträge, die von einem Rechnungskunden fällig sind, werden in grüner Farbe angezeigt. Alle Beträge, die für einen abgeschlossenen Vertrag fällig sind, werden in rot angezeigt. Wenn Sie auf „Alle Artikel anzeigen“ klicken, werden alle Artikel des Vertrags angezeigt und nicht nur der erste Artikel des Vertrags.

Angezeigte Station

Mit dieser Option können Sie nur Verträge für die gewünschte Station auswählen.

Status: 

Mit diesen Kontrollkästchen können Sie nur Verträge auswählen, die einen bestimmten Status haben.  Hinweis:  Einige Status sind bei der Ausführung bestimmter Funktionen nicht verfügbar, z. B. ist der Status „Abgeschlossen“ beim Schließen von Verträgen nicht verfügbar.

Historie verwenden:

In JET zeigt diese Option nur historische Verträge an. In SQL werden sowohl aktive als auch historische Verträge angezeigt.

Bezahlung:

Diese Kontrollkästchen ermöglichen die Auswahl nur von Verträgen für Rechnungs- und/oder Vorkassekunden.

Suche:

Mit dieser Schaltfläche werden alle Verträge gesucht, die den Auswahlkriterien entsprechen, die auf den markierten Kästchen basieren.

Hinweis:  Alle Beträge, die von einem Rechnungskunden fällig sind, werden in grüner Farbe angezeigt. Alle Beträge, die für einen abgeschlossenen Vertrag für einen Vorkassekunden fällig sind, werden in rot angezeigt. Für jeden Vertrag, der zur Lieferung oder Abholung markiert ist, wird der Vertragsstatus in fett angezeigt.

1. Der Vertrag wird auf dem Bildschirm angezeigt. Als nächstes werden Sie gefragt, ob die Reservierung rausgeht („Ja“ oder „Nein“). Sie müssen „JA“ oder „NEIN“ eingeben, um fortzufahren. Wenn Sie „NEIN“ eingeben, werden Sie zum Hauptmenü zurückgeleitet. Wenn Sie „JA“ eingeben, werden die Artikel aus dem Bestand genommen und der Vertragsstatus wird in „Offen“ geändert.

2. Die weiteren Schritte finden Sie in einem OFFENEN Vertrag ab Schritt 5.

 

***Sonderhinweis***

 

Reservierungen können bereits vor dem eigentlichen Liefertag versandt werden. Obwohl wir dies NICHT empfehlen, verstehen wir, dass es manchmal notwendig ist. Nachstehend finden Sie die Regeln und die Auswirkungen auf das Ausgabedatum und das Lieferdatum.

Wenn das Lieferdatum und das Ausgabedatum/Veranstaltungsdatum auf den aktuellen Tag fallen, werden keine Änderungen an Datum und Uhrzeit vorgenommen.

Wenn das Ausgabedatum morgen ist und Ihr Standardpreiscode „Partyformel“ ist, wird keine Änderung des Datums/der Uhrzeit vorgenommen.

Wenn das Ausgabedatum morgen ist und es heute nach 15 Uhr ist und Ihr Standardpreiscode NICHT „Partyformel“ ist, wird keine Änderung des Datums/der Uhrzeit vorgenommen.

 

Wenn eine Änderung des Ausgabedatums vorgenommen wird, wird als nächstes das Lieferdatum geprüft.

Wenn das Lieferdatum morgen ist und es heute nach 15 Uhr ist und Ihr Standardpreiscode „Partyformel“ ist, wird keine Änderung des Lieferdatums/der Uhrzeit vorgenommen.

Wenn das Lieferdatum morgen ist und Ihr Standardpreiscode NICHT „Partyformel“ ist, wird das Lieferdatum/die Lieferzeit aktualisiert.

 

Aus „Tag im Überblick“

Wählen Sie im Counter System die Option „Tag im Überblick“. Wählen Sie die Registerkarte „Reservierung“. Sie können entweder mit der rechten Maustaste klicken und „Reservierungen senden“ wählen oder Sie können einen der Verträge auswählen, indem Sie auf die Vertragsnummer klicken.



Wenn Sie auf das Pluszeichen klicken, werden alle Artikel dieses Vertrags angezeigt. 



Oder Sie können die Option „Artikeldetails anzeigen“ wählen



Alle Beträge, die von einem Rechnungskunden fällig sind, werden in grüner Farbe angezeigt.

Alle Beträge, die für einen abgeschlossenen Vertrag für einen Vorkassekunden fällig sind, werden in rot angezeigt.

Bei Verträgen, die zur Lieferung oder Abholung markiert sind, wird der Vertragsstatus in fett angezeigt.

Wenn Sie mit der rechten Maustaste auf einen Vertrag klicken und „Reservierung senden“ wählen, wird der Vertrag geöffnet. Sie erhalten eine Meldung mit der Frage, ob die Kundendaten in Ordnung sind. Wenn Sie „Ja“ wählen, wird zum Vertrag weitergeleitet. Wenn Sie „Nein“ wählen, können Sie die Informationen korrigieren. Wenn es sich um eine Selbstabholungsreservierung handelt und der Führerschein des Kunden abgelaufen ist, erhalten Sie eine Warnung.



Sobald Sie den Vertrag eingegeben haben, erhalten Sie die Meldung „Möchten Sie, dass die Rückgabezeit neu berechnet wird?“ Wenn Sie „Ja“ eingeben, werden diese Artikel auf der Grundlage der Fälligkeitszeit neu errechnet. Wenn Sie „Nein“ wählen, wird die Zeit nicht neu berechnet.

Der Status der Auftragsposition ändert sich von „Reservierung“ (gelb) auf „In Miete“ (grün). Die Bestandsartikel werden aus dem Bestand genommen.

Wenn für den Vertrag ein Betrag fällig ist, können Sie zum Zahlungsbildschirm gehen, um eine Zahlung zu leisten.
