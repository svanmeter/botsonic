Connect Moneris to Rental Essentials

 

 

 

This guide will walk you through how to connect a Moneris Credit Card Terminal over to Rental Essentials. 

 

From Moneris Merchant Resource Center

Go to the Customer Portal: https://www3.moneris.com/mpg/

 

Please have the customer grab the Store ID, API Token, and Hosted Token Profile ID. Here is information on how to grab these Items.

  • Store ID
    • Is on the top of the login screen normally 6-digit alpha code and 5 digits

 

 

  • API Token
    • This is found under Admin > Store Settings. The API Token will be on the top.

 

  • Hosted Token ID
    • This is found under Admin > Hosted Tokenization.
    • Click Create Profile, leave Source Domain blank.

  • The Domain will automatically Generate as “Source URL authentication bypassed." The Profile ID is the Hosted token Profile ID.

 

Set up on Customer Rental Essentials (V3.0+)

  • Log in to Billing Profile of Essentials.
  • Impersonate customer.
  • Go to Accounting > Credit Card Processing.

 

  • Under Update Merchant Credentials, click on the Moneris button.

 

 

  • Plug in the Moneris credentials underneath the area under “Moneris Credentials.” Click Save when complete.

 

 

Connect a Terminal to Essentials

  • Go to Stations > Registers.

 

  • Click the Pencil next to the Register we are connecting the terminal to.

 

 

  • Click the (+) underneath Register Configurations.

 

 

  • You will get this screen.

 

 

  • Name - You will need to enter a name (you can name this whatever you would like).
  • Moneris Terminal ID – This is going to be on the side of the terminal. You will need to get this from the customer, normally an 8-digit number.

 

  • Moneris Pairing Token – This is the token that is on the terminal. If the Pairing Token is not available, then they will have to contact Moneris to place the terminal back into Pairing mode. 
  • Once that is done, click on Pair and you should see that the pairing is complete.
