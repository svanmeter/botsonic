 Display the Price List for Items on Contracts and Invoices - You now have the ability to display rates on your contracts! If enabled, your contracts and invoices will display the current Daily, Weekly, and Monthly rates. You can find this setting by going to the Configuration icon > Company Configuration > Printing Options. By displaying item rates it can help reduce questions and confusion your customers may have about pricing. Please note, that rates will not appear on Receipts.
Now supported on Modern Contract Format (previously only available for the Traditional format).


UI Improvements - Small UI improvements for consistency.
  • Updated "SpecialInstructions" column header in Transaction List Overview report to render as "Special Instructions" and to wrap.
  • Update the formatting of the configuration option "ShowJobNumber" to "Show Job Number."

Bug Fix -  "Default Model Name" and "Default Model Number" now save when creating or editing an item record.

Bug Fix - Stripe payments now display properly in Essentials.

Bug Fix - Stripe payments no longer fail on Quick Sale.

Bug Fix - Initials on an eSignature no longer obscure contract terms.

Bug Fix - Deleting customers no longer gives an internal server error.

Note on Storefront Changes

We are constantly working to improve the user experience, performance, and visual layout of the Point of Rental Storefront service. As part of our ongoing efforts to improve the user experience, we have identified necessary structural changes to the Storefront HTML that include a high chance of impacting Custom CSS layouts. If you have Custom CSS, please follow the steps below to make sure you are not impacted.

 

  • Review your Custom CSS
  • Please check your storefront to confirm there are no conflicts with your custom CSS
  • Make any changes needed to ensure that your designs still function as intended
  • If you need to remove any custom CSS, please find the instructions here.
