Close All Fulfilled Items

 

 

 
 

Overview

 

When you close a contract, you now have the option to close all fulfilled items. This option closes all items that have been prepped out and prepped back in through contract fulfillment. 

 

Prerequisites 

This option is only available if you are licensed for the Contract Fulfillment module. Contact us to learn more.

 
 

Closing all fulfilled items

 

1. Go to Counter System > Close Contract and select the contract you want to close.

 

2. Select Close All Fulfilled Items from the menu.

 

 

3. This will only return those items that have completed the prep in process. Note: You can see the process of the order by looking under the fulfillment tab within the contract.

 

 

 

Note: If all the items have not been fulfilled yet, the contract will be treated as a partial return and remain in an open status.
