Data Management Tools – Data Field Clean-up

 

Select # 12 Data Field Clean-up from the Menu.

 

Select the Table you would like to Clean-up.  Your options are Customer file or Item file.

Select the Field in that table that you would like to clean-up.   The example we are showing here has selected the Item File and the Name field that we want to clean up.

Once you have made your selection you will see the data loaded in the Value list.  In the New, Column Enter the Replacement value for each Line.

Select Go and the records will be updated.  A window will pop up when letting you know the process is done.
