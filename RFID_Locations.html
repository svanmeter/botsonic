RFID Locations

 

Within Fulfillment, you now have the option of scanning items into locations within your operation. For example, you can trace a linen from the wash to the iron to being back on the rack and know how many are in each phase of this at any given time. Locations are set in Configurations>Item Configurations>Item Location Status. These can be different for every store.

 

*Note-This only works for RFID bulk serialized and true serialized items.

 

Defaults locations can also be assigned to a fulfillment status on a department basis. For example, every time anything in the dish department is ‘Prepped In’, it can also automatically be assigned to a Location of ‘Dirty’ or ‘Dish Pit’. These can be customized per store, department, and action.

 

Within Fulfillment, there is a secondary tab on the left-hand side dedicated to locations.

 

In addition to having default locations set up, you can also come to this tab and manually scan items into locations. Simply select the status in the drop down and begin scanning. The unique identifier, key, name will populate as well as the last location and the new one it is being scanned into. If the box is checked for a label it will also print a label for this item.

 

 

Where items are located is visible from two locations. First, while prepping items out in fulfillment, you may find you are short. By clicking on the ‘View Items Location’, you can see where the other chairs are located.

 

 

This view can also be accessed from Item Availability. From this view, simply select the ‘Current’ button to see the current location of all items.

.When you Click the Current button a window will open showing the quantities at each Store and Location.

If one of the departments becomes inactive, but is still tied to a location, it will indicate to the user that it is inactive when they open the default location setup. Any item assigned to an inactive department will not be scanned into the default location. Additionally, the inactive department will no longer be available in the Department drop down.
