Data Management Tools – Update Salesperson

 

Select # 8 - Update Salesperson from the Menu.

 

You can select to Update By -- Salesperson - When you do so you will have the option to select the Previous Salesperson. Then Select either Customer or Job Site. Then Select the New Salesperson.  If you Opt to update by Zip Code see the next step.

 

You can select to Update By - Zip Code - When you do so you will have the option to enter the Zip code. Then Select either Customer or Job Site.  Then Select the New Salesperson.

Select Go and all the Records will be updated.  A Window will pop up letting you know the process is done.
