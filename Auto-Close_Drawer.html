Auto-Close Drawer


 

This article will define and go through the Auto Close Drawer button. This option is only enabled after the default Carry Over Cash in Parameters is set to a number greater than zero. The default Carry Over Cash will automatically roll over to the next day's beginning drawer balance. Any bank deposits (deposit slips) will manually have to be created.




Topics include in the article
 

  • How does Auto-Close initiate?
  • Auto-Close Reports





How does Auto-Close initiate?

Auto Close can be initiated manually or automatically. 
 

  • Manually - To initiate the auto close, you can press the 'Auto Close Drawer' button. When selected, the close-for-the-day procedure will start with no further interaction necessary from you. 
  • This option is only enabled after the default Carry Over Cash in Parameters is set to a number greater than zero. The default Carry Over Cash will automatically roll over to the next day's beginning drawer balance. Any bank deposits (deposit slips) will manually have to be created.
 
  • Here's how to get there
    • Expert: Program Menu > #2 End of Day Processing
    • Elite: Left menu > End of Day
 
  •  Automatically - Point of Rental can be set up to initiate Auto Close for both Store 1 and remote stores.  For more information on Automatic Auto-Close, view Setting up Auto-Close.


     

    Auto-close Reports


    If you set up an Auto Close Drawer task via POR Task Runner (or Windows Task Scheduler), ONLY automatic reports will print. They may also email, fax, and/or save to PDF, XLS, CSV, HTML, XML, or RTF depending on their configuration.
     

    • Here's how to get there
      • Expert: Program Menu > Configuration > Reports and Notifications
      • Elite: Configuration gear > App Configurations > Reports and Notifications


    If you manually select the Auto Close Drawer option, the system will generate both automatic reports AND the End of Day History report, aka "Totals Summary."
     

    • Here's how to get there
      • Expert: Program Menu > #2 End of Day Processing
      • Elite: Left Menu > End-of-Day


    If you have an email set up in Account Numbers, that email account will receive a copy of the General Ledger at End of Day.
     

    • Here's how to get there
      • Expert: Program Menu > Configuration > Accounting Configurations
      • Elite: Configuration gear > App Configurations > Accounting Configurations
