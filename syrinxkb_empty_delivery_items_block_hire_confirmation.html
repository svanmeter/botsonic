If you are unable to confirm a contract because the system is issuing the following message: 

Delivery xxxxxx/2 has no items included on it. 

You will need to check the "Delivery Summary" tab of the Enter Hire Contract screen to verify that all items have been allocated to a delivery. If all items have been accounted for, you should then check the delivery note number mentioned in the error message in the "Multiple Deliveries" section of the "Delivery" tab of the hire contract. If that delivery is genuinely not required you can simply select it and then press the "Delete" button to delete the entry. You should then be able to confirm the hire contract.
