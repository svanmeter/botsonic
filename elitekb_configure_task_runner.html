Configure Task Runner




Overview
 

You can set up different types of tasks to run at certain times from within the Point of Rental system.
Tasks can be set to run once, daily, weekly, or monthly. You will have the ability to view, edit, add, or clone a task. You can enter the Name of the task, Recurrence, Start Time, and Description.


Here's how to get there:
Expert: Program Menu > Configurations > System Configurations > Configure Task Runner
Elite: Gear icon > App Configurations > System Configurations > Configure Task Runner

 


Note: When setting up the auto close as a task, Point of Rental will look to see if a close has been done in the past 8 hours. If not, then it will proceed to auto-close. If there has been a close within eight hours from the scheduled close, then no auto-close will be done.
