How To Properly Log Off The Server

Overview: When you are ready to close for the day, you should always close the Point of Rental program. This will ensure your updates and/or backups will work properly.  It is important for your employees to know the proper method.

 

Remote Desktop Connection

Do not click the X at the top of the screen. This will disconnect the session but Point of Rental will still be running. There is no good reason to ever disconnect a session. The correct thing to do is to log off.

Before logging off, you will need to close all programs currently running. You can see all running programs on your Task Bar. 

As soon as all programs are closed, click the Start button in the lower-left corner of the screen and then click the Sign out button.

 

Remote Apps

If you are using an Azure Based Remote App. To log off the server you will go to All Resources and click on the End Session or Logoff Session

 

If you are using the Azure Remote Desktop Application you can go to the Main Application and End your Session or Logoff Session
