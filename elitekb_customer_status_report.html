Customer Status Report




Overview
 

The Customer Status Report prints the customer’s key, number, name, driver’s license number, phone number, auto license, number of contracts, and current balance for all selected customers. The report can be sorted by customer name, key, number, or lifetime income.

Here's how to get there
Expert: Program Menu > Customer Reports > Customer Status Report
Elite: Right Menu > Customer Status Report

 

Customer Status Report

You will get the following screen:
 


1. Select your customers and payment type to report.

2. Select your 'Group By' type.

3. Select yearly or lifetime income.

4. Enter the report's before and after active dates.

5. Click the Generate button. The report will look like the following:



Report Fields
 

Customers to Report - This option allows you to select only certain customers based on their type or restriction: Allow Over 30 Days, Allow Over 60 Days, Allow Over 90 Days, Undesirable, Pay on Return, No Deposits, Account Closed, or Account Not Closed.
 

Payment Type to Report - This option allows you to select certain customers based on their payment type: account customers, cash customers, or both.
 

Sort - This option sets which column the report is sorted by: customer name, key, number, or lifetime income.
 

Yearly Income - This option allows you to report customers that fall within an income level. Once you check Yearly Income, you will be able to select Greater than, Equal to, or Less than and then enter the appropriate income level in the box.
 

Lifetime Income - This option allows you to report customers that fall within an income level. Once you check Lifetime Income, you will be able to select Greater than, Equal to, or Less than and then enter the appropriate income level in the box.
 

Active Date - This option allows you to select customers that have been active within a certain date range. If you want to select all customers that have been active since January 1, 2004, then enter 1/1/04 in the after box and leave the before box blank. If you want to select all customers that haven’t been active since December 31, 2003, then enter 12/31/03 in the before box. If you want to select all customers that have been active from January 1, 2005, to June 30, 2005, then enter 1/1/05 in the after box and 6/30/05 in the before box.
 

Printer Select - This button defines the output printer for the report.
 

Cancel - This button will cancel your selections and return you to the Program Menu.
 

Generate - This button will generate the report and display it on the screen. To print the report, click on the printer icon at the top of the screen after the report is displayed.
