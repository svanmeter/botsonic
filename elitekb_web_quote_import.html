Web Quote Import

Overview: One of the numerous benefits of the Point of Rental Website Module is the Web Quote Import feature. A customer can submit a quote from your integrated website right into your counter system. Terminals can also be placed in your store with your website up for customers to write a quote for themselves instead of standing in line on a busy day.

 

1. After entering in your ID, from the upper left-hand corner of the Counter System Main Menu, click Tools>Web Quote Import. This will bring up a window with all currently pending quotes that have yet to be imported. Select the one you wish and click OK.
 


 

From here the user can choose to import or remove the quote in question. If a quote is removed in error, simply check the box to show all and right-click on the removed quote to restore it.
 

2. Point of Rental will then check if the phone number the customer used matches any records. (A match is considered when the customer KEY, PHONE, or MOBILE fields match the phone number in the web quote). If two customers share that same number as their primary number, a list of results with any customer with that phone number will display in search results and you can choose the customer you want. If there is no record it will ask the question below: Click “Yes”. If you click ”No”, the customer Quote from Web will be selected instead.
 

 

3.   You will then be taken to the customer record with the information they filled out online populated in the correct fields. The customer is not asked for their driver's license info and at that point, you can scan their license if you have it in front of you.
 

 

4. Click “OK” and you will be taken to the contract processing screen with the correct items and rental length. By default the status is a quote, but the Change Status button at the top allows you to change it to an open contract or a reservation.


5. While importing, the file extensions will be marked as .InProcess.  Once the import is completed the extension will change to .Imported.  If an error occurs and the import does not complete, it will stay .InProcess.


*Note: Web Quote xml file requests, both imported and not, will be cleared from the Web Quote folder after 21 days.
