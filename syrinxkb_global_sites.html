Global Sites


Overview

Any sites (that equipment is to be delivered to) which may be applicable to more than one customer can be recorded on the system, so that they are then available to select on any new hire/sale for any customer.

Adding new Global Sites

To add a new global site, go into the Syrinx Configuration program, and select the "Global Sites" menu option on the "General" tab.:

Note: this menu option is available to those users with the "System Setup" system privilege.

 

Go to the blank record at the bottom, and enter the name of the site:

Then press the “Amend Address” button, and enter any further address, delivery direction or sub-location information on the various tabs:

 

Using Global Sites

Once set up, global sites are displayed at the bottom of the drop-down list when picking a site on the “Enter Hire Contract” and “Enter Advice Note” screens and are available to all customer records:


Removing Global Sites

If there are no more hires or sales to be entered against a particular global site, then you can un-tick the ‘Current?’ box against that site in the ‘Global Sites’ screen.  Then, the site will no longer appear in the list when entering new hires or sales.  Existing contracts will not be affected.
