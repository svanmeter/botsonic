Item Availability Reports



 

  • Price List – This option will print the Price List Report for this item’s category.
  • Items for Sale – This report will list all the items that have been marked for sale in their item record.
  • Print Label – This option will print an item barcode label to the label printer. You must have a label printer defined in the printer setup for this option to be enabled.
  • Sub-Rental Substatus – This option will show you a list of items that have been sub-rented to cover item overbooking. The report will show you (the item being sub-rented, contract no, contract status, customer name, qty rented, request qty owned)
