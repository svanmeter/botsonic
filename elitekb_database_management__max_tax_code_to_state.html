Database Management - Max Tax Code to State

 

Data Management Tools:

 

Map Tax Code To State - Used to link all Customers in a state to 1 tax Code. If you report multiple tax codes for a state you would not use this.

 

1. Select Map Tax Code to State from the Menu.

2. Select The state from the Drop-Down Menu

3. Select the Tax Jurisdiction that you would like to link to that State.

4. Note The Warnings Below the Drop-Down Menus.

Click Go and the Customer Records will be updated. You will see a pop-up window when the process is done.
