Guide on setting up and maintaining Staff Document Distribution Lists in Syrinx



Overview

You can now use Syrinx to record (for auditing purposes) the distribution of key documentation to staff members using distribution groups and lists.

You must first set up the names of the staff groups that documents may need to be sent out to (Syrinx Configuration / General / Document Distribution Groups):

You can then record which distribution groups each staff member belongs in


Create and Manage Distribution Lists
