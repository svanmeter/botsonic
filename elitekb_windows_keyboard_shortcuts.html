Windows Keyboard Shortcuts:

Below is a list of useful keyboard shortcuts you can use within your Point of Rental Software.
 

Tab moves you through data fields

SHIFT + Tab moves you in reverse through data fields

Esc or / takes you out of data entry and asks if you want to save changes

Alt + underlined letter will activate that option button

Alt + Tab will let you move between programs running on the toolbar

Ctrl + A will select all text in the active window or field

Ctrl + X will cut and copy an object or text

Ctrl + C will copy an object or text

Ctrl + V will paste an object or text that was copied

Ctrl + Print Screen will capture your entire screen in an image (paste into a document to see)

Alt + Print Screen will capture the active window on your screen in an image (paste into a document to see)
