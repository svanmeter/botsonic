Setting up a Zebra TLP 2824 

 

To print out the network settings page: turn on the power and wait until the button on the top is solid green.  Next, hold down the button on the top until it flashes once and then releases. 

Resetting the printer to factory defaults.  Hold down the green button on the top until it blinks 4 times and then release.  It will then turn red.  As soon as it turns red, hold the green button until it blinks once and then release.  There should be a new IP address. 

****You can remote into the Zebra’s LAN IPA and set it’s IP address.  You’ll be prompted for a username and password.  They are as follows:  

 

Username:  admin

Password:  1234 

 

 

Click on Print Server Settings -> Print Server -> TCP/IP Configuration = change or set the IP address to a static IP address of the Zebra printer

 

1.   Download the Zebra Setup Utility to the server. The setup utility can be found on the FTP site under /installable Software\Zebra TLP2824 Setup Utility.

2.   Double click on the folder and then double click on the Setup.exe install program contained in the folder

3.   A setup wizard will appear.  Run the setup wizard.  Be sure to install this application for the current user (preferably poradmin or administrator) only.  At the end of the wizard, be sure the Check Box “Run Zebra Setup Utilities now” is checked and press finish

4.   The Zebra Setup Utilities will automatically open once the Finish button is pressed.

5.   Attach the Zebra Printer with a USB cable to the server.  An Ethernet cable will need to be connected to their network and the printer as well.

6.   Click on the Configure Printer Connectivity button in the utility.

7.   Ensure the “Wired” option is selected and press the Next button.

8.   Select the “Internal Print Server (Print server is inside the printer)” option and press the Next button.

9.   Choose the “Static” option.  This will display the IP settings.  Find an available IP address on their network and enter it into the IP Address field.  Then fill out the Subnet mask and Default gateway with the correct network settings for their network.  Press the Next button.

10.  Press the Next button again

11.  When the Utility ask you if you would like to “Send connectivity configuration commands to the printer or save it to a file” choose printer and press the Finish button.

12.  Once the printer is done updating, turn off the printer, and then turn it back on.  Wait until the red light on the green button on the top turns to a solid green light to indicate it is ready for interaction.  Hold the green button until it flashes and then release it.  This will print out the Printer’s settings.  Ensure the network settings match to what was entered on the Zebra Network Utility

13.  Remove the USB cable from the server and the Zebra Printer.  Leave the Ethernet cable plugged into the printer and the customer’s network.

14.  Configure the printer on the server as normal.  You want to use the ZDesigner TLP 2824 Driver and create a TCP / IP port pointing at the LAN address of the Zebra as the print port

15.  Once the printer is configured, the settings will need to be changed in the Printer Properties

16.  In the printer properties, click on the Advanced Tab and then click on the button labeled Printing Defaults

17.  Click on the Advanced Setup Tab.  Click on the Check Box that says Top of form backup.  Enter .13 inches where it says Gap/Mark Height, and then press the Apply button.

18.  Next click on the Stocks Tab.  Create a new Stock entry by pressing the New button

19.  Name the Stock 2.00 x 1.00.  Everything can stay defaulted except where it says Right under the Unprintable Area.  Set Right to .04 and then press the OK button

20.  Go to the General Tab\Printing Preferences and do everything you did for the Advanced tab again on the General Tab
