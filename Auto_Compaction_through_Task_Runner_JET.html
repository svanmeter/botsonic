Auto Compaction through Task Runner (JET) [Auto Compaction through Task Runner (JET)]

Overview 

Auto compaction through Task Runner allows you to set a time to have Point of Rental compact your data (normally after hours). Data compactions would normally be run when you are constantly exceeding your database, or have white screens or slowness issues.
 
Note: Auto compactions can only be done for JET databases. This is not applicable for users on a SQL database.


How to get there:
Program Menu > Configuration > System Configuration > Configure Task Runner


Setting up auto compaction through Task Runner
 
1. A Create New Task window opens.
 

2. Enter the following information into the respective text boxes. The Start Time can be different depending on when you want the auto compaction to begin. The drive letter for the File to Run and Start in Path text boxes can be different, too, depending on the drive letter the Point of Rental program is installed on.
 

3. Click the Save button when done. You will be taken back to the Configure Task Runner window where you can see your newly created task.
 
4. Make sure the PORTaskRunner Status is running. If not, click the Start button. If this isn’t started the task(s) will not run. If this is the only store the company has then you are done.
 
Note: If you no longer want to run the task, you can not delete it. Simply check the box in the upper right corner to mark the task as inactive.
