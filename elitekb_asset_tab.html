Anlage-Tab

 

Im Tab „Anlage“ werden Informationen über den Artikel wie Kaufdatum, Seriennummer und Abschreibungsinformationen aufgeführt.



Aktiviert - Dieses Kontrollkästchen gibt an, ob der Artikel abgeschrieben werden soll. Unmarkiert bedeutet, dass keine Abschreibung berechnet wird. Aktiviert bedeutet, dass die Abschreibung jeden Monat berechnet wird.﻿

Gebraucht gekauft - Dieses Kontrollkästchen gibt an, ob der Artikel entweder neu oder gebraucht gekauft wurde.

Abschreibung - Wenn Sie auf dieses Feld klicken, öffnet sich das Fenster „Abschreibungsplan“.

Lizenznummer - Ermöglicht dem Benutzer die Eingabe der Lizenznummer für den Artikel.

Ablaufdatum der Lizenz - Ermöglicht die Eingabe des Ablaufdatums der Lizenz für den Artikel. Sie können einen Bericht erstellen, der alle Artikel mit abgelaufenen Lizenzen anzeigt.

Aufschlag in Prozent - Dies ist der Betrag des Aufschlags für Verkaufsartikel. Wenn Sie einen Aufschlag von 50 % auf den Einkaufspreis berechnen möchten, geben Sie in dieses Feld 50 ein. Immer wenn sich der Einkaufspreis ändert, wird der Verkaufspreis auf 50 % über dem Einkaufspreis aktualisiert. Dies ist ein optionales Feld. Wenn Sie den Verkaufspreis manuell festlegen möchten, lassen Sie dieses Feld leer.

Hersteller - Hier können Sie den Hersteller des Geräts eingeben.

Modellnummer - Hier können Sie die Modellnummer des Geräts eingeben.

Modelljahr - Hier können Sie das Modelljahr des Geräts eingeben.

Seriennummer - Hier können Sie die Seriennummer für den eingegebenen Artikel eingeben.

GPS-Nr. - Dies ist der GPS-Anbieter, der der GPS-Einheit dieses Geräts zugewiesen ist.

Geräte-IdNr. - Dieses Feld enthält die diesem Artikel zugewiesene Gerätekennnummer.

Fahrzeugtyp - Hier können Sie den Typ des Fahrzeugs eingeben, das Sie in den Bestand aufnehmen wollen.

Weblink - Hier können Sie einen Weblink eingeben.
