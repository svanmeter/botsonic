Available Inventory

Issue Description: Customer may report that the Inventory Availability column is providing incorrect numbers.

Possible Cause: The issue could be due to 'Automatic' suggested items assigned to the parent item in the Suggested Items tab. Automatically suggested items DO affect the availability of the parent item. This is the intended functionality. Image example below:

We can see that the parent item is itself IN STOCK.



We see that the parent item has a child item assigned to it as an AUTOMATIC suggested item.

We see that the lack of available inventory for child items is affecting the availability of parent items.


Resolution Steps

 

1. Check the Suggested Items tab of the item in question to see if there are any 'Automatic' suggested items assigned to it.

 

 

2. If you find any automatically suggested items, verify that the currently available inventory of the child items is showing enough inventory to support the ‘Automatic’ suggested item requirement of the parent item. If the child items have insufficient inventory, reassure the customer that the Inventory Availability column for the parent item is accurate, and the Automatic suggested items are impacting the parent item's availability as intended.


 

3. If the child items do not have enough inventory, advise the customer to update the inventory of the child items or modify the parent item's Automatic suggested items to resolve the discrepancy in the Inventory Availability column.
 

 

Note: Automatic Items did not always affect the availability of the Parent Item. It was a long-standing bug in the system, and many users (and customer support members) built their inventory around the assumption that this bug was an intended feature. As a result, there may be some initial confusion on the subject. Clarify this with the customer and provide any guidance to help them understand the updated functionality.
 


Reports

The best reports to use for finding conflicting overbooks are:

The Availability Report: This report shows you the availability of each item by date and location. You can use this report to check if there are any items that are overbooked or underutilized for a given date range.

The Reservation Detail Report: This report shows you the details of each reservation by date and location. You can use this report to check if there are any reservations that have overlapping dates or locations for the same item.

The Overbooked Items Report: This report shows you a summary of all the items that are overbooked for a given date range. You can use this report to see how many overbookings there are for each item and how much revenue they are affecting.
 

Note: You may need to also include maintenance log checks to this report as well as unavailability periods since that can affect overbookings.
