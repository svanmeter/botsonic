INTERNAL | QS2000 Troubleshooting Steps



This document will help resolve customer issues related to the new QS2000 Scanner by IDScan.net.  Document all troubleshooting efforts in the case.

 

Topics in this article include
 

  • New Install
  • Completed Install
  • How to Verify the File Names for the QS2000
  • Troubleshooting



New Install

Is this a new install?
 

1. If so, has the customer followed the steps from the IDScan.net manual from PORhelp.com?
 

2. If not, have the customer navigate to PORHelp.com and download the IDscan.net scanner and the IDScan.net manual.
 

3. Inform the customer to follow the instructions and call back if they need assistance.



Completed Install

If the customer has completed the installation and has followed the POR instructions but the scanner still does not work, follow the steps below.
 

1. Verify the scanner is selected in the WizzForm program.

2. Right-click the WizzForm icon in the Program Menu.
 


3. Hover over Image Scanning and select Page Reader Settings.
 


4. The Page Reader Settings will open. Select the drop-down and select the serial number. There should only be one option in the drop-down.
 



How to Verify the File Names for the QS2000
 

1. Verify the file path is correct in the WizzForm software.
 

2. Ask the customer to scan an image.
 

3. Open the POR\Pictures folder and note the name.
 

4. Open POR.
 

5. Select Counter System.
 

6. Select Tools at the top right, then select Printer Selection.
 

7. Click the Advanced tab.
 

8. Click Modify.
 

9. Enter POR creds.
 

10. Verify the file path, the picture folder, and verify the file name is the same as the image.

 

Scan Filename Image path: C:\POR\Pictures\[filename]*.jpb
OCR Filename File path: C\POR\Pictures\[filename].txt




Troubleshooting
 

1. Have you tried to restart the computer?

 

2. Is the WizzForm software running?

If not, start the program and test the scanner.
 

3. Is the device connected to the PC?

If not, plug the scanner into the computer.
 

4. When trying to scan, do you see the scanner flash?

If not, exit the WizzForm software, power cycle the device, and start the WizzForm program.
If this is a constant issue, send a ticket to IT support as “QS2000 No Wake Up.”
