Syrinx® - Inspection Templates

 

 

 

 

Overview: This guide is to help you with setting up Inspection templates in Syrinx.

 

 

 

 

Select the menu option Workshop > Setup > Inspection Test Specs.

 

 

If this new test applies to the hire stock items in a particular hire category, then enter the hire category code; otherwise leave it blank.

 

 

Enter a descriptive name for the test, for example, Simple Pass/Fail or Clamps, then press OK. The Amend Test Specification window is displayed.

 

 

Use the Test Items tab to enter the different items to be tested. If you are setting up a Simple Pass/Fail test, then enter a single item named Passed. Otherwise give each item a short descriptive name.

 

If a test value is to be entered for the item (for example, a tyre pressure reading), then tick the ‘Value?’ tick box, and optionally enter the permitted range of values. If the test result is outside of this range, the system will automatically mark the item as failed.

 

 

If you have chosen to print the test certificate, then you may optionally un-tick the ‘Print?’ tick box on certain test items if you do not want them to appear on the Test Certificate.

 

You can also mark a test item line as optional, by unselecting the ‘Req?’ flag from the line item in the test specification. When entering inspection results, optional tests can be set as ticked “pass”, unticked “failed” or unset "Not applicable.”

 

 

Optional tests when flagged as failed, will still be highlighted in red while pass or N/A will be shown in normal blue. To aid identification, optional tests will automatically include the text "(Optional)" in the "What to test for" notes. When a user confirms an inspection test that has optional items that have been set as N/A, the system will issue the following prompt:

 

 

When querying an inspection test, optional tests will be highlighted in Amber if marked as "Not Applicable" (i.e. value is left unset).

 

 

The inbuilt printed inspection certificate will show "N/A" against any optional test that is left blank.

 

  

Once the items have been entered, press the Close button to return to the list of Safety Inspection tests.

 

If the setup of the test is complete, then press the Make Live button to tell the system that the test is now operational. Alternatively, you may leave the test in its "Setting Up" status, and return to make it live later.

 

To link the specific inspections to fleet items you can update against the Misc tab of the category or the stock tab of the fleet item.
