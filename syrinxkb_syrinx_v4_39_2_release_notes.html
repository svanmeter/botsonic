1. Hire

Offhire Alert text added to hire contracts will now also be displayed when raising exchanges for those contracts as equipment is technically being offhired via the exchange process and users may need to be aware of any special instructions for returning items on that contract.

The system will now log changes to the "Work Type" code linked to confirmed hire contracts in the hire contract audit log.


 

  1. Sales

Users are now able to create custom property fields for advice notes via a new "Advice Note Properties" menu option in Syrinx SP Configuration\General.  The privilege for this new menu, "Cfg.User Properties.Advice Note User Properties", is automatically added to any role that already has access to any of the existing user property menus (e.g. Stock Item, Customer, Contract, etc.) in the configuration program.  Fields added via the new configuration menu option will be visible on a new "Misc" tab on the Enter and Query Advice Note screens.


 

  1. Accounts

A new version of the Syrinx "Transfer to Xero Accounts" tool has been created that supports authentication via OAuth2.  This version can be installed selecting the new "Xero (OAuth2)" entry in the transfer tool installer:

Once installed, the tool can be linked to the Xero tenant organisation via the "Xero Consent Tenant" button on the "Xero" tab of the transfer tool "Configuration Settings" screen:

When this button is pressed, the transfer tool will open a new screen prompting the user to log into their Xero organisation and grant the transfer tool access to the accounts data:

Once that process is completed successfully, the name of the linked Xero organisation will then appear as the "Connected to Tennant" value on that tab:

Going forward, the transfer tool should then use its consent token when connecting to the Xero data and the user should no longer be prompted for consent to access Xero during each transfer as was the case with earlier versions of the "Transfer to Xero Accounts" tool.

The new tool still provides support for business areas and the user simply must choose the correct Xero organisation that should be linked to the specific transfer tool configurations for each business area when linking to a Xero tenant organisation.

This new OAuth2 version of the tool also means that the tool can be set to run as a service and transfer data to the accounts package as and when it becomes available.

  1. Android Apps

A new system option "Show Included Items on Job" has been added to the "Driver App" tab of Syrinx SP Configuration\System Options\General which, when enabled, will show the items included on a mobile job on the job summary slide.  The system will also display the source document number along with the customer name against "Load" jobs.  Previously, the Driver App would show an item list on the summary page of a job for tablet devices that had larger screen resolutions.  The new option now forces the behaviour for all Driver App devices regardless of screen size.


 

  1. Syrinx CWA/CWS/CWO

A link can now be embedded in the invoice batch notification email sent to customers who have been set up to download their new invoices from SyrinxCWS which will immediately display the invoices as soon as the user successfully logs into SyrinxCWS.  For example, an HTML tag similar to the below can be added to the "Body" field on the "Syrinx WS" tab of Syrinx SP Configuration\Syrinx WS:

<a href="http://servername/SyrinxCWS/Invoicing/PrintBatch/|1">Click Here</a>

  1. Syrinx 365

Performance optimization improvements have been made to the Syrinx 365 Desktop and Web clients in order to improve the efficiency and speed with which data is returned to the Contract and Customer search grids.  

Individual users have also been given options within the “Settings” area of the Syrinx 365 Desktop and Web clients to control “Performance Optimization” for cases where they may be working in low-bandwidth environments and may need to tune the clients for optimal speed when accessing data via the search grids:

Default Search Result Size – Sets a  hard limit on the amount of rows returned by the search grids.  The slider can be toggled from a minimum of 100 rows to a maximum of 10000 rows.

Show Real Search Data on Open – Determines whether or not the client returns any intial data when the user opens an Amend or Query search grid.  If disabled, the user will be required to enter search criteria before the search grids will return any data.

Return Sorted Search Data – Determines whether or not the data returned to the client is sorted prior to being returned to the search grid.
