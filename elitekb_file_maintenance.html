File Maintenance

This program allows you to inquire, modify, add, or delete records in the major database files that are part of the POINT-OF-RENTAL™ system.  Depending on how your store owner has set the Data File Security options, you may not be permitted to perform these functions on all database files (See Security).  First, you will be asked to enter the password.  If you do not know the password, just enter an ENTER.  You will still be permitted to perform any functions that may be allowed without a valid password.

 

Under Modify or Add, all files use the same editing procedure and special character functions.  The special editing characters for both are as follows:

 

              \ (backslash) – exit record and save changes

              / (forward slash) – exit record and ask to save changes

              = (equal) – repeats the key field in the current field

              Tab – move to the next field

              Shift-Tab – move to the previous field

              Esc – exit record and ask to save changes

 

Select the file. This is presented to you in a menu:

 

 

Enter the underlined letter or use your arrow keys to select the desired menu item.

 

If you select the item, customer, or transaction file, you will be asked for the selection screen for the record you want.  If the record is found, all pertinent data will be displayed about that record.  If the record is not found in the Item or Customer file, a list of more items or more customers in search key order will be displayed.  Use your arrow keys to select the correct key or press ESC to exit the main menu.
