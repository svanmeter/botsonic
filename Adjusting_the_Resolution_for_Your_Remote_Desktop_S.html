Anpassen der Auflösung für Ihre Remotedesktop-Sitzung


Übersicht

Wenn Sie die Auflösung eines lokalen PCs anpassen, können Sie die Symbole und Programme des Remotedesktops größer oder kleiner machen, um sie an Ihre visuellen Präferenzen anzupassen. In diesem Artikel erfahren Sie, wie Sie die Auflösung Ihres Remotedesktops anpassen können, und was Sie dabei vermeiden sollten.

Die Themen in diesem Artikel umfassen:

  • Beste Option
  • Alternative Option
  • Was Sie vermeiden sollten

 

 

Beste Option

Die beste Option, die Größe der Symbole in Ihrem Remotedesktop anzupassen, ist die Anpassung der Auflösung Ihres lokalen PCs über die Anzeigeeinstellungen. Sie können Ihre Auflösung auf niedrig oder hoch einstellen. Die Wahl der Auflösung hängt davon ab, was Sie zu erreichen versuchen. Je niedriger die Auflösung, desto größer werden die Programme und Symbole in der Regel in Ihrer Remotedesktop-Verbindung angezeigt.

So gelangen Sie dorthin:

Start ⊞ > Einstellungen ⚙️ > System > Anzeige

Oder

Klicken Sie mit der rechten Maustaste auf eine beliebige Stelle auf Ihrem lokalen Desktop > Anzeigeeinstellungen. Das Bild unten zeigt das Menü, das je nachdem, ob Sie Windows 10 oder Windows 11 verwenden, angezeigt wird.



1. Stellen Sie sicher, dass Ihre Größe auf 100 % steht. 



2. Wählen Sie die gewünschte Anzeigeauflösung.

 

Alternative Option

Bearbeiten Sie die Auflösung des Remote Desktop-Symbols, indem Sie mit der rechten Maustaste darauf klicken und im Dropdown-Menü die Option „Bearbeiten“ auswählen.

1. Wählen Sie in der Dropdown-Liste die Registerkarte „Anzeige“.

2. Passen Sie unter der Auswahl Anzeigekonfiguration die Auflösung an.


Hinweis: Wenn Sie die Bildschirmauflösung nicht auf Vollbild einstellen, werden möglicherweise Bildlaufleisten um das Remotedesktop-Fenster herum angezeigt.


Was Sie vermeiden sollten ﻿

Vermeiden Sie es, die Auflösung auf dem Server selbst zu ändern, da dies Auswirkungen auf alle Remote-Benutzer hat. Vermeiden Sie es nach Möglichkeit, den Text größer zu machen, da sich dies negativ auf die Formulierung in Programmen und Berichten auswirken kann.
