Daily Totals Report




Overview
 

This article will discuss the Daily Totals Report. This report will print a summary of totals for each day of the month selected to be reported.
 

Note: This is a landscape report. If the report does not display in landscape mode, click the Printer Setup icon (top center of the screen) and select landscape.



Here's how to get there
Expert: Program Menu > Accounting Reports > Accounting Reports Menu > Daily Totals Report
Elite: Right Menu > Reports Icon > Daily Totals Report


You will be prompted to select the month you wish to report on.
