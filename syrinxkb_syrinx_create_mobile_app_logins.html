You can create a new login for a staff member from the "misc" tab of a staff member's record. You simply need to define a login name for the staff member and do not need to set a password unless the "Enable Passwords?" option is enabled on the "Android Apps" tab of Syrinx SP Configuration\System Options\General.

The licences for the driver app are related to the number of physical mobile devices connected to the database meaning that you can have as many driver app logins as you like.

Note: This is process is not required for POR One Operator App users
