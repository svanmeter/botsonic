This article will show you how to create themes for your documents, customer online storefront and kiosk features.


Topics in this article include:

  • Front End Theme
  • Applying Front End Theme
  • Document Theme
  • Kiosk Theme





Front End Theme


1. Go to Configuration > Themes. This will display a list of your themes. If you want to create a new one then click + New Theme. If this is for your storefront or Front End, you want to make sure the Front End option is highlighted in the Content Preview Examples.  

 

 

 

 

2. Enter a name for this theme, then select your primary and header font.

 

 

3. Next, you will select the color themes for each option. As you select the color, you will see what section changes in the Content Preview Example to your right. 

 

 

4. Be sure to check which feature this theme is to apply to.

 

 

5. If you need a certain text changed or footers removed, etc., switch to Advanced Mode and enter your Custom CSS code.   

  

 

Note: We do not support custom CSS. You or your website hosting person would need to know CSS before this can be applied. We do not guarantee the CSS code entered will work with the next Customer Online Storefront updates.

  

 

 
Applying Front End Theme
 

Once you have the Front End theme created and saved, navigate to Configuration > Front End Configuration and select the theme you want to use on your storefront.



 

Document Theme
 

1. Go to Configuration > Themes. This will display a list of your themes. If you want to create a new one then click + New Theme. If this is for your documents, you want to make sure the Document option is highlighted in the Content Preview Examples.  

  

 

 

 

 

2. Select your color themes for Primary Brand/Header and Title/Heading colors.

  

 

3. Make sure the Documents box is selected in the content to apply this theme to section.

 

  

4. If you need a certain text changed or footers removed, etc., switch to Advanced Mode and enter your Custom CSS code.  

  

 

Note: We do not support custom CSS. You or your website hosting person would need to know CSS before this can be applied. We do not guarantee the CSS code entered will work with the next document updates.

  

  

5. Once everything is selected, just click Save and this theme is automatically applied to your documents.



Kiosk Theme
 

1. Go to Configuration > Themes. This will display a list of your themes. If you want to create a new one then click + New Theme. If this is for your kiosk, you want to make sure the Kiosk option is highlighted in the Content Preview Examples.  

 

If you do not have a kiosk and would like to learn more, please contact our Sales department at Sales@rentalessentials.com

 

 

2. Don't forget to set your name and color themes. As you set your color themes, the example to your right will change accordingly.

 

 

3. If you need a certain text changed or footers removed, etc., switch to Advanced Mode and enter your Custom CSS code.  

  

 

Note: We do not support custom CSS. You or your website hosting person would need to know CSS before this can be applied. We do not guarantee the CSS code entered will work with the next kiosk updates.

  

  

4. Once you have made your selections, click Save and navigate to Stations > Kiosks and click the edit icon next to the kiosk you want to add this theme to. If you don't have kiosks created, please refer to our Kiosk Operations and Setup document.  

  

 

 

5. From the edit kiosk screen, click the edit icon next to the Kiosk Configuration you want this theme to apply to. Instructions on how to set up kiosk configurations can be found in our Kiosk Operations and Setup document.

  

  

6. Select the created theme from the Theme drop-down and click Save.

  

 
Now you have your storefront, documents, and kiosk set to the same theme as your company's theme.
