Item Options

 

One way users can enhance the way they search for items, is to utilize item options.  Assigning options to items allows the user to easily filter their inventory to find items that meet the criteria a customer requires.  

 

Security for ‘Item Options’ is turned on or off under the ‘Configurations’ area of security.  It will have to be turned on for the user to be able to see the option in the ‘Item Configurations’ area in browser configurations.  If the user has access to it, they can edit the option types, the options, and which are assigned to which items.  There is no further security.  

 

 

Options types are assigned on the header level.  Once they are assigned to the header, the specific options can be assigned to the serialized items.  For example, ‘Color’ is an option type that would be assigned to the header, and ‘Red’ is an option assigned to the serialized item.  Selecting multiple items will allow the user to assign option types/options to several items at once.  Once the option types/options are assigned the items will be automatically unselected so that a new group can be selected.  

 

If an item is moved from one header to another, it will lose its assigned options and options under the option types for the new header will need to be reassigned.  Should an item be changed from ‘Sales-Header’ or ‘Rental-Header’ to a different item type, it will also lose its option types as it is no longer eligible to hold them.  




 

 

A number of option types/options come with the feature via default.  However, users can add and delete these as required.  If the user searches for an option that does not exist, they ‘Add’ button will become available.  Hovering over any option type or option will reveal a pencil and a trash can; the former to edit, the latter to delete.



 

 

Once options are selected, they can be utilized within the inventory search screen.  The option types panel is collapsed by default, but it can be expanded by clicking on the side panel.  If options are available for the list of items contained in the search, they will be included.  If there are none assigned, the panel will be blank except for a message informing the user of that.  Checking and unchecking the option boxes will further filter the items until only those that meet the specified criteria are visible.
