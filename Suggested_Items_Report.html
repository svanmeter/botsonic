Suggested Items Report

 

This report lists all the suggested items that have been setup.  This report shows the item with and all the suggested items and quantities.  It will also list any special pricing that has been defined for any of the items.  The report is grouped by item category. This report is grouped by action type (On Add, On Send Out, and On Close) and then by category.
