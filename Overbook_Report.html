Overbook Report

 

This report allows you to print an Overbook report. It will display all items in your inventory that have been added to contracts and exceeded the quantity available in the item record. This report will show items that are overbooked company-wide, giving users an insight into items that need to be subrented or purchased for future needs. It will not reflect individual store overbookings. Store overbooking can be viewed in Item Availability. Note the following:

 

  • Logistic contracts are not included (items from logistics are reported from the rental contract).
  • Item types not included are Rental - Dynamic Items and Rental - Package. 
  • Cleaning delays are not included (cleaning delays are included in Item Availability).
  • The critical report uses the critical quantity on the item record.
