Category Totals for a Month

 

 

 

Overview
 

The Category Totals for a Month report shows each category’s income for a month for all the previous years on record. 

 

 

 

Generating the report

 

1. You can access the report by going to the following:


Expert: Program Menu > Analysis Reports > Income Analysis > Category Totals for a Month
 

Elite: Right Side > Reports > Category Totals for a Month
 

2. Select whether to include rental items, sales items, and/or parts items. 

 

 

3. Next, select the month.
