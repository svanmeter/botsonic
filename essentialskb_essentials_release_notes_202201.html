1/17/2022 - Version 2022.01


  • - Straight to Reservation - At Point of Rental Software, we have become very aware of how valuable your mouse clicks are to you. Previously, when you created a transaction at the counter you could only create a quote. If you really wanted a reservation, then you had to click create quote, then click reservation. So now, if you don't require payment to create a reservation, then you also have the option to create a reservation without having to do that extra mouse click. Think of all the time savings, not to mention less wear and tear on your mouse. 
 
  • - Workbench Web Orders - We have added a shared filter for those of you that use the storefront to filter only web orders. A great feature for you to be able to see those orders that were entered via the storefront.
 
  • Additional Fields for Bill of Lading - For those of you using this new feature, we have added the ability to type in several selections on the screen instead of having to handwrite them. Fields like Item Description, Quantity, Length/Width/Height, and several others.
