Optional Suggested Items

Overview: Allows the user to link another quote to a reservation or a quote to inform the customer that there are options available. This will be displayed in a separate section of the contract as optional items. While prices are displayed for each optional item, they do not affect the total contract value. Predefined quotes can be created and linked again and again saving time (i.e. Tenting Options, Painting Options, Digging Options, Staging Options, etc...). 

Setup of Optional Items Groups

File Maintenance->Item File->Add Item.

1.  Create a New Item

2.  Have the item key start with OPITEMS-

3.  Type - Enter whether it is a Rental or Sales type item

4.  Category = Optional Quote Items

Note-- May have to create category, See item category.

Setup of Optional Customers

Note – This will be used to filter your optional items when adding an optional suggested item on contract. More on this later.

1.  Go to File Maintenance>Item File>Add Customer

2.  Create New Customer

3.  The Key starts with OPITEMS-

4.  Type= OPITEMS-CUSTOMERS (Optional)

Note-- May have to create category, See customer category.

Setup Quote Templates

Overview: Used as a fast way to add a list of optional items depending on event.

Go to Counter System->New Contract->Quote.

1.  When asked for a date, choose any one date.

2.  For the customer select an OPITEMS customer.

3.  Add the items onto the quote.

4.  Select Optional Item Group from the drop-down menu.  *Note -- This is the item that you will add to the contract.

5.  After you have finished adding all items, add in a description and notes.

6.  Click Ok and you are done.

Link Optional Items to a Contract

Overview: This will teach you how to link a quote template, customer quote, or any quote from the system to a reservation or contract.

1.  When you are modifying or writing up a contract.

2.  Click Add Item

3.  Search for the OPITEMS and select the appropriate item.

4.  Next you can select the predefined quotes or search for all active customer quotes or use one of the current customer’s quotes.   

•Optional Item Quotes -- Will Show you a list of all quotes that in that item group

•All Active Quotes – Search for all active quotes.

•Customer’s Quotes – List of all the current customer’s quotes.

    5.  Select a quote you want to have listed as optional items and click link Quote.

When the contract prints out the optional items will print out in its own grouping.
