Mobile WorkForce Set Up

Before you can start using your Mobile WorkForce, you will have to set up the employees and what functionality they will be able to access once logged into the mobile workforce app. You can set different functions by store, groups, or employees. Depending on the group (Divers, Sales Rep) you can limit the accessibility within the workforce application. To get started, you would first set up the employees and assign them to a group.
