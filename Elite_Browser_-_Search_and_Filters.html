Elite Browser - Search and Filters

 

Searches are initiated from the primary search field.  You can select what you are searching for and then enter the name of it to bring up an initial search.  Or, you can just place the cursor in the search field and hit <enter> to get all results. Until you have set and saved filters, searches, by default, show everything based on your search criteria. Filters then, in most cases, limit the search results. There are some where we hide results by default but then give you options to ‘Show’ those items.

 

 

This initial search will bring up a list of all matches to what you entered.  In some cases there are ‘system’ filters that control those results:

  • Item searches default the Current Location to the Left Menu store selected.  This can be expanded with a Current Location filter.
  • Inactive things are hidden - but can be Shown with the ‘Inactive’ filter.

There are also ‘Default’ filters displayed.  By default, these filters do not have any values they are filtering by, they are just there for your convenience to have available.  If there is no value after the filter name, no filter is being applied. If there is a value displayed and you want to remove it but have the filter remain as an option, expand the filter and select the ‘Clear’ option at the bottom.  

 

 

Saved Searches

You can design your own ‘filter profiles’ called Page Views.  If you save one and set it as default, it will override any system defaults when you trigger a new search.  The options are almost endless.  From the columns you choose to display, the order they are in and the filters displayed at the top. They can all be customized and saved. You can have as many saved searches as you want.

 

Column Selection: Show or Hide Columns from the ‘hamburger’ menu to the top left of the search results grid (three horizontal lines).

 

Column Order: Click and drag into the position you want.

 

Add a filter: Select the ‘More’ option.  Search and select the filter you want to add - or leave blank.

 

Remove a filter: Select the ‘X’ and it will go away immediately - you will not get an ‘are you sure’ message.

 

Once you have everything as you want it, select the ‘hamburger’ menu to the top left of the search results grid (three horizontal lines).  From here you see the options available:

 

  • Save Page View: Displays options to Save it as a New Page View or Update the the one you are viewing (and is already saved).
  • Load Page View:  Displays all of the ones you have currently saved.  Select one to update your view.
  • Clear Page View: This will clear filter values, leave the selected filters intact, leave anything entered in the search field and trigger a new search.
  • Delete Page View: Displays your saved views to select and delete.

A saved search set as your default will be used when you initiate from the main search in the upper left.
