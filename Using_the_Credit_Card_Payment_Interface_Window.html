Using the Credit Card Payment Interface Window

 

The credit card Payment Interface window is where you choose how you wish to process a card transaction.  Notice the four options – Retrieve Card, Manual Entry, Customer Swipe, and Cancel.  This same interface is used when processing any card payment.

The credit card integration requires a CAYAN Genius credit card terminal, although a “soft terminal” is available upon request.  The soft terminal is an integrated on-screen web payment window. Owner authorization is required because it does not perform end-to-end encryption like the Genius terminals.  See the article “CAYAN Soft Terminal” for more information.

Retrieve Card– Allows the reprocessing of a stored card.  Stored cards are stored as tokens.

Manual Entry – Allows the manually entering of a card number. 

Customer Swipe – Allows the acceptance of EMV cards or magnetic swipe cards. 

Cancel– Allows canceling of the credit card transaction before card input. 

 

Upon clicking Manual Entry or Customer Swipe buttons you will see a banner stating “Credit card transaction in process, use the pin pad to complete a transaction.  If you need to cancel, use the pin pad.”

The transaction result is displayed in a separate window that will automatically close after a few seconds. 

The approved card payment is then posted in Point of Rental along with the token for future use.
