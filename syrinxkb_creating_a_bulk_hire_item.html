Syrinx® - Creating Bulk Hire Item




Overview: This guide is designed to help you to create a Bulk Hire Item.

 

 

Bulk hire assets are non-serialised or non-mechanical items which do not have an individual asset number allocated to the item.

 

You can add new, update, amend, and remove bulk hire items from the fleet tab of Syrinx. 

 

 

To create a new bulk hire item you first need to remove the Individual Items Only flag from the category they are to be linked to.

 

 

Select Bulk item from the Fleet tab, and select the category from the drop-down. You can now Add New. 

 

Create an asset number and description for Name. You can also enter hire rates if applicable and input replacement cost in the place of the sale price.

 

 

When entering the new bulk stock item you can also place default hire notes using the notes tab. These will show under the asset on all contracts the items are placed on. Select OK to confirm the creation of the bulk hire item.

 

 

To update the quantity in stock select Adjust Qty, and update the quantity into stock and leave a note to the reason.

 

 

 

If you don’t wish to track quantity in stock you can amend the item and mark it as a non-stock item. The quantity has to be 0 for you to enable this function.
