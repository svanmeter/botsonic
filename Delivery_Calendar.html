Specific Date

A specific number of deliveries/pickups can also be assigned on a per date basis. Select ‘Add’ at the bottom and the calendar box will appear, allowing the user to select the specified date. Once entered, the user can enter the number of deliveries or pickups for the particular date.

The user in the counter system will receive the same message "You have exceeded your number of delivery/pickup slots available on this day".

Dates are listed in descending order. If the user wishes to see the dates that have passed, simply check the box next to ‘Show Historical Dates’.
