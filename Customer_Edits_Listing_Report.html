Customer Edits Listing Report




Overview
 

The Customer Edits Listing report shows all the customer edits done between two selected dates. 


Here's how to get there:
Expert: Program Menu > Customer Reports > Customer Edits Listing
Elite: Right Menu > Customer Edits Listing
 

 

1. Select the Change button to adjust the beginning and end reporting dates. 

 

2. Select your Group By configurations for the report.

 

3. Select which store you want to report on.

 

Note: If you are not multi-store your dropdown will be greyed out. 

 

4. Select the Generate button.
