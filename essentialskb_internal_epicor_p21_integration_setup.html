Epicor P21 Integration Setup



Epicor P21 in Rental Essentials is programmed completely by Epicor. We do not currently have a test environment for their product.

Topics included in this article:
  • New Setup
  • Steps to add a company
  • Support


New Setup

When they request a new setup follow the steps below.
 

Note: It should come from someone with an email address ending in epicor.com


The following are required by them:
 

1. Company Name.

 

2. Number of Stores.

 

3. Number of users.

 

4. Number of users to allocate at each store (if more than one store).

 

5. Company Address/Phone.

 

6. At least 1 user for the company with email address.
 

Steps to add a company
 

1. Login to epicorrentalessentials.com
 

Note: If you don’t have creds, ask Joe W, Evan, or Robert F to provide for you.

 

2. Go to Add company, and add Parent company.
 

Note: Each setup must be setup as multi store regardless of the number of stores - Parent, HQ, Store 1, Store 2, etc.

 

3. Add rest of stores as you would any other multi store (HQ, Store 1, etc).

 

4. See below for licensing requirements.

Support

Epicor is Tier 1 support. Every request goes through them first per our signed agreement with them. Any connection issues from their product (P21) to Rental Essentials MUST go through Epicor first as we don’t have a test environment and they are Tier 1 support. Epicor is to reach out to our support team with anything they cannot solve. We should not engage the customer unless a meeting is requested from Epicor's team.
