Prerequisites 
 

  • This third-party software requires a license from Microsoft. 

  • You must be using a SQL database. All Elite users are on a SQL database. If you are using Expert on a SQL Database, you can integrate with Power BI as well. Please reach out to our Product Support Team for more information.  



What is Power BI?

Power BI is a third-party software that works in conjunction with Point of Rental. It is a Business Intelligence (BI) and Data Visualization tool for converting data from various data sources, like Point of Rental Software, into interactive dashboards and analysis reports. Power BI offers cloud-based services for interactive visualizations with a simple interface for end users to create their own dashboards and widgets.

For more information on Power BI, follow this link.


How can I get Power BI?

Power BI is a Microsoft Product and can be purchased through Microsoft. Once you have purchased Power BI, contact our Sales team to integrate Power BI within Point of Rental Elite or Expert.

For more information on Power BI, follow this link.


How can I access Power BI from the Elite software?

Before you can access Power BI, it must be installed on the same server as Elite, so that it can be launched from the Elite Interface.

Once installed, you'll need to make it visible. To do so, go to the Gear icon (top right corner of the screen) > Configuration Areas > System > Security > Left Menu > Power BI > Allow access to groups by using the 'Yes' dropdown option.

Once visible, you can access Power BI from Left Menu >Tools > Power BI. 


If Power BI is enabled but is not displaying, reach out to our Product Support Team.


 

How does Power BI work with Elite?

Point of Rental offers a link from Elite to Power BI, but does not offer customizations nor development in  Power BI. Point of Rental simply provides access to Power BI directly from the Elite Browser. To learn how to utilize Power BI, please refer to the link at the bottom of this page. 



How do I set up Power BI?

We discourage self-setup. Setting up and configuring Power BI is best left to professionals with a strong understanding of Power BI, SQL, and the underlying data models.

If you are interested in getting help, please reach out to our sales team, who can connect you with a partner that can assist you.

 

Using Power BI, can I customize information within the Elite Dashboard?
 

This is NOT possible. Power BI is simply a way for our customers to access their data from a program that allows you to create your own dashboards and widgets. So in short, the dashboards and widgets created within Power BI are only accessible from within the Power BI interface; no direct access from within the Elite Browser should be expected. 

If you would like to see a report or customized information within the Elite Dashboard, you can request for us to build it. This is billable. For more information please contact our Sales department.
