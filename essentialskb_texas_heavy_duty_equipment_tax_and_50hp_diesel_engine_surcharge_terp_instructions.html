Texas Heavy Duty Equipment Tax and 50HP Diesel Engine Surcharge (TERP) Instructions


The Texas Emissions Reduction Plan (TERP) Off-Road, Heavy-Duty Diesel Equipment Surcharge applies to the sale, use, lease, or rental of off-road, heavy-duty diesel equipment and is based on the sale, lease, or rental amount. The seller collects the surcharge along with the sales tax and remits it to the Comptroller.
 
Because Essentials does not allow a secondary tax to be set up in the system at this time, we will need to work around this. We will create a Virtual Inventory Item that will automatically attach itself to a Quote or Reservation when a TERP Taxable item is added. Please note that EACH TERP Taxable item will need its own Virtual Inventory Item created for this workaround to work.
 

Creating the Category

First, we will set up a Category for our TERP Tax Virtual Items. This helps with creating reports the location may need for Tax Records.
Inventory > Category > New Category > Name Category "TERP TAX" > Save

Creating The Inventory Item

Next, we create a Virtual Inventory Item in the system and give it a name that will help determine what item it will be attached to. This is especially important for Rental locations that have a large inventory that is TERP Taxable. In our example, I will be creating a TERP Tax Item for the Bobcat 753 Skidsteer. Therefore, I will name this Virtual Item "TERP Tax - Bobcat 753 Skidsteer".
Inventory > Inventory > New Item > Virtual Item > Name Item > Hide From Webstore > Set to "Tax Free"

We want to make sure it is using the same Pricing Sheet as the Bobcat.
Under the Pricing Tab, Check the price sheet that corresponds to the Price Sheet of your TERP Taxable Item

Item:

TERP TAX
Now we will need to calculate the TERP Tax that will be applied to our Rental. The daily rate for our Bobcat is set at $150.00. Currently, the TERP Tax rate is set at 1.5% as of January 2021. Multiply $150.00 by 0.015 to get the Daily TERP Tax for this item. $150.00 x 0.015 = $2.25. We have our Tax, let's go into the Pricing Tab to enter it in.
In your Pricing tab for the TERP tax,  Enter $2.25 into the Base Rate > Save.

 

Adding the TERP Tax to the Item

Let's attach this TERP Tax to our Bobcat.
In Inventory > Inventory, Search for the item you wish to add the Virtual Item Tax to. Click  Edit and go to the  Suggested Items tab.  Search for your Tax Item > Add Item > Set Type to "Automatic" and Quantity to 1 > Confirm.



Now our Bobcat has an Item that will automatically add in a "TERP Tax" when it is entered into a Quote or Reservation. Please note that this Tax will NOT appear in the Totals area. Since this is considered an Item in the system, it will also count as revenue. This will need to be factored in when calculating revenue from reports. Also note that if the rate of a TERP Taxable item changes, the Virtual Item Tax will also need to be manually changed to reflect the correct tax amount.
