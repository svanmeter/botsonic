Reservation Report


 

The Reservation Report produces a report of reservation, delivery, and pickup contracts.  The report can include reservations that are to be delivered, reservations that are not to be delivered, all reservations, reservations that were marked for deliveries that are now open contracts ready for pickup, quote contracts, or any combination of these for whatever time you choose.

 

You will see the following menu:

 

 

Beginning Date – This is the beginning date of the report.

 

Ending Date – This is the ending date of the report.

 

Change – This button allows you to change the beginning and ending date of the report.

 

Grouping – The report can be grouped by contract or by item.  When grouped by contract, each contract with the associated items and information is printed together.  When grouped by item, each type of item is grouped together.  For example, all the 8 foot tables will print together.  Note:  When grouping by item, the Delivery & Pickup information as well as the customer address will not be printed.

 

Sort Order – The report can be sorted by Delivery/Non-delivery, by Store, or Reservation Zip code. 

 

Delivery/Non-delivery: Sort by delivery code, non-deliveries first, followed by deliveries.  The non-delivery section and the delivery section will then be sorted by date.

By Store:  Each store’s contract will be printed in store order.

Reservation Zip:  The report will be sorted first by the Zip code of the contract and then by date.

 

Include Deliveries – checked includes contracts to be delivered in the report.

 

Print Customer Address – If this option is checked, the customer’s address is printed next to their contract information.  This option is not available if you are sorting by item.

 

Print Component Items – If this option is checked, the component items will be printed on the report.  A component item has the “Don’t print on contract” option checked in the item record.  Usually this would-be item such as poles and stakes.

 

Store to report – This option is only valid if you are running the multi-store version.  You can select a store to print or ALL stores.
