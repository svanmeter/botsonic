Task Authorization 

 

Overview: Mobile WorkForce allows for varying Menu Templates that dictate which WorkForce tasks are allowed for an employee.

 

*You must first set up your Operator ID correctly. (see Employee Setup)

*Note: It may take up to 15 minutes for changes to take effect and you must exit the configure utility for changes to save.

1.   From the Program Menu, select #10 Security.

2.   Select Configure WorkForce Security

 

 

3.  Make note of your WorkForce Company ID noted at the top of the configure screen. Your employee will need this to log in.

4.   In the Menu Template section, use the create button to create a new template for an employee or a group of employees. Templates are used to control what functions your employees’ will have access to when using Mobile WorkForce.

a.  Name the new template

b.  Select the actions that will be available to the employee by checking the appropriate box

c.  Save the template

• Note – if you are scheduling a crew, you can set a default template called (In/Out) and enter the selected Drivers and helpers to this list. This will allow them to log into the Mobile Workforce and clock in and out.

4.  If you modify an existing template (use the up/down arrows next to the Create button) be sure to save it when done.

5.  Select your employee by changing the Store, Group, and Employee selection

6.  Press the Assign to Template button to assign the employee selected to the selected template

7.  Repeat this process until all applicable employees are assigned to the appropriate templates.

8.  As you assign/select employees, you will see the assigned template (see image below).

 

 

 

Menu Template

Once the employees have been assigned to the proper group, you will need to set the template giving the employee which functions they will have access to.

Checking the box gives them access, unchecking the box will not give them access.
