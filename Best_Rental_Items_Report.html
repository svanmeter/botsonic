Best Rental Items Report

 

 

 

 

This report lists all the rental items with the last rented date, lifetime income, return on investment, average yearly return on investment, and repair costs. The report is helpful to determine the best rental items that you might want to purchase more of. 

 

 

How to Find the Best Rental Items Report

 

  • Go to Program Menu > Analysis Reports > Inventory Analysis > Best Rental Items Report. You will be asked to enter the minimum yearly ROI percentage to report.

 

 

  • Point of Rental will then generate the Best Rental Items report:
