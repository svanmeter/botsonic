Placing Inventory in Maintenance

1. Go to Inventory then select Inventory. 
2. Search for the item in question and select the edit icon next to the item name. 
3. Go to the Stock tab. 
4. Select Make Inventory Unavailable.

 

 

5. Fill out the information highlighted with a red asterisk. For Unavailability Type, select Maintenance.

 
6. Press Save, when finished.  

 

 

 

Putting Inventory back in Stock
 
1. To put inventory back into stock, go to the Stock Tab of the item.
 
2. Select Receive Inventory from Unavailability.
 
3. Fill out the required fields that have a red asterisk next to them.
 
4. Then, press Save, when done.  

 

 

This will return the inventory to being available for rent.
