Partial Return

 

Partial Return allows you to return an entire line item or a partial quantity of a line item. You specify which line item is being returned and how many. If the entire quantity is not being returned, a new line item is added to the contract with the quantity equal to the number being returned. The original line item quantity will be decreased by the same amount. For this type of close, you must have fewer than 200 line items on the contract.

 

Select Item

Enter quantity being returned

You will see that it will create a new line item showing the number of items being returned. Leaving the original line item showing what is still out on rent.

This can also be done for sales items. If something was sold in one store and returned in another those sales items will be added to the current item’s bulk sales item, not the selling store’s.

If this is done on a continuation contract that has been prebilled the user will have the option of zeroing out the price.
