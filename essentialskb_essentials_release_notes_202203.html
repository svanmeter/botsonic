2/28/2022 - Version 2022.03


  • Workbench Enhancements - We like the workbench. You like the workbench. It’s a win/win. Because of that, we have been on a roll enhancing it. Last release we enhanced it so you can set workbench as the default when you log in. We figured why stop there? Here are some nice new workbench features that will change your life, especially if you use events.
    • Event Name will now show on each contract on the workbench.
    • If Event Name is not used, then it will display the address.
    • You can now create a filter for specific events, or you can simply use the search function on the workbench.
  • - Push Price Sheets for Multi Store - “Money, so they say” - If you have items associated with particular price sheets, you can now push that price sheet to matching items at other stores. It’$ $o Money
  • Single Sign on Enhancement - We have removed the login and password section if you use single sign on as it wasn’t necessary, and was kinda wonky to have it there.
