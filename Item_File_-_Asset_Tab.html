Asset Tab 

 

The Asset Tab lists information about the item such as purchase date, serial number, and depreciation information.

 

 

Capitalized - This check box denotes whether to depreciate the item. Unchecked means that no depreciation is computed. Checked means that depreciation will be computed each month.

Bought Used - This check box indicates whether the item was bought either new or used.

Depreciation - Clicking on this box will open the Depreciation Schedule window.

License No - Allows the user to enter the license number of the item.

License Expire Date - Allows you to enter the license expiration date of the item. You will be able to generate a report to show all items with expired licenses.

Markup Percent - This is the amount of markup for sales items. If you want to charge 50% over the cost, then enter 50 in this field. Whenever the purchase price changes, the selling price will be updated to 50% over the purchase price. This is an optional field. If you want to manually set the selling price, then leave this field blank.

Manufacture - This allows you to enter the manufacture for the piece of equipment.

Model No - Allows you to enter the Model No of the equipment.

Model Year - Allows you to enter the model year of the equipment.

Serial No - This Allows you to enter the serialized number for the inventory item being entered.

GPS No - This is the GPS provider assigned to the GPS unit on this unit.

EIN No - This field holds the equipment identification number assigned to this item.

Vehicle Type - This will allow you to enter the type of vehicle you are entering into the inventory.

Web Link - This will allow you to enter a weblink.
