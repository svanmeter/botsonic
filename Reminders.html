Erinnerungen


Auf der „Erinnerungen“-Seite können Sie eine Erinnerung mithilfe der internen Aufgabenliste von Point-of-Rental, Microsoft Outlook, Microsoft Calendar oder Google Calendar erstellen. Die Erinnerung kann für Sie selbst, für einen Mitarbeiter, eine Mitarbeitergruppe oder für alle Personen mit einem E-Mail-Konto erstellt werden: 



Erinnerungsart: Dies ist die Methode, die Sie zum Versenden der Erinnerung verwendet wird.

Datum/Zeit: Dies ist das Datum und die Uhrzeit, zu der der Empfänger die Erinnerung erhalten soll.

Senden an: Hier können Sie den Empfänger auswählen: Ausgewählte Mitarbeiter, Weiterer Mitarbeiter und Mitarbeitergruppe sind jeweils Pulldown-Menüs.

 

Wenn Sie Mitarbeiter auswählen und auf die Schaltfläche „OK“ klicken, wird die Liste der Mitarbeiter angezeigt.

 

Station: Im Dropdown-Menü haben Sie die Möglichkeit, alle Stationen oder eine einzelne Station auszuwählen.

Gruppe: Sie können verschiedene Gruppenarten auswählen (Büro, Fahrer, usw.)

Wählen Sie die Person(en) aus, an die die Erinnerung gesendet werden soll. Wenn Sie das Häkchen gesetzt haben, klicken Sie auf „OK“ und die Erinnerung wird gesendet.



Erinnerung- Dies ist die Betreffzeile und der Text der Erinnerung.

    *Hinweis: Wenn Sie eine Aufgabe oder Erinnerung haben, die fällig ist, erscheint ein FETTROTER Link in der unteren rechten Ecke des Counter System-Menüs. Wenn Sie eine Aufgabe für eine andere Person erstellen und wissen möchten, wann diese Aufgabe erledigt worden ist, markieren Sie das Kästchen „Bei Erledigung benachrichtigen“ und Sie werden über eine POR-interne E-Mail-Nachricht über die Erledigung informiert.
