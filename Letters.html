LETTERS

 

Letters will be based upon the Tab you are viewing at the time you click on Letters. For instance, if you are on the Reservations Tab the Letters available are Reservation Verify and Reservation Reminder.

            *Note: The Email and Fax fields are part of the Mass Fax/Email Module. This is an optional module and if those fields are grayed out then you have not purchased this module.

 

If you click on a contract displayed in Day-at-a-Glance, you will get a menu with various options depending on the status and type of the contract. The options are as follows:

Print Contract: This option will print the contract to the End of Day printer

Cancel Reservation: This option will only be displayed for Reservation contracts. It will allow you to cancel the reservation.

Close Contract: This option will go into the Close contract mode.

Modify Contract: This option will go into the Modify contract mode.

Inquire Contract: This option will go into the Inquire/Reprint contract mode.

Pay Contract: This option gives the opportunity to pay this contract.

Select Serialized: This option allows the selection of serialized items when a header item was sent out. This only shows up under the Non-serialized tab and if the parameters are set to not force serialized items on sending out.

Send Out Reservation: This option will only be displayed for Reservation contracts. It will send the reservation out.

Customer Balance: This option will bring up Customer Balance for the selected customer.

Mark as Unlocked: This option will unlock the contract so that changes can be made to the contract without entering a password.

Mark as Locked: This option will lock the contract so changes cannot be made without entering a password.

Update Operation: This option will allow the Transaction Operation to be updated.

Update Item Sub status: This option will allow the Transaction Item Sub status to be updated.

Create a Reminder: This allows you to create a reminder for you or another employee for this contract.

Show Call Logs: This option shows all the Call Logs for this contract.

Transaction Edit: Display all the Transaction Edits for this contract

Print Item Prep Report: This option prints the Item Prep Report for just this contract.

Print Load Slip: This option reprints the Load Slip for the selected contract.

Change to Reservation: This option will take an Open Contract and convert it into a Reservation.

Confirm Reservation:  This option will set the reservation as being confirmed. When confirmed, the reservation date will be shown in red. This option is only available to non-delivery reservation contracts. For delivered contracts, us Update Delivery/Pickup option.

Unconfirmed Reservation: This option will set the reservation as being unconfirmed. When confirmed, the reservation date will be shown in red.  This option is only available to non-delivery reservation contracts. For delivered contracts use the Update Delivery/Pickup option.

Mark as Loaded: This option will set the contract as already loaded on the delivery track. Once a contract is marked as loaded no other modifications can be done to the contract with an override password.

Mark as Unlocked: This option will reset a contract marked as loaded as no longer being loaded. You must enter an override password.

Print Driving Directions: This option will print the Map-It driving directions to the delivery to address. This option will only be displayed to deliver or pick up contracts.
