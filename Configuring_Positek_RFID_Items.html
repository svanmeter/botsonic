Configuring Positek RFID Items

 

 

 

This article will go over how to go live with the Positek RFID module, how the module works on a daily basis, and how to configure items in Point of Rental for Positek RFID upload. 

 

 

Topics in this article include:

  • Prerequisites

  • Going live with the module

  • How the module works daily

  • Configuring tagged items in Point of Rental for Positek RFID upload




 

Prerequisites

 

This feature is available in software versions 2019 and later. It is not available in recent iterations of the software. 

 

Going live with the module

 

1. Install the Positek RFID module. You will need a TeamViewer session and password to the Point of Rental server. Please contact our support team to get this established.

 

2. Mark items as Bulk Serialized. See below on how to do this. If you already have a list of items marked as tagged, you can perform a mass update to assist with the initial implementation.

 

3. Upload tagged items into Positek RFID.

 

Note: Point of Rental will provide a shortcut for this.

 

4. Once items are tagged in Point of Rental, switch the module to live status.

 

 

How the module works daily

 

  • When a reservation with tagged items falls within a window of being sent out:
    • If the customer is not already uploaded into Positek RFID, they will be uploaded.
    • If any tagged items are not already uploaded into Positek RFID, they will be uploaded.
    • The contract will be uploaded into Positek RFID.
    • The window is a time frame that Flex Productions specifies. The default is 14 days from when the contract is being sent out.

 

  • Each time a reservation with tagged items is created, the module checks to see if it falls within the window.
    • If the reservation falls within the window, it is uploaded.

 

  • Each night a scheduled task looks for previously created reservations that now fall within the window.
    • If a previously created reservation falls within the window, it is uploaded.
    • In addition, all tagged items are updated each night.

 

  • If you add a tagged item after the initial install, you may choose to upload it immediately using the shortcut or wait for the nightly upload to catch it.

 

Note: This is normally the evening after it is marked as tagged in Point of Rental.

 

 

Configuring tagged items in Point of Rental for Positek RFID upload

 

Here's how to get there

Expert: Program Menu > File Maintenance > Item File

Elite: Configuration Gear > File Maintenance > Item File

 

1. Open the item in Item File Maintenance from the File Menu.

 

2. Click Functions then the Modify menu option.

 


3. Go to the Options tab and find the bulk item dropdown underneath the check box in the far right-hand column.

 

 

4. Choose the Bulk Serialized option from the dropdown.

 

 

5. Click the OK button and the item will be marked as tagged.

 

6. Double-click the Positek upload shortcut on your desktop to upload the tagged item(s) into the Positek system.
