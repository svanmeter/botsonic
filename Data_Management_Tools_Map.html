Data Management Tools – Map Tax Code to State

 

This would only be used if you wanted to link all Customers in a state to 1 tax Code.  If you report multiple tax codes for a state you would not use this.

 

Select # 10 - Map Tax Code to State from the Menu.

 

Select The state from the drop-down Menu

Select the Tax Jurisdiction that you would like to link to that State.

Note The Warnings Below the drop-down Menus.

Click Go and the Customer Records will be updated.  You will see a pop-up window when the process is done.
