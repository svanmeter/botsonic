Mass Update Salesperson

Data Management Tools - Update Salesperson

Â 

1. Select Update Salesperson from the Menu.

2. You can select to Update By -- Salesperson - When you do so you will have the option to select the Previous Salesperson. Then Select either Customer or Jobsite. Then Select the New Salesperson. If you Opt to update by ZipCode see next step.

3. You can select to Update By -- Zip Code - When you do so you will have the option to enter the Zipcode . Then Select either Customer or Jobsite. Then Select the New Salesperson.

4. Select Go and all the Records will be updated. A Window will pop up letting you know the process is done.
