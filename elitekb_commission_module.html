Commission Module




This article will show you how to set up customer and item commission levels in Point of Rental. It is often necessary to have varying commission schemes for sales reps to accommodate business goals or handle negotiated commission plans by sales reps. This can be used to persuade salespeople to sell specific products over others for promotional purposes, adjust commission payouts based on customer attributes (corporate customer, new customer, 2-year customer, etc.), or pay varying commission percentages by sales rep. The commission module allows item categories to be grouped into user-created commission levels and then assigned to sales reps where percentages can be set for commission calculations. Customer commission levels can be created and assigned to specific customers to adjust overall commission payouts.


Topics included in this article
 

  • Getting there
  • Customer Configuration
  • Item Commission Level



Getting there

Here's how to get there:
Expert: Configuration > Customer Configurations > Customer Commission Level
Elite: Gear icon > App Configurations > Customer Configurations > Customer Commission Level

 

Creating Customer Commission Levels

 

1. Create the custom commission levels under Configuration > Customer Configuration > Customer Commission Level.
 

2. Create different levels for different types as illustrated below. Double-click in each field and enter a percentage.

3. Customer Commission Levels can be assigned to Customer Pricing Types or to individual accounts.
 

a. Configuration > Customer Configuration > Customer Pricing Types.

b. File Maintenance > Customer File.

 

The percentage entered will adjust to the overall commission payout. For example, if the commission was $100 and the customer is set to a level defined at 80, the payout would adjust to $80 or 80%. A long-standing customer may not earn a sales rep a higher commission level as a brand-new customer.


 

Creating Item Commission Levels
 

1. Create item commission levels through Configuration > Item Configuration > Item Commission Level.

 

2. Click Add to create a new level. Item Commission Levels could be set up to handle parts, under $100 retail, over $100 retail, major rental equipment, minor rental equipment, tenting, tabletop, or any designation you choose.

 

3. Assign the appropriate commission level to each Item Category by going to Configuration > Item Configuration > Item Categories.
 

 

4. In Configuration > System Configuration > Sales Rep, each item's commission level can be defined under Exceptions. This allows the user to define commission percentages for the varying Item Commission Levels by sales reps.
 

Note: If an exception is not defined for a sales rep, the standard commission levels will be used in the commission calculations. If you only want to pay on the exceptions, you would set the standard commission percentages to zero.
 


 

5. Click the Exceptions button.
 

6. Click in each field and enter the percentage of commission to be paid for the corresponding level and type of revenue (Rent, Sale, Asset, etc.). Select OK when done. 

 

While setting exceptions, you will also need to set the base percentages that your sales reps are going to receive.

 

These will apply to all items that don’t have an exception.

 

For any period desired, run the Sales Rep Commission Report from Accounting Reports > Sales Rep Reports Menu > Commission Report. Be sure to select “Enhanced Reporting." If you do not have the commission module, this option will not be accessible. This report will only be accurate with live data after an End of Day has been completed.
 

Note: Commission totals will show accurately in training mode.
 

Note: Commission does not calculate retroactively.
