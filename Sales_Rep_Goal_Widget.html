Sales Rep Goal Widget


This widget allows users to set sales goals against a sales rep and track both how successful they are and how probable it is that they will succeed. It is available on a sales type dashboard. It is filterable on both a sales rep and store level from the dashboard.

 


Goal - Goal configured for sales rep(s)

Actual - Closed contracts within a time period

Probable Open - Contract end date or line item will bill within a time period

Probable Reservation - Contract end date falls within a time period

Probably Quotes - Contract end date falls within a time period.  Additionally, it takes into account the probability set against the quote.  If the quote is worth $1,000 and the probability is 60%, $600 will be included in the probable quote value.  If no probability is assigned to the quote, it accounts for the full value.

 

To see the dollar values, hover over the different areas of the bar. Clicking on the bar will open a separate modal with the list of all contracts that make up that value.  Users can interact with this screen and open the contracts, customers, and sales reps to access more information and sales rep goals.

 

To use this widget, goals must be set on the sales reps individually.

 

Set the year(s) at the top and then from Goal Type, select ‘Sales Revenue’ or ‘Rental Revenue’ and select ‘Add Goal’. This will further expand the grid to allow for entry of monthly goals. Once the values are entered, select ‘Save’ at the bottom right. This will need to be done individually for both Sales and Rental Revenue. The entered values will be saved.


 

The goals will then be compared to the actual values recorded in the chart below the goals.  This can be viewed on either a dollar or percent basis dispensing on the icon selected next to the ‘Actual vs Goal’ label.

 

 

Overtime, users will be able to compare months/years by manipulating what years are seen. Goals from previous years and actual values cannot be updated.  Security under the ‘Sales Rep’ security area label ‘Update Sales Rep Goals’ can be used to control who can modify these goals.
