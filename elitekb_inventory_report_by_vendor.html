Inventory Report by Vendor

 

This selection produces a report of your inventory grouped by vendor.  The report will list, for each item, the item key, number, name, purchase date, purchase price, quantity, current store, home store, and serial number.

 

Once selected the following screen will be displayed:

 

 

Category – this button allows you to select only a certain item category to be reported.

 

Rental Items – check this box if you want your rental items on the report.

 

Capitalized Only – check this box if you want only capitalized items (items that you compute depreciation on).  This option is only available when you have Rental Items checked.

 

Sales Items – check this box if you want your sales items on the report.

 

Parts Items – check this box if you want your parts items on the report.

 

Include items with no purchase price – this option allows you to include items that no purchase price has been set.  With this option unchecked, only items that the purchase price for the items has been set will be shown.

 

Include zero quantity Items – this option allows you to include items that you currently don’t have in stock.  With this option unchecked, only in-stock items will be shown.

 

Print Details/Print Summary – this option allows you to print a detailed report or just a summary.  With Print Summary checked, only the totals for each category will be displayed.  With Print Details checked, each item’s information will be displayed.

 

Sort – the report is sorted by item category.  Within each category, you can choose whether it is sorted by item key, name, number, or part number.

 

Purchase Date – this option allows you to define a beginning purchase date and an ending purchase date to be reported.  You can leave either field blank.

 

Store to report – for multi-store systems this option allows the selection of a store’s inventory to be printed.

 

Select Printer – this button defines the printer that the report will print to.

 

Cancel – this button will cancel your previous selections and return you to the Program Menu.

 

Generate – this button will generate the report and display the report on your screen.
