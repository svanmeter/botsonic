Price Listing

 

This program generates a price list of your inventory items.  The list will be sorted by item category and within each category by item name, item key, or item number.  The report will show the item key, number, name, selling price and deposit.  For rental items, the rental rates for various time periods (up to 6) will also be listed.  Some items may not contain a rate for each time, depending on how you have set up the price tables in the ITEMS file.  For example, if you have not defined a monthly rate for an item, then nothing will appear in the monthly column.

 

First you must choose category, types, sorting option, and rental periods to be listed by selecting from the following options:

 

 

Category - By clicking on the CATEGORY button, you will be given the opportunity to select only a certain category to print.

 

Customer Pricing Type – by selecting a Customer Pricing Type any items that have special rates defined for that Pricing Type will print the special rates.  If no special rates are defined then the standard rates will be printed.

 

Rental Items – with the Rental Items option checked, all the rental items for that category or categories will be printed.

 

Sales Items – with the Sales Items option checked, all the sale items for that category or categories will be printed.

 

Accessory Items – with the Accessory Items option checked, all the accessory items for that category or categories will be printed.

 

Show Serialized – with the Serialized option checked, all serialized items will be printed.  Unchecked only the header item will print.

 

Columns – you can define up to 6 columns to print on the Price Listing Report.  The default columns will be the same as you have defined in the Parameters for display on the Item MORE screen.  However, you can change any of them by checking or unchecking the box in front of the desired time.

 

Format – You can choose a format with one section, two sections, or three sections across the page.  The more sections the less columns that can be displayed for different rates; however, more items will fit on a page.  For example, twice as many items will fit on one page if you use the two sections across instead of the single section.

 

Show Item Picture – with the Show Item Picture option checked, the pictures will be displayed for any items that have pictures.

 

Sort – the Price Listing can be sorted by Item Key, Item Name, or Item Number within each category.

 

Store – for multi-store systems this option allows the selection of a store’s inventory to be printed.

 

Printer Select – this button defines the output printer for the report.

 

Cancel – this button will cancel the Price Listing Report and return you to the Program Menu.

 

Generate – this button will generate the report and display it on the screen.  Once displayed, you can print the report to your printer.
