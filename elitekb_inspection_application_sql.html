Inspection Application (For SQL only)

 

The inspection Application is designed to make it possible for an inspection of rental equipment upon return from the rent  to be created and recorded via a device with internet access. Series of questions can be created and assigned to template that are assigned to various equipment that are answered by an inspector and stored in the system. Internal repair, maintenance orders or service calls can be requested if the equipment requires additional attention.
