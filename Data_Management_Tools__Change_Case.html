Change Case

 

Data Management Tools:

 

Change Case -- Allows you to clean up a Field in the Database.

1. Select the Table that you want to clean up from the Table drop-down Menu

2. Select the field within that Table from the Field Drop Down Menu. In the example below, we have selected the Table - Customer file and now we have highlighted the Field of City.

3. Now we can select the operation that we would like performed. In this case, we are choosing to change that field to Proper Case.

4. Click the go box in the bottom right-hand corner. The system will now change that field in that table. When it is complete you will see a box that says Done. Click ok To complete.
