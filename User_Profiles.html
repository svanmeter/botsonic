User Profiles




This article will define and walk you through User Profiles. User Profiles are saved profiles that keep track of user settings and defaults like printer and cash drawer assignments, store number, scan file name, OCR file name, and signature file name. Each saved profile is tied to only one terminal. However, you can have multiple User Profiles, one for each store you work in.

Here's how to get there


Expert: Program Menu > Security > User Profiles

Elite: Configuration Gear > Security > User Profiles



User Profiles

 

 

You can manually create a user profile and assign a default store by pressing the Add button. User Profiles can be modified by clicking on the user.


Default user profile by store: If you use the terminal naming of Str#PC#, then Point of Rental will remember the store number from the username. If there isn’t a specific user profile created, then it will look for the user profile DefaultStr#.

 

You can have a user profile for every store. When you go to another store and then go to Printer Setup and save, you'll be prompted to create a StrXAdminUserName profile or to update the default profile. 

 

 

Printer Selection Screen


When you click on a user profile, a new window will appear called the Printer Selection Screen. This is where you can modify the assigned printers for the user and set other defaults.
