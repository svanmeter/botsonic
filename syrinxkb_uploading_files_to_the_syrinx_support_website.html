Uploading Files to the Support Website.
In the web browser of your choice, browse to the following address:

https://www.syrinxsupport.co.uk:8443

You will see a login screen. Enter the following details:

 

User name: upload

Password: $yr1nx (note this is case sensitive)

 

You should now see a web page which has an ‘Add files’ button which you will need to press:

 

A ‘File Upload’ browse folder will appear which will allow you to select the file you need to upload. Browse for the file and then press OK.

The file will then be shown in the ‘Files to upload screen’. Some file types are restricted, such as *.exe, *.bak. In order to upload a Syrinx backup file (*.bak), please first compress the file to zip or similar format. If you select a restricted file type, then you will see an error similar to below:

 

You can keep adding files to the upload list by using the ‘Add files’ button:

When you are ready to start the upload, press the ‘Upload’ button on the “Files to Upload” screen:

Once the upload is completed, press the Logout menu at the top of the page:

Finally close your web browser.
