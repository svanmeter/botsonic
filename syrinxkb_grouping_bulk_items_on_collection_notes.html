Overview

Items on a collection note are, by default, sorted in the order that they were added to a contract.  With Syrinx, you can flag certain equipment to be grouped on collection notes, so that they appear as a single line on the printed collection note, and are sorted together on the off hire screen.

System Setup.

To use the feature, select the Fleet/Collection Sorting menu option to open the ‘Collection Note Sorting’ screen:

The first screen is used to specify how the different categories are to be sequenced on the collection note.  For example, you may want to sort certain types of equipment before other types.

Once the sequence of the categories have been set, you need to use the ‘Set Sorting for Stock Items’ button on this screen to set the sort sequence of the items within each category:

The sort sequence for the items is defaulted automatically according the stock numbers, but may be amended as required.

Off Hire Screen.

When the sort sequences have been set, both the Off Hire and Partial Off Hire screens will list the items according to the sort sequence:

Collection Note.

When the sort sequences have been set, the collection notes for full/partial off hires will group the items by the equipment, in the order specified:

Non sorted Items.

If a contract contains a mixture of equipment with/without sorting specified, then the equipment without sort sequences will be listed first on the screen/collection note (in the order it was added to the contract) before all of the equipment with sort sequences defined.
