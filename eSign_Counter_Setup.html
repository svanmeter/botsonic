eSign Counter Set Up



Overview

This article will explain the eSign Counter setup process, as well as how to set up basic information in the Point of Rental software.
 

Note: The eSign module will need to be configured before use. If you do not have eSign please contact our Sales team for more information.
 

Topics included in this article
 

  • From the Computer
  • Getting eSign API and Username
  • eSign Setup in Point of Rental
  • Parameter Settings
  • On the Tablet
  • Where to Find the Register Name in Point of Rental
  • Setting up the Terminal for Countersign


 

Prerequisites
 

1. A Wifi network MUST be in place for the device to connect to.
 

Note: Should be the same as the company's WiFi.

 

2. Install a Chromium-based Browser on the device. You can download this from the Apple App Store or Google Playstore.


 

From the Computer

Add Your Company Logo
 

1. Go to https://rentalesign.com.
 

2. Login using your existing eSign username and password. You can use the forgot password link if you don't remember it.

3. Go to Configuration > Company Configuration and then click on Logo.
 

 

4. Upload your logo adhering to the dimension restrictions provided: maximum of 250px X 100px.

 


 

5. Once the logo has been uploaded, click Save.

 

Getting eSign API and Username

To connect your software to start using eSign you will need to grab the API Key as well as the username that is associated with the account.
 

1. Go to rentalesign.com
​​​​​​​

Note: This might be different if you are in a different region.
 

2. Go to your Configuration > Company Details.
 

From there you will be able to get the API Key and Username.

 



eSign Setup in Point of Rental
 

Here's how to get there
Expert: Program Menu > Configuration > Reports and Notification > System Email Setup > eSign Setup
Elite: Configuration Gear > Legacy (Core) Options > App Configuration > Reports and Notification > System Email Setup > eSign Setup
 

1. Enter the username and API Key in the appropriate fields.

2. If you have separate eSign accounts for each of your stores, the proper username and API Key needs to be entered for each.
 


 

eSign Username - This is the username you set up on your eSign web page.
 

eSign API Key - Enter the API Key from your eSign web page.
 

eSign Server - This can be left blank.
 

Notifications Email - Enter the email address that will receive notifications when sending or receiving an eSign document.
 

Notifications - You can send a notification to the 'sending employee,' sending store, or both.
 

Use High Security - Checking this box gives an extra level of security. The receiving party will be prompted with a message to enter their email, confirming they are the correct party to sign the document. With the box unchecked, no message will be displayed.


Parameter Settings

Here's how to get there
Expert: Program Menu > Configuration > System Configuration > Parameters
Elite: Configuration Gear > Legacy (Core) Options > App Configuration > System Configuration > Parameters


Setup Tab

Go to your setup tab to choose where to force signatures on contracts.



Contracts Tab

Choose the Default format for Countersign in the Contracts Tab. If you do not choose one it will choose the Store Default Settings.




On the Tablet
 

1. Open a Chromium-based browser on the tablet.
 

2. Enter the countersign URL:  https://rentalesign.com/countersign
 

Note: This might differ depending on the region you were set up in.
 

3. Login using your existing eSign username and password.
 


 

4. Enter a name for the device (i.e., the name of the tablet).
 

5. Enter the Register ID for the register associated with the device. The Register ID is the name of the Point of Rental user profile that will be using the device.



Where to Find the Register Name in Point of Rental

Expert: Look at the bottom of the Program Menu. It is to the right of the date.



Elite: Right Menu > '?' icon > 

You will find the terminal name under the operator information.




Setting up the Terminal for Countersign

Expert: Counter System > Tools > Print Setup > Advanced


 

1. Select Modify and enter “Countersign” in the Signature Filename Field.

 

2. Click OK.
 


Elite: Operator Information > Operator Information > Print Setup


 

1. Click on the Advanced Tab.

2. Click Modify and enter “Countersign” in the Signature Filename Field.
 


3. Click OK.
