System Edit Report

 

Allows you to see who is making changes to system bales such as parameters and taxes. Now you will know who is messing with your parameters.
 

1.  Go to Program Menu

2.  Select Database Management

3.  Under Logs, select System Edit.

 

 

System Edit Report.

 

 

The Report will show how many edits were made, followed by the date and Operator's Name, the type of Modification and were the modification took place.
