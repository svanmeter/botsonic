Register with Mozy for Offsite Data Backups

Backing up critical data offsite is a smart move to prepare for disasters.  Mozy is Point-of-Rental Systems’ recommended offsite data backup vendor because of their reliability.  Below you will find instructions for registering with Mozy.  After registering, if you need help configuring the Mozy program you can  Point-of-Rental Systems support for help at 800-944-7368.

 

Register

1.     Open an Internet browser and navigate to www.mozy.com. http://www.porhelp.com/

2.  Click the Sign Up link to register and complete the registration.  Be sure you select the Business (MozyPro)product during registration and sign up for a minimum of 10GB of offsite storage space. 

3.   Be sure to document your MozyPro email and password.

Install

1.     After registering you need to install the MozyPro client on the server hosting the Point-of-Rental Systems software.  Go to the server, open an Internet browser and navigate to www.mozy.com and log in using the email and password you documented.

2.  Once logged in navigate to Resources > Download MozyPro Client and download the latest version to the server.  You could also download the client from another computer, save it to a USB drive and plug it into the server.

3.  After downloading navigate to Control Panel > Install Application on Remote Desktop Server.  When the install wizard prompts you navigate to where you saved the MozyPro client and select it to start the install.  Follow the Mozy install wizard through completion.

Configure

1.     Once installed you will automatically be taken to the Mozy client in the Settings section where you can select which files and folders to backup.  You can also open the Mozy program later by browsing your start menu and clicking on the MozyPro Status shortcut.  When opened click the Settings button.

2.  We recommend to first select the entire \POR\ directory to be backed up.  If you wait a moment after selecting files the Mozy Settings window will indicate how much space you have available after backing up the selected files.  If you do not have enough space then folders   within the \POR\ directory could be un-selected.  See our What Files Need to be Backed Up document.  If you need it please call 800-944-7368 to request it.

3.   We recommend setting a backup schedule where the offsite backup runs at a specific time after hours.

4.   Check your backup logs every day.  Mozy will automatically prompt small Mozy Status window when users log into the server displaying the status of the last backup attempt
