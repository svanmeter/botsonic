This article will show you how to keep track of the inventory items you missed renting, for example, because you didn't own the item or you didn't own enough for the time. Missed Rental allows you to track the reasons customers did not rent from you. 

Topics included in this article:
 

  • Missed Rental
  • Add Different Reasons
  • Create a Missed Rental from a Contract
  • Reports



Missed Rental
 

1. Select the Missed Rental icon in the top menu bar. You will be prompted to search for the customer or create a new customer record. You also have the option to select No Customer.  

 

2. Select the missed rental item from the drop-down menu and the quantity. Note: If it is an item you do not carry, select N/A and fill out the Item Name field instead.  
 

    

3. On the right-hand side, select a reason and add any additional notes. 



4. Press Save when finished.  

 

 

 

Add Different Reasons
 

To add different reasons to the Reasons dropdown, go to Configuration > Configurable Dropdowns.
 


From here, you can add, remove and edit the reasons you create.
 

 

 

Create a Missed Rental from a Contract
 

You can track missed opportunities as we now support converting Quotes and Reservations into Missed Rentals. 
 

From the transaction screen, click on the Copy to Missed Rental button.
 


You’ll be able to convert the transaction along with assigning a Missed Rental reason.

 

 

Reports 

 

1. To see a list of submitted Missed Rentals, go to Reports in the top menu bar.

 

 

2. Under the Transaction Reports column, select Missed Rentals. From here you can search by item name or leave the item name field blank to see all items instead.

3. Select Run Report after you select the dates.
