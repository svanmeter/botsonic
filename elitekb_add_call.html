Add Call




Overview
                    
This article will explain how to add a call to contracts, customer records, and the customer dashboard. With Add Call, you can enter customer information into a call log for tracking and informational tracking purposes. You will be able to keep track of who reached out to a customer, how many times, and what the nature or purpose of the call was for, as well as any additional notes that were taken during the course of the call.


Topics included in this article
 
  • Adding New Call
  • Add Call through AR Manager
  • Add Call through Contracts
  • Call Log
  • Edit Call Log Security

Here's how to get there: 
Expert: Program Menu > File Maintenance > Customer File
Elite: Gear Icon > File Maintenance > Customer File


Adding New Call


1. From here you will select a customer and be brought to their dashboard.

2. Select the Calls tab and press the New Call button. 
 

New calls can also be added from within a contract, through the AR Manager, and within the Customer Dashboard. Next, we'll walk through adding calls in the AR Manager.


Add Call through AR Manager

Here's how to get there:
Expert: Program Menu > Accounts Receivable > Accounts Receivable Manager
Elite: Accounting > Accounts Receivable Dashboard

 
1. Find your desired customer.

2. With a right-click you can bring up a small window that will allow you to open the Customer Dashboard or create a new call. 
 

3. Selecting Calling Customer opens a new call log that you may fill out. You can even get to the customer dashboard by selecting the Customer Dashboard button in the bottom left corner.

4. If you instead select the Customer Dashboard, you will select the View Calls button found on the top bar of the menu.
 

5. Select New Call and you will be able to enter a new call log for that customer.
 

Now we'll walk through how to add calls through a contract. 


Add Call through Contracts

Here's how to get there:
Expert: Program Menu > Counter System > Modify Contract
Elite: Transactions > Modify Contract

 
1. Search for and select the contract you wish to modify.

2. On the contract processing screen, you will press the View Calls tab.
 

3. Press the New Call button in the bottom right corner to create a new call log. 
 


Call Log
 

Customer Call Log - You can enter a call log each time you talk to the customer. If they call in to make a change, you can enter the name of the person requesting the change. If you call the person because they are late or their payment is overdue, you can make an entry in the call log stating the particulars of the call.


Print Call - Will print the calls on this contract to the report printer.


New Call - Will allow another call to be entered on this contract. The date, time, and operator's name will automatically be entered.


Call Type - In Configurations and Call Log Types you can select different call types in this list to group certain call logs together.


Force Acknowledgment - When entering a call, you can click on Force Acknowledgment. Every time this contract is accessed, the call log will be displayed. You can also check this on a call log only pertaining to a customer and not a particular contract. Every time this customer is accessed, the call log will be displayed.


Note: Call logs entered on one contract in a continuation series are visible on all contracts in the series.

 



In Elite, call logs can also be added within customer and jobsite records in the browser. There they can be related to one another so that they are searchable by either. 




Edit Call Log Security

Here's how to get there:
Expert: Program Menu > Security > System Security
Elite: Gear icon > Security > System Security
 

1. If you are multistore you will need to select the store you wish to change settings for.

2. From the dropdown next to 'Edit call log modifications' you can select your password level.

 


3. Go to the customer record to retrieve the call log.

4. Select Modify from within the call log.

 


5. You will be required to enter your user id and password based on modification settings. After those have been entered you will be able to edit the call log.

Elite: Right Menu > Search Customer Call Log
