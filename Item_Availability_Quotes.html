Artikelverfügbarkeit Angebote

Auf diese Weise können Sie innerhalb von Artikelverfügbarkeit ein schnelles Angebot erstellen, ohne einen Angebotsvertrag zu erstellen.

 

Von hier aus können Sie einen Angebotsvertrag anzeigen, drucken oder erstellen. Wenn der Kunde entscheidet, dass er mit dem Angebot fortfahren möchte, klicken Sie einfach auf die Schaltfläche Angebot erstellen.


Angebot erstellen

1. Klicken Sie auf die Schaltfläche Angebot erstellen.

2. Geben Sie das Gültig-bis-Datum des Angebots ein.

3. Geben Sie die Angebots-Zeit ein

Hinweis: Der Angebotsvertrag wird angezeigt

4. Geben Sie die Menge ein

Hinweis: Sobald die Menge eingegeben wurde, beenden Sie den Vertrag wie gewohnt.
