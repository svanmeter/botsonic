Erstellen einer ACH/Direktzahlungsmethode

 


Übersicht

Dieses Dokument führt Sie durch den Prozess der Erstellung einer Auszahlung für eine ACH/Direktzahlungsmethode. Auszahlungen können als Zahlungsmethode für Zahlungen innerhalb von Verträgen oder durch Rechnungszahlungen verwendet werden.

Die Themen umfassen:

  • Erstellen einer ACH/Direktzahlungsmethode
  • Deaktivieren von Auszahlungen
  • Web-Zahlungen mit ACH-Zahlungen


Wie Sie dorthin gelangen:

Expert: Programmmenü > Konfiguration > Buchhaltungskonfigurationen > Auszahlungsartikel

Elite: Zahnradsymbol > App-Konfigurationen > Buchhaltungskonfigurationen > Auszahlungsartikel


Erstellen einer ACH/Direktzahlungsmethode


1. Sobald Sie bei den Auszahlungsartikeln sind, klicken Sie auf die Schaltfläche „Hinzufügen“, um eine Auszahlung zu erstellen.

2. Geben Sie die Stationsnummer, den Namen und das Sachkonto (falls zutreffend) ein. 


3. Geben Sie an, wo die Auszahlung als Zahlung verwendet werden kann, indem Sie ein oder beide Kästchen ankreuzen:


  •    Als Rechnungszahlungsmethode verwenden – Wenn diese Option aktiviert ist, können Sie diese Auszahlung als Zahlungsmethode verwenden, wenn Sie eine Zahlung über die Debitorenbuchhaltung entgegennehmen.
  •   Als Counter-Zahlungsmethode verwenden – Wenn diese Option aktiviert ist, können Sie diese Auszahlung als Zahlungsmethode verwenden, wenn Sie eine Counter-Zahlung vornehmen.


4. Um eine separate Sachkonten-Buchung für die Verwendung der Auszahlung zu erstellen, markieren Sie das Feld „Separate Sachkonten-Buchung pro Auszahlung“. Wenn nicht, fahren Sie mit dem nächsten Schritt fort.



5. Geben Sie Ihre maximalen Auszahlungsbeträge je nach Benutzerpasswort ein. In der obigen Abbildung kann ein Benutzer mit einem Passwort auf Managerebene diese Zahlungsmethode verwenden, um eine Zahlung von maximal 25,00 $ vorzunehmen. 


6. Klicken Sie auf OK. Die folgende Abbildung zeigt ein Beispiel für eine Rechnungszahlung, die mit einem Auszahlungsartikel vorgenommen wird.


Deaktivieren von Auszahlungen

Um eine Auszahlung zu deaktivieren, öffnen Sie die gewünschte Auszahlung und markieren Sie unter Sachkonto das Feld „Inaktiv“.



Web-Zahlungen mit ACH-Zahlungen

 

Rental Hosting unterstützt ACH als eine Web-Zahlungsoption über CAYAN.

Es muss ein Auszahlungsartikel mit dem Namen „ACH Web-Zahlung“ eingerichtet werden. In der Regel ist dies die Station „000“, es sei denn, der Benutzer benötigt unterschiedliche Sachkonten für jede Station.
