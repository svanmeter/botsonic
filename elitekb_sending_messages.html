Sending a Message



This article will go over the different ways you can send messages to your employees. You can send interoffice emails and instant messages. Instant messaging is only available for Elite and POR One users.



Topics in this article include
 

  • Interoffice Email
  • Messaging (Elite & POR One)



 

Interoffice Email

Depending on how the employee's Operator file is configured, they can receive the interoffice message via interoffice mail, text message, or email.

Here's how to get there
Expert: Counter System > Employee Email/Logs > Send Message
Elite: Left Menu > User Tools > Employee Email and Logs > Send Message

 

1. If you want to send an email, select the Send Message button.

2. Once selected, the Enter Email window will display, and you may type in your message. If you reach the end of a line, the system will automatically wrap around to the next line.

3. When you are finished, select the OK button.

4. A listing of operator names that can receive your email will be displayed like the below messages. By default, all active employees will be listed. You can optionally list only employees from a store or from a group. In either case, left-click the employee name to select an employee(s) or click All to select all employees displayed.
 

 

5. Once you have the desired employees selected, click the OK button.

 

Messaging (Elite & POR One)


Within Elite

Point of Rental has the ability for you to send instant messaging to your employees. This can be sent via the Elite Browser Interface from the Right Menu and then clicking on the  icon.

To use this feature, you have to log into Elite using your Single Sign-on credentials. This feature is not available using your core credentials. To learn how to create and login with SSO credentials, see our article Elite | Single Sign-On (SSO) and Global User Admin.


To create a new message, click on the New Message button and search for the user(s) that you want to contact.
 

 

If you are sending a group message, you will need to create a group chat name.
 

 

When you're ready, you can add your message and send images and files as well. Please note that there is a limit on the size of the file or image.


 

Within POR One

Click on the Messaging icon in the app.
 

 

Please make sure your notifications are enabled on your smart device to ensure you receive notifications and messages. For more information on changing your notification settings, visit the help pages linked below.
