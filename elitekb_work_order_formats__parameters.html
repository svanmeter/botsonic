Work Order Formats - Parameters


Location: Program Menu>>Configurations>.System Configurations>Parameters>Contracts Tab

Purpose: This allows you to set contract formats for different types of contracts.

Key Features:  Below is a list of different types of default formats you can select from.
 
Laser ContractCompleted
EmailWork Order
FaxeSign
QuoteCounter SIgn
ReservationDelivery eSign
OpenPDF Modification
ClosedLoad Slip


 After you created your various formats through Work Orders, you can set when a format is used.

Go to Configuration/Parameters/Contracts.  You will see a Formats Tab.  From there you may select default formats. From the drop-down menu, you can select (Store Default, Standard Contract, Delivery Contract, Pickup Contract, Email/Fax Contract, Without Amounts, Invoice, or PDF Modification)
