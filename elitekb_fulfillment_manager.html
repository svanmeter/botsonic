Fulfillment Manager


Add information on YES* and hovering over it Fulfillment Manager is a tool used to see contracts that have completed the fulfillment process and are ready for final review prior to invoicing.

At the top, define the number of days you want to look at. You will want to check the box for ‘Include Reviewed’ if you want to see contracts that have already gone through the review billing process. The data filter allows you to include or exclude information. If you have applied a filter, the data filter button with turn green.

The fulfilled column will give you the details of where a contract is in the fulfillment process.

Yes*-Indicates no prep out has been done, and thus, you expect no prep in (ie contracts that are just tables and chairs)

Yes-Indicates all departments have been fulfilled.

No-Indicates no departments have been fulfilled.

In Progress-Indicates some departments have been fulfilled.

 

By hovering the mouse over the status, a box will pop up that indicates which departments have been fulfilled and which you are still waiting on.


Individual departments can mark themselves as fulfilled by selecting the ‘Done’ button when they are finished checking in their part of the contract. This allows a department to acknowledge they are finished; this is important when there are missing items or when all items have been counted in but the department is still working through the inventory. This button is only visible when a specific department is selected. The button will then turn green to indicate it has been clicked.

Right-clicking on contracts within the Fulfillment Manager will give you the option to see an overview, inquire, modify, close, or access the customer dashboard.
