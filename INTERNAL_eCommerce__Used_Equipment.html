Overview


This article documents the various ways to add used equipment to websites.

 
 

From the customer side


In POR, the customer must mark rental items to populate the Used Equipment Sales page on the website.

  • Select a serialized item, not a header record
  • Click on the Tools drop-down
  • Click on Mark Item for Sale
    • They’ll be prompted to enter a sale price and reminded to enter the manufacturer, model, and model year.
 
 

Simple listing for stores with small used inventory


For stores with small inventory at any given time, use our one-page listing, forsale.asp.

This file is a simple plug & play and only needs the Title Tag and <metaDescription> updated.
 


Example

https://badgerrentalservices.com/forsale.asp 

 

Category listing for stores with extensive used inventory


For stores with a lot of inventory, we can show it by category to keep the list from being too long on the page.  For this, use used-equipment-sales.asp.
 

When using this option, you’ll need a rule in web.config:

<rule name="used-equipment-sales">

                 <match url="^used-equipment-sales.asp" />

                 <action type="Rewrite" url="forsale.asp?action=catpics" />

             </rule>
 

It also requires <!--#include virtual="porcommon/forsale_2020_c.asp"--> in forsale.asp.

 

You’ll need two include files: used-equipment-sales-header.inc and used-equipment-sales-footer.inc.  These files contain the customer-provided text that is shown at the top and bottom of the page. The text is optional, but the files are required. If the customer does not want any text in either spot, you can control it with keywords in StoreFlag(20); see below.

 

These files can be found on our server in porwebredesign/000Admin/Development/Used Equipment By Category



Example

https://actionrental.com/used-equipment-sales.asp cat-pics landing page.

We have at least one website, https://chasecorental.com/used-equipment-sales.asp  that shows just text on the landing page.

I see no reason to create pages like this, since the other option lets them add text at the top and bottom of the page with cat-pics in between. The ONLY exception I can think of would be if the customer wants a generic image on the page.


Values for StoreFlag(20) in customize.inc
 

ShowUsedEquipmentHours - shows hours of use.
 

ShowUsedEquipmentYear - shows model year.
 

ShowUsedEquipmentHWL - shows height, width, and length fields.  *requires POR v2020
 

ShowUsedEquipmentRetailPrice - shows retail price in addition to sale price.
 

ShowUsedMarkedHiddenItems - shows item even if it is marked Hide on Website.
 

ShowUsedEquipmentHeader - shows used-equipment-sales-header.inc text on used equipment page.
 

ShowUsedEquipmentFooter - shows used-equipment-sales-footer.inc text on used equipment page.
 

ShowUsedEquipmentInTable - shows items in row-by-row table format.
 

See Ideal Rent-All

SkipUsedEquipmentHours skip querying missing MTIN hours field on used equipment page on older POR stores

 

ExcludeUsedEquipmentForStores - excludes item records in home locations as set in the constant

                                                                            

Const ExcludeUsedEquipmentForStores

See AAA Rent-All (LA)

ShowAllStoresUsedEquipment shows items from all stores, regardless of the store selected for the main catalog.

See Aurora Rents – they are showing only Store 001 items in their catalog, but picking up items from all stores for their used equipment page.
