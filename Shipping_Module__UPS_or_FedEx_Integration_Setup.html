Shipping Module | UPS or FedEx Integration Setup & Process



 

This is an add-on module and is not part of the standard Point of Rental software. This article will go over the shipping process and explain the steps necessary to set up the data connections required for the UPS or FedEx integration for creating the Ship Link Data Connection.

 

Topics in this article include

 
  • How does it work on a contract?
  • Setting up the ODBC Connection from UPS/FedEx to Point of Rental
  • Processing a Shipment
  • Tracking shipments within Point of Rental




How does it work on a contract?

This is done by adding an item to the contract with an Item Key that starts with "UPS" or "FEDEX." It needs to be on the contract with a $0 price. Whenever the contract is pulled up later (modify, close, send out, etc), it will look up the shipping cost from Worldship, multiply that by the Vendor Markup Percentage, and put that price on the contract.

 

Setting up the ODBC Connection from UPS/FedEx to Point of Rental

Setting up an ODBC connection allows the UPS and FedEx software to read and write to your Point of Rental system. If you are using a cloud environment, then a VPN (virtual private network) needs to be configured.
 

1.  You must be logged in as an administrator.
 

2.  Run: c:\Windows\SysWOW64\odbcad32.exe
 

3.  Click on the System DSN tab.

 


 

4.  Click the Add… button.
 

5.  Select the driver named: Microsoft Access Driver (*.mdb). There are several variations of this name; be sure to select the correct one.
 

6.  Select the Finish button.

 


 

7.  Fill in the Data Source Name and Description.
 

8.  Click on the Select button to choose the database.
 


 

9. Navigate to the location of your PORShipLink.mdb file. It will be in the POR folder that is typically located on the C: drive. The drive letter will vary depending on your specific setup.  
 

10. Press the OK button.
 

 

11. When completed, your ODBC Data Source Administrator should have an entry for your newly added database.
 


 

Processing a Shipment

 

  1. Ensure the ‘Delivery Address’ fields on the contract have been filled out properly.
 
  1. Open the UPS WorldShip program.
 
  1. Select the Import/Export tab then click on the Keyed Import button like in the photo below and choose POPUPS.


 

  1. Enter the contract number in the import Key. This going to prompt POR to pull the shipping data for the customer.
 
  1. Fill in the remaining shipping information (UPS service, dimensions, weight, etc.) on the right side of the window. When you're done, select the Process Shipment option in the bottom right corner of the window.



Tracking shipments within Point of Rental


After you have completed a shipment, verify that the tracking information has been updated within Point of Rental.  
 

  1. Open the contract within Point of Rental.
 
  1. Select Tools, located in the top toolbar then select Track Shipment.


 

  1. A new window will open containing the tracking information for the contract.
