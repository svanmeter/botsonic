Data Management Tools

 

The database management tools is a list of different custom tools to make easy database edits/changes.

Change Case

This allows you to change cases in a certain field. You first select a table, then select the field within that table that you want to change, then select whether you are changing to UPPER CASE, lower case, Or Proper Case.

Merge Item Categories

This allows you to merge an item category into another item category. First, choose the old item category, then choose the new category to merge it into and it will merge the old category into the new.

Damage Waiver Change

This allows you to set a new damage waiver for all items or groups of items. You can set the division to change all items within a specific division, or you can set the category to change all items within that category, or you can do it for your entire inventory across the board. Just enter the item selection, then enter your old damage waiver, then enter the new damage waiver. If you enter a value in the Old DW % field it will only update items with that old damage waiver to the new damage waiver, otherwise check the all items box to update all items in that category/division.

Item Percentage Change

This allows you to update the percentage fee for items. You first select your division, then select the item category(you can also choose all categories and all divisions), then input the old percentage fee, and finally input the new percentage fee. If you input a percentage fee in the old item % field, it will update any item records with that percentage fee to the new percentage fee, otherwise check the all items box to update all the items in that category/division.

Item Markup Percentage Change

This allows you to change the item markup percentage for your items. First select your division, category, vendor, and or item type. Then enter a value into the change % from the field to select the old markup percentage you want to change(or check the box for all items), then you will enter your new markup percentage.

Sell Price Change

This allows you to change the sale price of your items. First select your division, category, vendor, item type. Then you will select how to update, either from item markup percent, percent of last purchase price, or percent of the current sales price. If you choose either of the latter two, you will need to input an updated percentage. If you want to make sure no current prices decrease, check the “Do not Decrease Sell Price” box. Lastly, enter what you want to round the price by. ie: if you enter 0.05, it will round to the nearest nickel. If the markup percent determines the new price would be $1.03, it will round to $1.05).

Mark Customer Closed

This allows you to mark inactive customers as closed. Select the customer status(either account or cash), then select the last active date of the customer. If you select a last active date of 1/1/10, it will mark a customer not active since that date is closed.

Update Salesperson

This allows you to change the salesperson via zipcode or transfer customers from one salesperson to another. First select whether you are updating customers within a zip code, or change customers from one salesperson to another. Then select whether you want to update the salesperson set on the Jobsite, or set on the customer file. Then either enter your zip code, or the current salesperson that you are changing from. Lastly, enter the new salesperson.

Contract/Statement Send To

This allows you to update either your mass send to or your statement sends to from one method to another. If you want to change your mass send to, select contract sends to, otherwise, select statement sends to. Then select the method you want to change, and lastly select which method it should update to

Map Tax Code To State

This allows you to map a tax code to an entire state. First, select your state, then select the tax code you want to set for that state. *warning: this will erase any pre-existing zip code links for that state*

Sub Rent Quantity Change

This allows you to change the sub rent qty of item files. First select your division, category, item type, then select bulk or serialized. Next, enter the old qty you want to update(or choose all items to update all sub rent qty amounts), then enter a new sub rent qty.

Data Field Clean-up

This allows you to overwrite specific fields in item file or customer file, determined by what is already in that field. Select your table, item file, or customer file, then select your field. Now it will show you all the different values that you have entered for that field. You can enter a new value to update every record that has that pre-existing value. Eg: if you run by item file and fuel field, the online items may say cGAS. It will say count to show you how many records have a fuel field that is set to cGAS. You can enter a new value to update all of those items to something else, like cGAS-1.

Depreciation Setup

o This allows you to quickly set up depreciation on items. Choose your division, category, item type, and or purchase price. Next, set your salvage value, years, schedule, and then method. This will update your selection with the new values.

Customer Rewards

This allows you to easily enroll customers in the rewards program. First, select your customers by status, type, group, or any combination of these things. Next, choose whether you want to exclude wanted/undesirable/closed accounts, then also choose whether you want to include credit hold accounts.

Convert to Base Rate

This allows you to change rates from a standard rate to base rates in mass. * Note - It is advised that you try in training first.

Service Update

When a route or a portion of a route needs to move from Tuesday to Wednesday or you want to suspend a route temporarily, you can do this in one screen vs modifying each individual contract separately.

Special Rate Update

This allows you to update already existing special rates. You can search for special rates by Items, Customer, Pricing Types, Job sites, and Store. By using the Update options in the Update section, you can select a field to update and enter its new value.

Rental Rate Adjustment

Allows the user to schedule future rate changes on rental items.
