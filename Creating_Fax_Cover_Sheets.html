Creating Fax Cover Sheets


 


Overview

This article will walk you through implementing a fax with the cover sheet page from the contract printer screen, as well as how to create custom fax cover sheets for use when faxing transactions to customers from your Point of Rental system.


Topics in this article include:
 

  • Fax with System Cover Page
  • Creating a New Fax Cover Sheet



Here's how to get there
You will be brought to the Contract Printer window upon the completed creation of a new contract, the modification of an existing contract, or the closing of a contract.



Fax with System Cover Page

From this Contract Printer window, you will select the Fax with System Cover Page option once your Fax Cover Sheets have been created and uploaded to Point of Rental. This is covered in the following section.
 



Creating a New Fax Cover Sheet

Here's how to get there
Windows Settings > Bluetooth & devices > Printers & scanners > Fax > Open print queue

 

the bluetooth & devices screen


the fax screen
 

1. Select Tools then Cover Pages.
 

 

2. Select Copy.
 

 

3. Choose a format to copy and double click the file. 
 

 

4. You can rename the file.
 

 

5. Select Open. You can then edit as necessary.

 

 

6. Select File and then Save.

 

7.  Select File and then Save As.

 

8. Browse to C:\POR\FaxCover and click Save.

 

Your new coversheet will now be available to select when faxing a contract from Point of Rental or Windows.
