Item Names
 

  • For SEO purposes, Item Name should always be in conversational format. i.e. Genie 50ft Towable Boom Lift instead of catalog format, i.e. Lift, Towable, Genie Boom Lift, 50'.
  • Apostrophes and quote marks should be avoided when possible.
 

Item Name vs Item Description

 

  • Some stores use the catalog format, but their staff is used to it, so they instead may want to use the description on the website. Some stores just like having more characters to work with.

  • Item Name can be up to 40 characters. Description can be up to 100 characters.

  • Item Name shows by default on websites. Description can be shown instead. This is site wide you cannot pick and choose by division, category, item, etc.

  • Consult with the Web Team if you're unsure which choice to make; ultimately the Web Team will need to enable this choice in the code.


 

Manufacturer & Model

 

  • If you’d like to display manufacturer/model info on your site, make sure the Manufacturer & Model fields in your Item Detail (Asset tab) records are filled in as much as possible. Let the WebTeam know you’d like this feature enabled.
 
 

Manufacturer Link

 
  • If you’d like to display a link to a manufacturer website, fill in the link in your Item Detail (Asset tab) records. Let the Web Team know you’d like this feature enabled.
     


Specs

 

  • Edit your Spec field in your Item Detail (Details tab) record, as these will be displayed as “comments” on your site.

  • The type of information you’d include in your Spec field is dimension, weight, general info about your rental, etc.  You may also enter keyword-rich text for SEO purposes.

  • Pressing the “Enter” key as you type in your specs causes a line break in the specs.

  • See All About Item Specs for more information.

  • This feature is enabled by default; please let the WebTeam know if you'd like this feature disabled.
     

 

Item Images

 

  • When inserting images for your Items into your POR Item Detail (Pictures tab) record, use images that are at least 250 pixels wide to give you the best quality on the site. 

  • 600x600px is the largest size POR will import.

  • To get the best quality, insert the pictures on your POR server, not from your terminals.

  • See Item and Category Images for more information.

 
 

Header Records
 

  • If you have header records created in your POR, we do NOT upload the corresponding serialized records to the web. This means that you only need to insert images & specs for header records and skip their serialized records (which will save you lots of time).

 

Pricing
 

  • If you do NOT want pricing to show for an item, enter ‘<NOPRICING>’ in the WebGroup field on the Item Detail (Options tab) record. Be sure to type <NOPRICING> exactly, with no spaces, all capital letters, and the leading and trailing arrows!
 
  • If you have other codes entered in the WebGroup field, this code may be entered after those codes.

 

Hide/Exclude Items
 

  • File Maintenance
    • Item File – Options Tab
      • Check the Hide on Website checkbox
 

New Items

 

For a 'What’s New' page on your site, you’ll mark your individual Item records:

 
  • File Maintenance
    • Item File – Options Tab
      • Enter ‘<NEWITEM>’ in the WebGroup field.  Be sure to type <NEWITEM> exactly, with no spaces, all capital letters, and the leading and trailing arrows!
      • If you have other codes entered in the WebGroup field, this code may be entered after those codes.
 
 

Used Items
 

For a Used Equipment page on your site, you’ll mark your individual POR Item records. You must mark the serialized item itself, not the header record:

  • File Maintenance
    • Item File - Tools pull-down
      • Mark items for sale option
 

You’ll be prompted to enter a sale price, then reminded to enter the Manufacturer, Model, and Model Year.  On the website, the manufacturer and model will be shown by default. Model Year and Hours of Use are optional; please notify the WebTeam if you’d like either of these enabled.
 

To remove an item from the Used Equipment page:

  • File Maintenance
    • Item File - Tools pull-down
      • Mark items for sale option
 

You’ll be notified that the item is already marked for sale and asked if you’d like to delete it from the list. Click ‘Yes.’

Entering zero-quantity items to show on your website


For items to NOT affect your GL, enter '9999' in the sub-rent field. Any item record with a value in that field is ignored by the GL but will be uploaded to the website.

You may also enable ALL zero-quantity/negative-quantity items to upload to the website in the Options dropdown in the Update Website process. But consider this carefully before you opt to do so!!


 

Entering Suggested Items

 

You may wish to suggest add-on items: either sale items or additional rental items, to show on the website with a particular rental item, i.e. a bucket with a loader.
 

  • File Maintenance
    • Item File – loader record - Tools pull-down
      • Suggested Items


Add items; these will then show on the website as Suggested Items on the loader item page. Items will show in the order entered.

You may then mark each suggested item record “Hide on Website” to prevent it from cluttering your categories on the website as standalone items. If you choose to do so, please notify the WebTeam.
