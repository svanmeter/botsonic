Database Files

• All .MDB Files are to be backed up

o  Two or more .MDB files will be in the root of the POR\ directory.  Make sure to back them all up.

 

 

Subfolders to Include in Backup

 

• POR\AppTasks\ folder contains the tasks initiated by the Point-of-Rental program.

•POR\Attachments\ folder if you use attachments.

o  (Note these are only links, the actual files “attached” would need to be backed up, wherever they are located)

•POR\Backup\ folder contains many scripts that may be tied to scheduled tasks.

•POR\Contracts\ folder if you are saving PDF versions of contract Modifications.

o  (This folder can take a long time to process due to thousands of small files)

•POR\Dash\ folder contains the settings for the Dashboard.

•POR\EmailAttachments\ folder contains any email attachments created by the customer used within Point-of-Rental.

•POR\EndOfPeriod\ Maintains PDFs of specified reports.

•POR\EndOfMonth\ Maintains PDFs of specified reports.

•POR\Instruct\ folder if you have any Item Instruction Documents.

•POR\Letters\ folder if you have made any Customer Letters, etc.

•POR\Map\ folder if you have the Dispatch Center module.

•POR\Maintenance\ folder If you have made any Maintenance To-Do Lists.

•POR\Messages\ folder contains the internal POR messages.

•POR\PcChargeConfigDisk\ folder if you have the Integrated Credit Card option.

•POR\QB\ folder if you export the GL entries import files to it.

•POR\Reports\ folder if you have created any custom Contract Formats.

•POR\ReportsCustom\ folder if you have created any custom Crystal Reports.

•POR\Statements\ folder if you want PDF versions of account statements.

•POR\Service\ folder if you are creating Service Schedules

•POR\Web\ folder if you have the Hosted Website option.

•POR\WebContent\ folder if you have Hosted Website option.

•POR\WebTasks\ folder if you have the Mobile WorkForce module.

 

If you are multi-company then make sure to get all of the above files and folders from the separate POR3\, POR4\, POR5\, etc. directories.  If any subfolders listed are not present it means that feature is not being used on your system.
