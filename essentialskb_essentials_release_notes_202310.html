Release Notes for v2023.10 (7/17/2023)

 

 

Our development team has been working hard and working overtime to come up with some really great features that have been asked for by you. We love our customers, so we have made the changes you have asked for. We have marked those features so that you know they came from you.


Once the release is complete, just refresh your browser to get the new update. Hit Ctrl-F5 in Firefox and IE, or Ctrl-Shift-R in Chrome.

  • Storefront Enhancements
    • Better Quality Storefront Thumbnails - As we have been working on improving the look and feel of the storefront, we upsized the quality of the thumbnails for your items with pictures on them.
    • Price Display on Storefront - We have enhanced the area around "Your Price" on the storefront. Once the start and end dates are selected, we now calculate 'Best Rate', and the select dates area changes to say 'Modify Dates'.
    • Rates per Item - We have changed the rate per item to show the first price with an expander that goes to the item detail page for more rates and more item information.
    • Date Selector
      • Changed from the full view of start and end dates to a ‘Select Dates’ that opens up those selections.
      • Changed text to ‘Start Date’ and ‘End Date’. Also made the view clear once dates have been selected.
      • Clear Dates - Added Clear Date button to start over if desired.
    • Earliest Dates - Changed "Minimum Rental Date" to "Earliest Start Date". Also changed "Minimum Delivery Date" to "Earliest Delivery Date"
    • Minimum End Date - Once a start date is selected, then the end date, it automatically goes to the minimum end date.
    • COMING SOON - Enhancements to Delivery/Pickup, and in-store selection process for your customers.

Note on Storefront Changes: We are constantly working to improve the user experience, performance, and visual layout of the Point of Rental Storefront service. As part of our ongoing efforts to improve the user experience, we have identified necessary structural changes to the Storefront HTML that include a high chance of impacting Custom CSS layouts. If you have Custom CSS, please follow the steps below to make sure you are not impacted.

  • Review your Custom CSS
  • Please check your storefront to confirm there are no conflicts with your custom CSS
  • Make any changes needed to ensure that your designs still function as intended
  • If you need to remove any custom CSS, please find the instructions here.

 

  • Serialized Inventory Aging Report - For your viewing pleasure, we have added the Serial Number as a selectable column in this report


Blast from the Past - Below are some features from the past that you may not know about.

 

  • General Ledger Account Numbers for Categories/Items: How does an Accountant stay out of debt? He learns to act his wage. If you are the accounting type, and you survived that Dad joke, you will really love this highly requested feature. You can now create your own account numbers for Categories and even go all the way down to the item level. It will also break out discounts by GL Income accounts. Here is a video of how it works. https://vimeo.com/597290784/d7baf40b60
 
  • ‘Pinning’ of Filters: If you create a non-shared filter in Inventory/Workbench/Customers, you can pin it so that it will stay on that selection when you come back to it and will hold that setting for your user. 
 
  • Undo Receive - We told Mom once that it’s better to ask forgiveness than to ask for permission. That didn’t work out so well. But with our software, we forgive you with no punishment. If you have received an item or items by mistake, you can now un-receive them. Not only will it show the item as still out, but it will change the contract status back to order if the contract is in received status.
