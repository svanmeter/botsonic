This article will show you how to create kits where specific serialized units go out together with other specific serialized units.


Topics in this article include:

  • Creating serialized items
  • Creating serialized kits

 



 

Creating serialized items

To create a kit, you will first have to add the individual serialized items to Essentials.
 

1. From Inventory on the menu bar, add each serialized item that will be included in your kit by clicking + New Item.

2. Select Serialized Item as the type of item you will be adding. Refer to our New Serialized Inventory document for instructions on how to add each serialized item. 

3. When adding the items, if you don't want your customers to be able to rent the items alone, be sure you choose Kit/Suggested Item Only in the Display drop-down menu.
 



 

Creating serialized kits
 

1. Once all the serialized pieces have been added to your inventory, add a new item. This time, instead of choosing a serialized item, choose Serialized Kit.

 


 

2. Fill in the basic information and pricing. Once you have filled in the necessary fields on the Basic Information screen, click the Pricing tab toward the top of the page. The pricing screen allows you to enter a base rate for the kit and choose which price sheet goes with this item if you have multiple price sheets created. If you have a deposit, replacement cost, or damage waiver dollar amount for the kit, be sure to enter those amounts here as well.

 

    

 

 

3. Add the component items to the kit by selecting the Components/Suggested Items tab. Next click the Add Item button once you have searched and found the piece you want to add and enter the quantity that comes with the kit. The type is Kit Component by default. This means the component items will be hidden from the contract the customer receives and no additional cost is added to the kit. You can also select Automatic, Optional, or Choose One Item. These will be added to the contract the customer receives and will add pricing to the kit unless specified with a 100% discount. Once you are done, hit Confirm. Repeat for all component items belonging to the kit. 

 

 

4. Once all the component pieces are added to the kit, you will see those in the list.

 


 

5. Next select the Kit Stock tab and add all the individual pieces for each kit that goes out. Click + New Serialized Kit to create the first group of kit stock. 
 

 

6. From here, enter a name for the kit stock along with a serial number. All other information is not required but can be helpful later. Once all the desired fields are completed, click Save.

 

 

7. Next, you will be asked to select which specific pieces of each item go into the kit stock. Select the specific piece by clicking on the serialized item. Once each serialized item for each main item has been selected, click Done. You will now see the first kit stock listed. To add more kit stock, repeat the steps until all units are assigned to a kit stock.

 

 

8. The number of kits you can make depends on how many component items you have in stock and how many are needed to create each kit.

 

9. Your customers will only see the kit item and any additional or optional items that were added to their order. The component items are hidden.
