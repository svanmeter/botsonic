Kappungsmodul

 

Übersicht

 

Mit dem Kappungsmodul kann der Mietpreis auf einen bestimmten Prozentsatz der Wiederbeschaffungskosten begrenzt werden. Hinweis: Dies funktioniert nur für Serienartikel, nicht für Massenartikel. Sobald der Mietpreis eines Artikels in einem Vertrag den gewünschten Kappungsprozentsatz erreicht, wird der Vertrag nicht mehr für den Artikel berechnet. Sie müssen lizenziert sein, um dieses Zusatzmodul nutzen zu können. Um eine Lizenz zu erwerben oder weitere Informationen zu erhalten, wenden Sie sich bitte an unsere Vertriebsabteilung.

 

Einrichten der Kappung

 

1. Geben Sie die Wiederbeschaffungskosten für alle Artikel an, für die eine Kappung gewünscht wird. 



2. Gehen Sie zu Kundenkonfigurationen und richten Sie Kundengruppen als Prozentsätze ein. Wenn Sie für bestimmte Kunden eine Obergrenze von 100 % der Wiederbeschaffungskosten festlegen wollen, im obigen Beispiel 28.000, dann richten Sie eine Kundengruppe von 100 ein. Wenn Sie eine Obergrenze von 75 % bei 21.000 Euro festlegen möchten, dann richten Sie eine Kundengruppe von 75 ein usw.



3. Weisen Sie nun diese Gruppen den Kunden zu, für die Sie im Kundendatensatz eine Obergrenze festlegen möchten.



4. Wenden Sie sich dann an unser Software-Supportteam, um das Programm so einzurichten, dass jeden Abend eine Kappung ausgeführt wird.
