Package Items

 

 

This option allows you to view or select the Package Items. The Package Items are set up just like the Suggested Items. The only difference between a Package Item and a Suggested Item is that the Package Items are automatically added to the contract in the quantity entered on this screen. The counter person does not have the opportunity to change the quantity until after the item has been added to the contract. When updating the quantity on the contract, the user can update the quantity of the first package item and will be prompted to automatically update the rest of the items in the package. Items can also be added to the package on the item by double clicking in the kit column on the contract processing screen. The user will be prompted to enter the line number of the package to tie the new item to. This can be cleared by double clicking on the kit field again and entering a zero.

You can define a Special Price for each line item of the Package Items. To do this check the Special Price check-box, then enter the pricing you desire. You can also define a discount dollar amount or a discount percentage amount.

 

*Note: You must use the Package Items to set the items to include in a Rental - Package item type.

** Note: Under the options tab, make sure the box is unchecked to suppress availability. If box is checked the quantity of the package item will display 99999

 

Extending package items

Assign the first 16 individual records to the main package record’s suggested records, as you normally do.

• For the extending package record:

- Create a dummy special package record and name its key to something like “108RNDLIN-EXT”

- Do not mark “hide on website”

- Enter <EXTENDKIT> in its WEBGROUP field (in the options tab) – make sure to enter < EXTENDKIT > without any spaces (you need to include the < and >)

- Assign this dummy package record to the same category as the main special package record (e.g. your LINEN category)

- Map remaining individual records (e.g. suggested item records #17..#20) as suggested items to this new dummy package

- Assign this extension package as suggested record #17 for the initial package record
