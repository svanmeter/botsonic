365 Homescreen


This is the main screen and various elements:

Please see below for what each section relates to.
 

1 | Left-hand Menu

This is where you can find options to access the areas of the system like Depots, Favourites, Hires, Operated, Sales, Fleet, and Stock. Clicking on one of these will expand a submenu.

 In the example below,  "Sales" has been selected, and a submenu appeared with Sales related options. Submenu options can be saved for quick access under the Favorites icon.
To close the submenu, click "Sales" again or the grey indent arrow on the right-hand side
.
Note - You may have to use the verticle scroll bar to view all of the left-hand options.

 

2 | Global Search

Global Search allows you to quickly search across multiple entities or a single entity without having to build up a search query (filters).  For more information on how the Global Search works, see our article 365 Global Search.
 

3 & 4 | Dashboards and Widgets

These two options are available for adding and Unhiding dashboards.
For more information, see the article  Syrinx 365 | Dashboard and Widgets.

5 | Company Name

This will display the company name against your database.
If you are a multi-depot, then this feature will display the depot you are logged into.
 

6 | Notifications

This will display notifications. Currently, it will display items like a successful saving of a dashboard but may also display errors you may have received. You can mark them as read and/or delete the notifications.


7 | User Settings

Use this option to access user settings.

Choose a theme - You can choose a theme of Default, Light, or Dark.

Menu Select - If you are an admin user and have access to the Configuration menu options, then you can toggle between the Main and Configuration menus (the left-hand menu will change accordingly).

Search Results Size and Search Options - These are settings for using Local Search by user. For more information, please refer to the Syrinx 365 | Local Search User Settings article.

 

8| Logged In/Out Users

This displays users who are currently logged in. Use the dropdown to get to the Log Out option. For more information, see the article Logging in and out of Syrinx 365  article.

9 | Right-hand Menu

This right-hand menu contains various non-transactional options, including 365 Reports, uLearn access, TeamViewer QuickSupport, uInnovate, Weather, and About Syrinx.
