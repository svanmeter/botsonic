Data Management Tools – Merge Item Categories

 

Select # 2 - Merge Item Categories

 

Select the Category you want to change from the Old Category Drop Down.

 

Select the Category you want to move those items into from the New Category Drop Down Menu.  In the example below, we have selected the old Category of Auto Related & Pullers and we are moving them into the Trailers Category.

 

Select Go in the Bottom Right-hand Corner.  When the program has finished running you will see a Pop-up window saying Done.  Click the OK button.  All of the items will now be in a new category.
