INTERNAL | How to Look Up Software/License Information for Expert/Elite



This article explains how to look up the license information in Salesforce for Expert and Elite customers. This will only apply for Expert/Elite customers.


 

To see the number of licenses a customer has in Salesforce:

 

1. Select the asset name.
 


 

2. Select the Software Info button.
 



3. Scroll down until you see POR Licensed User Count.

 


This will display how many user licenses can be logged into Point of Rental at the same time. The customer will need to talk to Sales to increase the number of licenses.

This is not the same as licenses for Windows or Mobile Workforce. The customer will need to reach out to Sales for any questions related to licensing.

If the license expiration is not specified, it indicates that the user is on an outdated package and may not receive updates. To learn more about how to check for updates, please refer to the article Check for Updates.

This information also shows important information like if the customer is on a physical server or a cloud server, accounting methods, and credit card processor Information to name a few.
