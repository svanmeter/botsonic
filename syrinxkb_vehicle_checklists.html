Syrinx® - Driver App Vehicle Checklists

 


 

 

Overview: This guide is designed to help you with creating vehicle checks that can be used in the driver app.

 

 

 

To view or create checklists, log into the Syrinx Configuration and go to Mobile Checklists.

 

This will display checklists already created. To add a new one click Add New.

 

Give your checklist a title and click OK.

 

You can then start to build it by clicking Add New.

 

 

Make the questions straightforward so the driver knows clearly what the answer should be, i.e., a simple yes-no feature.

 

When you have added the question, you then need to give answers.

 

In the example shown, I can click Add Yes/No. If it is a no answer, then that is an error/fault and the user will need to add notes of what the issue is. Once you have built your test, you will need to make it live. To make it live, just click Make Live at the bottom of the screen.

 

*Note: Only make a check live when you have finished building it.

 

To turn the checklist on for the app, you will need to go to Configuration > System Options > General > Android Apps (2) and use the drop-down in the vehicle check.

 

You can create multiple checks for different types of vehicles, for example, a tilt tray will have different checks to a low loader. To allocate different checks to the vehicles, you will need to go to SyrinxSPV4 > Fleet > Amend > Search for the Vehicle and on the Misc tab select the vehicle check if different from the standard.
