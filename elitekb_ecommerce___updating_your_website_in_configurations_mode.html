Overview 

There are situations in which you will need to run your Update Website process in Configuration Mode. This simply means you are manually running steps that normally run automatically in Auto mode (when you click the Update Site Data button).

 

To Update the Website in Configuration Mode
 

  • Configuration
    • System Configurations
      • Other Updates
        • Update Website
          • Click “OK” on any pop-up messages you see.
          • Click on the "Configuration Mode” box - this runs Step 1 of the Update Website process.  Once this completes, the boxes that were previously grayed out, Steps 2-6, will be available for selection.
          • You may then choose to run Steps 2,3,4, and/or 5 - depending on your goal:
            • Step 2. Select Web Categories allows you to select which categories you want to upload to the RentalHosting server, and which ones you do not. See Excluding Items and Categories for more information.
            • Step 3. Extract Web Item Pictures uploads item images to the RentalHosting server.
            • Step 4. Extract Customer Portal uploads Customer Portal data to the RentalHosting server.
            • Step 5. Extract Web Gallery Content uploads files in your gallery folders to the RentalHosting server.
 

Once you have run the intermediate steps (2-5) that you wish to run, you MUST RUN Step 6, "Upload to Website" TO FINISH THE UPDATE.
