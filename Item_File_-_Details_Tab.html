Details Tab - Item Record



Overview:  The details tabs, display information about a particular item record.

Key Features: Depending on the type of item record (Rental vs Sales) you can enter information into the appropriate fields to determine how the item record will calculate either rental rates vs sales cost. 
 

  • Rental Items - Retail Price, Damage Waiver, Fuel Tax, Road Tax, Deposit, Rental Rates.
  • Sales Items - Sell Price, Deposit


Other informational fields required to be filled out include ( Bin Location, Lookup/Alias, Header Key, Groups, Barcode). You will also have the option of entering any additional information into the item record by entering it under either the (Details Tab, Notes, or Print). * Note: Anything entered under the Print Tab, will print out onto the contract.  


Example



Description:

Bin LocationThe location where this item is located
Lookup/AliasDifferent variations of what the item can be called. Ex. Boom Lift, Cherry Picker
Header Key If a Serialized item, this will be where the header key is placed
GroupThis allows grouping of items to allow exchanging items on contracts
Count DateThe last time this item was counted
BarcodeThis field holds the barcode number allowing the use of a barcode gun
