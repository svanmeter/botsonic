Marketing Page

 

This form is used to send a marketing page with a contract. This document will need to be saved on your server prior to adding it into Point of Rental. Once the documents have been added to the server, you can load the document from the file.  Click on the three dots next to the load from the file box.

* Note: This form needs to be in a Rich Text Format (RTF) file. This form can be turned on or off to print under the laser contract format setup.
