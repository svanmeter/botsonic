Capping Module



 

Overview

 

The Capping Module allows the rental charge to be capped at a specified percentage of the replacement cost. Once the item rental price on a contract reaches the desired capping percentage, the contract will stop charging for the item. You must be licensed to use this add-on module. To become licensed or for more information, please contact our sales department. 
 

Note: This only works for serialized items, not bulk.

 
 

Setting up capping

 

1. Put replacement cost in all items that capping is desired on. 

 

 

2. Go to customer configurations and set up customer groups as percentages. If you want to set up certain customers to cap at 100% of the replacement cost, in the above example 28k, then set up a customer group of 100. If you want to cap at 75% at 21k, then set a customer group of 75, etc.

 

 

3. Now assign those groups to the customers you want to cap for from within the customer record.

 

 

4. From here, reach out to our software support team to get the program set up to run capping nightly.
