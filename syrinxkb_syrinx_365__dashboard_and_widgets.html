Syrinx 365 | Dashboards and Widgets



This article will look at using 365 dashboards and widgets.
 

Topics included in this article

  • Adding a Dashboard

  • Adding Widgets

  • Dashboard Settings

  • Unhide Dashboards

 

Adding a Dashboard

You will see this icon and message if you haven't created any dashboards. Click "Get Started" to add a dashboard. 



Click '"Add a new Dashboard." You'll be prompted to enter a Type and Title.

  • Type - Currently, this is either Custom or Sales. The vision is that Sales would be a pre-populated dashboard, and there may be other dashboard types of that nature added versus Custom, which allows you to select the widgets you want, currently, they are both working in the Custom style.

  • Title - This is simply the name you are giving the dashboard


Adding Widgets

Once you have added your Dashboard, you can start adding Widgets.

Note - you need to add at least one widget to a Dashboard to be able to save the dashboard. Closing out a dashboard with nothing on it could cause the dashboard to be lost.


Once you have created your dashboard, you'll have the option to pick a widget. You can select the area of the system from the dropdown, i.e., Invoicing, Workshop, etc, and then pick the required widget(s).

The selection window allows for a single selection, or if you click "Select Multiple", then you can make a multi-item selection.



Once you select the required item(s), there will be an "Add" button to add them to the dashboard.

Note - When adding widgets, the indicator in the bottom left displays the number of remaining widgets that can be added to the dashboard.


If you are adding a single widget, before clicking "Add", you can amend the title of that widget if you want to name it something more specific.


In the bottom right-hand corner of the dashboard will be the Cancel and Save buttons. Clicking Cancel will clear the work you have just done, and Save will save the dashboard in the state you are at.

Dashboard Settings

When you have a Dashboard open, there are severak options under "Dashboard Settings" at the top right of the window.

  • Edit Dashboard - If you want to make amendments like changing the dashboard name or removing widgets.

  • Add Widget - Allows you to add widget(s) in the same flow as earlier.

  • Copy Dashboard - This will create a copy of the dashboard, which you can then make the required amendments to.

  • Hide Dashboard - This will close the dashboard. It can be reopened at any point. See the next topic, Unhide Dashboards.

  • Delete Dashboard - This will permanently delete a dashboard.

 


Unhide Dashboards

If you have previously hidden your dashboard(s) and want to unhide one, then you can initiate the process from two places depending on what else you have open.

If you have the "Get Started" icon on the screen because you don't have any other tabs open, then you can click that. Or you can click the little plus icon at the top left of the main window.


You would then click "Hide/Unhide a Dashboard."


Then select the dashboard you want to open.
