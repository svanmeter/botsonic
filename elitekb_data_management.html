Data Management

Looking for a way to clean up your data? We offer easy-to-use tools for common and not so common updates, as well as more complex tools for advanced users with complex data needs. By exporting records to a spreadsheet directly from the Lookup Screen or Export Tool, you can filter results easily, modify the data, and import any changes back into your system using the Import Tool. For frequently repeated procedures, you also have the ability to save data layouts for quick import. In addition, you can import new inventory or customer records, or export predefined data sets to a spreadsheet for use in third party programs or reporting.
