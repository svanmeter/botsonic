Warnmeldungen

 

 

Übersicht

 

Dieser Artikel erklärt, wie Sie Warnmeldungen erstellen und versenden. Sie können Warnmeldungen an Kunden senden, z. B. bei Wettervorhersagen, Verkehrsverzögerungen oder schweren Unfällen. Benachrichtigungen sind eine gute Möglichkeit, mit Ihren Kunden über mögliche Verzögerungen oder Warnungen in Bezug auf die Lieferung, Abholung oder Veranstaltung Ihres Kunden zu kommunizieren. So können Sie Ihre Kunden stets über alle potenziellen Probleme informieren, die die Vertragserfüllung beeinträchtigen könnten.

 

 

In diesem Artikel enthaltene Themen

 

  • Erstellen von Warnmeldungen
  • Senden einer Warnmeldung

 

Voraussetzungen

 

Gilt nur für Benutzer mit einer Lizenz für das Modul MASS/FAX/EMAIL.

 

 

 

So gelangen Sie dorthin:

Expert: Programmmenü > Konfiguration > Systemkonfigurationen > E-Mail-Briefe

Elite: Zahnradsymbol > App-Konfigurationen > Systemkonfigurationen > E-Mail-Briefe

 

 

 

Erstellen von Warnmeldungen

 

1. Wählen Sie Hinzufügen, um eine Standard-Warnungs-E-Mail mit der Art des Hinweises zu erstellen, den Sie an Ihre Kunden senden möchten. Sie können mehrere E-Mail-Briefe für verschiedene Arten von Hinweisen einrichten.



2. Wählen Sie einen Nachrichtenstatus.

 

3. Wählen Sie eine oder alle Stationen.

 

4. Geben Sie die Beschreibung und den Betreff Ihrer Nachricht ein.

 

5. Geben Sie Ihre Nachricht in das Textfeld ein.



Senden einer Warnmeldung

 

1. Unter Tag im Überblick können Sie oben auf der Seite Briefe auswählen. Wählen Sie aus der Dropdown-Liste die Art der Benachrichtigung aus, die versendet werden soll. Dadurch wird automatisch eine Benachrichtigung an alle Kunden gesendet, die sich auf der aktuellen Registerkarte befinden.

 

Wenn Sie z. B. die Registerkarte „Reservierung“ geöffnet haben, wird eine Benachrichtigung an alle Kunden unter „Reservierungen“ gesendet. Wenn Sie die Registerkarte „Lieferungen“ geöffnet haben, wird eine Warnung an alle Kunden gesendet, die sich unter der Registerkarte „Lieferungen“ befinden.

 

Hinweis: Programmmenü > Counter System > Tag im Überblick



2. Sie können Ihre Auswahl beim Versenden von Warnmeldungen nach Abteilungen, Postleitzahlen, Kunden usw. verfeinern.

 

3. Aktivieren Sie für Abteilungen das Kontrollkästchen „Artikeldetails anzeigen“ und wählen Sie die Abteilung aus, an die Sie die Warnmeldung senden möchten.



4. Sie können auch mit der Maus über eine der ausgewählten grünen Spalten fahren und auf den schwarzen Dreieckspfeil klicken. Dadurch werden nur die Elemente in der Spalte angezeigt. Markieren Sie nur die Kästchen, an die Sie eine Benachrichtigung senden möchten.

 

5. Sie können mit der rechten Maustaste auf die grüne Kopfleiste klicken und das Kästchen für Postleitzahlen aktivieren. Fahren Sie dann mit der Maus über das Wort „Postleitzahl“. Es erscheint ein kleines schwarzes Dreieck. Klicken Sie auf das Dreieck, und es öffnet sich ein Feld, in dem nur die Postleitzahlen für diese Verträge angezeigt werden. Markieren Sie nur die Kästchen für die Postleitzahlen, an die Sie eine Benachrichtigung senden möchten.

 

Wenn zum Beispiel eine Windwarnung für ein bestimmtes Gebiet vorliegt, können Sie nur die Postleitzahlen auswählen, die in diesem Gebiet betroffen sind.
