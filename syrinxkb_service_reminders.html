Syrinx® - Service Reminders

 

 

 

 

Overview: This guide is to help you create and allocate service reminders to fleet within Syrinx.

 

 

 

 

Service reminders can be set up for fleet items and also customer equipment in Syrinx configuration under the workshop tab.

 

Navigate to the workshop tab within Syrinx configuration and select reminders from the servicing part of the ribbon. At the bottom, you can then select New to create a new service reminder type.

 

 

Now you can set your service type reminder based on the intervals of the reminder. You can either enter meter reading or date range to set reminders for the fleet.

 

 

 

Under the Categories tab of the service reminder, you can link the reminder to the categories for which the reminder will be linked to. When creating a new fleet item in that category, the service reminder will allocate to the new stock item created.

This does not link to existing items within the fleet.

 

 

To add the service reminder to fleet items you need to add the newly created service reminder against the asset in the reminders tab of the item. You also need to ensure that if using meter readings you have the log meter readings turned on against the assets category.

 

 

With the service reminder now set against the item, you can run service reminders from the workshop tab to view all items which are due or overdue based on the service parameters.

 

 

From the results screen, you can then create the service job by selecting raise service job at the bottom of the window.
