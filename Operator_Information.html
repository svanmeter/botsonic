Operator Information

 

Overview -   Allows an employee to set their own email, cell phone, and desired contact method. The employee can also change their own password and if they have forgotten their password, they can request a new one.

 

How to get there

Expert:Program Menu > Counter System > Tools > Operator Information.
Elite: Left Menu > User Options > Operator Information

 

Tools:

 

 

Modify Contact Info - From Tools, select Modify Contact Info will allow you to update operator contact information.

Change Password – Allows you to change your log in password. * Note: This will update the password under the Operator ID.

Forgot Password - Will send you a temporary password by email or interoffice email.

Print Setup - Allows the user to select their printers, cc terminal, store, etc.
