Price List

 

Under Reports/Inventory Reports, select Price List.
 

 
 

Interface for Price List Report


This report interface will come up. You can select your time periods using Duration (Ctrl/click will allow multiple selections). You can choose a specific category, whether to include all sales items, or exclude them by default. 

 

 
 

Exportable Options

You will have the option to run, download, email, or schedule the report.
